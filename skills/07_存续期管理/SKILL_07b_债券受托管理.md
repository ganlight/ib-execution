# SKILL_07b：债券受托管理

## 概述

债券受托管理人是债券存续期内代表债券持有人利益、监督发行人的法定角色。根据《公司债券发行与交易管理办法》（2023年修订）和NAFMII相关自律规则，受托管理人（通常由主承销商担任）在债券发行后至全额兑付完毕整个存续期内承担持续监督职责。核心使命是：**作为债券持有人的"看门人"，提前发现信用风险信号，在风险事件发生时组织持有人集体行动，最大化保护债权人的合法权益。**

受托管理不同于IPO持续督导——IPO持续督导关注规范运作和信息披露，受托管理聚焦偿债能力和信用风险。一个债券受托管理周期可能长达5-10年（债券存续期），任期内需要建立常态化的跟踪和风险排查机制。本模块覆盖受托管理人职责、定期报告、重大事项监测、持有人会议、信息披露督导、风险处置、资金管理和年度工作日历。对于需要更深层次信用风险分析，可调用 **bond-dcm-core** 技能。

## 一、受托管理人职责（全部）

根据现行法规，受托管理人的法定职责清单：

### 1.1 法定职责清单

| 序号 | 职责内容 | 法规要求来源 | 时限要求 |
|------|---------|------------|---------|
| 1 | 签署并保管《债券受托管理协议》 | 公司债管理办法 | 发行前 |
| 2 | 持续关注发行人的经营、财务和资信状况 | 公司债管理办法 | 持续 |
| 3 | 监督募集资金使用情况 | 公司债管理办法+募集说明书 | 募集资金使用期间持续监督 |
| 4 | 监督发行人信息披露义务履行情况 | 公司债管理办法+信息披露办法 | 持续 |
| 5 | 持续关注发行人是否存在重大事项 | 公司债管理办法 | 持续（定期+突发） |
| 6 | 在债券存续期内持续督促发行人按时偿付本息 | 受托管理协议 | 付息/兑付日前后 |
| 7 | 出现可能影响偿债能力的重大事项时，及时出具临时受托管理报告 | 公司债管理办法 | 事件发生后2个工作日内 |
| 8 | 定期出具受托管理事务报告（年报） | 公司债管理办法 | 每年6月30日前（上年度） |
| 9 | 受托管理事务报告中对偿债能力发表意见 | 公司债管理办法 | 年报中专项分析 |
| 10 | 监督和检查偿债保障措施的执行情况 | 受托管理协议+募集书 | 持续 |
| 11 | 召集债券持有人会议 | 持有人会议规则 | 触发事件发生时及时召集 |
| 12 | 根据持有人会议决议代表持有人行事 | 持有人会议规则 | 决议通过后执行 |
| 13 | 债券违约或重大风险事件时，及时启动处置程序 | 受托管理协议 | 发生即启动 |
| 14 | 代表持有人参加债务重组/破产程序 | 受托管理协议+公司法/企业破产法 | 进入程序后 |
| 15 | 至少每年对发行人进行1次现场回访/检查 | 合规要求 | 年度 |
| 16 | 建立受托管理工作底稿并妥善保管 | 公司债管理办法 | 债券存续期+结束后5年 |
| 17 | 履行中国证监会、交易所、NAFMII要求的其他受托管理职责 | 各类规则 | 按规则时限 |

### 1.2 NAFMII规则下的受托管理人
NAFMII体系内MTN/PPN等品种的受托管理由《非金融企业债务融资工具受托管理人业务指引》规定：
- MTN的受托管理人与公司债高度相似，但信息披露平台和报告格式不同
- NAFMII的受托管理人除上述职责外还需参与"NAFMII存续期信用风险分类管理"
- MTN受托管理人年度报告报送NAFMII并在指定平台向投资者公示

## 二、定期跟踪与报告

### 2.1 季/年报审阅清单

| 审阅节点 | 报告类型 | 发行方发布时限 | 受托管理人工作 | 审阅要点 |
|---------|---------|------------|-------------|---------|
| 每年4月30日前 | 年度报告 | 4月30日前 | 审核并出具《年报审阅意见》 | 全文审阅，重点关注偿债指标 |
| 每年8月31日前 | 半年度报告 | 8月31日前 | 审核并留存审阅记录 | 中期偿债指标趋势 |
| 每季度结束1个月内（银行间） | 季度财务报表 | 季度后1个月 | 审核并留存 | 季度间异常波动 |
| 每年6月30日前 | 受托管理事务年度报告 | 受托管理人出具 | 编制+报备+公示 | 对偿债能力的独立评估 |

### 2.2 受托管理事务年度报告框架
1. **受托管理人声明：** 声明独立性和审阅范围
2. **发行人基本情况更新：** 注册资本、经营范围、组织架构变动
3. **发行人经营和财务状况：**
   - 近两年核心财务指标对比表
   - 偿债能力指标分析（资产负债率/流动比率/速动比率/利息倍数/EBITDA等）
   - 现金流分析（经营/投资/融资三大现金流）
4. **募集资金使用情况：** 是否按照募集书约定的用途使用
5. **增信机制运行情况：** 担保人信用变化、抵押资产价值变动
6. **偿债保障措施执行情况：** 偿债资金专户、偿债准备金、股东支持等
7. **信息披露执行情况：** 是否按要求及时披露年报/半年报/重大事项
8. **债券持有人会议召开情况：** 是否召集、决议执行
9. **债券本息偿付情况：** 是否按时足额支付，有无拖欠
10. **信用风险排查结论：** 综合评估+风险分类（关注类/风险类/正常类）
11. **受托管理人情况变化：** 如受托管理人自身是否发生影响其履职的事件

## 三、重大事项监测（20+触发事件）

### 3.1 重大事项触发清单

以下事件发生后，受托管理人需在**2个工作日内**出具临时受托管理事务报告：

| 序号 | 触发事件 | 对偿债能力的影响 | 报告重点关注内容 |
|------|---------|---------------|---------------|
| 1 | 发行人名称/经营范围/注册资本变更 | 低 | 是否涉及经营主业变化 |
| 2 | 发行人变更董事长/总经理/法定代表人 | 低-中 | 新任管理层背景和战略方向 |
| 3 | 发行人发生超过上年末净资产10%的重大损失 | 高 | 损失来源、原因、对当年净利润的影响 |
| 4 | 发行人放弃债权或财产超过上年末净资产的10% | 高 | 放弃的原因和合理性 |
| 5 | 发行人股权结构/控股股东/实际控制人变更 | 高 | 新实控人背景、管理能力和授信情况 |
| 6 | 发行人涉及重大诉讼/仲裁——案件金额≥净资产10% | 高 | 案件性质、不利判决对现金流的影响 |
| 7 | 发行人被立案调查/受到重大行政处罚 | 高 | 罚款金额、经营限制、声誉风险 |
| 8 | 发行人减资/合并/分立/解散/申请破产 | 最高 | 对债券持有人的法律保护措施 |
| 9 | 发行人的债券出现实质违约/展期 | 最高 | 处置进展、回收率估计、持有人会议 |
| 10 | 发行人的债券评级被下调 | 中-高 | 下调原因、关注是否被触发交叉违约 |
| 11 | 担保人发生重大不利变化（如被降级、违约） | 中-高 | 担保人的偿付意愿和能力评估 |
| 12 | 发行人资产被查封、扣押或冻结——价值≥净资产10% | 高 | 涉及资产、原因、对经营的影响 |
| 13 | 发行人新增借款超过上年末净资产的20% | 中 | 新增债务原因、用途、偿付安排 |
| 14 | 发行人对外提供担保超过上年末净资产的20% | 中 | 被担保方资质、代偿风险分析 |
| 15 | 发行人发生安全生产事故/环保重大事故 | 中 | 停产损失、赔偿金、罚款估算 |
| 16 | 发行人涉隐性债务清理 | 中 | 对再融资能力和偿债来源的影响 |
| 17 | 发行人年报被审计师出具"非标意见" | 高 | 审计保留意见的具体内容 |
| 18 | 发行人财务负责人/会计师事务所变更 | 低-中 | 是否为审计争议导致的更换 |
| 19 | 经营方针和经营范围发生重大变化 | 中 | 跨行业转型的风险分析 |
| 20 | 生产经营的外部条件发生重大变化（行业政策/贸易限制等） | 中-高 | 量化分析对收入和利润的影响 |
| 21 | 发行人一次免除他人债务超过上年末净资产的10% | 高 | 放弃该资产是否属于变相转移 |
| 22 | 发行人主要资产被无偿划转 | 高 | 是否属于国有资本重组，是否获得补偿 |
| 23 | 发行人未按期披露定期报告 | 中 | 督促发行、必要时召集持有人会议 |

### 3.2 触发重大事项后的受托管理人工作流程
```
事件发生
  │
  ├→ ① 核实事件（联系发行人确认）
  │
  ├→ ② 分析对偿债能力的影响（定性+定量）
  │
  ├→ ③ 2个工作日内出具《临时受托管理报告》
  │   ├─ 内容：事件描述+影响分析+行动计划
  │   └─ 报送：交易所/NAFMII+在指定信息披露平台公示
  │
  ├→ ④ 是否需要召集持有人会议？→ 是 → 启动会议程序
  │
  └→ ⑤ 后续跟踪，直至事件影响消除或债券到期
```

## 四、持有人会议

### 4.1 召集触发情形

| 触发情形 | 召集义务人 | 召集时限 | 备注 |
|---------|-----------|---------|------|
| 发行人拟变更募集说明书约定（利率/期限/偿还方式等） | 受托管理人 | 及时 | 必须经持有人同意 |
| 拟修改《持有人会议规则》 | 受托管理人 | 及时 | 须2/3以上表决权通过 |
| 拟变更受托管理人 | 受托管理人 | 及时 | 须持有人会议决议 |
| 发行人发生重大资产重组/减资/合并/分立 | 受托管理人 | 及时 | 影响债券持有人利益 |
| 发行人不能按期支付本息 | 受托管理人 | 立即 | 实质违约 |
| 增信措施发生重大不利变化 | 受托管理人 | 及时 | 增信失效 |
| 债券评级下调/展望负面 | 受托管理人 | 评级调整后15个工作日内 | 触发加速到期/补充增信的讨论 |
| 发行人管理层无法正常履职 | 受托管理人 | 及时 | |
| 单独或合计持有债券总额10%以上的持有人书面提议 | 受托管理人（必须召集） | 收到请求后15个工作日内 | 持有人有召集权 |
| 受托管理人提议召集 | 受托管理人 | 自行决定 | 预防性召集 |

### 4.2 会议表决机制

| 事项 | 生效表决权要求 | 说明 |
|------|-------------|------|
| 一般事项（除以下特殊事项外） | 持有债券表决权≥1/2通过 | 较简易 |
| 变更债券利率/期限/偿付方式 | ≥2/3通过 | 核心条款 |
| 变更受托管理人 | ≥1/2通过 | |
| 发行人减资/合并/分立/解散/申请破产的应对方案 | ≥2/3通过 | 重大事项 |
| 修改《持有人会议规则》 | ≥2/3通过 | 程序性核心文件 |

### 4.3 持有人会议完整流程
```
发出会议通知（提前10个交易日）
       │
  债券登记日（确定出席持有人名单+表决权）
       │
  会议召开日（线上/线下/混合）
       │
  表决计票（委托律师见证）
       │
  决议公告（会议后2个工作日内）
       │
  受托管理人根据决议代表持有人执行
```

## 五、信息披露督导

### 5.1 受托管理人的信息披露监督职责
| 监督内容 | 具体工作 | 频率/触发条件 |
|---------|---------|------------|
| 年度报告 | 检查发行人是否在4月30日前披露，内容是否完备 | 年度 |
| 半年度报告 | 检查是否在8月31日前披露 | 半年 |
| 重大事项临时报告 | 检查发行人是否在事件发生后2个工作日内披露 | 事件驱动 |
| 付息/兑付公告 | 检查发行人是否按时发布付息/兑付公告 | 每次付息/兑付前 |
| 评级报告 | 检查评级机构是否按时出具跟踪评级报告 | 年度+事件驱动 |

### 5.2 信息披露纠正
如发现发行人未及时披露或披露内容有误：
1. **发送书面督促函**（含具体问题和整改时限）
2. 如发行人在时限内未整改 → **出具临时受托管理报告**公开说明情况
3. 必要时召集持有人会议讨论

## 六、违约与风险处置

### 6.1 风险分类体系

| 风险类别 | 判定标准 | 受托管理动作 |
|---------|---------|------------|
| **正常类** | 所有偿债指标正常，无重大不利变化 | 持续常规监控 |
| **关注类** | 出现个别负面信号（如经营性现金流转负、评级展望负面、原实际控制人变更等），但偿债能力暂未实质恶化 | 加强跟踪频率（月度→半月），建立专项档案 |
| **风险类** | 偿债能力出现实质性恶化（如连续2期净利润为负、高额受限资产、流动比率<0.8），或发生触发交叉违约的事件 | 启动应急预案，专项小组跟踪，必要时召集持有人会议 |
| **违约类** | 已发生本息逾期未付等实质违约 | 启动处置程序 |

### 6.2 违约处置流程
```
实质违约发生
      │
      ├→ Step 1: 立即通知所有债券持有人
      │
      ├→ Step 2: 宣布债券加速到期（根据持有人会议决议）
      │     └─ 选择不加速到期人（不赞成加速到期的持有人35%以上可反对）
      │
      ├→ Step 3: 启动法律程序
      │   ├─ 申请财产保全（冻结发行人账户/查封资产）
      │   ├─ 诉讼/仲裁（主张债权+DV）
      │   └─ 如发行人进入破产程序 → 代表持有人申报债权+参与债权人委员会
      │
      ├→ Step 4: 债务重组谈判（如可行）
      │   ├─ 展期/分期偿还/降息
      │   ├─ 转股（如有转股条款）
      │   └─ 新债务人承接（如有第三方收购）
      │
      └→ Step 5: 分配回收资金（按债券持有比例分配）
```

### 6.3 代位行权与加速到期
- **加速到期权：** 持有人会议决议宣布债券应到期，可要求发行人立即偿付全部本金利息
- **代位行权：** 受托管理人可在不经过持有人个别授权的情况下，以自己名义对发行人提起诉讼
- **破产程序中的代位：** 受托管理人可集中代表全部持有人申报债权（避免每个持有人单独申报）

## 七、付息兑付管理

### 7.1 付息/兑付专项工作清单

| 时间节点 | 工作内容 | 关键产出物 |
|---------|---------|----------|
| 付息/兑付日前T-30天 | 提醒发行人准备偿付资金 | 付息/兑付提示函 |
| T-15天 | 核实偿债资金专户余额 | 到账确认记录 |
| T-10天 | 督促发行人发布《付息/兑付公告》 | 公告 |
| T-5天 | 确认全部资金到位（利息金+本金+兑付手续费） | 资金到位确认函 |
| T-2天 | 向中央结算/上清所提交兑付材料 | 代付指令 |
| T日 | 完成代理付息/兑付 | 兑付完成确认 |
| T+1日 | 公告付息/兑付完成情况 | 完成公告 |

### 7.2 付息兑付应急预案
**情况1：发行人付息日前5天资金未足额到账**
- 发出紧急催款函
- 评估发行人自有资金和外部融资的可用性
- 与增信方沟通是否启用增信支持
- 准备临时受托管理报告（预判违约+应对方案）

**情况2：实际发生逾期（违约后）**
- 立即启动违约处置流程（参见第六节）
- 收取逾期利息（按募集书约定的违约利率）
- 在信用信息系统中进行不良记录报送

## 八、年度工作日历

受托管理人可按月安排例行工作：

| 月份 | 常规工作 | 特别关注 |
|------|---------|---------|
| 1月 | 上年四季度财务数据监控；年终总结；新年工作规划 | 年报季启动准备 |
| 2月 | 持续跟进年报编制进度；现场回访安排 | 春节前后沟通窗口 |
| 3月 | 现场回访（至少1次）；年度信用风险排查启动 | 密集发行期监控 |
| 4月 | **重点：** 督促发行人4月30日前发布年报；审阅年报 | **年报+评级跟踪报告密集发布月** |
| 5月 | 出具《受托管理事务年度报告》初稿 | 年报审阅汇总 |
| 6月 | **6月30日前向监管报送年度受托管理报告** | **受托管理报告强制截止日** |
| 7月 | 半年度财务数据跟踪 | 持续融资动态监控 |
| 8月 | **重点：** 督促发行人8月31日前发布半年报；审阅半年报 | **半年报发布月** |
