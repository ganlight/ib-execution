# SKILL_07c：并购整合跟踪

## 一、概述

并购重组完成后，独立财务顾问需要在业绩承诺期内持续跟踪标的资产的经营和业绩实现情况，关注商誉减值风险，监测整合进展。本技能定义跟踪框架和关键指标体系。

## 二、业绩承诺跟踪

### 2.1 业绩承诺基本框架

| 要素 | 说明 |
|-----|------|
| 承诺期限 | 通常3年（部分项目4-5年） |
| 承诺口径 | 扣非归母净利润（最常见） |
| 承诺金额 | 逐年增长型为主（部分阶梯型） |
| 补偿方式 | 股份补偿（优先）→现金补偿（补充） |
| 补偿计算 | （累计承诺净利润 - 累计实现净利润）÷ 累计承诺净利润 × 交易对价 |
| 减值测试 | 业绩承诺期满后，对标的资产进行减值测试，减值额 > 已补偿额时追加补偿 |

### 2.2 年度跟踪流程

```
第一步：获取标的公司年度审计报告
第二步：计算当年度实际实现的承诺口径净利润
第三步：对比承诺金额，计算完成率
第四步：判断是否触发补偿义务
第五步：出具专项核查意见（独立财务顾问+会计师）
第六步：如触发补偿 → 督促承诺方履行补偿 → 上市公司公告
第七步：更新业绩承诺跟踪档案
```

### 2.3 业绩承诺完成率判断

| 完成率 | 状态 | 处理 |
|--------|------|------|
| ≥ 100% | ✅ 达标 | 正常推进 |
| 90% ~ 100% | ⚠️ 接近达标 | 关注下一期完成情况 |
| 80% ~ 90% | 🟡 关注 | 分析原因，评估全年和累计完成预期 |
| < 80% | 🔴 风险 | 立即核查原因，评估对商誉减值的影响 |

**累计口径优先**：如当期完成率<100%但累计完成率≥100%，通常不触发当期补偿，但需密切关注剩余承诺期。

### 2.4 业绩承诺完成情况专项核查报告

**报告结构**：

```
一、本次重组基本情况
二、业绩承诺及补偿安排
三、标的资产当年业绩实现情况
  - 承诺净利润 vs 实际实现净利润
  - 完成率及差异原因分析
四、标的资产累计业绩实现情况
五、关于是否需要补偿的结论
六、保荐人/独立财务顾问核查意见
```

### 2.5 业绩补偿执行

- **补偿方式确定**：优先以本次交易获得的上市公司股份补偿，不足部分以现金补偿
- **股份回购注销**：上市公司以1元总价回购应补偿股份并注销
- **现金补偿**：承诺方在收到通知后30日内支付
- **执行监督**：独立财务顾问督促上市公司和承诺方履行

### 2.6 业绩承诺未达标的常见原因

| 原因 | 应对 |
|-----|------|
| 行业下行（外部系统性风险） | 区分系统性vs标的管理层责任 |
| 客户流失 | 核查流失原因，评估持续性 |
| 成本上升超预期 | 分析成本结构，评估可优化空间 |
| 新项目/新产品不及预期 | 核查研发进展和市场反馈 |
| 整合效果不佳 | 进入整合风险分析流程 |

## 三、商誉减值测试跟踪

### 3.1 商誉减值测试时间节点

| 时间 | 事项 |
|-----|------|
| 每年年末（12月31日） | 无论是否有减值迹象，必须进行年度商誉减值测试 |
| 业绩大幅低于承诺时 | 可能需要开展中期减值测试 |
| 行业/市场重大变化时 | 可能需要开展临时减值测试 |

### 3.2 减值测试关注要点

- 测试所采用的资产组/资产组组合是否合理
- 未来现金流预测假设是否合理（与评估时、上年测试时对比）
- 折现率是否合理（与评估时使用的折现率对比）
- 可比公司选择是否恰当
- 测试结论是否与标的业绩表现一致

### 3.3 商誉减值迹象识别

| 减值迹象 | 触发条件 | 核查建议 |
|---------|---------|---------|
| 业绩大幅低于承诺 | 实际完成率 < 80% | 必须进行减值测试并分析 |
| 行业政策重大不利变化 | 影响持续经营能力 | 立即开展减值评估 |
| 核心客户/供应商丢失 | 流失占比 > 30% | 分析现金流预测需下调幅度 |
| 核心团队流失 | 关键人员离职 | 评估经营持续性影响 |
| 核心技术被替代 | 技术路线重大变革 | 分析技术贬值影响 |

## 四、整合进展监测

### 4.1 整合监测维度

| 维度 | 关键指标 | 数据来源 | 目标 | 跟踪频率 |
|-----|---------|---------|------|---------|
| 收入协同 | 交叉销售额、新客户数量 | 销售数据 | 高于独立经营时的增速 | 季度 |
| 成本协同 | 采购成本降幅、费用率下降 | 财务数据 | 达到整合方案预期 | 季度 |
| 组织整合 | ERP系统上线、制度统一进度 | 项目管理 | 按期完成 | 月度 |
| 人员整合 | 核心人员留存率、员工满意度 | HR数据 | 核心人员留存≥90% | 季度 |
| 文化整合 | 员工满意度、投诉率 | HR数据 | — | 半年 |

## 五、重大不利变化监测

### 5.1 监测触发事项

以下事项一旦发生，需立即评估对标的资产持续经营能力和上市公司的影响：

| 触发事项 | 行动 |
|---------|------|
| 标的公司主要经营资质被吊销/到期未续 | 评估经营合法性风险 |
| 标的公司重大诉讼/仲裁/行政处罚 | 评估财务影响和声誉影响 |
| 标的公司核心客户/供应商终止合作 | 评估对收入和供应链的影响 |
| 标的公司核心技术/专利被无效/侵权 | 评估技术壁垒持续性 |
| 承诺方（业绩补偿义务人）财务状况恶化 | 评估补偿能力 |
| 上市公司对标的公司管理失控 | 录为值得特别关注风险 |

### 5.2 重大不利变化处置

1. 初步评估影响程度
2. 与上市公司、承诺方沟通
3. 督促上市公司进行信息披露
4. 出具独立财务顾问核查意见
5. 评估是否触发重组前置条款的MAC条款
6. 记录入跟踪档案

## 六、对赌期/或有对价管理

### 6.1 或有对价（Earnout）机制

部分重组可能约定，除基础对价外，根据标的公司未来业绩表现追加支付额外对价（earnout）：

- Earnout条件的逐条跟踪
- Earnout期间标的公司的行为约束（防止操纵业绩）
- Earnout支付金额的计算和确认

### 6.2 Earnout常见纠纷及预防

| 纠纷类型 | 预防措施 |
|---------|---------|
| 业绩认定口径争议 | 协议明确约定计算口径 |
| 会计政策变更影响 | 约定按原政策调整 |
| 承诺方对期间行为的争议 | 协议约定标的独立运营约束条款 |
| 支付时间的争议 | 约定审计报告出具后XX天内支付 |

## 八、文档归档要求

- 每年出具《业绩承诺实现情况专项核查意见》并公告
- 每年审阅商誉减值测试过程，保留核查底稿
- 整合进展报告按季归档
- 重大事项核查记录实时归档
- 跟踪档案至少保存至业绩承诺期满后5年
