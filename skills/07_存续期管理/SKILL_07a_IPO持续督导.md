# SKILL_07a：IPO持续督导

## 一、概述

IPO持续督导是保荐人在发行人上市后需履行的持续监督和指导义务。根据《证券发行上市保荐业务管理办法》，持续督导期间为上市当年剩余时间及其后2个完整会计年度（创业板/科创版为3个完整会计年度）。持续督导工作的质量直接影响保荐人的执业评价和后续项目承揽。

## 二、定期报告审阅

### 2.1 年度报告审阅

**时限要求**：年报公告后15个工作日内出具年度督导报告

**审阅重点**：
- 财务数据的合理性（同比、环比分析）
- 与招股书预测数据的对比
- 审计意见类型（是否仍为标准无保留）
- 关联交易及定价公允性
- 募集资金使用情况
- 内部控制情况（内控鉴证报告是否继续出具）
- 重大承诺事项的履行进展
- 是否存在应披露未披露的重大事项

**审阅记录要求**：
- 对年报逐项审阅，制作审阅工作底稿
- 记录审阅发现和分析判断
- 对异常事项形成专项核查底稿

### 2.2 半年度报告审阅

**时限要求**：半年报公告前协助审阅

**审阅重点**：
- 中期财务数据的合理性
- 与上年同期的比较分析
- 募集资金半年度使用进展
- 半年度期间发生的重大事项

## 三、募集资金使用监控

### 3.1 募集资金专项账户管理

- 确认募集资金已存入专项账户
- 募集资金的支出经过规定的审批程序
- 不存在违规挪用募集资金的情况
- 临时补充流动资金是否符合规定（不得超过12个月，不得影响募投项目正常进行）

### 3.2 募投项目实施进度监控

**监控内容**：

| 监控维度 | 数据来源 | 异常判断标准 |
|---------|---------|-------------|
| 投资进度 | 募集资金使用台账 | 实际投入远低于计划进度 |
| 建设进度 | 项目现场考察 | 工期延误超过6个月 |
| 预期效益 | 投产后财务数据 | 实际效益低于可行性研究报告预期50%以上 |
| 项目变更 | 发行人公告/会议 | 募投项目变更或终止 |

### 3.3 募集资金使用变更程序

如发生以下情形，需要履行变更程序：
- 取消或终止原募集资金投资项目，实施新项目
- 变更募集资金投资项目实施主体
- 变更募集资金投资项目实施方式
- 募集资金用途变更超过募集资金净额的一定比例

变更程序：
1. 董事会审议
2. 独立董事、监事会、保荐人发表意见
3. 股东大会审议通过
4. 公告相关决议和意见

### 3.4 募集资金专项核查报告

- **频率**：每半年出具一次
- **时间**：中期报告/年度报告公告时同步披露
- **内容**：募集资金基本情况和存储情况、本年度/半年度使用情况、募集资金投资项目进展情况、变更情况、保荐人核查意见

## 四、关联交易监控

### 4.1 需关注的关联交易

| 交易类型 | 关注阈值 | 核查重点 |
|---------|---------|---------|
| 日常关联交易（采购/销售/服务） | 超出年度预计额度 | 是否重新履行审批程序 |
| 资产收购/出售 | >净资产5% 或 >3000万 | 评估公允性、必要性、决策程序 |
| 对外担保 | 任何关联担保 | 反担保安排、决策程序、是否损害上市公司利益 |
| 资金拆借 | 任何资金往来 | 利率公允性、是否构成资金占用 |
| 共同投资 | 与关联方共同设立公司 | 是否构成同业竞争、决策程序 |

### 4.2 关联交易审查程序

1. 审查交易的商业必要性（是否具备独立商业目的）
2. 价格公允性分析（与市场第三方价格对比）
3. 决策程序的合规性（回避表决、独董事前认可）
4. 信息披露的充分性

## 五、重大事项关注

### 5.1 应关注的重大事项类别

根据《证券发行上市保荐业务管理办法》，以下情形属于需要特别关注的重大事项：

| 序号 | 事项 | 关注要点 |
|-----|------|---------|
| 1 | 经营业绩大幅下滑（营收/净利润下降>50%） | 查明原因，判断是临时性还是持续性 |
| 2 | 重大资产重组/再融资 | 审阅方案，出具核查意见 |
| 3 | 关联交易超出预计额度 | 督促履行程序 |
| 4 | 为他人提供担保（>净资产10%） | 评估担保风险 |
| 5 | 重大诉讼/仲裁（>净资产10%） | 评估影响 |
| 6 | 控股股东/实际控制人/董监高被立案调查 | 评估影响及披露充分性 |
| 7 | 发生重大亏损或重大损失 | 评估原因及影响 |
| 8 | 募集资金投向变更 | 督促履行程序 |
| 9 | 产业政策重大变化 | 评估对公司经营的实质影响 |
| 10 | 核心技术人员/高管离职 | 评估是否影响持续经营 |

## 六、现场检查

### 6.1 基本要求

- **频率**：每年至少1次
- **时间**：通常在年度报告审计期间进行
- **内容**：
  - 查阅公司治理、内控制度运行记录
  - 查阅募集资金使用凭证和台账
  - 实地查看募投项目进度
  - 访谈管理层及相关人员
  - 查阅重大合同和关联交易审批文件

### 6.2 现场检查报告

现场检查后应出具报告，内容包括：
- 检查时间、检查人员、检查内容
- 检查发现的问题
- 发行人整改措施（如有问题）
- 核查结论

## 七、年度督导报告

### 7.1 年度督导报告结构

```
一、保荐人/保荐代表人基本情况
二、发行人基本情况及本年度概况
三、保荐人对上市公司信息披露审阅的情况
四、保荐人对募集资金使用情况的核查意见
五、保荐人对关联交易/对外担保的核查意见
六、保荐人对收购兼并（如有）的核查意见
七、保荐人对承诺事项履行情况的核查意见
八、保荐人现场检查情况
九、保荐人关注的重大事项及核查意见
十、保荐结论
```

### 7.2 年度督导报告时限

- 年报公告后15个工作日内出具
- 向交易所提交，并通知上市公司

## 八、保荐代表人变更

### 8.1 变更情形

- 保荐代表人离职
- 保荐代表人因违规被处罚，不能继续履职
- 保荐代表人岗位调整
- 发行人要求变更（需合理理由）

### 8.2 变更程序

1. 保荐人内部确认新任保荐代表人
2. 原保荐代表人与新任保荐代表人做好工作交接
3. 保荐人出具《变更保荐代表人说明》
4. 通知发行人并公告
5. 新任保荐代表人签署新的持续督导文件

## 九、持续督导工作底稿

### 9.1 底稿保存内容

```
持续督导工作底稿/
├── 1_定期报告审阅/
│   ├── 2026年年报审阅底稿/
│   └── 2027年半年报审阅底稿/
├── 2_募集资金监控/
│   ├── 募集资金台账/
│   ├── 银行对账单/
│   └── 专项核查意见/
├── 3_关联交易审查/
│   └── 关联交易核查记录/
├── 4_现场检查/
│   ├── 2026年度现场检查报告/
│   └── 2027年度现场检查报告/
├── 5_重大事项/
│   └── 重大事项核查记录/
├── 6_年度督导报告/
└── 7_变更记录/
```

### 9.2 保存期间

- 持续督导工作底稿保存期限不少于10年
- 关联交易、重大事项等重要底稿保存不少于20年
