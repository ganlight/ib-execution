# SKILL_05d：重组审核流程

## 一、概述

上市公司重大资产重组的审核分为两类：发行股份购买资产的需经交易所审核+证监会注册；现金购买资产（不构成重大资产重组或仅构成重大但纯现金）仅需交易所信息披露审核。本技能聚焦于需审核的重组类型。

## 二、审核路径分类

### 2.1 审核路径一览

| 交易类型 | 审核机构 | 审核方式 | 主体流程 |
|---------|---------|---------|---------|
| 发行股份购买资产 | 交易所审核+证监会注册 | 重组委审议 | 申报→审核→重组委→注册 |
| 纯现金重大资产重组 | 交易所信息披露审核 | 问询函 | 申报→问询回复 |
| 不构成重大资产重组 | 不需审核 | 信息披露 | 董事会+股东大会 |
| 简易程序（同时满足条件） | 交易所审核（简易程序） | 免重组委 | 申报→快速审核→注册 |

### 2.2 简易程序适用条件

申请适用简易程序需**同时满足**：
1. 最近12个月累计交易金额不超过5亿元
2. 最近12个月累计交易金额不超过上市公司最近一期经审计净资产的20%
3. 不构成重组上市
4. 交易不涉及发行股份
5. 不存在《重组办法》规定的不适用简易程序的情形

> 简易程序不需提交重组委审议，审核时限大幅压缩。

## 三、重组审核完整流程

### 3.1 标准流程（发股购买资产）

```
第一步：申报前程序
  ├── 董事会审议重组预案/草案（首次董事会）
  ├── 与交易对方签署附生效条件的交易协议
  ├── 独立财务顾问出具核查意见
  ├── 审计/评估/法律等专业机构出具报告
  └── 股东大会审议重组方案

第二步：申报受理
  ├── 向交易所提交申请文件
  ├── 交易所5个工作日内形式审查
  └── 受理 → 出具受理通知

第三步：交易所审核
  ├── 审核问询（1-2轮，通常1轮）
  ├── 反馈回复（时限通常1-2个月）
  └── 审核报告 → 提交重组委

第四步：并购重组委审议
  ├── 重组委会议审议
  ├── 表决结果（同意/反对/暂缓）
  └── 通过 → 进入注册

第五步：证监会注册
  ├── 交易所报送审核意见
  ├── 证监会注册部门审阅
  └── 注册决定 → 批文有效期12个月

第六步：实施交割
  ├── 满足交割条件
  ├── 股权/资产过户
  └── 新增股份登记上市
```

### 3.2 时限基准

| 阶段 | 时限 |
|-----|------|
| 受理→首轮问询 | 15-20个工作日 |
| 首轮回复时限 | 1-2个月 |
| 问询回复→重组委会议 | 约1-2个月 |
| 重组委通过→提交注册 | 1-2周 |
| 证监会注册 | 约1-2个月 |
| **总周期** | **约4-8个月**（含回复时间） |

## 四、重组上市（借壳上市）审核

### 4.1 定义

上市公司自控制权发生变更之日起36个月内，向收购人及其关联人购买资产，导致上市公司发生以下根本变化的情形之一即构成重组上市（借壳）：
- 购买的资产总额占上市公司控制权变更前一个会计年度经审计的合并财务会计报告期末资产总额的比例达到100%以上
- 或营业收入/净资产/净利润/发行股份达到100%以上
- 或主营业务发生根本变化

### 4.2 审核特点

重组上市的审核标准**等同于IPO**：
- 标的资产应符合IPO条件
- 持续经营时间、盈利能力、合规性等全面对标IPO
- 审核更加严格，关注交易背景是否有规避IPO审核的嫌疑
- 通常问询2-3轮

## 五、并购重组委常见问询

### 5.1 问询频率TOP 5

| 排名 | 重组委关注点 | 出现频率 | 典型问法 |
|-----|-------------|---------|---------|
| 🥇 1 | **评估/估值公允性** | **~95%** | "本次评估增值率较高的原因和合理性" |
| 🥈 2 | **业绩承诺可实现性** | **~90%** | "业绩承诺的可实现性及承诺方的履约能力" |
| 🥉 3 | **是否存在同业竞争** | **~80%** | "本次交易后是否新增同业竞争，解决措施是否有效" |
| 4 | **标的资产质量和合规性** | ~75% | "标的公司生产经营是否合法合规" |
| 5 | **交易的必要性** | ~70% | "本次交易是否符合上市公司发展战略，是否有利于增强持续盈利能力" |

### 5.2 其他常见问询

- 交易对方穿透至最终权益人（是否存在关联关系或利益安排）
- 商誉确认及减值风险
- 关联交易定价公允性
- 标的公司会计政策与上市公司差异
- 过渡期损益安排
- 配套融资合理性（如涉及）
- 盈利预测假设的合理性

## 六、重组终止/撤回的常见情形

| 情形 | 占比 | 预防措施 |
|-----|------|---------|
| 评估/估值无法取得双方一致 | ~30% | 提前锁定评估核心假设 |
| 市场环境/行业变化 | ~25% | 交易方案设置保护条款 |
| 审核未通过（上会被否） | ~15% | 提前与审核员充分沟通 |
| 交易对方违约/不配合 | ~15% | 合同明确违约责任 |
| 标的资产业绩变脸 | ~10% | 设定过渡期保护条款 |
| 其他（政策变化等） | ~5% | 持续跟踪政策动态 |

## 七、与IPO审核流程的主要差异

| 维度 | 重组审核 | IPO审核 |
|-----|---------|--------|
| 审核核心理念 | 交易方案合理性 | 发行人质量达标 |
| 问询侧重 | 评估+业绩承诺+整合 | 业务财务+独立性 |
| 审核轮次 | 通常1-2轮 | 通常2-3轮 |
| 上市委/重组委 | 重组委（证监会/交易所） | 上市委（交易所） |
| 否决率 | 约5-10% | 约10-15% |
| 申报前审批环节 | 需股东大会 | 需股东大会 |
| 辅导阶段 | 不需要辅导 | 辅导备案和验收 |
