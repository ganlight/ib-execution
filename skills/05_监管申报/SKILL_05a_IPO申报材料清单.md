# SKILL_05a：IPO申报材料清单

## 一、概述

IPO申报文件是向交易所和证监会提交的全套法定文件。根据《首次公开发行股票注册管理办法》和各板块上市审核规则，申报文件分为招股说明书、发行人申请文件、保荐人文件、会计师文件、律师文件等几个大类。

本清单基于**全面注册制**下的最新要求编制，各板块差异处另行标注。

## 三、文件格式要求

### 3.1 电子版申报

- 格式：PDF文件（含书签/目录导航）
- 命名：与文件清单序号对应的规范命名
- 上传：通过交易所审核系统上传

### 3.2 签章要求

- 发行人：公章 + 法定代表人签字或盖章
- 保荐人：公章 + 法定代表人、保荐业务负责人、保荐代表人签字
- 会计师事务所：公章 + 两名签字注册会计师签字盖章
- 律师事务所：公章 + 两名签字律师签字盖章

### 3.3 纸质版要求

- 份数：按交易所要求报送（通常3-5份）
- 装订：A4纸张，胶装成册
- 封面：统一格式（含文件名称、发行人名称、保荐人名称、申报日期）

## 四、申报前完整性核对清单

### 4.1 必备文件核对

- [ ] 招股说明书（含签字页）
- [ ] 保荐人推荐书
- [ ] 保荐工作报告/发行保荐书
- [ ] 发行人董事会决议
- [ ] 发行人股东大会决议
- [ ] 审计报告（审计期在有效期内：基准日后6个月内）
- [ ] 内部控制鉴证报告
- [ ] 非经常性损益核验报告
- [ ] 纳税情况鉴证报告
- [ ] 法律意见书
- [ ] 律师工作报告
- [ ] 保荐人内核意见
- [ ] 保荐人尽调工作底稿目录
- [ ] 板块特殊要求文件

### 4.2 签章完整性核对

- [ ] 发行人公章 + 法定代表人签字
- [ ] 全体董事、监事、高级管理人员在招股说明书上签字
- [ ] 保荐人公章 + 法定代表人、保荐业务负责人、保荐代表人签字
- [ ] 会计师公章 + 两名签字注册会计师签字
- [ ] 律师公章 + 两名签字律师签字

### 4.3 时效性核对

- [ ] 审计报告基准日距今不超过6个月
- [ ] 内控鉴证报告基准日与审计报告一致
- [ ] 法律意见书签署日距今合理（通常不超过1个月）
- [ ] 董事会/股东大会决议在有效期内

### 4.4 数据一致性核对

- [ ] 招股书财务数据 = 审计报告数据
- [ ] 招股书法律数据 = 法律意见书数据
- [ ] 保荐工作报告数据 = 招股书数据
- [ ] 募集资金总额 = 各项目之和
- [ ] 发行后股本 = 发行前股本 + 本次发行股数

## 五、申报系统提交流程

1. 通过交易所审核系统创建项目
2. 逐一上传申报文件（按系统目录结构）
3. 提交受理申请
4. 交易所形式审查（5个工作日内完成）
5. 形式审查通过 → 正式受理 → 披露预披露文件（招股书申报稿）
6. 形式审查补正 → 10个工作日内补正 → 重新提交
