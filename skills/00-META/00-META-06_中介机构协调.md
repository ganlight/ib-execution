## 二、文件产出顺序与依赖链

### 2.1 关键路径图

```
时间轴 →

[会计师] 审计报告（初稿）
    │
    ├──→ [会计师] 内控鉴证报告 ← 依赖审计报告财务数据
    │
    ├──→ [律师] 法律意见书 ← 依赖审计的关联方/担保数据
    │         │
    │         ├──→ [券商] 招股书-历史沿革 ← 依赖法律意见书
    │         ├──→ [券商] 招股书-关联交易 ← 依赖审计+法律
    │         └──→ [券商] 招股书-公司治理 ← 依赖法律意见书
    │
    ├──→ [券商] 招股书-财务会计信息 ← 依赖审计报告
    ├──→ [券商] 招股书-管理层分析(MD&A) ← 依赖审计报告
    │
    └──→ [评估师] 评估报告 ← 依赖审计报告的资产数据
              │
              └──→ [券商] 招股书-资产评估章节（如适用）

    [券商] 整合所有内容 → 招股说明书（完整稿）
      └──→ 全套申报文件包
```

## 三、常见协调问题与解决方案

### 问题一：审计报告延迟交付

```
┌─────────────────────────────────────────────────────────────┐
│ 典型场景：                                                  │
│  年报审计尚未结束，审计报告需延迟2-3周                      │
│  但招股书其他章节已基本完成                                 │
├─────────────────────────────────────────────────────────────┤
│ 影响分析：                                                  │
│  🔴 直接影响：招股书财务章节、MD&A、内控报告无法定稿       │
│  🟡 间接影响：法律意见书关联数据、评估报告基准日数据        │
├─────────────────────────────────────────────────────────────┤
│ 解决方案（三级递进）：                                      │
│                                                             │
│  Level 1 - 使用未审数据先行（★★★★★ 最常用）               │
│    券商：以发行人提供的未审管理层报表为基础，先行撰写MD&A  │
│    律师：以未审关联交易数据先行撰写法律意见书关联交易部分   │
│    条件：会计师需书面确认未审数据的编制基础                 │
│    风险：未审数与审定数可能有差异                           │
│                                                             │
│  Level 2 - 分批交付                                          │
│    会计师：先交付审定财务报表（不含附注）                   │
│    券商：先用审定报表更新核心数据                           │
│    会计师：附注和完整报告随后交付                           │
│                                                             │
│  Level 3 - 内核稿使用未审数据 + 内核后替换                   │
│    时间最紧时的做法，风险较高，需保荐机构内核会同意         │
│                                                             │
│  ⚠️ 红线：申报稿必须使用审定数据，不得有任何未审数据        │
└─────────────────────────────────────────────────────────────┘
```

### 问题二：法律意见书滞后

```
┌─────────────────────────────────────────────────────────────┐
│ 典型场景：                                                  │
│  律师尽调发现历史沿革问题，需要额外核查时间                  │
│  法律意见书节奏明显落后于其他文件                           │
├─────────────────────────────────────────────────────────────┤
│ 解决方案：                                                  │
│                                                             │
│  Strategy A - 分拆撰写                                       │
│    法1 先行定稿无争议部分：公司设立、历次增资、经营范围     │
│         这些可独立于复杂问题先行交付                         │
│    法2 复杂问题专项处理：                                   │
│         - 历史上的股权代持 → 单独出具核查报告               │
│         - 知识产权争议 → 分拆为专项法律意见                 │
│    法3 招股书对应章节先用初步法律结论                       │
│         - 标注"以最终法律意见书为准"                         │
│                                                             │
│  Strategy B - 加速核查                                       │
│    - 增派律师人手（需提前协议约定费用）                     │
│    - 或由券商法务协助基础资料整理                           │
│    - 发行人专人对接工商/法院档案调取                        │
└─────────────────────────────────────────────────────────────┘
```

### 问题三：中介机构之间意见分歧

```
┌─────────────────────────────────────────────────────────────┐
│ 典型争议：                                                  │
│  Ⓐ 关联方认定范围 → 会计师vs律师                            │
│    会计师主张按会计准则（控制/重大影响）                    │
│    律师主张按上市规则（更宽泛，含历史关联方）               │
│                                                             │
│  Ⓑ 收入确认时点 → 会计师vs券商                              │
│    会计师按审计准则给出确认时点                             │
│    券商从披露充分性角度认为需要更详细的解释                 │
│                                                             │
│  Ⓒ 估值方法选择 → 评估师vs券商                              │
│    评估师偏好传统方法                                       │
│    券商建议考虑市场法对标（以便定价参考）                   │
├─────────────────────────────────────────────────────────────┤
│ 解决方案：                                                  │
│                                                             │
│  原则：保荐机构拥有最终决定权（法规赋予）                   │
│                                                             │
│  处理流程：                                                 │
│  Step 1: 各方书面陈述立场和依据（各1页以内）               │
│  Step 2: 保荐机构组织四方专题会议（线上即可）              │
│  Step 3: 保荐机构形成最终决议，记录理由                    │
│  Step 4: 持异议方可在各自文件中保留意见                    │
│          （但申报文件应以一致口径呈现）                     │
│                                                             │
│  升级路径：                                                 │
│  重大分歧 → 咨询第三方专家（其他律所/会所）                 │
│  根本分歧 → 提前与交易所沟通窗口指导意见                    │
│  无法调和 → 评估项目可行性，避免带病申报                   │
│                                                             │
│  实战案例：                                                 │
│  某创业板项目：会计师认为某大额合同收入确认应在验收后，     │
│  券商认为根据合同条款可在发货后确认。最终券商组织双方与     │
│  交易所预沟通，交易所倾向于会计师的保守处理，各方接受。     │
└─────────────────────────────────────────────────────────────┘
```

### 问题四：发行人配合度低

```
┌─────────────────────────────────────────────────────────────┐
│ 典型表现：                                                  │
│  - 资料提供不及时（承诺3天，实际拖2周）                     │
│  - 关键人员难以约到访谈时间                                 │
│  - 对尽调问题的回复模棱两可                                 │
│  - 实控人/董事长不重视，把IPO当成中介的活                  │
├─────────────────────────────────────────────────────────────┤
│ 解决方案：                                                  │
│                                                             │
│  🟢 日常级（每日/每周推）：                                 │
│    - 建立资料需求清单+负责人+截止日期（在线共享表格）       │
│    - 每日傍晚邮件/微信推送"今日待完成事项"                  │
│    - 每周五发送"本周进展+下周计划+逾期警示"                 │
│    - 保荐代表人每周与发行人对接人通话1次                    │
│                                                             │
│  🟡 升级级（逾期>1周）：                                    │
│    - 保荐机构发送正式书面催告函                             │
│    - 约见发行人实控人/董事长当面沟通                        │
│    - 明确告知进度延迟对申报时间窗口的影响                   │
│    - 建议发行人设立专职IPO工作组（可从各部门借调）          │
│                                                             │
│  🔴 严重级（持续不改善）：                                  │
│    - 由保荐机构负责人向发行人实控人发出风险提示             │
│    - 暂停部分现场工作以释放信号                             │
│    - 评估是否需要终止辅导/暂停项目                          │
│    - 若发行人确有上市诚意但执行力不足 → 加派驻场人员       │
└─────────────────────────────────────────────────────────────┘
```

## 五、协调最佳实践

### 5.1 黄金原则

1. **保荐机构说"最后一句话"** — 文件可以各有表述，但申报文件口径必须一致
2. **问题不过夜** — 中介间争议24小时内启动沟通，72小时内要有结论
3. **书面留痕** — 所有重要沟通用邮件确认，口头沟通后补书面纪要
4. **提前量思维** — 预留20%缓冲时间应对突发延迟
5. **一个坑一个责任人** — 每项任务只设一个主责机构

### 5.2 工具推荐

| 工具 | 用途 | 备注 |
|------|------|------|
| 飞书/企业微信 | 日常即时沟通 | 建立项目专用群 |
| 腾讯文档 | 资料清单/进度在线共享 | 方便发行人随时查看 |
| 腾讯会议 | 线上协调会 | 每周至少1次全体同步 |
| 邮件 | 正式沟通/催交/决策 | 所有正式决策必须邮件留痕 |
