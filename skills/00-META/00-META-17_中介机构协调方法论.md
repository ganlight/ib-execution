# META-17: 中介机构协调方法论

> 定位：投行作为项目总协调人，系统性管理各中介机构的方法论和最佳实践。

## 一、投行作为总协调人的角色定位

### 1.1 为什么投行是总协调人？

```
                 ┌──────────────────┐
                 │   客户（发行人）   │
                 └────────┬─────────┘
                          │
              ┌───────────┼───────────┐
              │                       │
     ┌────────▼────────┐     ┌───────▼────────┐
     │   投行（总协调）│     │  独立财务顾问   │
     │  承销/保荐/     │     │  (如适用)      │
     │  重组财务顾问   │     └────────────────┘
     └────────┬────────┘
              │
  ┌───────────┼───────────────┬──────────────┐
  │           │               │              │
┌─▼──┐   ┌───▼───┐   ┌──────▼──────┐ ┌─────▼─────┐
│律师│   │会计师 │   │ 评级机构     │ │评估机构   │
│   │   │       │   │  (债券必备)  │ │(重组必备) │
└───┘   └──────┘   └─────────────┘ └──────────┘
```

**投行是唯一的"全程参与者"** — 从项目启动到存续期管理，而各中介仅在特定阶段介入。

### 1.2 总协调人的核心职责

| 职责 | 说明 |
|------|------|
| 制定项目计划 | 制定总时间表，分解各中介任务 |
| 协调信息流 | 确保信息在各中介间畅通流动 |
| 整合最终产品 | 汇总各中介产出，形成统一申报文件 |
| 质量把关 | 对各中介产出进行第一轮审核 |
| 客户界面管理 | 统一对外沟通口径 |
| 问题升级处理 | 中介间分歧的裁决 |

## 二、中介选聘标准

### 2.1 各中介资质要求

| 中介类型 | 必备资质 | 补充要求 |
|----------|----------|----------|
| **律师事务所** | 证券法律业务资格 | 近3年无重大处罚 |
| **会计师事务所** | 证券期货相关业务资格 | 审计团队经验 |
| **信用评级机构** | 人民银行/证监会核准 | 市场认可度 |
| **资产评估机构** | 证券期货相关业务资格 | 相关行业评估经验 |
| **独立财务顾问** | 投行资质 | 独立性要求 |

### 2.2 中介选聘评分维度

| 评估维度 | 权重 | 评分要素 |
|----------|------|----------|
| **项目经验** | 30% | 同类项目数量、规模、成功率 |
| **团队配置** | 25% | 签字人资历、团队稳定性、人员充足 |
| **市场声誉** | 15% | 监管评价、客户口碑、行业排名 |
| **服务报价** | 15% | 报价合理性、付款节奏 |
| **独立性** | 15% | 是否与发行人/其他中介有利益关系 |

### 2.3 独立性检查清单

| 检查项 | 红线 | 处理方式 |
|--------|------|----------|
| 审计师与发行人 | 连续审计超过规定年限 | 更换审计师 |
| 律师与发行人 | 长期独家法律顾问 | 需独立履职 |
| 评级机构与发行人 | 存在利益关联 | 更换评级机构 |
| 评估机构与交易方 | 与交易对方有关联 | 更换评估机构 |

```
┌──────────────────────────────────────────────────────────────┐
│                    分工边界矩阵                                │
├──────────┬──────────┬──────────┬──────────┬──────────────────┤
│ 工作内容  │  投行     │  律师     │  会计师  │  评级/评估       │
├──────────┼──────────┼──────────┼──────────┼──────────────────┤
│ 尽职调查  │ 业务DD   │ 法律DD   │ 财务DD   │ 评估DD(重组)     │
│ 文件撰写  │ 招股书/  │ 法律意见 │ 审计报告 │ 评级报告/        │
│          │ 募集书   │ 书       │ 内控鉴证 │ 评估报告         │
│ 方案设计  │ 交易方案 │ 法律合规 │ —        │ —               │
│ 监管沟通  │ 主沟通   │ 专业沟通 │ 财务沟通 │ 必要时           │
│ 反馈回复  │ 统筹回复 │ 法律部分 │ 财务部分 │ —               │
│ 发行执行  │ 主执行   │ 法律审核 │ —        │ —               │
│ 存续期    │ 持续督导 │ —        │ —        │ 跟踪评级         │

```

**说明**：
- ✅ = 主导责任
- ⚠️ = 辅助/配合
- — = 不涉及

### 3.2 各产品线下分工差异

| 工作内容 | IPO | 债券 | 并购 |
|----------|-----|------|------|
| **法律DD深度** | 全面+历史沿革 | 资产权属+担保 | 标的权属+合规 |
| **财务DD焦点** | 收入真实性 | 偿债能力 | 标的估值 |
| **报告类型** | 法律意见+审计+内控 | 法律意见+审计+评级 | 法律意见+审计+评估 |
| **中介数量** | 3-4家 | 3-4家 | 4-6家(含交易对方) |

## 四、沟通机制

### 4.1 正式沟通安排

| 沟通类型 | 频率 | 参与方 | 主要内容 | 产出 |
|----------|------|--------|----------|------|
| **周例会** | 每周 | 全体中介 | 进度汇报、问题讨论 | 会议纪要 |
| **月度总结** | 每月 | 全体中介+客户 | 月度进展、下月计划 | 月度报告 |
| **专题讨论会** | 按需 | 相关中介 | 解决特定问题 | 专题报告 |
| **内核沟通会** | 重大节点 | 投行内部 | 项目风险评估 | 内核意见 |
| **重大问题即时沟通** | 事件触发 | 相关方 | 重大事项通报 | 沟通记录 |

### 4.2 周例会标准议程

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  第X周项目例会  YYYY-MM-DD
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 上周待办事项跟进（5min）
2. 各中介进度汇报（各5-10min）
   - 投行：整体进度、方案更新
   - 律师：法律尽调进展
   - 会计师：财务尽调进展
   - 评级/评估：（如适用）
3. 问题讨论与决策（15-20min）
4. 下周工作安排（5min）
5. 客户沟通议题确认（5min）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 4.3 沟通纪律

| 规则 | 内容 |
|------|------|
| 会前准备 | 各中介提前1天提交进度更新 |
| 纪要标准 | 24小时内发出，明确待办事项和责任人 |
| 信息共享 | 重要信息同步推送，不得仅在个别中介间流传 |
| 统一口径 | 对客户和监管统一口径，由投行协调 |
| 信息隔离 | 涉及内幕信息的，严格遵守信息隔离规则 |

## 六、收费结构

### 6.1 行业惯例概览

| 费用类型 | 投行 | 律师 | 会计师 | 评级机构 | 评估机构 |
|----------|------|------|--------|----------|----------|
| **前期费** | 有 | 无 | 无 | 无 | 无 |
| **阶段费** | 部分 | 按阶段 | 按阶段 | 发债前 | 按阶段 |
| **成功费** | 是 | 否 | 否 | 否 | 否 |
| **存续期费** | 是 | 否 | 否 | 跟踪评级 | 无 |

### 6.2 费用结构详解

#### 投行承销费

```
总承销费 = 发行规模 × 承销费率

其中承销费率参考范围：
  IPO        3% - 8%  （取决于规模和复杂度）
  债券        0.3% - 1.5%
  并购财务顾问 交易金额 × 阶梯费率
```

#### 律师费

```
律师费 = 固定费用 + 工作量(超出范围的按小时计)

通常以阶段付费：
  尽调阶段       30%
  文件撰写阶段   40%
  申报+发行       30%
```

#### 审计费

```
审计费 = 年度审计费 + IPO专项审计费

IPO专项审计 = 近3年财务核查费用 + 申报期加期审计
```

### 6.3 费用管理要点

| 原则 | 说明 |
|------|------|
| 报价比较 | 同项目中至少获取3家报价 |
| 波动预留 | 预算留10-15%弹性空间 |
| 存续期费 | 提前与客户沟通持续费用 |
| 超支预警 | 费用接近预算时提前预警 |

## 七、冲突管理

### 7.1 中介意见分歧的类型与处理

```
常见分歧类型：

1. 法律问题分歧（投行 vs 律师）
2. 财务处理分歧（投行 vs 会计师）  
3. 估值结果分歧（投行 vs 评估机构）
4. 信息披露分歧（各中介意见不一致）
```

### 7.2 分歧处理规则

| 分歧类型 | 排序规则 | 升级机制 |
|----------|----------|----------|
| **法律判断** | 律师 > 投行 | 投行可建议，但法律结论以律师为准 |
| **财务判断** | 会计师 > 投行 | 审计结论以会计师为准，投行可质疑并要求说明 |
| **估值结论** | 多方协商 | 投行可提出不同方法论，最终由客户决策 |
| **披露程度** | 最严格标准 | 不同中介意见有不同的披露标准时，采用最严格标准 |
| **风险评估** | 投行 > 其他中介 | 投行作为承销商/保荐人，最终风险评估以投行为准 |

### 7.3 冲突升级机制

```
Level 1: 直接沟通
  └─ 发现分歧 → 投行牵头 → 相关中介直接讨论

Level 2: 专题会议
  └─ L1无法解决 → 召开全体中介专题会

Level 3: 专家咨询
  └─ L2仍有分歧 → 引入行业专家/独立第三方

Level 4: 客户决策
  └─ 将各方案及其利弊呈报客户 → 客户决策

Level 5: 内核裁决
  └─ L4后仍有重大分歧 → 提交内核裁定
```

### 7.4 分歧管理最佳实践

| 原则 | 实践 |
|------|------|
| **问题前置** | 不要等到申报前夕才发现分歧 |
| **有据可查** | 所有分歧的讨论过程和结论留有书面记录 |
| **信息充分** | 确保各方在做决策前获得充分信息 |
| **明确定论** | 不能留下"待定"的不确定性问题 |

## 八、使用指南

### 8.1 关键时点的中介协调动作

| 项目时点 | 投行应做的协调动作 |
|----------|--------------------|
