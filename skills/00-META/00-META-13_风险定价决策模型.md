# META-13: 风险定价决策模型

> 定位：系统化的风险定价决策框架，帮助项目组在定价环节做出有数据支撑的理性决策。

## 一、定价影响因素金字塔

定价是投行的核心能力。理解影响定价的因素层次，是精准定价的前提。

```
                  ┌─────────────────────────────────────┐
                  │  顶层：市场因素                      │
                  │  - 市场供需状况                     │
                  │  - 宏观利率环境                     │
                  │  - 投资者风险偏好                   │
                  │  - 同类型项目发行节奏               │
                  │  "市场给什么价？"                   │
                  ├─────────────────────────────────────┤
                  │  中层：定性因素                      │
                  │  - 公司治理水平                     │
                  │  - 业务稳定性/可持续性              │
                  │  - 管理层能力与诚信                 │
                  │  - 担保增信安排                     │
                  │  - 行业政策环境                     │
                  │  "凭什么值这个价？"                 │
                  ├─────────────────────────────────────┤
                  │  底层：定量因素                      │
                  │  - 财务指标体系                     │
                  │  - 信用评级结果                     │
                  │  - 行业基准/可比公司                │
                  │  - 历史交易参考                     │
                  │  "数字说了什么？"                   │
                  └─────────────────────────────────────┘
```

### 1.1 各层因素的决策权重

| 因素层次 | 债券决策权重 | 股票决策权重 | 并购决策权重 |
|----------|-------------|-------------|-------------|
| 底层（定量） | 60% | 40% | 35% |
| 中层（定性） | 25% | 35% | 30% |
| 顶层（市场） | 15% | 25% | 35% |

### 1.2 关键定量指标体系

#### 债券定价核心指标

| 指标类别 | 具体指标 | Normal范围 | 信号意义 |
|----------|----------|------------|----------|
| 偿债能力 | 利息保障倍数 | >3倍 | 越高定价越低 |
| 杠杆水平 | 资产负债率 | <70%（一般企业） | 越低定价越低 |
| 流动性 | 流动比率 | >1.5 | 越高定价越低 |
| 盈利能力 | EBITDA利润率 | >15% | 越高定价越低 |
| 现金流 | 经营现金流/总债务 | >0.2 | 越高定价越低 |

#### 股票估值核心指标

| 指标类别 | 具体指标 | 分析要点 |
|----------|----------|----------|
| 盈利能力 | 净利润、毛利率、ROE | 趋势分析、行业比较 |
| 成长性 | 收入增长率、利润增长率 | 可持续性评估 |
| 资产质量 | 应收账款周转率、存货周转率 | 与同行业比较 |
| 现金流 | 经营现金流/净利润比 | >1.0为健康指标 |

### 2.2 信用利差计算模型

```
信用利差 = 无风险利差基准 + 行业利差 + 个券利差

其中：
· 无风险利差基准 = 同期国债收益率
· 行业利差 = 同行业/同评级平均利差 - 无风险基准
· 个券利差 = 发行人特定风险溢价
  └─ 包含：财务风险溢价 + 治理风险溢价 + 流动性溢价
```

### 2.3 债券各品种典型定价区间

| 品种 | 基准参考 | 典型利差范围 | 定价影响因素 |
|------|----------|-------------|-------------|
| 超短期融资券(SCP) | 同期限Shibor | +50~150bp | 信用、期限、规模 |
| 短期融资券(CP) | 同期限国债 | +60~180bp | 信用、行业、规模 |
| 中期票据(MTN) | 同期限国债 | +80~250bp | 信用、期限、行业 |
| 企业债 | 同期限国债 | +100~300bp | 信用、担保、项目 |
| 公司债 | 同期限国债 | +80~250bp | 信用、行业、交易所流动性 |
| 定向工具(PPN) | 同期限国债 | +100~300bp | 流动性折价+信用 |

## 三、股票定价框架（注册制下）

### 3.1 注册制下IPO定价流程

```
┌──────────────────────────────────────────────────────────────┐
│                    注册制IPO定价流程                           │
│                                                              │
│  第一步：可比公司估值                                         │
│  ┌─────────────────────────────────────────────────────┐    │
│  │  方法：PE估值 / PB估值 / PS估值  （根据行业特性选择）  │    │
│  │  产出：估值区间 [A_low, A_high]                       │    │
│  └─────────────────────────────────────────────────────┘    │
│                          │                                   │
│                          ▼                                   │
│  第二步：路演/询价反馈                                        │
│  ┌─────────────────────────────────────────────────────┐    │
│  │  - 面向机构投资者的询价                              │    │
│  │  - 收集投资故事反馈                                 │    │
│  │  - 修正：基于反馈调整估值                            │    │
│  │  产出：修正后区间 [B_low, B_high]                    │    │
│  └─────────────────────────────────────────────────────┘    │
│                          │                                   │
│                          ▼                                   │
│  第三步：最终定价区间确定                                      │
│  ┌─────────────────────────────────────────────────────┐    │
│  │  综合考虑：                                          │    │
│  │  · 估值模型结果                                     │    │
│  │  · 询价反馈                                         │    │
│  │  · 市场窗口（指数水平、行业热度）                    │    │
│  │  · 可比同业的近期发行价                             │    │
│  │  · 发行人预期                                       │    │
│  │  产出：最终发行价区间 [Price_low, Price_high]        │    │
│  └─────────────────────────────────────────────────────┘    │
└──────────────────────────────────────────────────────────────┘
```

### 3.2 估值方法选择矩阵

| 行业类别 | 首选估值方法 | 次选估值方法 | 不适用方法 | 原因 |
|----------|-------------|-------------|-----------|------|
| 制造业 | PE | PB | PS | 有成熟盈利 |
| 科技/互联网 | PS | PE（如盈利） | PB | 轻资产，成长期 |
| 金融（银行） | PB | PE | PS | 重资产，ROE驱动 |
| 消费品 | PE | PS | PB | 品牌价值PB不反映 |
| 生物医药 | rNPV | PS | PE/PB | 管线估值 |
| 房地产 | NAV | PB | PS | 重资产 |

### 3.3 询价策略

| 询价策略 | 适用场景 | 风险 |
|----------|----------|------|
| **积极定价** | 需求旺盛、供给稀缺 | 发行失败风险 |
| **保守定价** | 需求谨慎、供给充裕 | 客户满意度低 |
| **阶梯定价** | 开价低，逐渐上移 | 谈判时间延长 |

### 3.4 定价的风险管理

| 风险类型 | 管理措施 |
|----------|----------|
| 定价过高 → 发行失败 | 提前测试投资者意向 |
| 定价过低 → 客户不满 | 合理价值论证+发行后表现预测 |
| 市场骤变 → 窗口关闭 | 发行时机选择+弹性定价区间 |

## 四、对赌条款设计逻辑（并购）

### 4.1 对赌条款全景图

```
                 ┌──────────────────┐
                 │   对赌条款体系    │
                 └────────┬─────────┘
                          │
          ┌───────────────┼───────────────┐
          │               │               │
  ┌───────▼──────┐ ┌──────▼──────┐ ┌──────▼──────┐
  │  业绩补偿    │ │  减值补偿    │ │  股权回购    │
  │             │ │             │ │             │
  │ 净利润不达标│ │ 标的价值下降│ │ 触发回购条件│
  │ → 现金/股份 │ │ → 补偿差额  │ │ → 回购义务  │
  └─────────────┘ └─────────────┘ └─────────────┘
```

### 4.2 三种对赌机制对比

| 维度 | 业绩补偿 | 减值补偿 | 股权回购 |
|------|----------|----------|----------|
| 触发条件 | 承诺期净利润不达标 | 减值测试发现减值 | 特定事件触发 |
| 补偿方式 | 优先股份补偿/现金 | 现金补偿为主 | 现金回购 |
| 计算公式 | (累计承诺-累计实现)/累计承诺 × 交易对价 | 减值金额 - 已补偿金额 | 约定回购价格 |
| 保护对象 | 上市公司 | 上市公司 | 上市公司/交易对方 |
| 常见存续期 | 3年 | 3年+额外测试 | 触发后执行 |
| 近年趋势 | 监管收紧，鼓励非业绩承诺 | 标准条款保留 | 谨慎使用 |

### 4.3 业绩补偿条款设计要点

#### 核心要素

```
业绩承诺 = {
  承诺主体：交易对方（原股东）
  承诺期间：通常3年（可延长至4年）
  承诺指标：扣非归母净利润（主要）
  补偿方式：优先股份补偿，不足部分现金补足
  ─────────────────────────────────
}
```

#### 累进式补偿计算示例

| 实际完成率 | 补偿比例 | 说明 |
|------------|----------|------|
| ≥100% | 无需补偿 | 目标达成 |
| 80-100% | 差额全额补偿 | 小幅不达标 |
| 60-80% | 差额全额+额外10% | 中幅度不达标 |
| <60% | 差额全额+额外20% | 大幅不达标 |

#### 补偿公式

```
当期应补偿金额 = (截至当期期末累积承诺净利润数 - 截至当期期末累积实现净利润数)
                 ÷ 业绩承诺期内各年承诺净利润总和
                 × 本次交易总对价
                 - 累积已补偿金额
```

### 4.4 减值补偿条款

**触发时机**：业绩承诺期届满时进行减值测试

```
减值测试流程：
1. 聘请评估机构对标的价值进行评估
2. 计算：期末减值额 = 交易对价 - 期末评估价值
3. 比较：减值补偿 = 期末减值额 - 已支付的业绩补偿总额
4. 如果减值补偿 > 0 → 交易对方需进行额外补偿
```

### 4.5 股权回购条款

| 触发事件 | 典型条款 | 回购价格 |
|----------|----------|----------|
| 标的公司核心资产/资质丧失 | 丧失核心业务资质 | 交易对价+固定利率 |
| 交易对方重大违约 | 财务造假、重大隐瞒 | 交易对价+违约金 |
| 标的公司持续经营能力丧失 | 停产、破产等 | 届时评估价值 |
| 合同约定的其他情形 | 协议约定 | 协议约定 |

### 4.6 对赌条款设计实务建议

| 建议 | 理由 |
|------|------|
| 承诺指标聚焦核心利润 | 监管偏好扣非归母净利润 |
| 承诺梯度合理 | 20-30%年增长率为合理上限 |
| 补偿上限设有（或不设） | 监管趋严，上限保护逐步减弱 |
| 分年解锁的锁定期 | 与业绩承诺期匹配 |

## 五、定价情景分析法

### 5.1 定价敏感性分析框架

对于一个债券/股票发行，建议至少做三种情景分析：

| 情景 | 假设条件 | 定价影响 |
|------|----------|----------|
| **乐观情景** | 市场环境好、投资者认购积极 | 定价上沿 |
| **基准情景** | 正常市场环境、合理定价 | 定价中枢 |
| **悲观情景** | 市场环境差、投资者认购谨慎 | 定价下沿 |

### 5.2 市场窗口分析

```
影响发行窗口的关键变量：

1. 利率走势预测
   - 上升期 → 尽早发
   - 下降期 → 可稍等

2. 同类供给情况
   - 供给集中 → 可能推高利率
   - 供给稀缺 → 有利于定价

3. 资本市场情绪
   - 乐观 → 风险偏好高 → 利好
   - 悲观 → 风险偏好低 → 利差扩大

4. 政策因素
   - 信用紧缩 → 利差扩大
   - 信用宽松 → 利差收窄
```

## 六、使用指南

### 6.1 常用计算速查

**YTM（到期收益率）**
```
P = Σ[C/(1+YTM)^t] + F/(1+YTM)^n
其中：P=全价, C=每期票息, F=面值, n=剩余期限, t=期数
```

**修正久期**
```
MD = MacD / (1 + YTM/m)
其中：MacD=麦考利久期, m=年付息次数
```
