# SKILL_00：债券文件总览

## 概述

债券发行文件是债券项目能否通过监管审核、获得投资者认可的核心载体。与IPO以"招股说明书"为单一核心文件不同，债券发行文件是一个**文件体系**——由10余项相互关联的文件构成，任何一项出现重大错漏都可能导致审核被驳回或发行失败。

本文件作为债券线第四阶段（04_文件撰写）的路由入口，阐述债券发行文件全景、募集说明书的核心地位与7大核心内容、文件质量检查清单以及各品种文件差异。详细的募集说明书撰写指引见 `SKILL_04_债券募集说明书.md`，文件框架参考见 `../../references/bond/bond-prospectus-detail.md`。

## 一、债券发行文件全景（10项核心文件+每项撰写责任方）

债券发行涉及至少10项核心文件，分布在不同撰写责任方之间。以下全景表展示每份文件的撰写责任、审核重点和相互引用关系：

| 序号 | 文件名称 | 撰写责任方 | 审核关注重点 | 与其他文件的引用关系 |
|:----:|---------|----------|------------|----------------|
| **1** | **募集说明书** | 主承销商（主笔） | 发行人的核心信息披露 | 引用审计报告、评级报告、法律意见书中几乎所有核心章节 |
| **2** | 评级报告 | 评级机构 | 发行人/债项信用等级及评级理由 | 被募集说明书核心章节引用 |
| **3** | 法律意见书 | 律师事务所 | 法律合规性验证 | 被募集说明书"法律情况"章节引用 |
| **4** | 审计报告 | 审计机构 | 近2-3年财务数据 | 被募集说明书财务报表章节引用 |
| **5** | 主承销商核查意见 | 主承销商DCM内核 | 合规+风险验证 | 与募集说明书的信息披露一致 |
| **6** | 受托管理协议 | 主承销商+发行人 | 存续期管理和权益保护 | 附件形式与募集说明书同步披露 |
| **7** | 持有人会议规则 | 主承销商 | 持有人权利保障机制 | 附件形式与募集说明书同步披露 |
| **8** | 发行说明书要点 | 主承销商 | 发行要素摘要（发行公告精简版） | 提炼自募集说明书核心要素 |
| **9** | 信用评级说明 | 评级机构 | 评级方法论和关键假设 | 被募集说明书评级章节引用 |
| **10** | 担保协议（如有） | 担保人+发行人 | 增信措施法律效力 | 被募集说明书增信章节引用 |

### 常规信用债文件数量速查

| 品种 | 核心文件数 | 增量文件 |
|------|:-------:|---------|
| MTN/CP/SCP | 9项 | 无 |
| PPN | 8项 | 无需受托管理协议（私募定向） |
| 公募公司债（大公募/小公募） | 10项 | 全部覆盖 |
| 私募公司债 | 8项 | 小公募（无需受托管理协议or规则简化） |
| 企业债 | 11项 | +募投项目相关文件（项目审批/可行性研究） |
| ABS/ABN | 12项 | +SPV设立文件+资产买卖协议+资产服务协议 |

## 二、募集说明书是核心——必须覆盖的7大核心内容

募集说明书（Prospectus/Master Book）是所有债券发行文件中的**绝对核心**。监管审核、投资者决策、发行定价都围绕募集说明书展开。

### 7大核心内容与撰写要点

| 序号 | 核心内容 | 最低要求 | 常见审核关注 |
|:----:|---------|---------|------------|
| **1** | **发行人基本情况** | 近3年主营业务+股权结构+实际控制人 | 实控人变更/重大资产重组是否披露 |
| **2** | **财务会计信息** | 近2年+一期经审计合并报表 | 非标审计意见/会计政策变更/商誉 |
| **3** | **信用评级** | 主体评级+债项评级+评级展望 | 评级模型/关键假设/与发行人级别的匹配性 |
| **4** | **募集资金用途** | 具体用途+金额+时间安排 | 募投项目合规性/补流测算/资金监管安排 |
| **5** | **偿债保障措施** | 现金流+资产+增信全覆盖 | EBITDA利息倍数/偿债来源分析/极端情景 |
| **6** | **风险因素** | 本期债券特有的全部风险 | 评级下调风险/再融资风险/增信失效风险 |
| **7** | **发行条款** | 期限/利率/赎回/回售/付息 | 含权条款完整性/票面利率确定方式 |

### 各品种募集说明书差异

| 品种 | 披露深度 | 特殊章节 | 披露简化项 |
|------|---------|---------|---------|
| **MTN/CP/SCP** | 标准深度 | 无特殊 | - |
| **PPN** | 简化深度 | 定向投资人范围说明 | 可不披露完整财务附注 |
| **公募公司债** | 高深度 | 信息披露最严格 | - |
| **私募公司债** | 中等深度 | 简化财务附注 | 审计可仅覆盖近1年 |
| **企业债** | 高深度+募投项目 | 募投项目详细章节（立项/可研/环评等） | 需完整项目合规文件 |
| **ABS** | 特殊结构 | 资产池章节+SPV结构+现金流模型 | 资产逐笔/抽样信息 |
| **可转债** | 高深度+转股条款 | 转股价格确定/向下修正/强制赎回 | 需盈利预测+稀释性分析 |

## 三、文件质量检查清单（15项格式/内容/合规检查）

在文件签发前，必须完成以下15项质量检查。任何一项不通过都必须修改：

### A. 格式检查（5项）

| # | 检查项 | 合格标准 | 不合格后果 |
|:-:|--------|---------|----------|
| **A1** | 文件一致性 | 同一数据在审计报告/评级报告/募集说明书三处必须完全一致 | 审核质疑财务数据真实性 |
| **A2** | 签章完整性 | 所有盖章页/骑缝章/法定代表人签字必须齐全 | 形式审核驳回 |
| **A3** | 日期逻辑 | 审计报告出具日→评级报告出具日→募集说明书签署日→发行日必须符合时间顺序 | 审核质疑数据时效性 |
| **A4** | 语言规范 | 无"可能"/"大概"/"通常"等模糊表述（募集说明书事实陈述部分） | 审核质疑信息披露质量 |
| **A5** | 排版统一 | 字体/字号/行距/页眉页脚在整套文件中一致 | 专业性质疑 |

### B. 内容检查（6项）

| # | 检查项 | 合格标准 | 不合格后果 |
|:-:|--------|---------|----------|
| **B1** | 40%红线计算 | 募集说明书披露的累计债券余额≤净资产×40%，如有超出需说明 | 审核直接驳回 |
| **B2** | 资金用途合规 | 补流部分≤募资总额30%/项目投资≤总投资×60-70% | 审核质疑募资用途 |
| **B3** | 偿债措施不空洞 | 必须有可量化/可执行的偿债安排 | 审核质疑担保有效性 |
| **B4** | 风险揭示充分 | 风险因素章节必须覆盖本期债券特有风险不少于5项 | 审核质疑充分性 |
| **B5** | 评级依据充分 | 评级报告的分析逻辑必须与募集说明书"信用评级"章节一致 | 反馈要求补充 |
| **B6** | 重大诉讼完整 | 涉案金额>净资产5%的未决诉讼必须完整披露 | 审核质疑重大性 |

### C. 合规检查（4项）

| # | 检查项 | 合格标准 | 不合格后果 |
|:-:|--------|---------|----------|
| **C1** | 决议文件 | 发行人有权机构（董事会/股东会）发行决议原件 | 申报被退回 |
| **C2** | 审计意见 | 近2年审计报告无保留意见（无法表示意见/否定意见→无法申报） | 直接驳回 |
| **C3** | 失信查询 | 中国执行信息公开网查询无发行人/实控人失信记录 | 直接驳回 |
| **C4** | 行政处罚 | 近3年重大行政处罚已完整披露（如有）+整改完成 | 反馈质疑+可能驳回 |

## 五、Gate Check——文件撰写阶段三大关键验证

完成文件撰写后、申报前必须验证：

### Gate 1：募集说明书内核通过

| 验证项 | 通过标准 |
|-------|---------|
| 内容完整性 | 7大核心内容全部覆盖 |
| 数据一致性 | 审计报告/评级报告/募集说明书三处数据完全一致 |
| 合规性 | 40%红线/资金用途/风险因素均通过格式检查 |
| 表述质量 | 无重大事实性错误/无潜在诉讼风险 |

### Gate 2：法律意见书出具

| 验证项 | 通过标准 |
|-------|---------|
| 合规结论 | 近2年无重大违法违规/无正在进行的重大诉讼 |
| 决议有效性 | 发行决议已取得有权机构批准 |
| 担保（如有） | 担保文件法律效力无瑕疵 |

### Gate 3：评级报告定稿

| 验证项 | 通过标准 |
|-------|---------|
| 评级结论 | 主体/债项评级符合发行准入要求 |
| 评级展望 | "稳定"或"正面"，"负面"需充分风险揭示 |
| 评级与募集说明书一致 | 评级理由与募集说明书披露一致 |

## 六、文件撰写工作流与中介协调机制

### 6.1 债券文件撰写六步流程

```
第1步              第2步              第3步              第4步              第5步              第6步
资料收集  ──→  初稿撰写  ──→  内部审阅  ──→  中介会稿  ──→  内核审查  ──→  终稿定稿
(1-2周)          (2-3周)          (1周)            (1周)            (3-5天)          (2-3天)
```

**第1步：资料收集（1-2周）**

| 资料类型 | 来源 | 典型文件 |
|---------|------|---------|
| 财务资料 | 发行人财务部+审计机构 | 近2-3年审计报告、内审报告、银行授信函、征信报告 |
| 法律资料 | 发行人法务部+律师 | 营业执照、公司章程、股东决议、重大合同、诉讼仲裁文件 |
| 业务资料 | 发行人经营管理层 | 主营业务描述、行业分析、竞争格局、关联交易明细 |
| 信用资料 | 评级机构 | 评级报告初稿、信用分析模型、可比企业对比 |
| 增信资料（如有） | 担保人/评估机构 | 担保人审计报告、资产评估报告、抵押登记文件 |

**第2步：初稿撰写（2-3周）**
- 主承销商撰写团队（通常2-3人）分工撰写
- 以募集说明书为核心文件，其余文件同步推进
- 撰写顺序：①发行人基本情况 → ②财务会计信息 → ③信用评级 → ④募集资金用途 → ⑤偿债保障 → ⑥风险因素 → ⑦发行条款

**第3步：内部审阅（1周）**
- 主承销商内部法审、合规、风控三条线同步审阅
- 重点审阅：合规风险、信息披露充分性、事实准确性
- 审阅意见汇总后，由撰写团队统一修改

**第4步：中介会稿（1周）**
- **所有中介机构联席会议**（主承销商+律师+审计师+评级机构+发行人）
- 核对各机构出具文件的数据一致性（财务数据/法律事实/评级结论）
- 形成《文件一致性确认函》（所有中介机构签字）

**第5步：内核审查（3-5天）**
- 主承销商内核委员会审查全套文件
- 内核委员提出修改意见
- 撰写团队根据内核意见完成终稿修改

**第6步：终稿定稿（2-3天）**
- 全套文件终稿打印、装订
- 逐页签字盖章（公章+法定代表人签字+骑缝章）
- 电子版+纸质版同步准备

### 6.2 中介机构分工与协调机制

| 中介机构 | 负责文件 | 与其他文件的交叉引用关系 | 协调节点 |
|---------|---------|---------------------|---------|
| **主承销商（券商/银行）** | 募集说明书（核心）、发行公告、承销协议、受托管理协议、持有人会议规则、主承销商核查意见 | 引用审计报告、评级报告、法律意见书的核心结论 | 全程主导协调 |
| **会计师事务所** | 审计报告（近2-3年） | 被所有其他文件引用（财务数据来源） | 审计报告出具日是所有文件的时间锚点 |
| **律师事务所** | 法律意见书 | 验证发行人法律合规性，被募集说明书引用 | 与募集说明书法律章节交叉核对 |
| **评级机构** | 评级报告（主体+债项） | 被募集说明书评级章节引用 | 评级结论与募集说明书偿债分析章节一致 |
| **担保人（如有）** | 担保合同/担保函 | 被募集说明书增信章节引用 | 担保协议条款与募集说明书一致 |

**协调三大黄金规则：**

1. **前置规则：** 审计报告必须最先出具（所有其他文件的财务数据均引用自此）→ 评级报告第二（依赖审计报告数据）→ 法律意见书和募集说明书最后（依赖前两者）
2. **一致性规则：** 所有中介机构出具文件中引用的同一数据必须100%一致（如营业收入、净利润、资产负债率等）
3. **时间戳规则：** 审计报告出具日 → 评级报告出具日 → 法律意见书出具日 → 募集说明书签署日 → 必须时间递增，不得倒置

### 6.3 各品种文件撰写的特殊协调需求

| 品种 | 特殊协调需求 | 增量中介 | 协调重点 |
|------|------------|---------|---------|
| **企业债** | 募投项目文件需与证券公司/发改委同步协调 | +项目可行性研究机构 | 项目文件审批周期远长于其他文件，需提前2-3个月启动 |
| **ABS** | SPV设立文件/资产买卖协议/资产服务协议需律师+评级机构+发行人三方协调 | +资产服务机构+SPV管理人 | 底层资产池的逐笔核查需律师+审计师+评级机构三方交叉验证 |
| **绿色债** | 绿色认证报告需第三方认证机构出具 | +绿色认证机构 | 绿色认证周期约2-3周，需与评级报告同步推进 |
| **CRMW增信品种** | CRMW创设协议需与创设机构签订 | +CRMW创设机构 | CRMW创设与债券发行必须同步完成，任一环节滞后将导致发行失败 |
| **可转债** | 转股价格确定+稀释性分析+盈利预测 | +保荐机构（有时是同一家券商） | 可转债文件复杂度远高于普通公司债（需额外转股/赎回/回售条款） |
