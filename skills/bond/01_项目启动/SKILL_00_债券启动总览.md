# SKILL_00：债券启动总览

> **版本：** v2.0 | **最后更新：** 2026年5月 | **适用范围：** 债券承做全品种启动阶段总览

## 概述

债券项目启动是债券承做全流程的入口环节。与IPO股权项目启动相比，债券启动有独特的工作节奏、审核路径和风险控制要点。本文件作为债券线第一阶段（01_项目启动）的路由入口，框架性阐述债券启动的全景流程、决策要点和跨阶段影响。详细的启动操作手册见 `SKILL_01_债券项目启动.md`。

对于涉及城投/隐债/评级门槛/品种选择等深度分析场景，末尾委托接口指向已有债券专项技能（ib-bond-executor、bond-dcm-core 等）。

## 一、债券项目启动 vs 股权项目启动：五大核心差异

债券承做与股权承做虽然同属投行业务，但启动阶段的工作逻辑截然不同。以下五大差异贯穿项目始终：

### 差异1：核心定位不同——"偿债能力" vs "持续盈利能力"

| 维度 | 债券启动 | 股权启动（IPO/再融资） |
|------|---------|---------------------|
| 客户筛选标准 | 信用等级≥AA-、有稳定现金流、有资产担保 | 行业前景好、盈利增长确定、内控规范 |
| KYC核心问题 | "3年后能不能还本付息？" | "上市后能不能持续增长？" |
| 财务分析重点 | 有息负债结构、EBITDA覆盖率、经营活动现金流 | 收入增长率、毛利率趋势、ROE/ROA |
| 退出路径 | 到期还本付息后自然完成 | 通过二级市场退出或并购退出 |

### 差异2：监管路径不同——"分品种多通道" vs "单一注册路径"

- **股权：** 单一通道（交易所审核→证监会注册），路径清晰但周期长（6-18个月）
- **债券：** 三套监管体系同时可选（NAFMII/证监会/发改委），品种选择本身就是启动期决策
- **影响：** 启动阶段如未准确判断适合的品种+监管通道，后期可能需要推倒重来

### 差异3：评级前置——"必须拿到评级才能推进"

- **债券：** 评级是启动期的刚性约束——评级不达标（如城投<AA-），项目立即中止
- **股权：** 评级不是IPO硬性条件（评级主要用于发行定价，非准入门槛）
- **操作：** 启动第一周完成"评级预沟通"，确认可获目标评级后才能继续

### 差异4：城投隐性债务——特有的红线筛查

- **债券（城投类）：** 必须前置完成银保监会名单核查+财政部隐债系统排查+政府债务率四档分色
- **股权：** IPO不涉及融资平台/隐债概念——上市公司必须是市场化法人主体
- **操作：** 城投类发行人启动前必须拿到"不承担政府融资职能"的声明/证明

### 差异5：中介团队配置不同

| 中介角色 | 债券 | IPO |
|---------|------|-----|
| 主承销商 | 必选（银行or券商，取决于品种） | 必选（券商） |
| 评级机构 | **必选**（几乎所有品种） | 可选（仅在发行时用于定价） |
| 律师事务所 | 必选 | 必选 |
| 审计机构 | 必选（近2-3年审计报告） | 必选（近3年审计报告） |
| 担保机构 | 可选（低评级/城投常用） | 几乎不涉及 |
| 资产评估 | 可选（抵质押场景） | 必须（改制/重组场景） |
| 绿色认证 | 可选（绿色债场景） | 不涉及 |

## 二、债券启动流程图（五步推进）

```
┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
│ 第1步    │───→│ 第2步    │───→│ 第3步    │───→│ 第4步    │───→│ 第5步    │
│ 初步接洽 │    │ KYC      │    │ 预评沟通 │    │ 初步方案 │    │ 立项内核 │
│ (1-2天)  │    │ (3-5天)  │    │ (3-5天)  │    │ (5-7天)  │    │ (3-5天)  │
└──────────┘    └──────────┘    └──────────┘    └──────────┘    └──────────┘
     │               │               │               │               │
     ▼               ▼               ▼               ▼               ▼
 客户需求摸底    KYC判断        评级预判        方案比选       内核通过
 基本信息收集   隐债筛查        评级预沟通      品种选择      签署承销协议
 承揽决策        红线判断        出具预评级      发行规模+期限  正式启动尽调
                                 沟通函          定价初测
```

### 第1步：初步接洽（1-2天）

| 工作项 | 具体内容 | 产出物 |
|-------|---------|-------|
| 客户需求了解 | 融资规模、期限偏好、资金用途、时间紧迫性 | 客户需求纪要 |
| 基本信息收集 | 营业执照、近2年审计报告（简版）、征信报告、存量债券清单 | 基础资料包 |
| 初步判断 | 是否具备发债基本条件（评级/净资产/合规记录） | 初步判断记录 |
| 品种方向判断 | 根据企业类型+规模+用途初步判断可行品种范围 | 品种推荐方向 |

### 第2步：KYC——发行人核心画像（3-5天）

**KYC三大支柱：**

1. **主体信用画像**
   - 行业地位与竞争格局
   - 近3年核心财务数据（总资产/净资产/收入/EBITDA/有息负债/经营现金流）
   - 控股股东与实际控制人
   - 银行授信总额及使用率

2. **隐性债务排查（城投专项）**
   - 银保监会融资平台名单查询（在名单/退出/从未入）
   - 财政部隐性债务系统查询
   - 地方政府债务率四档分色（绿/黄/橙/红）
   - 企业是否出具"不再承担政府融资职能"声明

3. **合规红线筛查**
   - 失信被执行人查询（中国执行信息公开网）
   - 重大行政处罚（近3年）
   - 重大诉讼/仲裁
   - 存量债券是否有违约/展期记录

### 第3步：评级预沟通（3-5天）

**此步骤为债券启动的刚性约束——评级不达标，项目即止。**

| 工作项 | 操作要点 | 时限 |
|-------|---------|------|
| 选定评级机构 | 中诚信/联合/新世纪/大公/东方金诚择一 | 1天 |
| 签署预评级合同 | 预评级费用通常由发行人承担（5-20万） | 1天 |
| 提交预评级资料 | 审计报告+征信报告+公司简介+存量债务明细 | 1天 |
| 评级机构出具预评级沟通函 | 给出主体/债项预评级结论及关键限制条件 | 3-5天 |
| 评估结论 | 如预评级低于可发行品种的最低要求→项目中止 | 即时 |

**各品种最低评级门槛速查：**

| 品种 | 最低评级要求 | 备注 |
|------|-----------|------|
| SCP/CP | 主体A+以上 | 个别AA-也可尝试 |
| MTN | 主体AA-以上 | 首次注册建议AA以上 |
| PPN | 无硬性要求 | 但AA-以下难以找到投资者 |
| 公募公司债（小公募） | 主体+债项AA以上 | - |
| 公募公司债（大公募） | 主体+债项AAA | 评级刚性 |
| 私募公司债 | 主体AA以上（市场惯例） | 无法规硬性要求但实务必达 |
| 企业债 | 主体AA-以上 | 需匹配募投项目评级 |

### 第4步：初步方案设计（5-7天）

在KYC+预评级结论基础上，形成《发行初步方案》。方案包含：

1. **品种选择**：根据发行人类型、评级、融资需求匹配1-2个推荐品种 + 1个备选品种
2. **发行规模**：首次注册通常为基础额度的60-80%（预留后续发行空间）
3. **期限结构**：单一期限or多期限组合（如3Y+5Y）
4. **增信方式**：是否需要担保/抵质押/CRMW（信用风险缓释凭证）
5. **利率区间预判**：根据当前市场基准利率+信用利差测算
6. **关键假设与风险**：评级变动风险、行业政策变化、再融资环境变动

详细方案设计决策框架见 `SKILL_00_债券方案总览.md`（第三阶段）。

### 第5步：立项与内核（3-5天）

| 步骤 | 内容 | 责任方 |
|------|------|-------|
| 项目立项申请 | 提交《项目立项报告》（含KYC小结+预评级结论+初步方案） | 项目组 |
| 内核委员会审查 | 3-5位内核委员审核，重点关注：风险等级/合规性/盈利性 | 内核委员会 |
| 内核反馈 | 内核委员提出问题/补充尽调要求 | 内核委员 |
| 签署承销协议 | 内核通过后，与发行人签署《承销协议》 | 法审+双方授权代表 |
| 正式启动尽调 | 项目进入第二阶段（02_尽职调查） | 项目组+中介 |

## 三、阶段 Gate Check——五项拦停规则

启动阶段必须通过以下5项"拦停规则"，任何一项未通过则立即中止项目推进：

### Gate 1：发行人资质不达标

| 拦停条件 | 判断标准 | 处理 |
|---------|---------|------|
| 评级不达标 | 预评级低于目标品种最低要求 | 中止，或更换品种重新评估 |
| 净资产不足 | 净资产<发行额×2.5（触及40%红线时） | 下调发行规模 |
| 经营年限不足 | 非金融企业成立不满3个完整会计年度 | 中止（NAFMII/公司债硬性要求） |
| 审计报告非标 | 近2年审计报告存在保留意见/否定意见/无法表示意见 | 中止（大部分品种硬性要求） |
| 近3年连续亏损 | 扣非净利润连续3年为负+经营性现金流持续为负 | 高风险，除非有强担保或特殊行业豁免 |

### Gate 2：隐性债务未化解（城投专项）

| 拦停条件 | 判断标准 | 处理 |
|---------|---------|------|
| 在银保监会名单内 | 属于"监管类"融资平台 | 企业债直接禁止；MTN/公司债仅限借新还旧 |
| 隐债未出清 | 财政部系统中仍有未化解隐性债务余额 | 需先完成隐债化解（提供化解方案+财政确认） |
| 政府违规兜底 | 地方财政出具担保函/承诺函/安慰函 | 违规文件无效，需整改并剥离 |
| 涉红色区域 | 所在地方政府债务率为红色档 | 企业债禁止/交易所公司债严重受限 |
| 市场化转型不充分 | 政府补贴收入占利润>50%或经营性收入<30% | MTN/公司债审核重点，需提供转型方案 |

### Gate 3：评级预判失利

| 拦停条件 | 说明 | 处理 |
|---------|------|------|
| 预评级沟通函为负面 | 评级机构告知无法达到目标评级 | 与评级机构沟通上调路径或更换机构二次预评 |
| 存在评级下调风险 | 近期有重大不利事件（行业下行/大额亏损/监管处罚） | 评估影响并决定是否暂停 |
| 双评级差异>2个notch | 两家评级机构预评结论差距大（如AA vs AA-） | 需澄清差异原因，必要时增加第三家 |

### Gate 4：合规红线

| 拦停条件 | 判断标准 | 处理 |
|---------|---------|------|
| 失信被执行人 | 发行人/控股股东/实控人被列入失信被执行人名单 | 中止（几乎所有品种禁止） |
| 重大违法违规 | 近3年发生被证监会/银保监/交易所行政处罚且属于"重大" | 中止（公司债/企业债明确禁止） |
| 债务违约 | 近3年有债券/贷款违约记录 | 中止或仅限高评级发行人有担保通道 |
| 环保/安全重大事故 | 近3年发生产能过剩行业重大环境违法/安全事故 | 中止相关行业品种审核 |

### Gate 5：项目经济性不达标

| 拦停条件 | 判断标准 | 处理 |
|---------|---------|------|
| 规模过小 | 首期发行<3亿（公司债）/ <2亿（MTN）/ <1亿（SCP） | 费率不足以覆盖中介成本，中止 |
| 承销费过低 | 承销费率<0.1%/年（行业底线） | 商业不可行，中止 |
| 执行风险极高 | 预计成功率<50%（如首次发债AA-民企无担保） | 风险收益不匹配，中止 |

## 四、关键决策的跨阶段影响

启动阶段的决策将深远影响后续5个阶段的走向。以下梳理关键决策链条：

### 决策1：品种选择→决定后续路径

```
启动期选 MTN → 进入NAFMII路径 → 银行主承 → 尽调聚焦偿债/资产质量
启动期选 公司债 → 进入证监会路径 → 券商主承 → 尽调还需关注信息披露
启动期选 企业债 → 进入发改委路径 → 券商主承 → 尽调还需验证募投项目合规
```

- 品种决定了主承销商类型（银行vs券商）——启动期选错主承，后期换将成本巨大
- 品种决定了尽调重点和文件体系——见 `SKILL_00_债券文件总览.md`

### 决策2：增信方式→决定后续尽调与文件范围

- 选择"信用发行（无担保）"：尽调完全聚焦发行人主体信用，文件范围最小
- 选择"第三方担保"：需额外尽调担保人（规模约为发行人尽调的30-40%），文件多一份担保合同+担保人审计
- 选择"抵质押担保"：需资产评估报告+抵押登记+保险，文件体系复杂度翻倍
- 选择"CRMW"：需与CRMW创设机构签订协议，增加创设费用评估

### 决策3：注册额度拆分→影响后续发行执行

- 首次注册高额度但分批发行：后期有更多弹性空间但首期规模可能偏小
- 一次性全额发行：启动期需更充分的投资者沟通和市场窗口判断
- DFI/TDFI注册：启动期可直接启用储架额度，简化后期逐次注册流程

### 决策4：评级策略→决定发行成本区间

- 目标评级提高1个notch（如AA→AA+）：发行利率可降低30-80BPS
- 是否有双评级：可拓宽投资者基础（保险/境外投资者偏好双评级发行人）
- 评级展望"稳定"vs"负面"：影响投资者信心和认购倍数
