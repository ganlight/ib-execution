# SKILL_03：债券品种选择

> **版本：** v2.0 | **最后更新：** 2026年5月 | **适用范围：** 债券承做全品种方案设计阶段

## 概述

债券品种选择是融资方案设计的核心环节，直接影响后续的监管审核路径、时间周期、融资成本、投资者结构和发行成功率。中国债券市场存在多套平行的监管体系（NAFMII主管的银行间市场 / 证监会主管的交易所市场 / 发改委主管的企业债），不同品种的审核标准、时效要求和投资者偏好差异显著。品种选择失误可能导致：① 申报材料审核周期延长3-6个月；② 因不满足特定品种准入条件而被退回；③ 发行利率高于预期导致发行失败；④ 额度浪费或后续融资空间被压缩。

本模块提供系统化的品种选择决策框架，涵盖决策树、品种匹配矩阵、11品种×8维度完整对比、三维匹配框架、监管审批流程对比、城投政策红线、实战案例。对于需要更深层次DCM分析（如信用利差建模、投资者分层、特定行业政策研判）的场景，可调用 **bond-dcm-core** 技能。

## 一、品种选择决策树（五步法）

按以下五步走，每一步的输出作为下一步的输入：

### Step 1：发行人类型识别

| 发行人类型 | 子类 | 典型特征 |
|-----------|------|---------|
| SOE/央企 | 央企本級 | 集团直发，AAA评级，额度充裕 |
| SOE/央企 | 央企子公司 | 视集团担保/自身资质，AA+~AAA |
| LGFV/城投 | 省级平台 | 省会级城投，一般AA+以上 |
| LGFV/城投 | 市级平台 | 地级市主平台，AA~AA+ |
| LGFV/城投 | 区县级平台 | 区县主平台，AA-~AA，受限较大 |
| 上市公司 | 产业类 | 主业明确，需关注行业周期 |
| 上市公司 | 房地产 | 受行业政策限制，需满足"三道红线"等 |
| 民企 | 头部民企 | 行业龙头，AA+以上 |
| 民企 | 一般民企 | 需强担保或信用缓释工具 |
| 金融机构 | 银行 | 二级资本债、永续债等特定品种 |
| 金融机构 | 券商/保险 | 次级债、短期融资券 |

**操作要点：**
- 城投平台需同时判断"行政层级"（省/市/区县）和"是否涉及隐债"
- 房地产企业需核查"三道红线"踩线数量及行业融资窗口开放情况
- 金融机构发行债券品种与一般企业差异巨大，需单独研究

### Step 2：信用评级判断

| 评级 | 可用品种范围 | 关键限制 |
|------|-------------|---------|
| AAA | 全部品种通畅（MTN、CP、SCP、公司债、企业债、PPN、ABS等） | 无实质限制 |
| AA+ | 主流品种通畅，永续中票/可续期公司债需看发行案例 | 永续类品种需评估投资者接受度 |
| AA | 公司债受限（需增信或选择私募）、企业债需担保、MTN/PPN可操作 | 公募公司债大公募基本不可行 |
| AA- | 企业债（需担保）、PPN（特定投资者）、私募公司债、部分ABS | 公募品种基本需要增信 |
| AA-以下 | 极受限制，主要考虑私募公司债+强担保、ABN | 需重点设计增信方案 |

**评级动态管理：**
- 评级展望为"负面"时，即使主体评级达标，发行难度和成本也将上升
- 关注评级机构的跟踪评级时间点，避免在审核期内被降级

### Step 3：募资用途判定

| 用途 | 推荐品种 | 说明 |
|------|---------|------|
| 补充流动资金 | CP/SCP（短期补流）、MTN（中长期补流）、公司债 | SCP补流无用途限制，MTN需不超净资产40% |
| 偿还到期债务 | MTN、公司债、企业债 | 借新还旧需披露原债务明细 |
| 项目建设 | 企业债（专项债）、绿色债、项目收益债 | 企业债与固定资产投资项目挂钩最强 |
| 绿色/碳中和 | 绿色中票、绿色公司债、绿色企业债 | 需绿色认证，有审批绿色通道 |
| 并购/股权出资 | 并购中票、并购公司债 | 需符合并购贷款规则 |
| 补充资本金（项目） | 企业债 | 企业债允许部分项目资本金用途 |
| 降低资产负债率 | 永续中票、可续期公司债 | 会计上可计入权益（需条款设计达标） |

**用途审核要点：**
- 偿债用途需穿透至原债务，避免"偿还债务"成为资金挪用通道
- 项目建设用途需与发改委审批文件、环评、用地等挂钩
- 补流用途需做流动资金缺口测算

### Step 4：发行金额与期限

| 规模范围 | 期限建议 | 品种建议 | 备注 |
|---------|---------|---------|------|
| ≤5亿 | ≤1年 | SCP/CP（银行间）、短期公司债（交易所） | 适合滚动发行 |
| ≤5亿 | 2-3年 | PPN、私募公司债 | 小规模中长期品种选择有限 |
| 5-15亿 | 3年 | MTN、公募公司债 | 主流品种竞争选择 |
| 5-15亿 | 5年 | MTN、公募公司债、企业债 | 需比较审核周期和成本 |
| 15-30亿 | 3-5年 | MTN、公募公司债、企业债 | 规模适中，各品种均可 |
| >30亿 | 3-10年 | MTN（DFI统一注册）、公司债（大公募）、企业债 | 大规模建议分期限分期发行 |

**额度管理核心规则：**
- NAFMII体系：MTN+CP+SCP+PPN合计余额不超过发行人净资产的40%
- 公司债体系：公司债余额不超过发行人净资产的40%
- 企业债体系：企业债余额不超过发行人净资产的40%
- 三套体系分别计算，但实践中需统筹管理避免过度负债

### Step 5：市场窗口研判

| 维度 | 判断标准 | 影响 | 操作建议 |
|------|---------|------|---------|
| 利率走势 | 观察10Y国债、LPR、同评级中债估值周/月变动 | 决定发行时机 | 利率下行期优先选择长期限品种 |
| 监管窗口 | CSRC/NAFMII/NDRC近期审核偏松或偏紧 | 影响审核周期 | 窗口收紧期提前储备额度 |
| 可比发行 | 最近2周同评级同期限同行业的发行利率 | 定价参照 | 作为发行利率区间参考 |
| 季节性因素 | 季末/年末流动性偏紧；降准/MLF窗口 | 影响发行成功率 | 避免季末首日发行 |
| 信用事件 | 同行业/同区域是否有违约事件 | 影响投资者情绪 | 需等待市场情绪消化 |

**窗口期操作策略：**
- 优质发行人：可择机等待最佳窗口，不必急于发行
- 中低评级发行人：有窗口即发行，避免错过机会
- 政策敏感型发行人（城投/地产）：密切关注监管动态

### 决策树完整流程

```
Step 1：发行人类型识别
    ↓
Step 2：信用评级判断
    ↓
Step 3：募资用途判定
    ↓
Step 4：发行金额与期限
    ↓
Step 5：市场窗口研判
    ↓
转入"品种匹配矩阵"查表 + "三维匹配框架"细化
```

|-----------|:----------:|:------:|:------------:|:---------:|:-----:|:---:|:------:|:---------:|
| 央企/本部 | ✓ | ✓ | ✓ | ✓（少用） | ✓ | ✓（少用） | ✓ | ✓ |
| 央企子公司 | ✓ | ⚠ | ✓ | ✓ | ⚠ | ✓ | ⚠ | ⚠ |
| 省级城投 | ✓ | ✓ | ⚠ | ⚠ | ✓ | ✓ | ✗ | ✗ |
| 市级城投 | ✓ | ⚠ | ⚠ | ✓ | ✓ | ✓ | ✗ | ✗ |

| 上市公司（房地产） | ⚠ | ✗ | ⚠ | ⚠ | ✗ | ⚠ | ✓（CMBS） | ✗ |

| 一般民企 | ✗ | ✗ | ✗ | ⚠（需担保） | ✗ | ⚠ | ⚠ | ✗ |

**图例：** ✓ = 常规可做 | ⚠ = 有条件可做（需增信/特定政策窗口/额外审批） | ✗ = 当前政策下不可做或极难做

**重点提示：**
- 区县城投：交易所公司债实行"红黄绿"分档管理，红色区域区县级平台基本无法新增公司债
- 房地产企业：需满足"三道红线"+企业性质要求，民企地产发债几乎停滞
- 涉隐债主体：需关注财政部隐债清单，隐债主体发债受限（企业债禁止，MTN/公司债有条件）
- 银保监"15号文"：涉及隐债主体的流动资金贷款类融资受限，对CP/SCP补流用途有影响

## 三、11品种×8维度完整对比表

| 品种 | 注册/审核机关 | 发行门槛 | 审核周期 | 融资成本区间（AAA/3Y） | 增信方式选项 | 信息披露复杂度 | 募集资金用途限制 | 二级市场流动性 |
|------|-------------|---------|---------|---------------------|------------|--------------|---------------|--------------|
| **MTN（中期票据）** | NAFMII（交易商协会） | 主体评级AA及以上；净资产≥15亿；近3年连续盈利 | 30-60天（常规注册）<br>15-20天（DFI绿色通道） | 2.5%-3.5% | 第三方担保、资产抵押、CRMW、差额补足 | 中等（募集书+定期披露） | 补流/偿债/项目投资均可；合计余额≤净资产40% | 极深（日均成交活跃） |
| **CP（短期融资券）** | NAFMII | 主体评级AA及以上；近3年盈利 | 15-30天 | 2.0%-3.0% | 第三方担保、CRMW | 中等 | 补流/偿债；期限≤1年 | 深 |
| **SCP（超短期融资券）** | NAFMII | 主体评级AA及以上；无需年度审计报告 | 10-20天 | 1.8%-2.8% | CRMW为主 | 较轻 | 用途无限制；期限≤270天 | 深 |
| **公募公司债（大公募）** | 证监会/交易所 | 主体AAA；债项AAA；近3年年均可分配利润≥债券1年利息 | 2-4个月 | 2.5%-3.8% | 第三方担保、资产抵押、差额支付 | 重（年报+半年报+重大事项） | 补流/偿债/项目投资；余额≤净资产40% | 深 |
| **公募公司债（小公募）** | 交易所 | 主体AA+及以上；债项AA+及以上；近3年年均可分配利润≥债券1年利息 | 1.5-3个月 | 2.8%-4.2% | 同上 | 重 | 同上 | 中深 |
| **私募公司债** | 交易所（备案制） | 无硬性评级要求；面向合格投资者 | 15-30天（备案） | 3.5%-6.0% | 第三方担保、资产抵押、差额支付、结构化 | 轻（定向披露） | 较灵活；需说明用途 | 浅（合格投资者交易） |
| **企业债** | 发改委 | 主体AA及以上；净资产≥12.5亿；近3年连续盈利；有合规项目 | 3-6个月 | 2.8%-4.0% | 第三方担保、资产抵押、政府专项偿债基金 | 重（与项目挂钩） | 须用于固定资产投资项目；债券资金≤项目总投资70% | 中等 |
| **PPN（定向债务融资工具）** | NAFMII | 无硬性评级要求；面向特定机构投资者（≤200人） | 15-30天 | 3.0%-4.5% | 第三方担保、CRMW | 轻（定向披露） | 较灵活 | 浅（定向交易） |
| **ABS（资产支持证券）** | 交易所 | 基础资产合规、可产生稳定现金流；原始权益人主体评级≥AA | 2-4个月 | 优先档2.8%-4.5%；次级档6%-10% | 优先/次级分层、超额抵押、差额支付、外部担保 | 中等（资产池披露+定期） | 用于购买基础资产；循环购买需说明 | 中等 |
| **ABN（资产支持票据）** | NAFMII | 同ABS | 2-3个月 | 优先档3.0%-4.5% | 同ABS | 中等 | 同ABS | 中等 |
| **永续中票** | NAFMII | 主体AA+及以上优先；需有明确会计处理意见 | 45-75天 | 3.5%-5.0%（含利率跳升溢价） | 第三方担保（较少用） | 中等 | 同MTN | 中等（投资者较窄） |
| **可续期公司债** | 交易所 | 主体AA+及以上优先；需有明确会计处理意见 | 2-3个月 | 3.5%-5.0% | 第三方担保 | 重 | 同公司债 | 中等 |

**关键对比结论：**
1. **审核效率：** SCP > CP/PPN > MTN > 私募公司债 > 小公募 > ABS/ABN > 大公募 > 企业债
2. **融资成本：** SCP < CP < MTN < 公募公司债 < 企业债 < PPN < 私募公司债 < ABS优先档 < 永续类
3. **资金用途灵活度：** SCP(无限制) > PPN/私募公司债 > MTN/公募公司债 > 企业债(项目绑定)
4. **投资者覆盖：** MTN/公募公司债 > CP/SCP > 企业债 > ABS > PPN/私募公司债 > 永续类

## 五、监管审批流程对比

### 5.1 NAFMII（交易商协会）体系

#### 5.1.1 注册流程

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  材料准备   │ →  │  预审反馈   │ →  │  正式注册   │ →  │  每期发行   │
│ （T-45天）  │    │ （7-15天）  │    │ （15-30天） │    │   备案      │
└─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘
```

**关键节点说明：**
- **材料准备：** 募集说明书、法律意见书、评级报告、近2年审计报告
- **预审反馈：** 协会秘书处出具预审意见，发行人修改完善
- **注册会议：** 协会召开注册会议审议，出具《接受注册通知书》
- **发行备案：** 每期发行前提交发行方案备案（T-2个工作日）

#### 5.1.2 重大事项报告制度

| 事项类型 | 报告时限 | 报告内容要求 |
|---------|---------|------------|
| 生产经营重大变化 | 事项发生后2个工作日 | 事项性质、影响程度、应对措施 |
| 财务状况重大变化 | 季度报告后5个工作日 | 主要财务指标变动分析 |
| 信用评级调整 | 调整后1个工作日 | 调整原因、后续影响评估 |
| 重大诉讼仲裁 | 诉讼发生后3个工作日 | 案件基本情况、涉案金额、判决预期 |
| 对外担保/资产抵押 | 事项发生后5个工作日 | 担保对象、金额、反担保措施 |

#### 5.1.3 投资人保护机制

| 机制类型 | 触发条件 | 保护措施 |
|---------|---------|---------|
| 交叉违约保护 | 发行人其他债务违约≥5000万（或净资产3%） | 投资者可要求提前到期 |
| 控制权变更条款 | 实际控制人发生变更 | 投资者可要求回售 |
| 财务指标承诺 | 资产负债率≥75%或EBITDA利息倍数<2x | 投资者可要求增信或提前到期 |
| 信息披露违约 | 未按时披露定期报告 | 投资者可要求召开持有人会议 |

### 5.2 证监会/交易所体系

#### 5.2.1 审核流程

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  受理申请   │ →  │  审核问询   │ →  │  上会审议   │ →  │  注册批复   │
│ （T-0）     │    │ （1-2轮）   │    │ （上市委）  │    │ （证监会）  │
└─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘
                         ↓
              ┌─────────────┐
              │  反馈回复   │
              │ （每轮15天）│
              └─────────────┘
```

**审核时间轴：**
- 大公募：受理→审核→反馈→上会→注册，全程约3-5个月
- 小公募：交易所审核→证监会注册，全程约2-3个月
- 私募公司债：交易所备案制，约15-30天

#### 5.2.2 储架发行制度

| 制度要素 | 内容 |
|---------|------|
| 适用品种 | 公募公司债（大公募/小公募） |
| 发行额度 | 申请总额度，分期发行 |
| 有效期 | 注册批复后12个月内有效 |
| 发行条件 | 每期发行前需符合发行条件 |
| 信息披露 | 首期发行披露完整募集书，后续期次可简化 |

#### 5.2.3 分类审核机制

| 类型 | 主体要求 | 债项要求 | 审核特点 |
|------|---------|---------|---------|
| 大公募 | AAA | AAA | 审核最严格，投资者范围最广 |
| 小公募 | AA+及以上 | AA+及以上 | 审核相对简化，面向合格投资者 |
| 私募 | 无硬性要求 | 无硬性要求 | 备案制，面向合格投资者 |

### 5.3 发改委（企业债）体系

#### 5.3.1 审批流程

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  申报受理   │ →  │ 第三方评审  │ →  │  审批注册   │ →  │  发行备案   │
│ （省发改委）│    │ （专业机构）│    │ （国家发改委）│   │ （中央结算）│
└─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘
```

**关键节点说明：**
- **省级转报：** 省级发改委初审后转报国家发改委
- **第三方评审：** 委托专业机构（如中央结算、交易商协会）进行技术评审
- **批复文件：** 国家发改委出具《关于核准发行企业债券的批复》
- **发行备案：** 发行前向中央结算公司备案

#### 5.3.2 额度管理制度

| 项目类型 | 债券资金占项目投资比例 | 额度限制 |
|---------|---------------------|---------|
| 一般固定资产项目 | ≤70% | 单次申请额度有上限 |
| 保障性安居工程 | ≤60% | 需纳入省级计划 |
| 小微企业增信集合债 | ≤80% | 需银行托管 |
| 项目收益债 | ≤项目总投资 | 以项目收益覆盖为前提 |

#### 5.3.3 项目绑定要求

| 要求类型 | 具体内容 |
|---------|---------|
| 项目审批 | 需取得立项批复/核准/备案文件 |
| 土地手续 | 需取得土地使用权证或用地预审 |
| 环评批复 | 需取得环评批复或环评登记表 |
| 规划许可 | 需取得规划许可证（部分项目） |
| 收益测算 | 项目收益需覆盖债券本息 |

### 5.4 三套体系对比汇总

| 对比维度 | NAFMII体系 | 证监会/交易所体系 | 发改委体系 |
|---------|-----------|-----------------|-----------|
| **审核周期** | 30-60天（最快） | 60-120天 | 90-180天 |
| **审核重点** | 发行人资质+偿债能力 | 信息披露完整性+合规性 | 项目合规性+收益覆盖 |
| **资金用途** | 相对灵活 | 需详细披露 | 必须与项目挂钩 |
| **投资者范围** | 银行间全市场 | 交易所市场+跨市场 | 跨市场 |
| **后续管理** | 重大事项报告制度 | 受托管理人制度 | 债权代理人制度 |
| **政策敏感度** | 中等（城投政策变化快） | 高（地产/城投窗口明显） | 中等（项目导向相对稳定） |

## 六、城投再融资政策红线

### 6.1 政策红线速查表

| 政策文件 | 核心限制 | 影响品种 | 应对策略 |
|---------|---------|---------|---------|
| **15号文**（银保监） | 隐性债务主体不得新增融资平台类债务 | CP/SCP补流、银行贷款 | 确认是否在隐债清单；不在清单可正常申报 |
| **40号文**（国务院） | 产业类城投界定：政府收入占比<30%+市场化经营 | 所有债券品种 | 调整收入结构，做实产业类定位 |
| **协会15号文**（NAFMII） | 区县级城投限制；隐债主体穿透审核 | MTN/CP/SCP/PPN | 市级平台作为发行主体优先 |
| **交易所红黄绿** | 红色区域区县级平台暂停新增公司债 | 公募公司债、私募公司债 | 转向企业债或银行间品种 |
| **交易所高收益债** | 主体评级A以下需特殊条件 | 私募公司债 | 增加增信措施，引入高收益投资者 |
| **隐债清单** | 清单内主体企业债禁止、MTN/公司债有条件 | 企业债、MTN、公司债 | 借新还旧品种、隐债化解后再新增 |

### 6.2 15号文核心要点

**适用范围：**
- 承担地方政府隐性债务的融资平台
- 银行保险机构向融资平台提供融资

**核心限制：**
1. **不得新增融资平台类债务**：隐债主体不得新增以公益性项目或政府应收账款为主要还款来源的融资
2. **分类管理**：对不涉及隐债的平台，按照一般企业标准执行
3. **穿透审核**：对疑似隐债主体，需穿透至最终还款来源

**实操影响：**
- CP/SCP补流用途需严格论证，不得变相新增融资平台债务
- 银行贷款审批趋严，债券融资成为重要替代渠道
- 隐债化解后可恢复新增融资资格

### 6.3 40号文产业类城投界定

**界定标准：**
| 指标 | 标准要求 | 说明 |
|------|---------|------|
| 政府收入占比 | <30% | 主营业务收入中来自政府的部分 |
| 市场化经营 | 有实质性经营业务 | 非单纯融资平台职能 |
| 资产质量 | 经营性资产为主 | 非公益性资产为主 |
| 现金流 | 经营性现金流为正 | 政府补贴不占主导 |

**界定意义：**
- 符合产业类标准 → 可按一般企业标准申报债券
- 不符合产业类标准 → 按融资平台标准执行，受限较多

### 6.4 协会15号文审核要点

**区县级城投限制：**
- 原则上不受理区县级城投新增MTN/CP/SCP注册
- 市级平台控股的区县级子公司可由市级统一申报
- 优质区县级平台（百强县前50）可视情况放宽

**隐债主体穿透审核：**
- 需披露隐债化解方案和进度
- 募集资金不得新增隐性债务
- 需提供不新增隐债承诺函

### 6.5 交易所红黄绿分档管理

**分档标准（示例，具体以交易所最新通知为准）：**
| 分档 | 区域特征 | 审核政策 |
|------|---------|---------|
| 绿色 | 债务率低、财政实力强 | 正常审核 |
| 黄色 | 债务率中等、财政实力中等 | 严格审核，控制新增规模 |
| 红色 | 债务率高、财政实力弱 | 区县级暂停新增，市级借新还旧 |

**实操建议：**
- 发行前先向交易所沟通确认分档颜色
- 红色区域优先考虑企业债或调整融资结构
- 黄色区域控制申报规模，优化用途披露

## 七、品种组合策略

### 7.1 MTN + CP 组合策略（"长短搭配"）
- **适用场景：** 有中长期资金需求同时需要短期流动性补充
- **操作：** MTN（3-5年期）覆盖中长期资金 + CP/SCP（≤1年）滚动操作覆盖短期流动性
- **要点：** 注意40%净资产的额度分配，建议 MTN:CP = 7:3
- **典型方案：** 注册MTN额度15亿（5年期）+ 注册CP额度5亿（1年期），CP到期续发

### 7.2 SCP 滚动发行策略
- **适用场景：** 现金流波动大、日常营运资金补充、规避40%限制（SCP余额不计入MTN额度）
- **操作：** 注册SCP额度后，到期续发/提前发行新一期偿还上一期
- **注意：** 需保持滚动连续性，避免断档导致资金链紧张
- **市场波动风险：** 若发新还旧时市场利率大幅上行，可能面临成本上升

### 7.3 DFI统一注册策略
- **适用场景：** AAA/AA+优质企业，预计有频繁发行需求
- **操作：** 向NAFMII申请DFI（债务融资工具统一注册），一次注册、分期发行
- **优势：** ①注册后2年内可随时择机发行；②品种可灵活切换（MTN/CP/SCP/PPN）；③省去每次审核
- **额度管理：** DFI无事先额度限制，但每次发行后仍需计入"净资产40%"约束

### 7.4 跨市场双线推进策略
- **银行间+交易所双线推进：** 在公司债注册的同时，同步准备MTN注册
- **目的：** ①选择发行窗口更优的市场；②扩大投资者覆盖面；③分散单一市场监管风险
- **注意：** 银行间MTN与交易所公司债合计余额均受40%净资产限制，需合并计算

### 7.5 绿色通道策略
- **碳中和/绿色债：** NAFMII和交易所均设有绿色审批通道，审核周期缩短至15-20个工作日
- **条件：** ①需提供第三方绿色认证报告；②募集资金用途符合绿色债券支持目录
- **案例：** 风电/光伏/储能等新能源项目优先考虑绿色公司债/绿色中票

## 八、常见选型错误

### 错误1：区县城投直接申报公募公司债
- **场景：** 某中部省份区县级城投（AA-），看到其他城投发公司债成功，直接启动公募公司债申报
- **问题：** 未核实交易所"红黄绿"区域分档，项目在预审核阶段直接被退回（红色区域）
- **修正：** 应首先确认区域分档颜色；红色区域或隐性债务主体，应先处理合规问题或选择企业债（如需发改委审批且有项目支撑）

### 错误2：忽视40%净资产红线
- **场景：** 发行人一年内先后注册MTN 15亿、公司债 20亿，合计35亿，净资产仅80亿，已用43.75%
- **问题：** 第二笔公司债在注册阶段被否决，因超过净资产40%
- **修正：** 品种选择初期即应建立额度预算表，MTN+公司债+企业债+PPN合计不超过净资产40%

### 错误3：隐债主体选择不适格品种
- **场景：** 某城投在财政部隐债清单内，主承推荐其申报MTN用于偿还隐债
- **问题：** NAFMII对隐债主体新增融资实行穿透式审核，偿还隐债虽政策上有支持但实操难度大，最终审核被否
- **修正：** 隐债主体应优先考虑借新还旧类公司债（且只能用于偿还交易所债券）或依托项目申报企业债

### 错误4：房地产民企选择公募公司债
- **场景：** 某头部民营房企（AA+，"三道红线"踩一条），2022年申报小公募公司债
- **问题：** 交易所对民营房企发债实行暂停审核（当时政策窗口基本关闭，仅国企地产可操作）
- **修正：** 应转而通过CRMW（信用风险缓释凭证）+PPN组合方案，或在银行间以"中债增信"担保方式发行

### 错误5：过度依赖单一市场品种
- **场景：** 企业连续3年只发MTN，每次注册-发行-到期续发，从未开拓交易所品种
- **问题：** 遇到NAFMII窗口指导收紧（如城投政策收紧）时，立即出现融资断档
- **修正：** 应分散品种布局，在政策友好期提前注册公司债额度作为备用

## 九、实战案例

### 案例1：AA+地级市核心城投，10亿5年期融资

**发行人背景：**
- 某中部省份地级市核心城投平台
- 主体评级：AA+（中诚信）
- 净资产：85亿元
- 主要业务：城市基础设施建设、土地整理
- 是否涉隐债：否（已完成隐债化解）
- 行政区域：地级市主平台，非区县

**五步决策推演：**

**Step 1：发行人类型识别**
- 类型：市级城投平台
- 特征：地级市主平台，业务以基础设施为主
- 关键判断：行政层级符合要求，非区县级限制范围

**Step 2：信用评级判断**
- 主体评级：AA+
- 债项评级预期：AA+（无担保）/ AAA（若有省级担保）
- 可选品种：MTN、小公募公司债、企业债均可

**Step 3：募资用途判定**
- 用途：偿还到期债务+补充流动资金
- 债务明细：到期银行贷款8亿+到期债券2亿
- 判定：适合银行间MTN或交易所公司债

**Step 4：发行金额与期限**
- 金额：10亿元
- 期限：5年期
- 额度测算：净资产85亿×40%=34亿，现有债券余额15亿，剩余额度19亿>10亿，额度充足

**Step 5：市场窗口研判**
- 利率环境：10Y国债2.7%，AAA城投5Y中债估值3.2%
- 可比发行：近期AA+城投MTN 5Y发行利率3.5%-3.8%
- 监管窗口：NAFMII对市级城投审核正常，交易所红黄绿为黄色档

**品种选择结论：**
| 品种 | 优势 | 劣势 | 结论 |
|------|-----|------|------|
| MTN | 审核周期短（45天）、银行间投资者基础好 | 需每年披露定期报告 | **推荐** |
| 企业债 | 期限可更长、跨市场投资者 | 审核周期长（4-6个月）、需项目支撑 | **备选** |
| 小公募 | 交易所投资者、储架发行 | 审核周期较长、黄色区域审核趋严 | 不推荐 |

**最终方案：**
- 主品种：MTN（银行间中期票据）5年期，10亿元
- 备选品种：企业债（若项目审批完备，可作为第二品种储备）
- 发行结构：单一5年期，含第3年末投资者回售权+发行人调整利率权

### 案例2：AAA省高速集团，300亿额度需求

**发行人背景：**
- 某东部省份高速公路集团
- 主体评级：AAA（联合评级）
- 净资产：1,200亿元
- 主要业务：高速公路投资建设运营、服务区经营
- 融资需求：未来2年投资计划需300亿资金支持

**五步决策推演：**

**Step 1：发行人类型识别**
- 类型：省级AAA国企
- 特征：主业清晰（高速公路）、现金流稳定、政府支持力度大

**Step 2：信用评级判断**
- 主体评级：AAA
- 债项评级：AAA
- 可选品种：全品种通畅，重点考虑DFI统一注册

**Step 3：募资用途判定**
- 用途：高速公路新建/改扩建项目投资
- 期限匹配：项目建设期5-8年，需中长期资金
- 多元化需求：短期营运资金+长期项目资金

**Step 4：发行金额与期限**
- 总需求：300亿元
- 额度测算：净资产1,200亿×40%=480亿，现有债券余额150亿，剩余额度330亿>300亿
- 期限结构：短期（≤1年）50亿+中期（3-5年）150亿+长期（7-10年）100亿

**Step 5：市场窗口研判**
- 利率环境：10Y国债2.6%，AAA国企5Y中债估值2.8%
- 可比发行：近期AAA高速MTN 5Y发行利率2.9%-3.1%
- 监管窗口：NAFMII对AAA企业审核绿色通道

**品种选择结论：**

**核心策略：DFI统一注册+多品种组合**

| 品种 | 规模 | 期限 | 用途 | 优势 |
|------|-----|------|------|------|
| **DFI统一注册** | 200亿额度 | - | 统一额度池 | 一次注册，2年内灵活发行 |
| **SCP** | 滚动发行50亿 | 270天 | 营运资金补充 | 发行效率最高，资金灵活 |
| **MTN** | 150亿 | 3-5年 | 项目投资+偿债 | 主力品种，投资者接受度高 |
| **企业债** | 50亿 | 7-10年 | 重点项目投资 | 期限长，匹配项目建设周期 |

**最终方案：**
1. **第一步：** 向NAFMII申请DFI统一注册，额度200亿，有效期2年
2. **第二步：** 在DFI额度内分期发行MTN，3-5年期，每期20-30亿
3. **第三步：** 同步注册SCP额度50亿，用于短期资金周转
4. **第四步：** 对于7年以上超长期资金需求，申请企业债50亿（需项目支撑）

## 十、债券技能委托

当品种选择涉及以下深层次分析时，调用 **bond-dcm-core** 技能：

| 委托场景 | 具体内容 | 说明 |
|---------|---------|------|
| 信用利差建模 | 发行人信用利差曲线构建、同评级行业利差对照 | 需要量化模型支撑 |
| 投资者分层分析 | 银行/保险/基金/理财对不同品种的需求分析 | 影响发行窗口和定价 |
| 特定行业政策研判 | 城投/地产/钢铁/煤炭等行业最新监管口径 | 政策变化频繁，需实时跟踪 |
| 结构化方案设计 | ABS/ABN资产池构建、现金流分层、增信机制 | 复杂结构化产品 |
| 境外债品种 | 点心债/美元债/自贸区债 | 跨境发行 |
| 创新品种 | 科创债/乡村振兴债/革命老区债等专项品种 | 政策窗口类产品 |
| 偿债能力深度建模 | 现金流预测、压力测试、敏感度分析 | 调用 **bond-modeler** |
| 募集书撰写指导 | 特定品种募集书模板、审核意见回复 | 调用 **bond-documenter** |

**调用方式：** 在 ib-execution 流程中识别到上述场景时，直接调用相应技能获取分析结果，然后将结果整合回 ib-execution 的整体方案。
