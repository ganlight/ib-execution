# SKILL_00：债券方案总览

> **版本：** v2.0 | **最后更新：** 2026年5月 | **适用范围：** 债券承做全品种方案设计阶段总览

## 概述

债券方案设计是连接"尽调结论"与"文件撰写"的核心决策环节。在不尽相同的发行人资质、市场环境、监管约束之间找到最优解，是债券承做团队的核心价值所在。方案设计的本质是**五步决策链**：品种选择→规模确定→期限设计→增信安排→定价策略——每一步都牵一发动全身。

本文件作为债券线第三阶段（03_方案设计）的路由入口，阐述方案设计的全流程决策框架、各品种方案矩阵、方案比选方法和阶段Gate Check。详细的品种选择决策手册见 `SKILL_03_债券品种选择.md`，品种全景对比见 `../../references/bond/bond-product-matrix.md`，定价计算工具见 `../../../scripts/bond_pricing_calc.py`。

## 一、方案设计五步决策链

债券方案设计遵循严格决策顺序，前一步的输出是后一步的输入。**任一环节的决策都会倒逼前序环节的重新评估。**

```
第1步          第2步          第3步          第4步          第5步
品种选择  ──→  规模确定  ──→  期限设计  ──→  增信安排  ──→  定价策略
  │              │              │              │              │
  ▼              ▼              ▼              ▼              ▼
MTN/CP/SCP   净资产40%    ≤1Y/3Y/5Y   信用/保证/  利率区间+
公司债/企业债  监管限额/    /7Y/10Y/    抵质押/    发行时间窗口
PPN/ABS等     市场需求     永续        CRMW/CDS    簿记策略
```

### 第1步：品种选择

品种选择是方案设计的"第一因"，决定后续所有决策的框架边界。

**品种选择四问：**
1. **发行人是谁？** 央企/国企/城投/民企/金融机构？
2. **评级是多少？** AAA/AA+/AA/AA-？
3. **钱用来做什么？** 补充流动资金/偿还债务/项目建设/收购并购？
4. **时间有多紧？** 1个月内/3个月内/6个月内/时间充裕？

具体决策矩阵见本文第二章"各品种方案矩阵"。品种选择工具见 `../../../scripts/bond_product_matcher.py`。

### 第2步：规模确定

| 约束维度 | 计算方式 | 典型结果 |
|---------|---------|---------|
| **40%红线** | 本次发行后累计债券余额≤净资产×40% | 如净资产100亿→债券余额上限40亿，已发行25亿→本次≤15亿 |
| **监管限额** | 各品种单次发行上限（公司债≤60亿/MTN无明确上限） | 依据品种细则 |
| **市场需求** | 同评级同品种市场最大单次发行规模 | AAA主体可达50亿+，AA+约15-30亿，AA约5-15亿 |
| **资金用途匹配** | 补流≤募资总额30%（公募公司债）；项目建设=项目总投×60-70%（企业债） | 依据品种 |
| **分期策略** | 首期发行≈总额度的60-70%，预留后续发行空间 | 30亿注册→首期发行18-21亿 |

### 第3步：期限设计

| 期限 | 适用品种 | 典型场景 | 注意事项 |
|------|---------|---------|---------|
| ≤270天 | SCP | 超短期流动性补充 | 成本最低但再融资风险最高 |
| ≤1年 | CP | 短期营运资金需求 | 不适合长期项目融资 |
| 2-3年 | MTN/公司债/PPN | 中期融资（最主流期限） | 流动性最好，认购倍数通常最高 |
| 5年 | MTN/公司债/企业债 | 中期项目/资本开支 | 需匹配项目周期 |
| 7-10年 | 企业债/公司债（大公募） | 长周期基建项目 | 投资者接受度降低，需AAA评级+利率溢价 |
| 永续/N年 | MTN（永续中票）/可续期公司债 | 降低资产负债率（计入权益） | 利率跳升机制必须有，否则投资者不接受 |

**含权结构设计：**
- 发行人赎回权（Call）：3+2（3年到期发行人可选择赎回或延期2年）
- 投资者回售权（Put）：3+2中"2"部分赋予投资者回售权
- 调整票面利率选择权：与回售权配套使用
- 含权结构增加了方案复杂度但提升了灵活性

### 第4步：增信安排

| 增信方式 | 适用评级 | 效果 | 成本 |
|---------|:-------:|------|------|
| **信用发行（无增信）** | AA+以上 | 发行成本最低，流程最简单 | 0 |
| **第三方保证担保** | AA及以下+有AAA担保人 | 评级可跃升至担保人等级 | 担保费率0.3-1.0%/年 |
| **资产抵质押** | AA及以下+有优质资产 | 发行成功率明显提升 | 评估费+登记费+保险费 |
| **CRMW（信用风险缓释凭证）** | AA民企/首次发债 | 投资者接受度大幅提升 | 创设费率0.3-0.8%/年 |
| **多重增信（保证+抵质押）** | AA-及以下 | 极端情况下保障发行成功 | 综合成本较高 |

**增信决策关键判断：**
- AA+及以上产业类 → 大概率信用发行
- AA城投 → 一般需要至少省级AAA平台担保
- AA-及以下 → 必须强增信，否则发行可行性极低
- 民企首次发债 → CRMW几乎成为标配

### 第5步：定价策略

| 定价要素 | 确定方式 | 工具/参考 |
|---------|---------|----------|
| 无风险利率基准 | 同期限国债/国开债收益率 | 中债估值 |
| 信用利差 | 同评级同品种中债估值利差 | `../../references/bond/credit-analysis-framework.md` |
| 流动性溢价 | 私募/小额/首次发债→额外溢价20-50BPS | 市场经验 |
| 利率区间 | 中枢±20-30BPS | `../../../scripts/bond_pricing_calc.py` |
| 簿记策略 | 认购倍数→区间位置 | 见SKILL_06簿记建档 |

## 二、各品种方案矩阵

不同品种对应完全不同的决策路径。以下矩阵展示六大品种的方案设计差异。

### MTN方案决策路径

```
发行人类别→评级→用途        →方案结论
央企/AAA/WC+偿债           → 3Y/5Y MTN，信用发行，区间同期AAA利差
省级城投/AA+/WC+偿债      → 3Y MTN（主流），信用发行or AAA担保备选
市级城投/AA/WC            → 3Y MTN，需增信，区间中上限
产业国企/AA+/WC           → 3Y MTN，信用发行
民企/AA+/WC               → 3Y MTN + CRMW，利率上限+20BPS
```

**MTN方案特征：** 3年期主流、5年期次之、超大规模（单笔30-50亿）常见、银行主承为主。

### CP/SCP方案决策路径

```
发行人类别→时间紧迫性       →方案结论
AAA/超短期补流 → SCP(≤270天) → 信用发行，利率下限
AA+/≤1年补流   → CP(1年)     → 信用发行，AA+区间
AA/短期补流     → CP(1年)+增信→ 可能需要外部增信
```

**CP/SCP方案特征：** 周期最短（2-3周审核）、规模最大（单笔可达100亿+）、成本最低、但再融资风险最高。

### 公募公司债方案决策路径

```
发行人类别→评级→用途        →方案结论
AAA产业/AAA/通用            → 大公募，3Y/5Y/7Y可选，信用发行
AA+产业/AA+/通用           → 小公募，3Y（主流），信用/担保备选
AA产业/AA/WC+偿债          → 小公募，3Y，担保增信
城投/AA+/WC+偿债+补流      → 小公募，3+2含权，关注城投审核
```

**公司债方案特征：** 含权结构更常见（3+2为主流）、小公募vs大公募门槛差异大（大公募AAA+利息覆盖≥1.5x）、审核关注信息披露充分性。

### 企业债方案决策路径

```
发行人类别→用途→项目情况     →方案结论
省级城投/AA+以上/基建项目    → 发改委注册，7-10Y，省级转报
市级平台/AA+/市政项目        → 省级发改委转报+国家发改委审批，5-7Y
产业国企/AA+/产能项目       → 审核关注行业合规+产能审批，3-5Y
```

**企业债方案特征：** 期限最长（5-20年）、必须匹配募投项目、审批最复杂（省级转报+国家发改委）、用途以固定资产投资为主、城投占比最高。注意2023年后过渡期安排。

### ABS方案决策路径

```
资产类型→池质量→结构        →方案结论
优质信贷/分散度高/历史违约低 → AAA优先级为主，次级比例5-10%
供应链账款/期限短(≤1年)    → 循环购买结构，次级比例5-15%
物业租金/REITs结构         → 双SPV结构，长周期(5-10年)
```

**ABS方案特征：** 方案设计重心不在发行人而在资产池、次级档比例是关键（评级机构要求）、现金流模型必须覆盖压力情景、分层设计（优先A/优先B/次级）。

### PPN方案决策路径

```
发行人类别→评级→紧迫性      →方案结论
AA+/省属国企/3个月内       → PPN（灵活），1-3Y，定向投资人
AA/市属国企/不急于推进      → 建议走MTN路径（成本更低+流动性好）
AA民企/→不建议PPN         → 投资者基础太窄，建议加CRMW走公募
```

**PPN方案特征：** 灵活度高但成本高（+50-100BPS）、投资者限于定向投资人（≤200人）、信息披露大幅简化、监管审核速度最快（3-5周）。

## 三、方案比选Checklist（8维度权重评估）

当存在2个以上可行方案时，用以下8维度权重评分法进行比选：

| 维度 | 权重 | 评分1-5 | 加权分 | 说明 |
|------|:---:|:------:|:-----:|------|
| **融资成本** | 25% | /5 | | 票面利率+中介总费率，低→高分 |
| **审核确定性** | 20% | /5 | | 审核通过率+反馈轮数预期，高→高分 |
| **时间效率** | 15% | /5 | | 从申报到发行到账的总周期，短→高分 |
| **发行规模** | 15% | /5 | | 可获批的最大额度是否满足发行人需求 |
| **增信负担** | 10% | /5 | | 是否需要增信+增信成本（信用发行=5分） |
| **后续灵活性** | 5% | /5 | | 分期发行弹性+储架额度预留空间 |
| **投资者接受度** | 5% | /5 | | 市场需求热度+预期认购倍数 |
| **监管政策面** | 5% | /5 | | 当前政策窗口是否有利 |

**使用方法：**
1. 对每个方案逐维度打分（1=最差，5=最优）
2. 计算加权总分 = Σ(权重×评分)
3. 得分最高者为推荐方案，得分次高者为备选方案
4. 两方案分差<0.3 → 建议同时推进，利用时间差策略

**示例：某AA+市级城投10亿融资**

| 维度 | 权重 | MTN方案评分 | MTN加权 | 公司债方案评分 | 公司债加权 |
|------|:---:|:---------:|:------:|:-----------:|:--------:|
| 融资成本 | 25% | 4 | 1.00 | 3 | 0.75 |
| 审核确定性 | 20% | 4 | 0.80 | 2 | 0.40 |
| 时间效率 | 15% | 4 | 0.60 | 4 | 0.60 |
| 发行规模 | 15% | 5 | 0.75 | 5 | 0.75 |
| 增信负担 | 10% | 4 | 0.40 | 3 | 0.30 |
| 后续灵活性 | 5% | 4 | 0.20 | 3 | 0.15 |
| 投资者接受度 | 5% | 4 | 0.20 | 4 | 0.20 |
| 监管政策面 | 5% | 5 | 0.25 | 3 | 0.15 |
| **加权总分** | 100% | | **4.20** | | **3.30** |

→ 结论：MTN方案总分4.20 vs 公司债方案3.30，推荐MTN方案，公司债作为备选。

## 四、Gate Check——方案设计阶段三项硬性验证

进入文件撰写前必须验证：

### Gate 1：品种合规性

| 验证项 | 通过标准 | 参考 |
|-------|---------|------|
| 发行人资质满足品种准入的最低要求 | 评级/净资产/经营年限/审计意见均达标 | `../../references/bond/bond-product-matrix.md` |
| 城投发行人不在银保监会名单"监管类" | 或品种为借新还旧（名单内发行人唯一通道） | `SKILL_01` KYC成果 |
| 募投项目手续齐全（企业债） | 立项/可研/环评/土地/规划/能评全部取得 | `../../references/bond/bond-filing-regulators.md` |
| ABS资产池满足合格标准和分散度要求 | 入池资产不合格率<5% | `SKILL_00_债券尽调总览.md` Gate 3 |

### Gate 2：规模可行性

| 验证项 | 通过标准 | 参考 |
|-------|---------|------|
| 40%红线未触及 | 累计债券余额（含本次）+跨品种合计≤净资产×40% | 合并口径计算 |
| 本次发行+存量债券在NAFMII/证监会/发改委单一体系不超限制 | 按品种分别计算 | `SKILL_05_债券申报注册.md` |
| 发行规模具备市场消化能力 | 同评级同品种市场可比成功案例 | DCM市场经验 |
| 资金用途与规模匹配 | 补流有缺口测算支撑/项目资金不超过总投资60-70% | 按银保监流动资金贷款办法 |

### Gate 3：偿债保障

| 验证项 | 通过标准 | 参考 |
|-------|---------|------|
| 还本付息现金流覆盖 | 在保守假设下，经营活动现金流+在手授信≥到期本息合计 | `../../../scripts/bond_stress_test.py` |
| 再融资假设不过度依赖 | 再融资假设占比≤偿债来源的30% | 偿债保障分析 |
| 极端情景下仍有最低保障 | 极端情景（收入下降20%+融资冻结）仍可覆盖80%本息 | `../../../scripts/bond_stress_test.py` |
