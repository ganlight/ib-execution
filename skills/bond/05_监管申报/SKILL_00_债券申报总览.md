# SKILL_00：债券申报总览

> **版本：** v2.0 | **最后更新：** 2026年5月 | **适用范围：** 债券承做全品种监管申报阶段总览

## 概述

债券监管申报是债券项目进入监管审核的正式入口。三套监管体系（NAFMII/证监会/发改委）各有独特的申报体系、文件清单和审核节奏。选错通道、文件缺漏或错过最佳时间窗口，都可能导致数月时间和数百万元中介成本的浪费。

本文件作为债券线第五阶段（05_监管申报）的路由入口，速览三套申报体系的核心差异，提供申报文件清单总表，阐述审核时间轴和申报前内核清单。详细的申报注册操作手册见 `SKILL_05_债券申报注册.md`（增强版），三套体系横向对比见 `../../references/bond/bond-registration-comparison.md`，申报清单工具见 `../../../scripts/bond_filing_checklist.py`。

## 一、三套申报体系速览

### NAFMII体系（银行间债券市场）

| 维度 | 内容 |
|------|------|
| **主管机构** | 中国银行间市场交易商协会（NAFMII） |
| **核心品种** | MTN、CP、SCP、PPN、ABN、永续票据 |
| **审核方式** | 注册制（自律管理） |
| **关键表格编号** | MB表（募集说明书）、MS表（推荐报告）、DFI/TDFI（统一注册框架） |
| **主承销商** | 商业银行投行部（MTN/CP/SCP）或券商（PPN/ABN） |
| **投资者市场** | 银行间债券市场 |
| **审核周期** | SCP/CP：3-6周；MTN：6-12周；PPN：3-5周 |
| **注册有效期** | 2年（DFI）/ 注册后6个月内首期发行 |

### 证监会体系（交易所债券市场）

| 维度 | 内容 |
|------|------|
| **主管机构** | 中国证监会 + 沪深交易所 |
| **核心品种** | 公募公司债（大公募/小公募）、私募公司债、ABS、可转债/可交债 |
| **审核方式** | 交易所预审核 + 证监会注册（注册制） |
| **关键文件** | 募集说明书 + 交易所申请文件封卷表 + 证监会注册申请文件 |
| **主承销商** | 券商（全部品种） |
| **投资者市场** | 沪深交易所 |
| **审核周期** | 小公募：2-3个月；大公募：3-4个月；私募：1-2个月（备案制） |
| **批文有效期** | 24个月，首期发行12个月内 |

### 发改委体系（企业债）

| 维度 | 内容 |
|------|------|
| **主管机构** | 国家发展和改革委员会（过渡期：职责已宣布划入证监会） |
| **核心品种** | 一般企业债、专项企业债（县城新型城镇化/战略性新兴产业等） |
| **审核方式** | 审批制（中央/省级发改委双层审核） |
| **关键文件** | 发债申请报告 + 项目可行性研究报告 + 省级发改委初审意见 |
| **主承销商** | 券商 |
| **投资者市场** | 银行间+交易所（跨市场） |
| **审核周期** | 3-6个月（省级+国家发改委合计） |
| **批文有效期** | 24个月，首期发行12个月内 |

### 三体系快速对比

| 对比维度 | NAFMII | 证监会/交易所 | 发改委 |
|---------|:------:|:------------:|:------:|
| 审核速度 | ★★★★（最快） | ★★★（中等） | ★★（最慢） |
| 监管风格 | 自律规则，进化最快 | 框架式，市场化程度高 | 项目导向+审批联动 |
| 主承类型 | 银行为主（MTN/CP/ABN） | 券商 | 券商 |
| 创新品种支持 | 最强 | 较强 | 一般 |
| 城投支持度 | 高 | 中（视城投类型） | 高（传统城投主战场） |

## 三、审核时间轴（五节点时间预期）

债券审核流程在三个体系略有差异，但均遵循以下五节点结构：

```
节点1         节点2           节点3              节点4         节点5
申报提交  ──→  受理/形式审核  ──→  实质审核/反馈    ──→  反馈回复  ──→  注册/批文
(启动日)      (T+5工作日)      (T+4~8周)            (回复后)      (T+8~15周)
   │             │                │                  │             │
   │          受理/退回补正      1-3轮反馈           1-15工作日     │
   │          受理=正式启动     每轮反馈=时间重置   反馈回复      │
```

### 各节点详细说明

| 节点 | 内容 | NAFMII | 证监会/交易所 | 发改委 |
|------|------|:------:|:------------:|:------:|
| **T+5个工作日** | 受理/形式审核 | 受理通知 | 受理通知/补正 | 省级发改委受理 |
| **T+3~6周** | 第一轮反馈 | 4-8周 | 1-3个月 | 1-3个月（省级+国家） |
| **反馈回复时限** | 收到反馈后 | 15个工作日 | 15个工作日 | 30个工作日 |
| **常见反馈轮数** | - | 1-2轮 | 1-2轮 | 1-3轮 |
| **T+8~15周** | 取得批文 | SCP/CP最快2-3周 | 2-4个月 | 3-6个月 |

### 审核周期的影响因素

| 因素 | 加速效应 | 减速效应 |
|------|---------|---------|
| 评级越高（AAA） | ★★★★★ | - |
| 有完善募投项目文件 | 企业债显著加速 | 缺文件→无限延长 |
| 绿色/碳中和品种 | ★★★NAFMII/交易所快捷通道 | - |
| 城投类发行人 | - | 隐债问题→额外核查 |
| 首次发债AA主体 | - | 额外尽调+审慎审核 |
| 审计报告非标准意见 | - | 直接驳回或长期停审 |
| 房地产/产能过剩行业 | - | 额外行业专项审核 |

## 四、申报前内核清单（必须通过的15项检查）

在正式向监管机构提交申报材料之前，必须完成以下15项内核检查。任何一项不通过都必须解决：

### A. 文件完整性检查（5项）

| # | 检查项 | 工具/方法 |
|:-:|--------|---------|
| **1** | 全部申报文件已按该品种标准清单准备齐全 | `../../../scripts/bond_filing_checklist.py` |
| **2** | 所有文件已加盖公章/骑缝章，法定代表人签字齐全 | 逐页检查 |
| **3** | 日期逻辑正确（审计报告→评级报告→募集说明书→决议日期） | 时间轴核查 |
| **4** | 纸质版+电子版（PDF+Word）材料齐全，份数满足要求 | 份数清点 |
| **5** | 跨市场品种（如企业债跨市场）已同步准备两套材料 | 市场确认 |

### B. 数据与内容检查（5项）

| # | 检查项 | 工具/方法 |
|:-:|--------|---------|
| **6** | 40%红线计算正确（合并口径，扣除ABS待偿还余额） | `../../../scripts/financial_ratio_calc.py` |
| **7** | 审计报告、评级报告、募集说明书三处核心数据完全一致 | `../../../scripts/cross_ref_check.py` |
| **8** | 资金用途符合品种规定（补流≤30%，项目投资≤60-70%） | 条款核查 |
| **9** | 无失信被执行人记录（最新查询截图入卷） | 中国执行信息公开网 |
| **10** | 重大诉讼/行政处罚已完整披露（如有） | 律师核查意见 |

### C. 合规与程序检查（5项）

| # | 检查项 | 工具/方法 |
|:-:|--------|---------|
| **11** | 发行人有权机构（董事会/股东会）决议在有效期内 | 决议原件核查 |
| **12** | 主承销商已取得证监局备案或相关资质 | 主承资格确认 |
| **13** | 主承销商内核委员会已出具内核通过意见 | 内核意见原件 |
| **14** | 审计报告为无保留意见（保留/否定/无法表示→立即中止申报） | 审计意见类型核查 |
| **15** | 律所法律意见书已出具且结论为"合规" | 法律意见书原文 |

## 五、申报系统操作指南

### 5.1 三大申报系统入口

| 监管体系 | 系统名称 | 网址/入口 | 适用品种 | 关键功能 |
|---------|---------|----------|---------|---------|
| **NAFMII** | 综合业务和信息服务平台（孔雀开屏系统） | NAFMII官网→注册发行 | MTN/CP/SCP/PPN/ABN | 注册文件提交/审核进度查询/反馈回复/注册通知书下载 |
| **证监会/交易所** | 公司债券项目信息系统（上交所/深交所） | 上交所/深交所官网→债券项目信息 | 公司债/ABS/可转债 | 受理/审核/注册进度查询/反馈意见下载/批文下载 |
| **发改委** | 国家发改委政务服务大厅 | NDRC官网→政务服务 | 企业债 | 申报受理/省级转报/审批进度查询 |

### 5.2 申报材料制作要点

| 材料类型 | 制作要点 | 常见错误 |
|---------|---------|---------|
| **纸质版材料** | ① A4纸双面打印 ② 每份文件独立装订 ③ 公章+骑缝章完整 ④ 按要求份数准备（通常3-5份） | 公章缺失/份数不足/装订松散 |
| **电子版材料** | ① PDF格式（不可编辑版本） ② Word格式（可编辑版本） ③ 文件名统一规范（如：发行人简称_文件类型_日期.pdf） | 格式错误/文件名不规范/文件加密无法打开 |
| **光盘/U盘** | ① 所有电子版文件刻录至光盘或U盘 ② 标签注明发行人+品种+日期 | 光盘无法读取/内容不全 |

### 5.3 NAFMII孔雀开屏系统操作步骤

```
第1步：登录系统
       ↓
第2步：选择"注册发行"模块
       ↓
第3步：选择品种（MTN/CP/SCP/PPN/ABN）
       ↓
第4步：上传全部申报文件（MB表/R表/MS表+附件）
       ↓
第5步：系统自动校验文件格式→通过/不通过
       ↓
第6步：提交→NAFMII受理→生成受理编号
       ↓
第7步：后续通过系统跟踪审核进度/接收反馈意见/提交回复
```

### 5.4 证监会公司债系统操作要点

| 步骤 | 内容 | 注意事项 |
|------|------|---------|
| **系统申报** | 通过上交所/深交所公司债券项目信息系统提交 | 需使用CA证书登录 |
| **文件上传** | 逐件上传申报文件（必须与纸质版完全一致） | 上传后不可修改，如需修改需撤销后重新提交 |
| **受理确认** | 交易所5个工作日内出具《受理通知书》或《补正通知》 | 补正需在5个工作日内完成 |
| **审核追踪** | 通过系统实时查看审核进度/反馈意见 | 交易所审核后转证监会注册环节同样通过系统追踪 |

### 5.5 申报常见错误与解决方案

| 序号 | 常见错误 | 后果 | 解决方案 |
|------|---------|------|---------|
| 1 | 文件签名/盖章不全 | 形式审核驳回 | 逐页检查公章+骑缝章+法定代表人签字 |
| 2 | 材料份数不足 | 退回补正 | 严格按监管机构要求准备（核实最新要求） |
| 3 | 日期逻辑错误 | 审核质疑文件时效性 | 审计报告出具日→评级→法律意见书→募集说明书须递增 |
| 4 | 电子版与纸质版不一致 | 审核质疑真实性 | 提交前交叉核对电子版/纸质版 |
| 5 | 40%红线计算错误 | 审核直接退回 | 合并口径复核+审计师确认 |
| 6 | 城投申报缺少隐债清理证明 | 审核中止 | 提前取得财政/审计部门证明 |
| 7 | 评级报告有效期已过 | 需重新评级 | 评级报告有效期通常6个月，确保在有效期内 |
| 8 | 审计报告缺少最近一期 | 补正延迟 | 确保审计报告覆盖最近2个完整年度+最新一期 |
| 9 | 募投项目文件不齐全（企业债） | 长时间延迟 | 提前2-3个月启动项目审批手续 |
| 10 | 反馈回复超时 | 审核终止 | 严格按15个工作日内完成回复 |
