# SKILL_00：债券发行总览

> **版本：** v2.0 | **最后更新：** 2026年5月 | **适用范围：** 债券承做全品种发行执行阶段总览

## 概述

债券发行执行是"从批文到到账"的最后冲刺阶段。前期3-6个月的努力（尽调/方案/文件/申报）的成果，都取决于发行执行这2-4天的成败。发行执行的核心是**簿记建档**——通过集中收集投资者订单来确定发行利率和配售方案。与IPO的累计投标定价和竞价发行不同，债券发行有其独特的操作节奏、定价机制和投资者管理逻辑。

本文件作为债券线第六阶段（06_发行执行）的路由入口，阐述发行执行全流程、债券vs IPO发行执行的差异、定价方法选择和发行前核验清单。详细的操作手册见 `SKILL_06_债券簿记建档.md`（增强版），路演大纲见 `../../references/bond/bond_roadshow_outline.md`（如有）。

## 一、发行执行全流程（路演→簿记→定价→配售→登记→上市）

```
路演           簿记建档日 (T日)    配售缴款       登记上市
(T-5→T-1)     (T日全天)          (T+1日)        (T+2→T+3日)
   │              │                 │               │
   │ 投资者沟通    │ 订单收集         │ 资金清算        │ 债券到账
   │ 发行公告发布  │ 定价决策         │ 募集资金到账    │ 上市流通
   ▼              ▼                 ▼               ▼
预热期           定价日            清算日           上市日
(3-5天)          (1天)            (1天)           (1-2天)
```

### 阶段1：路演与预热（T-5至T-1日）

| 时间 | 工作事项 | 产出物 |
|------|---------|-------|
| T-5至T-2 | 一对一/一对多投资者路演 | 路演纪要+投资者反馈 |
| T-2 | 发布《发行公告》（含利率区间） | 正式发行公告 |
| T-1 | 向核心投资者发送全套发行文件 | 投资者材料包 |
| T-1 | 确认可投标投资者清单（白名单） | 投资者白名单 |
| T-1 | 收集非正式预订单（预判需求热度） | 初步订单底稿 |

### 阶段2：簿记建档日（T日）

| 时间段 | 工作事项 | 关键动作 |
|--------|---------|---------|
| 9:00-9:30 | 簿记系统开放 | 投资者通过北金所/交易所系统提交第一轮订单 |
| 9:30-11:30 | 第一轮订单收集 | 监控申购金额+申购利率+投资者结构 |
| 11:30 | 第一轮汇总 | 统计总申购量/边际利率/加权平均利率 |
| 11:30-12:00 | 如认购不充分→调整策略 | 启动第二轮引导/联系追加订单 |
| 13:30-15:30 | 最终轮订单收集 | 吸收观望投资者+追加订单 |
| 15:30 | 簿记截止 | 停止接收新订单 |
| 15:30-16:30 | DCM定价会议+发行人确认 | 确定最终票面利率 |
| 16:30-17:00 | 发布《票面利率公告》 | 公告发行结果 |

### 阶段3：配售与缴款（T+1日）

| 时间 | 工作事项 | 责任方 |
|------|---------|-------|
| 上午 | 按配售规则分配中标份额 | DCM中台 |
| 上午 | 向每位中标投资者发送《缴款通知书》 | DCM销售 |
| 下午16:00 | 投资者缴款截止 | 投资者 |
| 晚间 | 资金汇集→划付发行人指定账户 | 主承销商 |

### 阶段4：登记与上市（T+2至T+3日）

| 时间 | 工作事项 | 产出物 |
|------|---------|-------|
| T+2上午 | 提交登记托管材料 | 发行结果公告 |
| T+2全天 | CCDC/CSDC/上清所完成债券登记 | 债券代码+投资者到账 |
| T+2傍晚 | 交易所品种发布《上市公告书》 | 上市公告 |
| T+3 | 正式上市流通/银行间可交易 | 二级市场可买卖 |

## 二、债券 vs IPO 发行执行：五大核心差异

债券发行执行与IPO发行执行虽然都是"公开发售"，但逻辑完全不同：

### 差异1：定价机制不同

| 维度 | 债券发行 | IPO发行 |
|------|---------|---------|
| **定价方式** | 簿记建档（利率投标） | 累计投标定价（价格区间→最终定价） |
| **定价对象** | 票面利率（固定或浮动） | 发行价格（P/E倍数） |
| **定价周期** | T日当天（4-8小时） | T-3→T（路演+询价3-5天） |
| **定价弹性** | 利率区间内±10-20BPS弹性 | 价格区间±10%弹性（注册制） |

### 差异2：投资者结构不同

| 维度 | 债券发行 | IPO发行 |
|------|---------|---------|
| **主要投资者** | 银行金融市场部、保险公司、公募基金、理财子公司 | 公募基金、券商、保险资管、QFII/北向 |
| **投资者数量** | 通常10-40家机构 | 通常50-200+家机构 |
| **最小认购单位** | 100万面值（银行间）/ 10万面值（交易所） | 500股（A股一般）→约数千至数万元 |
| **投资者沟通** | DCM销售一对一/一对多路演 | 分析师路演+管理层路演+投价报告 |

### 差异3：配售规则不同

| 维度 | 债券发行 | IPO发行 |
|------|---------|---------|
| **配售规则** | 利率优先+时间优先+比例配售 | 投资者分类（A/B/C类）+摇号抽签 |
| **可分配总量** | 发行额的全量 | 网上+网下分池：网上70%起+网下30% |
| **特殊限制** | 单一投资者上限（通常为发行额的20-30%） | 网上限购市值+中签率约束 |
| **关联方限制** | 绝对禁止或需充分披露 | 券商关联方可参与网下但需隔离 |

### 差异4：失败后果不同

| 维度 | 债券发行 | IPO发行 |
|------|---------|---------|
| **认购不足处理** | 主承销商余额包销（如在承销协议中约定） | 中止发行，退回投资者申购款 |
| **定价偏离区间** | 可突破区间发行（如发行人接受更高利率） | 不可突破注册价格区间 |
| **取消发行后果** | 1-2周后可重新发行 | 需重新申报（周期性重启困难） |
| **恢复时间** | 快速恢复（市场好转可立即重启） | 缓慢恢复（审核流程重新启动） |

### 差异5：发行后流动性不同

| 维度 | 债券发行 | IPO发行 |
|------|---------|---------|
| **上市交易所** | 银行间/沪深交易所（品种决定） | 沪深/北交所（板块决定） |
| **首日交易特征** | 通常在发行价格±2%范围内成交 | 首日不限涨跌幅（新股效应） |
| **二级市场流动性** | 机构间交易为主，流动性好但T+1到账 | 散户+机构双向交易，流动性好 |
| **做市商** | 银行间市场做市商制度 | 交易所竞价交易（无做市商） |

## 三、定价方法选择

债券发行有三种定价方法，选择取决于品种、投资者基础和市场流动性：

### 3.1 簿记建档（Book Building）——最主流

**适用品种：** MTN、CP/SCP、公司债（公募+私募）、PPN、企业债、ABS优先级

| 维度 | 内容 |
|------|------|
| **操作方式** | 投资者通过系统提交"申购金额+申购利率"，主承收集所有订单后确定利率 |
| **定价依据** | 所有订单的边际利率和加权平均利率——市场需求直接决定定价 |
| **优势** | 透明度高、弹性大、利率发现机制最市场化 |
| **劣势** | 需活跃的市场和投资者参与（冷门品种不适合） |
| **典型周期** | T日当天（约4-8小时） |
| **使用频率** | 占中国信用债发行的90%+ |

### 3.2 竞价发行（Auction）

**适用品种：** 超短期融资券SCP（部分）、央行票据、国债

| 维度 | 内容 |
|------|------|
| **操作方式** | 投资者直接报价（价格/利率），按"从高价到低价"（价格投标）或"从低利率到高利率"（利率投标）分配 |
| **定价依据** | 美国式/荷兰式竞价：中标利率=各自报价利率（美国式）或统一最低可接受利率（荷兰式） |
| **优势** | 定价极为市场化、透明度最高 |
| **劣势** | 参与者需较强的定价能力、可能产生"赢家的诅咒" |
| **使用频率** | 国债为主，信用债SCP部分使用 |

### 3.3 协议定价（Negotiated Pricing）

**适用品种：** 私募公司债、PPN（部分）、挂钩互换的特殊品种

| 维度 | 内容 |
|------|------|
| **操作方式** | 主承销商与少数核心投资者（≤5家）直接协商确定利率，再由小范围配售 |
| **定价依据** | 可比债券利率+双方协商——发行人议价能力相对更强 |
| **优势** | 执行速度快、无需大规模路演、适合非公开发行 |
| **劣势** | 不市场化，可能高于公开市场利率20-50BPS、透明度低 |
| **使用频率** | 私募品种使用较多 |

### 定价方法选择决策树

```
品种是公募还是私募？
├─ 公募 → MTN/CP/公司债/企业债
│   └─ 簿记建档（主流，90%+使用）
│
└─ 私募 → 私募公司债/PPN
    ├─ 如投资者基数>5家 → 簿记建档（方向仍为主流）
    └─ 如投资者基数≤5家 → 协议定价

特殊品种：
└─ 国债/政金债 → 竞价发行（荷兰式）
└─ SCP（部分规模较大）→ 簿记建档 / 竞价发行
```

## 四、发行前10项最终核验

在簿记建档日之前，必须完成以下10项核验。任何一项未完成都将导致发行无法正常启动：

### 系统与流程核验（4项）

| # | 验证项 | 通过标准 |
|:-:|--------|---------|
| **1** | 簿记建档系统就绪 | 北金所集中簿记建档系统/交易所发行系统账号正常、权限就绪 |
| **2** | 发行公告已发布 | 《发行公告》已在指定信息披露平台正式发布 |
| **3** | 投资者白名单确认 | 适当性审查+反洗钱已完成，投资者白名单已录入系统 |
| **4** | 发行监控到位 | DCM主管/合规/风控人员已就位，发行应急方案已确认 |

### 文件与信息核验（3项）

| # | 验证项 | 通过标准 |
|:-:|--------|---------|
| **5** | 募集说明书终稿已定 | 无重大改动、已在信息披露平台可查 |
| **6** | 利率区间合理 | 区间中枢与市场基准利率+信用利差吻合，上下区间30-50BPS |
| **7** | 发行要素核对一致 | 发行金额/期限/付息方式/计息基准在所有文件中一致 |

### 市场与策略核验（3项）

| # | 验证项 | 通过标准 |
|:-:|--------|---------|
| **8** | 可比券信息就绪 | 同评级+同期限+同行业可比券最近2周发行结果 |
| **9** | 紧急联系机制就绪 | 如在簿记中发生市场巨变（利率大幅波动/投资者大规模撤单），可立即联系发行人决策 |
| **10** | 投资者预订单评估 | T-1日已收集非正式预订单，预判认购倍数和热度 |
