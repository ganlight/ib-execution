# SKILL_06b：债券簿记建档

## 概述

簿记建档（Book Building）是中国信用债券发行的核心定价机制，通过集中收集投资者订单来确定发行利率，区别于IPO的累计投标定价和竞价发行。簿记建档从启动到缴款登记一般经历4个工作日（T-1预热、T簿记建档日、T+1配售缴款、T+2登记流通），整个过程需要主承销商DCM团队对利率走势、投资者行为、认购倍数和边际利率进行实时判断和调整。

债券簿记建档的成败直接关系到：①发行能否顺利完成（避免发行失败）；②融资成本高低（利率定价过高增加企业成本，过低可能导致发行失败）；③投资者关系管理（配售是否公平透明）。本模块覆盖从簿记前准备到簿记后公告的完整流程、定价方法和应急预案。对于需要深层次DCM分析（如信用利差建模、投资者分层），可调用 **bond-dcm-core** 技能。

## 一、簿记建档全流程（Day-by-Day）

### T-1日：发行前一日

| 时间 | 工作事项 | 责任方 | 产出物 |
|------|---------|-------|-------|
| 上午 | ① 发布发行公告（含利率区间） | 主承+发行人 | 《发行公告》 |
| 上午 | ② 向核心投资者发送正式发行文件（募集说明书+发行公告+评级报告） | DCM销售 | 投资者材料包 |
| 上午 | ③ 确认可投标投资者清单（投资人适当性审查+反洗钱） | 合规 | 白名单 |
| 下午 | ④ 投资者路演/反路演（如需） | DCM+发行人 | 路演纪要 |
| 下午 | ⑤ 收集投资者预订单（非正式）以预判需求 | DCM销售 | 初步订单底稿 |
| 全日 | ⑥ 确认发行系统（北金所集中簿记建档系统/交易所系统）就绪 | DCM | 系统测试记录 |

**T-1关键点：**
- 利率区间一般为基准利率 + 信用利差 -+ 流动性溢价，上下区间差值通常为30-50BPS
- 预订单不是正式订单，但可以判断投资者热度和利率接受度
- 如收集的预订单价格多集中在下限，可适当压缩区间；较多在上限，可能破上限甚至调整区间

### T日：簿记建档日

| 时间 | 工作事项 | 关键动作 |
|------|---------|---------|
| 9:00-9:30 | 系统开放，簿记开始 | 投资者通过北金所系统或交易所系统提交第一轮订单 |
| 9:30-11:30 | 第一轮订单收集期 | 监控每笔订单：投资者名称、申购金额、申购利率 |
| 11:30 | 第一轮订单汇总 | 统计总申购量/边际利率/加权平均利率 |
| 11:30-12:00 | 如认购不充分→启动第二轮/下调利率引导 | DCM与发行人沟通是否调整策略 |
| 12:00-13:30 | 午间暂停簿记（如成交量充分）或继续接受订单 | 吸收下午交易员入场的新订单 |
| 13:30-15:30 | 第二轮/最终轮订单收集 | 部分投资者观望等待最后时刻下单 |
| 15:30 | 簿记截止（大多数品种） | 停止接收新订单 |
| 15:30-16:00 | 边际分析+定价决策 | DCM内部会议确定最终票面利率 |
| 16:00-16:30 | 发行人确认定价结果 | 发行人与主承签确定价确认函 |
| 16:30-17:00 | 发布《票面利率公告》 | 在指定信息披露平台公告发行结果 |

**T日关键决策点：**
- **认购倍数判断：** 认购倍数 = 总申购金额/计划发行金额
  - ≥2.0x：市场热度高，定价可偏向下限
  - 1.5-2.0x：正常水平
  - 1.0-1.5x：轻微不足，可能需定价偏上限
  - <1.0x：认购不足，需启动应急预案
- **边际倍数：** 最后一档利率对应的订单量/剩余可分配金额，边际倍数>1.5x说明定价支撑充分
- **非银/银行订单结构：** 银行和保险订单较稳定（不轻易撤单），基金/理财订单较灵活

### T+1日：配售与缴款

| 时间 | 工作事项 | 关键动作 |
|------|---------|---------|
| 上午 | ① 投资者配售分配 | 根据配售规则分配金额给中标投资者 |
| 上午 | ② 发送《配售确认及缴款通知书》 | 向每位中标投资者发送缴款通知 |
| 下午 | ③ 投资者缴款 | 投资者将认购款划至主承销商指定账户 |
| 下午16:00前 | ④ 确认所有投资者缴款完毕 | 如未到账，电话催缴 |
| 晚间 | ⑤ 缴款资金汇集 | 主承将募集资金划付至发行人指定账户 |

**配售规则（一般规则）：**
1. **利率优先：** 报价利率较低的投资者优先分配（簿记建档方向）
2. **时间优先：** 同一利率档次内，先下单的投资者优先分配
3. **比例配售：** 边际利率档次的订单按比例分配剩余额度（等比例/按规模比例）
4. **投资者分散化：** 对单一投资者的配售比例有上限（如不超过发行额的20-30%，防范集中度风险）
5. **关联方限制：** 发行人的关联方不得参与认购（或需做特殊信息披露）

### T+2日：登记与流通

| 时间 | 工作事项 | 产出物 |
|------|---------|-------|
| 上午 | ① 向登记托管机构提交登记材料 | 发行结果公告+债券基本要素 |
| 全天 | ② CCDC/CSDC完成债券登记 | 债券代码 |
| 全天 | ③ 完成债券初始登记 | 投资者账户到账 |
| 傍晚 | ④ 发布《上市公告书》（如交易所） | 上市公告 |
| T+3日 | ⑤ 交易所品种挂牌上市、银行间品种上市流通 | 可交易 |

## 二、簿记前准备

### 2.1 发行公告内容要点
- 发行人名称+债券简称+债券代码（如已获）
- 本期发行金额（如有超额配售权说明）
- 债券期限和票面利率类型（固定/浮动）
- **利率区间（申购区间）**：如 2.50% - 3.00%
- 簿记建档日期和时间
- 配售规则说明
- 缴款日+登记日+上市日
- 投资者适当性要求
- 主承销商和联系方式

### 2.2 投资者路演
在簿记建档日前至少完成2-3天的路演安排：
- **一对一/一对多路演：** 面向银行金融市场部、保险资管、公募基金固收部、理财子公司等核心买方
- **路演材料准备：** 发行人信用亮点（PPT/PDF）+ 行业分析 + 财务数据概要 + 可比债券对比
- **路演后跟进：** 发送电子版募集说明书+发行公告
- **反路演：** 如为ABS或有特殊资产，安排投资者实地尽调

### 2.3 定价参照——可比债券分析
| 参照维度 | 数据来源 | 分析要点 |
|---------|---------|---------|
| 同评级同期限中债估值 | 中债信息网 | 发行人所属收益率曲线的中位数和近期走势 |
| 同行业同评级近期新发 | Wind/NAFMII公告 | 最近2周同评级同行业的发行票面利率 |
| 发行人存量债券二级估值 | 外汇交易中心/Wind | 发行人已发行债券的二级市场到期收益率 |
| 国债/政金债基准曲线 | 中债估值 | 10Y国债、5Y国债、3Y国债相应期限的收盘收益率 |
| 信用利差走势 | 兴业/中金等研究 | 同评级利差压缩/走阔趋势 |

### 2.4 发行人信用刻画
在簿记前形成"C-Sheets"（信用要点表），便于DCM团队和投资者沟通：
1. 发行人简介（1页PPT）
2. 控股股东/实际控制人
3. 核心财务指标（近3年）：总资产、收入、净利润、EBITDA、资产负债率、EBITDA利息倍数
4. 评级与展望
5. 近期重大事件
6. 存量债券列表

## 三、簿记日操作

### 3.1 订单收集流程
投资者通过簿记建档系统（北金所/交易所）提交订单：
- 每笔订单包含：**投资者信息 + 申购金额 + 申购利率**
- 申购利率可以是一个具体数字（如2.75%）也可以是"区间内有效"（按区间上限等比例）
- 订单可修改和撤销（簿记截止前）

### 3.2 价格发现与订单分析
DCM团队需要对订单池进行实时分析：

| 分析维度 | 指标 | 决策意义 |
|---------|------|---------|
| 总量 | 总申购金额 vs 计划发行金额 | 超额倍数 |
| 利率分布 | 订单在各利率档次的分布量 | 定价位置 |
| 边际利率 | 最后一个全部分配的利率档次 | 定价基准 |
| 投资者结构 | 银行/保险/基金/其他 各类占比 | 稳定性判断 |
| 大单情况 | 单笔≥发行额10%的订单 | 是否存在依赖性 |
| 撤单风险 | 非银订单比例 | 非银>60%需警惕撤单 |

### 3.3 DCM定价会议
簿记截止后，DCM团队召开定价会，由DCM主管/销售负责人/项目负责人参加：
1. 销售团队报告订单汇总情况（总量、分布、结构）
2. 与参照利率进行交叉验证（同评级中债估值+最近发行成交价）
3. 与发行人沟通定价建议
4. 最终确定票面利率并取得发行人书面确认

## 四、定价方法

### 4.1 理论定价框架
```
票面利率区间中枢 = 无风险利率(同期限国债收益率)
                   + 信用利差(发行人评级对应的利差)
                   + 期限溢价(如有)
                   + 流动性溢价(如为私募或规模较小的品种)
                   + 新品溢价/新发溢价(首次进入该市场的发行人)
```

### 4.2 信用利差估算方法
| 方法 | 计算公式/数据源 | 适用场景 |
|------|---------------|---------|
| 中债估值法 | 中债同评级同期限收益率 - 国债同期限收益率 | 常规发行人 |
| 可比券法 | 同一发行人存量券的到期收益率 - 国债同期限收益率 | 有存量券的发行人 |
| 可比公司法 | 同行业同评级同期限最近新发票面利率均值 | 无存量券或市场定价参照不足 |
| OAS法（ABS） | 期权调整利差模型 | 结构化/含权产品 |

### 4.3 定价区间设定实战
**案例：** 某AAA产业类发行人发行3年期MTN 10亿元

- 3年期国债收益率（当前）：1.80%
- AAA信用利差（中债，产业类）：0.45%
- 期限溢价：0%
- 流动性溢价：0.05%（规模正常，市场活跃）
- 新发溢价：0%（该发行人存量MTN活跃）

→ **定价中枢：** 1.80% + 0.45% + 0.05% = 2.30%
→ **利率区间：** 2.10% - 2.50%（±20BPS）
→ **如认购倍数2.0x+ →** 定价可在下限附近：2.15-2.20%
→ **如认购不足1.3x →** 应在中上限：2.40-2.50%

## 五、投资者结构与配售

### 5.1 不同类型投资者的特征

| 投资者类型 | 典型投资规模 | 定价行为特征 | 配售偏好 |
|-----------|------------|------------|---------|
| 商业银行（金融市场/资管） | 单笔5千万-5亿 | 主要看利率，定价敏感，订单稳定 | 偏好AAA/AA+、MTN |
| 保险公司 | 单笔5千万-3亿 | 强资产负债匹配要求，偏好中长期限 | AA+以上、5Y/7Y |
| 公募基金（固收） | 单笔2千万-2亿 | 受排名和赎回压力，相对灵活 | 全评级段，含AA、PPN |
| 银行理财子公司 | 单笔5千万-5亿 | 零售资金驱动，风险偏好高于母行 | AA+以上为主 |
| 券商资管/自营 | 单笔1千万-1亿 | 赚取利差回报，撤单风险较高 | 全品种波段操作 |
| 境外机构（QFII/RQFII/债券通） | 单笔1千万-2亿 | 对流动性要求高，基本只投AAA/利率债 | 仅AAA或类主权 |
| 私募基金/信托 | 单笔500万-5千万 | 愿意接受高风险高收益溢价 | AA及以下、私募债、PPN |

### 5.2 配售执行
- 由DCM中台计算配售结果（Excel配售模型），分管DCM负责人审批
- 配售结果需与《发行公告》中的配售规则保持一致（规则不可在簿记后随意更改）
- 发行结果公告中包含中签率（有效申购金额/发行金额=有效申购倍数）
- 小单保护原则：对于申购金额较小的投资者可适当倾斜（拓宽持有人基础）

## 六、缴款与登记

### 6.1 缴款流程
| 步骤 | 内容 | 时限 |
|------|------|------|
| 发送缴款通知书 | 含缴款金额（认购金额+应计利息如适用）、缴款账户、缴款截止时间 | T日簿记结束后立即 |
| 投资者划款 | 将认购款汇至主承销商收款账户 | T+1日16:00前 |
| 资金归集 | 主承将所有投资者缴款资金汇至发行人指定账户 | T+1日 |
| 资金到账确认 | 发行人确认收到募集资金 | T+1日结束前 |

### 6.2 登记托管——CCDC vs CSDC
| 维度 | CCDC（中债登） | CSDC（中证登） |
|------|---------------|---------------|
| 覆盖品种 | 企业债、部分ABS | 公司债、交易所ABS |
| 银行间MTN/CP/SCP | 否（上清所） | 否 |
| 登记时间 | T+2日 | T+2日 |
| 结算方式 | DVP（券款兑付） | DVP |
| 操作界面 | 中债综合业务平台 | 中证登CA系统 |

**上清所（SHCH）说明：** NAFMII品种（MTN/CP/SCP/PPN等）由上清所登记托管

### 6.3 DVP结算机制
DVP（Delivery Versus Payment，券款对付）：
- 投资者付款与债券登记同步进行，保证双方资金安全
- T+1日16:00前完成缴款→T+2日债券到账
- 如投资者未及时缴款，主承商需启动差额补足或信用支持机制

## 七、应急预案

### 7.1 认购不足（认购倍数<1.0x）
| 场景 | 应对措施 | 责任方 | 后果 |
|------|---------|-------|------|
| 轻微不足（0.8-1.0x） | ① 定价上调至利率区间上限 ② 联系未投标投资者追加订单 ③ 主承销商余额包销（如在承销协议中约定） | DCM | 成功 |
| 中度不足（0.5-0.8x） | ① 延长簿记建档时间 ② 紧急联系发行人沟通定价方案 ③ 评估是否启动主承余额包销或取消发行 | DCM+发行人 | 可能取消 |
| 严重不足（<0.5x） | 宣布取消本次发行，重新评估发行窗口和利率区间 | 发行人决定 | 取消发行，1-2周后视情况重启 |

### 7.2 过度认购（认购倍数>3.0x）
- **积极应对：** 定价下移至下限或突破下限（在注册允许范围内），将票面利率尽量降至最低
- **超额配售权：** 如注册通知书中有超额配售安排（如基础发行10亿+超额配售不超过5亿），可启动超额配售
- **不启动超额配售：** 按比例配售，避免市场信号认为"公司急需资金"

### 7.3 大单集中撤单
- **预防：** 关注订单池中大单投资者，及时DCM销售进行"确权"（确认订单不会撤单）
- **发生撤单后：** 立即启动补单程序，联系未投标投资者
- **如撤单超过发行额的30%：** 评估是否继续或取消发行

### 7.4 定价与区间偏离
在簿记建档中可能遇到：投资者订单全部集中在利率区间上限或超出区间：
- 如区间为2.50-3.00%，订单包括"3.10%"的报价超出上限
- **处理方案：** 定价发行人可以接受超过区间上限的利率（但需在发行的注册/审批文件中不违反规则）
- 或者在承销协议条款下由主承销动用"自由裁量定价权"以区间上限价格由主承认购

### 7.5 缴款失败
某中标投资者T+1日未及时缴款：
- 启动中间级补缴：主承销商先行垫付该投资者的认购款（承销余额包销条款视情况）
- 之后向该投资者追偿（含违约金）
- 该投资者将纳入DCM黑名单不再接受未来投标

## 八、簿记后

### 8.1 定价公告（T日）
在同日傍晚前发布《票面利率公告》，汇总：
- 票面利率（最终确定的发行利率）
- 有效申购总量（总申购金额）
- 有效申购倍数
- 投资者中标情况分布

### 8.2 上市/流通申请
| 类型 | 上市/流通场所 | 申请时间 | 申请文件 |
|------|------------|---------|---------|
| 公司债（公募） | 交易所 | T+2日 | 上市公告书+发行结果公告 |
| 公司债（私募） | 交易所（挂牌转让） | T+2日 | 挂牌转让申请 |
| MTN/CP/SCP | 银行间 | T+2日（自动） | 上清所自动流通 |
| 企业债 | 交易所/跨市场 | 登记日+2日 | 上市申请书 |
| ABS/ABN | 交易所/上清所 | 登记后1-2日 | 挂牌申请 |

### 8.3 投后管理

- 对核心投资者进行持续关系管理
- 关注发行后债券二级市场流动性（是否出现折价/溢价交易）

## 十、簿记建档操作手册

### 10.1 定价区间设定

定价区间设定是簿记建档的核心准备工作。区间过宽会导致定价效率低下，区间过窄则可能无法覆盖市场真实利率水平。

**定价区间设定四步法：**

| 步骤 | 内容 | 方法 | 输出 |
|------|------|------|------|
| **第1步：确定基准利率** | 同期限国债/国开债收益率 | 中债估值实时数据 | 无风险利率基准值 |
| **第2步：估算信用利差** | 发行人评级对应的信用利差 | 中债同评级利差+可比券利差 | 信用利差中位数 |
| **第3步：调整流动性溢价** | 私募/小额/首次发债额外溢价 | 市场经验值：私募+20-50BPS、小额+10-20BPS | 调整后信用利差 |
| **第4步：设定区间** | 中枢±20-30BPS | 预留上下浮动空间 | 利率区间（低值-高值） |

**区间宽度参考：**

| 品种 | 标区间宽度 | 市场活跃时 | 市场波动时 |
|------|:--------:|:--------:|:--------:|
| MTN（AAA） | 20-30BPS | 15-20BPS | 30-40BPS |
| MTN（AA+） | 30-40BPS | 20-30BPS | 40-50BPS |
| MTN（AA） | 40-50BPS | 30-40BPS | 50-60BPS |
| SCP（AAA） | 15-20BPS | 10-15BPS | 20-30BPS |
| 公募公司债 | 30-40BPS | 20-30BPS | 40-60BPS |
| 私募公司债 | 40-60BPS | 30-40BPS | 60-80BPS |
| PPN | 50-70BPS | 40-50BPS | 70-100BPS |

### 10.2 投资者建档

**投资者建档六个环节：**

| 环节 | 时间 | 操作要点 | 负责人 |
|------|------|---------|-------|

**利率确定五步决策：**

| 步骤 | 内容 | 决策依据 | 输出 |
|------|------|---------|------|

## 十一、定价策略详解——4步定策法

票面利率的最终确定不是简单的"投资者出价平均"，而是一个复杂的多因素权衡过程。以下是DCM团队的4步定策法：

### 第1步：市场需求分析

| 分析维度 | 判断标准 | 策略含义 |
|---------|---------|---------|
| **认购倍数** | ≥2.0x→市场需求旺盛 | 定价可偏向下限（-10-30BPS） |
| | 1.5-2.0x→需求正常 | 定价在中枢附近（±10BPS） |
| | 1.0-1.5x→需求偏弱 | 定价偏向上限（+10-30BPS） |
| | <1.0x→需求不足 | 需启动应急预案（见第十二节） |
| **边际倍数** | >2.0x→底部支撑极强 | 定价可再压缩5-10BPS |
| | 1.0-2.0x→底部有一定支撑 | 定价在中枢附近 |
| | <1.0x→底部无支撑 | 定价不能下调，甚至需要上调 |
| **投资者结构** | 银行/保险订单占比>50% | 订单稳定性高，定价可稍激进 |
| | 银行/保险订单占比<30% | 撤单风险高，定价需保守 |
| **大单依赖度** | 单笔>20%发行额的订单 | 存在集中度风险，配售需分散 |

### 第2步：可比券利率分析

| 分析维度 | 数据源 | 关键判断 |
|---------|-------|---------|
| **同评级同期限中债估值** | 中债信息网 | 发行人所属收益率曲线中位数→定价中枢基准 |
| **同行业同评级最近新发** | Wind/NAFMII公告 | 最近2周同评级同行业发行票面利率→市场公允利率 |
| **发行人存量债券二级估值** | 外汇交易中心/Wind | 发行人已有债券的二级市场YTM→自身定价锚 |
| **信用利差走势** | 中金/兴业研究 | 同评级利差是压缩还是走阔→未来趋势判断 |

**可比券分析实例：**

假设某AAA央企发行3年MTN 10亿：
- 3年国债收益率：1.80%
- AAA（产业类）中债估值利差：0.40%（最近一周均值）
- 最近2周同评级MTN新发票面利率：2.15-2.35%
- 该发行人存量3Y MTN二级YTM：2.25%
- 信用利差趋势：最近1个月同评级利差压缩了5BPS

→ **定价中枢判断：** 2.20-2.25%（略低于存量券YTM，利差压缩趋势下）

### 第3步：发行人议价能力评估

| 维度 | 高议价能力（可压价） | 低议价能力（需让步） |
|------|-------------------|-------------------|
| **评级** | AAA/超级央企 → 利率下限 | AA/首次发债 → 利率上限 |
| **市场知名度** | 高（市场活跃发行人） | 低（首次进入市场） |
| **发行历史** | 多次成功发行，历史认购倍数>2x | 首次发行/上次发行困难 |
| **投资者关系** | 有稳定的核心投资者群体 | 投资者基础薄弱 |
| **市场窗口** | 利率下行期/资金面宽松 | 利率上行期/资金面紧缩 |
| **融资紧迫性** | 不急（可等好窗口） | 急需资金（资金到账时间紧） |
| **品牌溢价** | 央企/知名企业，投资者愿接受低利率 | 一般企业，投资者要求额外溢价 |

### 第4步：最终票面利率决策

| 情景 | 认购倍数 | 可比券利率 | 议价能力 | 推荐定价策略 |
|------|:------:|----------|:------:|------------|
| **情景A：最佳** | ≥2.5x | 低于可比券5-10BPS | 高 | 定价下限-5BPS |
| **情景B：良好** | 2.0-2.5x | 与可比券基本持平 | 中-高 | 定价下限附近 |
| **情景C：正常** | 1.5-2.0x | 略高于可比券 | 中 | 定价中枢附近 |
| **情景D：偏弱** | 1.2-1.5x | 高于可比券10-20BPS | 低 | 定价上限附近 |
| **情景E：不足** | <1.2x | 大幅高于可比券 | 极低 | 定价上限+评估是否取消 |

**最终票面利率确定检查清单：**

- [ ] 票面利率是否在《发行公告》的利率区间内？
- [ ] 认购倍数是否≥1.0x（正常发行）？
- [ ] 票面利率是否低于发行人可接受的上限？
- [ ] 与可比券利率的差异是否在合理范围内（±15BPS）？
- [ ] 投资者结构是否合理（银行/保险占比>30%）？
- [ ] 是否存在单笔大单依赖（单笔>20%）？

## 十二、配售分配规则——5类投资者的配售权重与优先级

配售分配不是简单的"按比例分"，而是需要综合考虑投资者类型、订单质量、市场关系维护等多重因素。

### 12.1 五类投资者的配售权重与优先级

| 投资者类型 | 配售优先级 | 推荐配售权重 | 原因 |
|-----------|:--------:|:----------:|------|
| **商业银行** | ★★★★★ | 30-40% | 订单最稳定（不撤单）、长期持有、有利于二级市场做市 |
| **保险公司** | ★★★★ | 15-25% | 长期持有（资产负债匹配需求）、订单稳定 | 
| **公募基金（固收）** | ★★★ | 15-20% | 流动性提供者，但撤单风险略高 |
| **银行理财子公司** | ★★★ | 10-20% | 新兴力量，规模增长快，分散持有人基础 |
| **券商资管/自营** | ★★ | 5-15% | 交易性投资者，折价风险高，但可提供流动性 |
| **其他（私募/信托/境外）** | ★ | 5-10% | 分散持有人基础，但资质参差 |

**配售优先级规则：**

1. **核心投资者（商业银行+保险公司）：** 优先配售，确保其认购不低于申报量的60-70%
2. **重要投资者（公募基金+理财子）：** 按申报量比例配售，确保认购不低于申报量的50-60%
3. **一般投资者（券商/私募等）：** 按剩余额度比例配售
4. **单一投资者配售上限：** 不超过发行总额的20-30%（防范集中度风险）
5. **小单保护原则：** 申购金额≤500万的投资者优先全额配售（拓宽持有人基础）

### 12.2 配售分配实战步骤

**步骤1：确定配售总金额** = 发行金额 + 超额配售（如有）

**步骤2：核对有效订单**
- 剔除关联方订单（禁止关联方认购）
- 剔除无效订单（超出利率区间、资料不完整等）
- 确认所有有效订单的投资者信息

**步骤3：按优先级分配**
- 第1轮：核心投资者（银行+保险）按60-70%比例配售
- 第2轮：重要投资者（基金+理财子）按50-60%比例配售
- 第3轮：一般投资者（券商/私募/其他）按剩余额度配售
- 第4轮：调整配售（考虑集中度风险、小单保护等）

**步骤4：调整配售**
- 单一投资者配售上限 = 发行总额 × 20-30%
- 如任一投资者配售比例超过上限，调减其配售额
- 调减部分按比例分配给其他投资者

**步骤5：生成配售结果**
- 导出配售结果表（Excel）
- DCM中台核对配售结果
- DCM负责人审批
- 向投资者发送《配售确认及缴款通知书》

### 12.3 配售分配注意事项

| 注意事项 | 说明 |
|---------|------|
| **公平原则** | 配售规则必须与《发行公告》一致，不可临时更改 |
| **透明度** | 配售结果需在发行结果公告中披露（无暗箱分配） |
| **反洗钱** | 所有投资者必须通过反洗钱审查 |
| **关联方禁止** | 发行人关联方不得参与认购（必须100%筛选） |
| **记录留存** | 保留完整配售决策记录（至少5年） |

## 十三、发行失败预案

### 情景1：簿记认购不足——最紧急情景

**触发条件：** 簿记建档截止时，有效认购倍数<1.0x

**分级应对：**

| 级别 | 倍数区间 | 紧急程度 | 应对措施 | 后果 |
|------|:------:|:------:|---------|------|
| **黄色预警** | 0.8-1.0x | 中等 | ① 定价上调至利率区间上限 ② 联系未投标投资者追加订单 ③ 联系已投标投资者追加申购金额 | 大概率成功 |
| **橙色预警** | 0.5-0.8x | 高 | ① 延长簿记建档时间（如发行公告允许） ② 紧急联系发行人沟通定价方案 ③ 评估启动主承余额包销（如在承销协议中约定） ④ 联系新的投资者 | 50%概率成功 |
| **红色预警** | <0.5x | 极高 | ① 宣布取消本次发行 ② 向监管机构报告取消发行原因 ③ 向市场发布取消发行公告 ④ 1-2周后重新评估发行窗口和利率区间 | 取消发行 |

**认购不足原因分析及应对：**

| 原因 | 应对策略 |
|------|---------|
| **利率区间太低** | 上调利率区间（如发行公告允许）/ 下次发行设定更宽的区间 |
| **市场整体环境不佳** | 等待市场好转（信用利差缩小/资金面宽松） |
| **发行人信用问题** | 重新评估发行人信用（评级下调/负面新闻）/ 引入担保增信 |
| **投资者沟通不足** | 增加路演场次/一对一投资者沟通 |
| **可比券替代效应** | 分析可比券发行情况（是否同期有更优质的同期限债券发行） |

### 情景2：利率倒挂——中期风险情景

**触发条件：** 簿记建档中，投资者报价利率普遍高于可比券利率20BPS以上，且大部分订单集中在上限附近

**应对措施：**

| 步骤 | 内容 | 责任人 |
|------|------|-------|
| **① 分析原因** | 是否因为市场利率上升？/ 投资者对发行人信用有疑虑？/ 可比券利率参考有误？ | DCM分析师 |
| **② 交叉验证** | 重新验证可比券利率（是否有更新的同评级新发利率？） | DCM分析师 |
| **③ 与发行人沟通** | 向发行人说明市场情况，评估是否接受较高利率 | DCM+发行人 |
| **④ 重新评估** | 是否继续发行？/ 是否调整利率区间？/ 是否推迟发行？ | DCM+发行人 |

**关键决策：**

| 利率倒挂幅度 | 决策建议 |
|:----------:|---------|
| +20-30BPS | 接受较高利率，继续发行 |
| +30-50BPS | 与发行人充分沟通，评估融资紧迫性 |
| >+50BPS | 建议推迟发行，等待更有利的市场窗口 |

### 情景3：发行窗口恶化——长期风险情景

**触发条件：** 在T-5至T-1日（路演预热期），市场利率出现大幅波动（如国债收益率单日涨跌>10BPS、信用利差单周走阔>20BPS、股市/汇率出现异常波动等）

**事前预警信号：**

| 预警信号 | 阈值 | 影响 |
|---------|------|------|
| 10Y国债收益率 | 单日涨跌>10BPS | 基准利率变动，影响所有固定收益定价 |
| AAA信用利差 | 单周走阔>20BPS | 同评级信用风险溢价上升 |
| 7D回购利率 | 连续3日>政策利率+50BPS | 资金面紧缩，投资者融资成本上升 |
| 人民币汇率 | 单日贬值>0.5% | 外资流出，影响境外投资者需求 |
| 股市 | 单日跌幅>3% | 风险偏好下降，固收需求增加（也可能避险情绪利好债市） |

**应对预案：**

| 情景 | 应对 |
|------|------|
| **轻微恶化**（信用利差走阔5-10BPS） | 利率区间上调5-10BPS，继续推进 |
| **中度恶化**（信用利差走阔10-20BPS） | 与发行人紧急沟通，评估是否调整区间或推迟 |
| **严重恶化**（信用利差走阔>20BPS或出现黑天鹅事件） | 建议推迟发行，1-2周后重新评估 |

**发行窗口的判断工具：**
- 中债估值曲线（Real-time）
- 银行间7D回购利率走势
- 同评级信用利差走势
- 可比券最近2周发行结果
- 外汇交易中心市场情绪指数

## 十四、实战案例

### 案例1：央企30亿MTN簿记成功（约500字）

**背景：** 某AAA央企（电力行业）拟发行3年期MTN 30亿元，用于偿还到期债务。该央企市场知名度高、信用等级顶级、存量债券流动性好，是银行和保险的核心配置品种。

**簿记建档全流程：**

**T-5至T-2（路演预热期）：**
- 与12家银行金融市场部、5家保险公司、8家公募基金进行一对一/一对多路演
- 路演材料突出：① 央企信用（国资委直管、无隐债风险）；② 行业垄断地位（电力行业核心企业）；③ 财务稳健（资产负债率45%、EBITDA利息倍数8.5x）；④ 银行授信充裕（未使用额度>本次发行金额的3倍）
- 收集预订单：央行1亿/招商2亿/浦发3亿/人寿2亿/人保1亿/华夏基金1亿...共约15亿预订单

**T日（簿记建档日）：**
- 发行公告：3年期MTN、利率区间2.00%-2.40%、规模30亿
- 9:00-9:30：簿记系统开放，投资者开始提交订单
- 10:30：首小时订单已达25亿（利率区间分布：下限2.00%有5亿、2.05-2.10%有12亿、2.15-2.20%有8亿）
- 11:30：第一轮汇总→订单总量已达38亿（认购倍数1.27x）、加权平均利率2.12%
- 13:30-15:30：午后订单继续流入→最终订单总量52亿（认购倍数1.73x）

**DCM定价会议（15:30-16:00）：**
- 订单汇总：52亿申购、认购倍数1.73x
- 边际分析：最后一档利率2.20%订单8亿，边际倍数2.67x（底部支撑极强）
- 投资者结构：银行52%+保险18%+基金20%+其他10%（结构健康）
- 交叉验证：同评级同期限中债估值2.15%、最近2周同评级MTN新发票面利率2.10-2.30%、该发行人存量MTN二级YTM 2.18%
- **定价建议：2.15%**（在中枢偏下，体现市场需求旺盛+优质发行人议价能力）
- 发行人确认：接受2.15%

**T+1（配售缴款日）：**
- 配售：52亿申购→30亿发行额（中签率58%）
- 核心投资者（银行+保险）配售占比65%（19.5亿）
- 缴款：30亿×2.15% = 6,450万利息/年
- 所有投资者16:00前全部缴款完毕

**发行结果：**
- 票面利率：2.15%（利率区间下限仅高15BPS，体现了优质发行人的议价能力）
- 认购倍数：1.73x
- 投资者结构：银行65%+保险18%+基金15%+其他2%
- 发行结论：成功

**经验总结：**
- 央企+AAA+财务状况优秀→投资者高度认可，定价在下限附近
- 路演预热充分（12家银行+5家保险+8家基金）→认购倍数高
- 投资者结构健康（银行+保险占比>80%）→订单稳定
- DCM定价会议充分交叉验证→定价合理
- 全程无撤单→市场和发行人双重认可

### 案例2：城投10亿企业债认购不足应对（约500字）

**背景：** 某AA市级城投平台拟发行企业债10亿元（5+2年期，第5年末可赎回），用于市政基础设施建设。该城投平台位于中西部地级市，所在地方政府债务率为橙色档（中等风险）。评级为AA（无担保增信），同期市场可比券利率为4.20-4.50%。

**簿记建档过程与问题：**

**T-5至T-2（路演预热期）：**
- 路演对象：6家券商资管、4家私募基金、3家银行理财子公司
- 路演反馈：投资者普遍关注隐债风险、地级市财政实力、城投市场化转型进展
- 预订单收集：仅3家券商和2家私募表示有兴趣，预订单总额仅约3亿（远低于10亿发行额）

**T日（簿记建档日）：**
- 发行公告：5+2年期企业债、利率区间4.00%-4.80%、规模10亿
- 9:00-11:30：第一轮订单仅收到4家投资者订单，总额4.2亿（认购倍数0.42x）
- 11:30-12:00：DCM紧急分析原因：
  ① 所在地方政府债务率橙色→保险机构（通常偏好城投）不敢投
  ② 无担保增信→纯AA信用吸引力不足
  ③ 5+2年期含权结构→投资者偏好纯5年期（含权结构增加定价复杂度）
- 12:00-13:30：DCM与发行人紧急电话会议：
  - 发行人希望继续推进（融资紧迫）
  - DCM建议：延长簿记时间+调整利率区间 → 发行人同意

**应对措施：**

**措施1：调整利率区间**
- 原区间：4.00%-4.80%（上下40BPS）
- 新区间：4.50%-5.20%（整体上调50BPS，上下35BPS）
- 公告方式：发布《调整发行利率区间的公告》（发行公告允许调整）

**措施2：延长簿记时间**
- 原截止时间：15:30
- 新截止时间：17:00（延长1.5小时）
- 理由：给予投资者更长时间评估新的利率区间

**措施3：追加路演+投资者沟通**
- 联系原有意向但未下单的2家投资者（说明新的利率区间）
- 联系1家新投资者（地方性银行，对本地城投较熟悉）
- 向现有订单的4家投资者确认是否追加申购

**措施4：启动主承销商余额包销**
- 主承销商承诺：如最终认购仍不足，主承销商将余额包销（按承销协议条款）

**发行结果：**
- 延长簿记+调整利率区间后→收到新增订单3.8亿，总订单8.0亿
- 主承销商余额包销2.0亿→最终完成10亿足额发行
- 票面利率：5.00%（略低于调整后的区间上限）
- 认购倍数：0.8x（调整为有效认购+主承包销）

**经验总结：**
- 城投债在橙色区域无担保增信情况下，投资需求天然不足→应提前预判
- 调整利率区间+延长簿记时间→给予了投资者二次评估机会
- 主承销商余额包销→发行"兜底"机制发挥了重要作用
- 建议下次发行：引入AAA担保/CRMW增信+选择市场窗口更好的时期
- 关注地方政府债务率四档分色对城投债的影响（橙/红区域投资者大幅缩减）
