# SKILL_00：债券尽调总览

> **版本：** v2.0 | **最后更新：** 2026年5月 | **适用范围：** 债券承做全品种尽调阶段总览

## 概述

债券尽职调查是债券承做全流程的核心技术环节。与股权尽调（IPO/再融资）追求"全面价值发现"不同，债券尽调以**信用导向**为唯一核心——所有核查工作均围绕"发行人能否按时足额还本付息"展开。本文件作为债券线第二阶段（02_尽职调查）的路由入口，阐述债券尽调的全景框架、四维尽调模型、品种差异化的尽调重点和阶段Gate Check。

详细的尽调操作手册见 `SKILL_02_债券尽调.md`。信用分析方法论见 `../../references/bond/credit-analysis-framework.md`。尽调清单生成工具见 `../../../scripts/dd_checklist_gen.py`。

## 一、债券尽调 vs 股权尽调：六大核心差异

债券尽调的逻辑链与股权尽调根本不同——理解这些差异是做好债券尽调的基础。**关键词：信用导向 vs 价值发现。**

### 差异1：尽调目标不同

| 维度 | 债券尽调 | 股权尽调（IPO） |
|------|---------|---------------|
| 核心问题 | "3-5年后能不能还钱？" | "这个公司值多少钱？" |
| 尽调哲学 | 下行风险保护（Downside Protection） | 上行价值发现（Upside Discovery） |
| 财务分析逻辑 | 现金流覆盖→偿债保障→违约概率 | 收入增长→盈利质量→估值倍数 |
| 尽调结论 | 信用等级 + 发行可行性判断 | 是否符合上市条件 + 估值区间 |

**实战隐喻：** 股权尽调像"体检+潜能评估"——能否跑得更快；债券尽调像"信用评分"——能否按时还信用卡。

### 差异2：尽调深度分布不同

| 尽调维度 | 债券尽调权重 | IPO尽调权重 | 差异说明 |
|---------|:---------:|:---------:|---------|
| 财务尽调 | **60%** | 35% | 债券大幅偏重财务偿债指标 |
| 增信/担保尽调 | **20%** | 0% | 债券特有的增信核查（IPO无此概念） |
| 法律尽调 | 10% | 40% | 债券法律尽调范围远小于IPO |
| 业务尽调 | 10% | 25% | 债券更关注行业稳定性而非业务增长 |

### 差异3：红线标准不同

| 红线类型 | 债券尽调 | 股权尽调 |
|---------|---------|---------|
| 财务红线 | 资产负债率>80%且无强担保 → 中止 | 资产负债率偏高不是绝对障碍 |
| 合规红线 | 失信被执行人 → 中止 | 同一标准 |
| 业务红线 | 经营性现金流连续3年为负 → 高风险 | 更关注是否有持续经营能力 |
| 资产红线 | 受限资产>总资产50% → 高风险 | 更关注资产权属是否清晰 |
| 特有红线 | **隐债未化解 / 担保代偿记录 / 资产池集中度超标** | 关联交易不公允 / 同业竞争 / 股权不清晰 |

### 差异4：尽调周期不同

| 品种类型 | 债券尽调周期 | IPO尽调周期 |
|---------|:----------:|:---------:|
| 常规信用债（MTN/公司债/CP） | 3-5周 | - |
| 担保增信型债券 | 4-6周（+担保人尽调） | - |
| ABS/ABN | 5-8周（+资产池尽调） | - |
| 企业债（含募投项目） | 4-6周（+项目尽调） | - |
| IPO | - | 8-16周（含辅导期则更长） |

### 差异5：尽调工作底稿不同

| 底稿类别 | 债券尽调 | IPO尽调 |
|---------|---------|---------|
| 总量 | 约IPO的40-55% | 全套7大类 |
| 核心增量 | +信用分析底稿 +增信文件 +担保人底稿 | +历史沿革底稿 +关联交易底稿 +同业竞争分析 |
| 缺失项 | 无辅导备案底稿 / 无盈利预测底稿（非必需） | 全套均有 |

### 差异6：中介团队分工不同

| 中介角色 | 债券尽调中的角色 | IPO尽调中的角色 |
|---------|---------------|---------------|
| 券商（主承） | 信用分析+文件撰写核心，尽调主导者 | 辅导+保荐+文件撰写全部角色 |
| 评级机构 | **尽调重要参与者**（信用评级→发行必须条件） | 辅助角色（仅定价参考） |
| 律师事务所 | 法律合规验证，但范围远小于IPO | 法律尽调核心 |
| 审计机构 | 财务数据验证（基础） | 财务尽调核心（更深度） |
| 担保机构（如有） | **全新角色**：担保人尽调对象 | 几乎不涉及 |

## 二、四维尽调模型

债券尽调不是一维的"看财报"，而是四维交叉验证。任何一维出现问题都可能触发风险评估升级。

### 维度1：主体信用尽调（权重60%）

**目标：** 判断发行人自身的独立偿债能力

| 尽调模块 | 核心指标 | 检查方法 |
|---------|---------|---------|
| **盈利能力** | EBITDA/营业收入、近3年扣非净利润趋势 | 审计报告+同行业对比 |
| **偿债能力** | EBITDA利息倍数（≥2x安全）、有息负债/EBITDA | 明细账+利息支出核对 |
| **现金流** | 经营活动净现金流/总债务、投资活动现金流合理性 | 现金流量表+银行流水抽凭 |
| **资产质量** | 应收账款周转天数、受限资产占比（>30%→黄灯） | 资产评估报告+抵押登记查询 |
| **负债结构** | 短期/长期债务比、一年内到期债务占比 | 明细账+借款合同 |
| **银行授信** | 未使用授信额度/总债务（>30%→缓冲充分） | 银行授信函+征信报告 |

### 维度2：担保增信尽调（权重20%）

**目标：** 判断增信措施的有效性——担保人的真实偿债能力、抵质押物价值与流动性

| 增信类型 | 尽调重点 | 关键验证 |
|---------|---------|---------|
| **第三方保证担保** | 担保人信用等级（应≥发行人等级）、担保意愿、与被担保方关系 | 担保人审计报告+征信报告+是否有关联 |
| **资产抵质押** | 资产估值合理性、变现能力、是否已设定其他权利负担 | 评估报告+登记查询+保险单 |
| **CRMW/CDS** | 创设机构信用等级、CRMW覆盖率 | 创设协议+创设机构资质 |
| **差额补足/流动性支持** | 差额补足方的履约能力和法律效力 | 协议文本+法律意见书 |
| **政府支持（城投）** | 支持意愿+支持能力+政府债务空间 | 36号文/35号文口径+财政数据 |

**担保有效性判断三问：**
1. 担保金额是否在担保人承受范围内？（担保人对外担保总额+本次<净资产×40%）
2. 担保人自身是否有重大不利事项？
3. 抵质押物的市场变现周期是否匹配债券期限？

### 维度3：资产池尽调——ABS/ABN专项（权重视品种而定）

**仅适用于ABS/ABN/REITs等结构化品种。常规信用债不涉及。**

| 尽调维度 | 核心指标 | 预警阈值 |
|---------|---------|---------|
| **资产合格性** | 逐笔资产的合同有效性、权属清晰性、可转让性 | 不合格率>5%→调减入池规模 |
| **统计特征** | 加权平均利率/期限/账龄、行业/地域集中度 | 单一行业>30%→集中度风险标红 |
| **历史表现** | 静态池违约率、早偿率、回收率的时间序列 | 违约率>同类型资产行业75分位→增厚超额利差 |
| **现金流模型** | 正常情景/压力情景/极端情景下的违约率和损失率分布 | 压力情景下优先级覆盖倍数<1.2x→评级下调风险 |
| **资产管理人** | 管理人的团队、系统、历史管理规模 | 管理规模<20亿或无同类型经验→审慎 |

### 维度4：合规尽调（权重10%）

**目标：** 排除法律法规层面的硬性障碍

| 尽调领域 | 核心核查项 | 核查方式 |
|---------|----------|---------|
| **失信与违约** | 失信被执行人、近3年债券/贷款违约 | 中国执行信息公开网+征信报告 |
| **行政处罚** | 近3年是否被证监会/银保监/交易所行政处罚 | 公开信息检索+发行人声明+律师核查 |
| **重大诉讼** | 涉案金额>净资产5%的未决诉讼 | 裁判文书网+天眼查+律师核查 |
| **环保安全** | 产能过剩/重污染行业近3年是否有重大事故 | 环保部门网站+安全生产监管部门 |
| **关联方占用** | 大股东/关联方非经营性占用资金 | 审计报告附注+律师核查 |
| **40%红线** | 本次发行后累计债券余额≤净资产40% | 合并口径逐品种累加 |
| **公司治理** | 发行决议是否经有权机构（董事会/股东会）审批 | 决议原件+公司章程 |

## 三、尽调清单总表（按债券品种差异化）

不同债券品种的尽调深度完全不同。以下矩阵帮助快速定位尽调重点：

| 品种 | 主体信用尽调 | 担保增信尽调 | 资产池尽调 | 合规尽调 | 募投项目尽调 | 总体周期 |
|------|:--------:|:--------:|:--------:|:------:|:--------:|:-------:|
| **MTN/CP/SCP** | ★★★★★ | ★★☆ | 无 | ★★★ | 无 | 3-4周 |
| **（超）短期CP** | ★★★★ | ★★ | 无 | ★★★ | 无 | 2-3周 |
| **PPN** | ★★★ | ★★★ | 无 | ★★ | 无 | 2-3周 |
| **公募公司债** | ★★★★★ | ★★☆ | 无 | ★★★★ | ★★（如需） | 4-5周 |
| **私募公司债** | ★★★★ | ★★★ | 无 | ★★★ | ★（如需） | 3-4周 |
| **企业债** | ★★★★★ | ★★★★ | 无 | ★★★★★ | ★★★★★ | 4-6周 |
| **ABS（交易所）** | ★★★ | ★★★ | ★★★★★ | ★★★★ | 无 | 5-8周 |
| **ABN（NAFMII）** | ★★★ | ★★★ | ★★★★★ | ★★★ | 无 | 5-8周 |
| **可转债** | ★★★★★ | ★★ | 无 | ★★★★★ | ★★★ | 6-10周 |
| **绿色债** | ★★★★ | ★★★ | 无 | ★★★★ | ★★★★（绿色认证） | 3-5周 |
| **境外债** | ★★★★★ | ★★★★ | 无 | ★★★★ | 无 | 4-8周 |

> ★数量表示尽调深度，★数越多=投入工作量越大。无=完全无此项尽调。

### 品种差异化尽调提示

**信用债通用（MTN/CP/公司债/PPN）：**
- 尽调重心~80%在主体信用分析
- 合规尽调聚焦40%红线+失信+重大违法违规
- 无募投项目时不涉及项目尽调

**企业债特有增量：**
- 募投项目尽调（新增工作量40-60%）：项目立项+可研+环评+土地+规划+能评全套手续
- 发行人合规更高要求：发改委口径需查是否有项目违规
- 城投附加：省级发改委转报意见必须取得

**ABS/ABN特有增量：**
- 资产池尽调（新增工作量100%+）：需逐笔/抽样核查底层资产的合同、回款记录、违约记录
- 法律尽调重心转移：SPV结构+破产隔离+真实出售
- 评级尽调双线并行：主体评级+各档优先级证券评级

## 四、Gate Check——尽调阶段三大硬性验证

阶段性核查必须在进入下一阶段前完成。以下三项任一未通过则必须解决后才能推进方案设计。

### Gate 1：信用等级验证

| 验证项 | 通过标准 | 未通过处理 |
|-------|---------|----------|
| 主体信用等级≥目标品种最低要求 | 见 `SKILL_00_债券启动总览.md` Gate 3 | 更换品种 / 引入担保增信 / 中止 |
| EBITDA利息倍数≥1.5x | 近1年及最新一期均达标 | 如仅靠担保通过，需充分揭示风险 |
| 经营活动净现金流覆盖到期债务 | 近2年至少1年为正覆盖 | 增加流动性支持文件 / 下调规模 |

### Gate 2：担保有效性验证

| 验证项 | 通过标准 | 未通过处理 |
|-------|---------|----------|
| 担保人信用等级≥发行人 | 低1个notch则需增厚担保覆盖率 | 更换担保人 / 多重增信（保证+抵质押） |
| 担保人对外担保总额<净资产×40% | 确保本次担保后可正常履约 | 降低本次担保金额 / 增加反担保措施 |
| 抵质押物市场价值≥担保债务×1.3 | 覆盖率为1.3-1.5x（视变现难度） | 追加抵质押物 / 置换为第三方保证担保 |

### Gate 3：资产池分散度验证（ABS/ABN专项）

| 验证项 | 通过标准 | 未通过处理 |
|-------|---------|----------|
| 单一借款人/资产占比≤15% | 满足评级机构分散度要求 | 剔除超标资产/调减入池 |
| 单一行业/地域≤30% | 降低行业/地域集中度风险 | 超额利差+超额抵押方案 |
| 压力情景优先级覆盖倍数≥1.0x | 最极端假设下的还本付息保障 | 调减优先级发行规模/增厚次级比例 |

## 五、跨阶段衔接

尽调结论是后续所有工作的基础：

```
尽调结论 → 方案设计（03）→ 确定品种/规模/期限/增信方案
尽调结论 → 文件撰写（04）→ 募集说明书"发行人情况"章节（财务+业务+风险）
尽调结论 → 监管申报（05）→ 提供申报文件中的尽调报告+评级报告+法律意见书
```

**不同尽调结论对应的后续路径：**

| 尽调结论 | → 方案设计 | → 文件撰写 | → 监管申报 |
|---------|-----------|-----------|-----------|
| AAA主体、财务优秀 | 直接公募AAA公司债/MTN，利率定价下限 | 文件质量要求最高（公众投资者保护） | 审核最快（1-2轮反馈） |
| AA+主体、财务稳健 | MTN+私募公司债双通道 | 正常文件撰写流程 | 2-3轮反馈 |
| AA主体、城投/有担保 | 必须有增信措施，利率中上限 | 增信文件占比30%+ | 城投审核重点关注（隐债/转型） |
| AA-主体、强担保 | 私募为主，公募通过率低 | AAA担保方尽调30-40%增量 | 拉长审核周期 |
| 资产池尽调 | → ABS方案设计 | → ABS计划说明书 | → ABS挂牌确认（证监会+交易所） |
