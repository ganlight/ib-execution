# SKILL_02：债券尽调

> **版本：** v2.0 | **最后更新：** 2026年5月 | **适用范围：** 债券承做全品种尽职调查阶段

## 概述

债券尽调是债券承做中最核心的技术环节——不同于IPO尽调以"持续盈利能力+内控有效性+信息披露真实性"为三大支柱，债券尽调以**偿债能力**为唯一核心逻辑链，所有核查工作均围绕"发行人能否按时足额还本付息"展开。本文件作为债券尽调的总入口，覆盖四大专项模块：（一）信用尽调（财务偿债+资产质量+负债结构）、（二）担保与增信尽调（担保人+抵质押物+其他增信）、（三）城投债尽调专项（政府关系+业务性质+资产结构）、（四）ABS/REITs资产池尽调专项（资产合格性+统计特征+现金流压力测试）。（五）尽调问题追踪（RAG分级—债券版）。

本文件应与 `SKILL_02_尽职调查总览.md`（IPO尽调总框架）互补使用——共享通用尽调方法论（资料收集、管理层访谈、中介协调、底稿管理），但聚焦于债券特有的核查深度和技术要点。具体单个指标的信用评级方法，同步参考 `references/bond/credit-analysis-framework.md`。

## 一、债券尽调总框架（与IPO尽调的本质差异）

### 1.1 核心差异全景

| 维度 | IPO尽调 | 债券尽调 |
|------|---------|---------|
| **核心关注** | 成长性、盈利能力、内控有效性 | 偿债能力、现金流稳定性、增信措施 |
| **尽调深度** | 全面尽调（3年一期），法律/财务/业务三线并行 | 以偿债为核心聚焦，财务线深度远高于法律/业务线 |
| **关键红线** | 虚增收入、关联交易不公允、持续经营存疑 | 隐债未化解、对外担保超标、受限资产过高、评级门槛 |
| **监管导向** | 信息披露真实性、中小投资者保护 | 投资者保护、还本付息保障、资金用途合规 |
| **工作底稿** | 全套7大类 | 聚焦3类：信用类+增信类+现金流类（总量约为IPO的40-50%） |
| **尽调重心分布** | 法律40%+财务35%+业务25% | **财务60%**+增信20%+法律10%+业务10% |
| **尽调周期** | 6-12周 | 3-5周（债券聚焦偿债）+ABS/REITs场景+2-3周 |

### 1.2 债券尽调三阶段

```
阶段1：初步尽调（T+0至T+7工作日）
  → 重点：阻断因素排查（隐债/评级/失信等），产出初步尽调结论
阶段2：全面尽调（T+7至T+21工作日）
  → 信用尽调（二）←最深度模块（2-2.5周）
  → 担保增信尽调（三）+城投专项（四）+ABS专项（五）←按场景
阶段3：补充尽调（T+21至T+28工作日，按需延长）
  → 监管反馈/评级关注函/负面舆情后的补充核查
```

## 二、信用尽职调查

> **债券尽调中最重要的模块，耗时最长（~2-2.5周），产出信用评估结论。**

### 2.1 财务偿债能力

#### 2.1.1 五大偿债核心指标

| # | 指标 | 公式 | 🟢安全 | 🟡关注 | 🔴危险 |
|---|------|------|:---:|:---:|:---:|
| 1 | **EBITDA利息保障倍数** | EBITDA/利息支出（含资本化部分） | >5x | 3-5x | <1.5x |
| 2 | **有息负债/EBITDA** |（短借+长借+应付债券+长期应付款有息）/EBITDA | <2.5x | 2.5-5x | >8x（产业）/12x（城投） |
| 3 | **经营性现金流/有息负债** | 经营净现金流/期末有息负债 | >0.25x | 0.10-0.25x | <0.05x或负值 |
| 4 | **货币资金/短期有息负债** |（货币-受限-质押存单）/短期有息负债合计 | >1.5x | 0.7-1.5x | <0.5x |
| 5 | **速动比率（修正版）** |（货币+交易金融资产+应收款）/流动负债 | >1.2x | 0.5-1.2x | <0.4x |

| | 公用事业/能源 | <3x | 3-6x | >6x |
| | 制造业 | <2.5x | 2.5-5x | >5x |

| **经营性现金流/有息负债** | 城投 | >0.15x | 0.05-0.15x | <0.05x |
| | 公用事业/能源 | >0.25x | 0.15-0.25x | <0.15x |
| | 制造业 | >0.20x | 0.10-0.20x | <0.10x |

| | 制造业 | >1.2x | 0.7-1.2x | <0.7x |
| | 房地产 | >0.8x | 0.4-0.8x | <0.4x |

**行业阈值说明：** 城投各指标可放宽0.5-1档（有政府隐性支持），但指标3和4（现金流相关）不可豁免——城投经营现金流长期为负且货币资金不足覆盖短期到期→红旗。

#### 2.1.3 经营活动现金流质量分析（4维核验）

| 维度 | 公式 | 🟢健康 | 🟡关注 | 🔴危险 |
|------|------|:---:|:---:|:---:|
| **收现比** | 销售商品收到现金/营业收入（含税） | >1.0x | 0.8-1.0x | <0.8x |
| **净现比** | 经营活动净现金流/净利润 | >1.0x | 0.5-1.0x | <0.5x或持续为负 |
| **FCF连续性** | 近3年FCF(经营净现金流-资本支出)为正的年数 | ≥2/3年 | 1/3年 | 0/3年 |
| **经营现金流/EBITDA** | 经营净现金流/EBITDA | >0.8x | 0.5-0.8x | <0.5x |

#### 2.1.4 盈利波动性分析

| 方法 | 计算 | 🟢稳定 | 🟡波动 | 🔴剧烈波动 |
|------|------|:---:|:---:|:---:|
| **变异系数（CV）** | 近3年净利润标准差/均值 | <30% | 30-60% | >60% |
| **最大单年降幅** | Max(第1-2年降幅,第2-3年降幅) | <15% | 15-40% | >40% |
| **毛利率波动** | 近3年毛利率标准差 | <3pct | 3-8pct | >8pct |

### 2.2 资产质量核查

#### 2.2.1 受限资产——对债权人保护的影响

| 核查维度 | 公式 | 🟢安全 | 🟡关注 | 🔴红旗 |
|---------|------|:---:|:---:|:---:|
| **受限资产/总资产** |（抵押+质押+冻结+保证金）/总资产 | <20% | 20-40% | >40% |
| **受限资产/净资产** | 受限资产总额/净资产 | <40% | 40-80% | >80% |
| **受限资产变现覆盖** | 变现价值/有息负债总额 | >1.5x | 1.0-1.5x | <1.0x |

**逐笔核查清单：**
- 受限原因：银行借款抵押/非银质押/法院冻结
- 资产类型：土地（流动性差）/房产（变现快）/应收账款（不确定最高）/股权（处置复杂）
- 合规性：是否有重复抵押/二押？是否已在征信系统登记？
- 解限计划：是否有明确解限时间表？

#### 2.2.2 应收账款/其他应收款对手方穿透

| 核查项 | 🟡关注阈值 | 🔴红旗 |
|--------|:--------:|:-----:|
| **应收账款/总资产** | >25%（产业）/35%（城投） | >40% |
| **前5大对手方集中度** | >60% | >80%+任何一家>30% |
| **对手方是否关联方** | 关联方>30% | 关联方>50%+无商业实质往来 |
| **对手方是否地方政府（城投核心）** | 政府应收>60% | >80%+回款滞后>2年 |
| **账龄—1年以上占比** | >40% | 3年以上>20%+未提足坏账 |
| **坏账计提低于行业均值** | >5pct低 | >10pct低+大额长账龄未单项计提 |

#### 2.2.3 商誉减值风险

| 核查项 | 🟡关注 | 🔴红旗 |
|--------|:----:|:---:|
| **商誉/净资产** | >15% | >30% |
| **减值测试假设合理性** | 增长率假设高于行业但无充分理由 | — |
| **被收购子公司实际vs承诺业绩** | 连续2年低于承诺>15%但未提减值 | 连续3年低于承诺>30% |

#### 2.2.4 投资性房地产公允价值计量合理性

| 核查项 | 🟡关注 | 🔴红旗 |
|--------|:---:|:---:|
| **投资性房地产/总资产** | >15%（产业）/30%（城投） | >40% |
| **公允价值变动贡献利润** | >15% | >30%（利润被非现金变动"注水"） |
| **估值方法变更** | — | 近3年内从成本法转公允价值法且估值显著高于周边可比案例 |

## 三、担保与增信尽调

### 3.1 担保人尽调

#### 3.1.1 担保人分类评估

| 担保人类型 | 增信力度 | 尽调要点 |
|-----------|:------:|---------|
| **关联—集团母公司** | 强至非常强 | 集团合并AAA→核查：①绝对控制权；②连带责任vs一般保证（前者>>后者） |
| **关联—同集团兄弟** | 弱至中等 | **一般不接受**（风险高度叠加不分散） |
| **非关联—外部担保** | 取决于担保人信用 | 核查：①是否有隐性关系→可能"形式外部实质内部"；②商业逻辑合理性 |
| **专业担保机构** | 强（AAA+） | 核查：①担保总额集中度；②备付准备金；③监管评级 |
| **政府及其派出机构** | ❌不合法 | **一旦发现→立即中止→见SKILL_01阻断因素** |

#### 3.1.2 担保人自身信用评估速查

| 指标 | 🟢担保能力强 | 🟡一般 | 🔴危险 |
|------|:---:|:---:|:---:|
| **担保人主体评级** | AAA/AA+ | AA+/AA | AA-/A+及以下 |
| **对外担保总额/净资产** | <30% | 30-50% | >50% |
| **担保人资产负债率** | <65% | 65-75% | >75% |
| **担保人流动比率** | >1.5x | 1.0-1.5x | <1.0x |
| **含反担保** | ✓足额→风险对冲 | 不足额→需查反担保人履约力 | ❌无反担保（裸担） |

#### 3.1.3 担保协议核心条款审查

| 审查项 | 标准 | 🔴危险信号 |
|--------|------|---------|
| **连带责任vs一般保证** | 连带责任→法律确定性高 | 一般保证→"几乎无增信"（需先穷尽司法救济） |
| **保证期间** | ≥债券期限+诉讼时效（3年）=T+3年 | 过短（<1年）→持有人面临免责窗口风险 |
| **代偿触发** | "到期未偿还→宽限期→书面通知" | 需先取得法院判决/会计师确认→拖延陷阱 |
| **担保范围** | 本金+利息+违约金+损害赔偿+实现债权费用 | 仅覆盖"本金"→大打折扣 |
| **独立性与不可撤销** | 写明"独立于主合同，不因主合同无效而无效"+"不可撤销" | "主合同无效→担保无效"；留有"担保人有权撤销"口子 |

### 3.2 抵质押物尽调

#### 3.2.1 抵质押物类型与LTV上限

| 类型 | 评估方法 | LTV上限 | 变现周期 |
|------|---------|:-----:|:------:|
| **土地使用权** | 市场比较法+基准地价法 | 60%（核心平台）/50%（其他） | 3-6个月 |
| **商业房产** | 收益法+市场比较法 | 55%（一线）/45%（其他） | 6-12个月 |
| **住宅房产** | 市场比较法 | 50% | 3-6个月 |
| **上市公司股权** | 市价法（20日均价/收盘孰低） | 40%（主板）/30%（中小创） | 较快（但大宗减持+限售股需额外折价） |
| **非上市公司股权** | DCF+市场法 | ≤30%（流动性折价极高） | 3-12个月+，不宜作主要增信 |
| **应收账款** | 债务人信用+历史回款率+账龄 | 70%（AAA/政府）/≤50%（其他） | 配合结算周期 |
| **收费权/特许经营权** | DCF折现未来现金流 | 50%（运营期）/30%（建设期） | 🔴极难—需政府审批 |
| **采矿权** | DCF（储量×价格×采收率-成本） | 40% | 🔴极长—自然资源部门审批数月到1年 |

#### 3.2.2 各品种抵质押率上限

| 品种 | 上限 | 来源 |
|------|:---:|------|
| 公募公司债 | 无硬性上限—但>65%→募集书须风险揭示+持续监测 | 交易所上市规则 |
| 私募公司债 | 协议自由—市场LTV<50% | — |
| 企业债 | ≤60%+评估基准日距申报≤6个月 | 发改委相关指引 |
| MTN | ≤65% | NAFMII中期票据指引 |
| ABS/ABN | 优先级LTV≤50%（超额抵押增信） | 专项计划合约 |

#### 3.2.3 权利完整性核查

| # | 核查项 | 操作 | 如有问题→ |
|---|--------|------|---------|
| 1 | 权证齐全性 | 不动产权证书/股权质押工商登记/中登网动产登记 | 估值折价≥50%或不得作为增信 |
| 2 | 无重复抵押 | 不动产登记中心产权查询 | 先次序方优先受偿→本次仅剩残值 |
| 3 | 无查封/冻结/扣押 | 登记查询确认 | 🔴不可作抵押物→更换担保或放弃 |
| 4 | 无第三方权利争议 | 核查资产购买合同链条+所有权纠纷 | 暂停方案（待诉讼结果） |

### 3.3 其他增信措施

#### 3.3.1 差额补足 vs 连带担保

| 维度 | 连带责任保证 | 差额补足承诺 |
|------|:---------:|:----------:|
| **法律确定性** | ★★★★★（民法典第688-700条） | ★★★☆☆（依赖条款措辞严谨度） |
| **触发速度** | 债务人到期未履行→立即触发 | 需先确认"缺口"→可能有争议/需独立方确认→延迟 |
| **评级增信** | +2至3子级 | +1至2子级（取决于措辞） |

#### 3.3.2 各类支持函的法律效力对比

| 支持类型 | 法律效力 | 评级增信效果 |
|---------|:------:|:--------:|
| **安慰函** | ★☆☆☆☆（几乎无法强制执行） | **0子级（不计入）** |
| **流动性支持函** | ★★★☆☆（临时流动性—不覆盖资不抵债） | +0.5至1子级 |
| **维好协议（KIB）** | ★★☆☆☆（美元债常用—道德约束为主） | +0.5至1子级（近年削弱→至多+1子级） |
| **连带担保** | ★★★★★ | +2至3子级 |

#### 3.3.3 CRMW/CDS安排

| 工具 | 创设机构 | 保护范围 | 费率 | 适用 |
|------|---------|---------|:--:|------|
| **CRMW（凭证）** | NAFMII备案（约30+家） | 单一特定债券，可转让 | 30-100bps | 最主流—民企增信 |
| **CRMA（合约）** | 同上 | 一对一不可转让 | 同CRMW+定制折价 | 双边定制 |
| **CDS** | 银行间备案 | 覆盖一类债务 | — | ⚠️国内市场仍极不活跃→不假定可用 |

#### 3.3.4 结构化分层保护程度

| 分层结构 | 保护程度 | 适用 |
|---------|:------:|------|
| 优/次=90%/10% | ★★☆☆☆（表面性） | 仅AAA基础资产 |
| 优/次=80%/20% | ★★★☆☆（标准） | 常规ABS |
| 优/次=70%/30% | ★★★★☆（较强） | 中等风险—消费/小微/租赁应收 |
| 优/次=60%/40% | ★★★★★（极强，但次级成本>优先级400-800bps） | 高风险/波动资产 |

> ⚠️ **核心警示**：分层保护取决于**基础资产回收率**——危机时期违约率↑+回收率↓↓——即使50%次级厚度也可能不足。

### 4.1 政府关系维度

#### 4.1.1 行政级别与支持评估

|:------:|:---------:|:---------:|:--------:|

| **省会/副省级** | ★★★★☆（核心平台—市融资工具） | ★★★★☆（一般预算≥100亿） | **+1至2子级** |

| **区县级（强）** | ★★★☆☆（保底倾力→资源有限） | ★★★☆☆（百强县自有留存多） | **0至+1子级** |
| **区县级（弱）** | ★★☆☆☆ | ★☆☆☆☆ | **0→可能负向** |

#### 4.1.2 地方政府支持核心指标

| 指标 | 公式 | 🟢良好 | 🟡关注 | 🔴危险 |
|------|------|:---:|:---:|:---:|
| **财政补贴/利润总额** | 近3年政府补贴合计/利润总额合计 | <30% | 30-70% | >70%→自身造血弱 |
| **政府回款时效** | BT/代建/PPP近3年平均回款周期 | <6个月 | 6-12个月 | >12个月+大量逾期 |
| **地方政府债务率** |（显性+隐性）/综合财力 | <120%（绿） | 120-200%（黄/橙） | >200%（红→几乎不可新增） |
| **银保监平台名单** | 名单内/外 | 不在→一般国企 | 不在+隐债未出清 | 🔴在名单内→企业债禁止 |

### 4.2 业务性质维度

#### 4.2.1 业务分类与偿债依赖性

| 业务类型 | 现金流特征 | 转型方向 | 偿债依赖 |
|---------|---------|:------:|---------|
| **纯公益性**（道路/公园/绿化/路灯） | 0收入+完全依赖政府 | 🔴需剥离 | 完全依赖政府信用 |
| **准公益性**（水务/燃气/公交/供热）| 政府定价+补贴→稳定 | ✅→公用事业企业 | 项目收益+补贴双支撑 |
| **经营性—园区运营** | 租金+服务费→稳定 | ✅→低风险 | 自身经营现金流 |
| **经营性—房地产** | 市场化→周期波动 | ⚠️高风险 | 市场化—不可依赖 |
| **经营性—贸易/金融投资** | 低利润+大额垫资/超出能力圈 | 🔴高风险 | — |

#### 4.2.2 基建代建业务市场化程度

| 维度 | 🟢市场化 | 🟡有限 | 🔴纯政府行为 |
|------|:------:|:----:|:---------:|
| **代建管理费率** | >5% | 2-5% | <2%（象征性） |
| **招商运营收入贡献** | >20% | 10-20% | <10% |
| **特许经营权**（水务/燃气/公交） | ≥2项+覆盖运营+部分偿债 | 1项勉强维持 | 无→不可靠市场化偿债 |

### 4.3 资产结构维度

| 资产项 | 核查要点 | 🟢健康 | 🟡关注 | 🔴红旗 |
|--------|---------|:---:|:---:|:---:|
| **存货—土地资产占比** | 权证完整+土地用途合规 | <40% | 40-60% | >60%+大量无证土地 |
| **其他应收款—政府欠款** | 对手方穿透+账龄+回款时效 | <25% | 25-40% | >40%（政府大量占款） |
| **无形资产—土地使用权** | 会计核算口径+土地证+估值 | <15% | 15-30% | >30%+评估入账（不偿债） |
| **公益性资产vs有效资产** | 公园/免费学校/市政道路→不得计入 | 公益0% | 5-15% | >15%（→虚增净资产→必须剔除） |

#### 城投资产"剃光头"测试——最终有效资产才是偿债基础：

```
最终有效资产 = 总资产
              - 公益性资产（0价值）
              - 无证权证土地（≥50%折价→几乎不能处置）
              - 政府应收账款（>2年→催收两难→打折50%）
              - 已抵押受限资产（归属原银行→只计剩余价值）
```

## 五、ABS/REITs资产池尽调专项

### 5.1 入池资产准入标准

| 标准 | 要求 | 🔴不合格 |
|------|------|:------:|
| **权属清晰** | 原始权益人完全、无争议所有权 | 有所有权争议/被查封/第三方权利主张 |
| **可转让性** | 可合法转让给SPV | 合同含"不得转让"且债务人不同意 |
| **无权利限制** | 无抵押/质押/司法查封 | 存在且无法入池前解限 |
| **资产分散度** | 单一≤15%/行业≤30%/地域≤35% | 单一>25%无额外缓解 |
| **历史表现数据** | ≥3年违约/早偿/回收率数据 | <1年（→假设极保守→优先级可能发不出） |

### 5.2 资产池统计特征监控

| 统计特征 | 计算 | 🟡关注 |
|---------|------|:---:|
| **加权平均剩余期限（WAL）** | Σ(剩余期限×本金)/Σ本金 | <5年安全，>10年→长期不稳定 |
| **加权平均利率** | Σ(利率×本金)/Σ本金 | 与优先级票面利差<200bps→安全垫薄 |
| **加权平均账龄** | Σ(已过月数×本金)/Σ本金 | 账龄越长→越依赖预测而非历史 |
| **地域集中度（前3）** | 前3省份余额/总池 | >65%→地域风险高度集中 |
| **行业集中度（前3）** | 前3行业余额/总池 | >55%→行业共振风险 |

### 5.3 现金流模型核验——5情景压力测试

| # | 情景 | 参数设定 | 优先级覆盖倍数要求 |
|---|------|---------|:---------------:|
| 1 | **正常** | 历史平均违约率/早偿率/回收率 | >1.2x ✅ |
| 2 | **利率上行100bp** | 优先级票面+100bp，违约率↑20%，早偿率↓15% | >1.1x ✅ |
| 3 | **违约率×2** | 违约率×2，回收率↓15%（危机折价） | >1.05x ✅ |
| 4 | **早偿率×2** | 早偿率×2，压扁利息缓冲 | >1.1x ✅ |
| 5 | **组合压力** | ①+②+③+④叠压 | >1.0x ✅（极端底线——只要≥1.0即pass） |

**覆盖倍数释义：**
- >1.2x → 优先级高安全度
- 1.0-1.2x → 勉强→必须确保假设极度保守
- <1.0x → 🔴不合格→需调整分层/排除高风险资产/增加外部增信

## 六、尽调问题追踪（RAG分级—债券版）

### 6.1 分级标准

| 等级 | 影响范围 | 处理方式 |
|:---:|---------|---------|
| 🔴 **Red** | 影响发行可行性或触发阻断 | **立即升级→承做+发行人高管→书面"阻断认定"→暂停或设定重启条件** |
| 🟡 **Amber** | 影响条款/成本/增信/评级（不致阻断） | **项目例会行动项→量化目标→限时解决（1-3月）→到期复查** |
| 🟢 **Green** | 需持续关注，目前不影响 | **每月追踪→恶化则转🟡→记录在项目周报** |

### 6.2 示例库

| 等级 | 发现 | 处理 |
|:---:|------|------|
| 🔴 | 隐债未化解—仍在財政部系统且无化解方案 | →阻断—若不→停→等待→每年核查 |
| 🔴 | 评级不达标—AA-/无担保→达不到品种门槛 | →启动担保方案→选≥2家AAA→评估可得→无则停 |
| 🔴 | 担保人不具备担保能力（对外担保/净>120%） | →更换担保人/放弃增值→转PPN等低门槛品种 |
| 🟡 | 对外担保40-60%（不阻断但偏高） | →逐笔审定：①反担保→降低敞口；②被担保方信用；③担保圈即将爆库→🟡转🔴预警 |
| 🟡 | 资产大量受限30-40%（对无担保持有人保护偏薄） | →募集说明风险揭示→与银行交涉解限→评级确保不下调 |
| 🟡 | 评级边缘（刚好AA，一个负面事件→跌至AA-/不达标） | →预判触发→一旦事件→提前启动担保方案→投资者沟通 |
| 🟢 | 行业周期下行（有序下行非自由落体） | →每月监测+DCM市场沟通→确认评级稳定→持续跟踪 |
| 🟢 | 主要客户集中 | →排查top5→确保关键客户稳定→有分散计划→持续跟踪 |
