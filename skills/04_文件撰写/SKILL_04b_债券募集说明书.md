# SKILL_04b：债券募集说明书撰写

## 概述

债券募集说明书是固定收益产品发行的核心披露文件，是监管部门审核的重点，也是投资者定价决策的基础。与股权招股说明书不同，债券募集书的逻辑主线是"偿债能力"而非"成长性"——发审委/交易所/NAFMII的审核人员最关注的是"发行人未来能否按时还本付息"。撰写高质量的募集说明书，既要求对发行条款设计精准把握，也要求对财务数据披露、风险揭示、偿债保障措施等内容有深入理解。

本模块覆盖各品类募集书的结构对比、章节撰写要点、监管审核常见意见、以及发行条款设计方法。对于更深层次的文件撰写（如特定行业模板、历史数据比对、财务模型论证），可调用 **bond-documenter** 技能。

## 二、公募公司债募集书章节详解

以下以一般公募公司债为例，逐章说明撰写要点：

### 2.1 重大事项提示（撰写篇幅：2-3页）
**要求：** 突出与本次债券相关的核心风险、特殊条款、关键财务指标变动
**审核热点：**
- 必须独立成章，不能用"详见某节"替代实质性揭示
- 风险揭示需定量（如"EBITDA利息保障倍数从5.2x降至3.1x"），不得仅写"存在风险"
**常见失误：**
- 复制招股说明书的风险因素章节，没有根据债券特点重新组织
- 把非重大事项也列入，导致重点不突出

### 2.2 风险因素（撰写篇幅：15-25页）
**要求：** 至少覆盖：①与发行人相关的风险（行业、经营、财务、管理）；②与本次债券相关的风险（利率、流动性、偿付、增信失败）
**审核热点：** 风险揭示是否"充分+具体+定量"，行业特性风险是否体现
**常见失误：** 财务风险部分只罗列比率，不分析变动原因和趋势

### 2.3 发行人基本情况（撰写篇幅：40-60页）
**要求：** 包括历史沿革、控股股东/实际控制人、组织架构、主要业务板块、行业地位、主要经营数据
**审核热点：**
- 关联交易的定价公允性和占比
- 行业竞争优势的论述是否夸大（如自称为"行业龙头"但市占不超5%）
**常见失误：** 重复年报内容但未从债权人角度重述（如应侧重"稳定现金流来源"而非"收入增长率"）

### 2.4 财务会计信息（撰写篇幅：30-50页）
**要求：** 近3年及一期合并+母公司报表、主要财务指标、管理层讨论与分析
**审核热点：**
- 资产负债率/有息负债/流动比率等偿债指标的纵向变动趋势
- 受限资产占总资产比例（超过30%需特别说明）
- 对外担保占净资产比例（超过20%需重点分析或有负债风险）
- 非经常性损益对利润的贡献度（如扣非净利润为负则严重负面）
**常见失误：** 忽略母公司报表（银行间/交易所均要求合并+母公司双重呈现）

### 2.5 募集资金运用（撰写篇幅：5-10页）
**要求：** 明确的募集资金使用计划、资金管理安排、对财务状况的影响
**审核热点：**
- 偿还到期债务：需列出拟偿还债务的具体信息（债权人、金额、到期日）
- 补充流动资金：需测算流动资金缺口（银保监"流动资金贷款管理办法"口径）
- 不可用于权益性投资、金融资产、房地产开发（非地产企业）、购买理财产品

### 2.6 偿债计划及保障措施（撰写篇幅：8-15页）
**要求：**
- ①偿债资金来源（经营活动现金流、投资活动现金流、外部融资能力等）
- ②应急保障措施（银行授信额度、可快速变现资产、股东支持等）
- ③偿债资金专户管理安排
**审核热点：** 保障措施是否"可执行"而非"原则性表述"；如声称"银行授信充裕"，需提供具体银行和额度

## 三、MTN募集说明书特殊要求

### 3.1 NAFMII表格体系
NAFMII采用"表格体系"（MB、MS、R、P系列），不同表格对应不同发行方式和注册阶段：
- **MB表：** 中期票据募集说明书（含永续中票、并购中票等衍生品种）
- **MS表：** 资产支持票据募集说明书
- **R表：** 公开发行注册报告
- **P表：** 定向发行注册报告

不同表格有固定模板和必填信息点，不得擅自增减或调整顺序。NAFMII定期更新表格版本，撰写前需确认使用最新版。

### 3.2 DFI/TDFI 统一注册体系
- **DFI（统一注册）：** 适用于符合条件的成熟企业（一般AAA/优质AA+），一次注册后2年内可自主选择品种分期发行
- **TDFI（储架发行）：** 更高层级的统一注册，发行前无需再向NAFMII备案
- 募集说明书在DFI框架下提供了"基募集书"和"各期发行文件"两个层次，每期发行前需补充当期募集说明书（通常是基募集书的更新+当期特别信息）

### 3.3 MTN募集书不同于公司债的关键差异
| 要素 | MTN | 公募公司债 |
|------|-----|----------|
| 募集书长度 | 60-100页 | 150-250页 |
| 评级报告 | 另附，不在募集书正文中 | 作为附件或正文摘要 |
| 律师意见 | 另附法律意见书 | 作为附件集中呈现 |
| 财务数据年限 | 近2年（实质审核会看3年） | 近3年及一期 |
| 募集资金用途披露 | 中度详细 | 高度详细 |
| 参考模板 | NAFMII表格体系 | 交易所+《公司债管理办法》 |

## 四、企业债募集书特殊要求

### 4.1 与项目审批的联动
企业债募集说明书的最大特殊性在于"项目导向"：
- 募集书中必须包含完整的**项目可行性分析**（建设内容、投资概算、审批文号、建设周期、预期收益）
- 需附项目审批/核准/备案文件（发改、规划、土地、环保等）
- 项目总投资与募集资金使用比例需合规（债券资金不超过项目总投资的70%一般项目、60%保障房等，具体看项目类型）

### 4.2 NDRC专项债品种
NDRC鼓励的专项品种各有不同募集书要求：
- **城市停车场专项债：** 需列明停车场类型（公共/配建）、泊位数、收费定价依据
- **地下综合管廊专项债：** 需附入廊收费标准及管线单位入廊协议
- **养老产业专项债：** 需说明服务内容、床位数、收费模式
- **双创孵化专项债：** 更注重入驻企业数和孵化成功率

### 4.3 企业债募集书常见特殊章节
- **第十六章/专项偿债计划：** 详细说明项目的自身偿债资金来源（项目运营收入、土地出让返还、政府补贴等）
- **担保增信（如适用）：** 第三方担保或资产抵押需包含完整信息
- **社保基金/养老基金投资信息**（如适用）

## 五、ABS/ABN募集书特殊要求

### 5.1 资产池详情披露
- **逐笔/分层披露：** 基础资产清单（借款合同编号、借据号、本金余额、利率、到期日、担保方式、五级分类等）
- **资产池统计特征：** 加权平均利率、加权平均剩余期限、地域分布、行业分布、单户集中度
- **影子评级/信用质量分布**

### 5.2 现金流瀑布（Waterfall）
- 税收/费用 → 优先A级利息 → 优先B级利息 → 优先A级本金 → 优先B级本金 → 次级（典型瀑布结构）
- 需与分配顺序条款一致（避免优先/次级之间的歧义）
- 需展示正常情景+压力情景下不同分层可获得的现金流

### 5.3 信用增级机制
| 增级类型 | 内容 | 撰写要点 |
|---------|------|---------|
| 优先/次级分层 | 内部增信，次级档为优先档提供保护 | 披露分层比例、偿付顺序、是否可向下现金流 |
| 超额抵押 | 基础资产本金超过发行金额 | 披露超额抵押比例、释放条件 |
| 差额支付承诺 | 原始权益人/关联方差额补足 | 需披露差额补足方的财务状况和履约能力 |
| 外部担保/保险 | 第三方担保或保险 | 措辞需与信用评级假设一致 |
| 流动性支持 | 流动性提供方承诺 | 区分流动性支持（非信用）与信用支持 |

### 5.4 现金流预测与压力测试
- 模型的假设前提（早偿率、违约率、回收率、利率假设）
- AAAsf评级对应的压力倍数
- 敏感性分析（上述假设变化对现金流的影响）

## 六、债券募集书常见审核意见（30条精选）

| 序号 | 问题类别 | 典型审核意见 | 修改方法 |
|------|---------|------------|---------|
| 1 | 风险因素 | "风险因素不够具体，请结合发行人实际进行定性和定量分析" | 将行业通用风险替换为发行人特有数据 |
| 2 | 风险因素 | "未充分揭示行业周期性风险" | 补充近3年行业景气度变化数据 |
| 3 | 风险因素 | "利率风险仅以'基准利率可能上升'归因过于简略" | 测算不同利率情景下利息负担变化 |
| 4 | 偿债计划 | "偿债资金来源分析不够充分，请量化各来源的可实现性" | 补充经营活动现金流趋势+压力测试 |
| 5 | 偿债计划 | "将再融资作为主要偿债来源，未做压力情景分析" | 增加再融资失败时的替代来源分析 |
| 6 | 财务分析 | "请说明应收账款周转率下降的原因及风险" | 拆解客商结构变化、账龄迁移等因素 |
| 7 | 财务分析 | "受限资产金额较大，请说明对偿债能力的影响" | 在偿债保障措施部分专项分析 |
| 8 | 财务分析 | "请解释经营活动现金流为负的原因及改善措施" | 结合在手订单/合同负债等前瞻指标 |
| 9 | 关联交易 | "请充分披露关联交易的定价公允性" | 提供第三方可比交易/评估报告 |
| 10 | 关联交易 | "关联方占款较高，请说明回收风险" | 列表债权人、金额、账龄、坏账计提 |
| 11 | 对外担保 | "对外担保规模较大，请逐笔披露并分析代偿风险" | 按被担保方评级和经营状况分类 |
| 12 | 行业分析 | "行业竞争分析不够深入，请补充行业进入壁垒分析" | 区分政策/资金/技术/客户转换成本 |
| 13 | 募集资金 | "流动资金缺口测算不够详实，请按照监管口径重新测算" | 用银保监公式（营运资金量-自有资金-现有贷款） |
| 14 | 募集资金 | "募集资金用于偿还债务的，请详细列示拟偿还债务清单" | 债权人+金额+利率+到期日+是否关联 |
| 15 | 增信措施 | "请说明增信方的履约能力和信用评级" | 提供增信方最近2年财务数据+评级报告 |
| 16 | 公司治理 | "请披露近3年董事、高管变动情况及对公司的影响" | 列表变动人员/时间/原因/后续稳定性 |
| 17 | 诉讼仲裁 | "请更新截至目前尚未了结的重大诉讼案件" | 更新至最近日期并分析判决不利情景 |
| 18 | 合规问题 | "报告期内是否有重大违法违规被处罚" | 自行梳理并主动披露、做有利性分析 |
| 19 | 评级调整 | "请关注近期是否有评级调整或负面展望" | 如是则需重点解释并作投资者警示 |
| 20 | 交叉违约 | "发行条款中交叉违约触发情形是否充分披露" | 在募集书"重要条款"处完整呈现 |
| 21 | 持有人会议 | "持有人会议召开情形和表决机制需完整列举" | 按照《持有人会议规则》逐条列出 |
| 22 | 信息披露 | "请明确信息披露的具体方式和频率" | 区分定期报告（年报/半年报）+临时报告触发事件 |
| 23 | 税收 | "投资者税负分析是否覆盖各类型投资者的纳税义务" | 企业投资者 vs 个人投资者 vs 境外投资者 |
| 24 | 其他中介 | "请补充债券评级报告全文、律师法律意见书、审计报告" | 作为附件或备查文件 |
| 25 | 数据一致性 | "募集说明书中的财务数据与审计报告不一致" | 核对全部数字与审计报告打勾 |
| 26 | 数据一致性 | "募集说明书中的关联方范围与法律意见书不一致" | 以法律意见书为准统一口径 |
| 27 | 发行条款 | "回售/赎回/利率调整等条款的设置是否与中国法律一致" | 律师补充出具专项法律意见 |
| 28 | 特定行业 | "请补充煤炭/钢铁/水泥等过剩产能行业合规证明文件" | 提供发改委/工信部相关批文 |
| 29 | 城市建设 | "城投企业请说明是否涉及隐瞒债务及政府隐性担保" | 律师+会计师分别出具专项说明 |
| 30 | 信息披露 | "请补充英文名称/境外信息披露（如面向境外投资者）" | 增加英文全文或英文摘要 |

## 七、发行条款设计

### 7.1 债券基本要素设计清单

| 要素 | 选择/设计要点 | 实务范围 |
|------|------------|---------|
| 债券期限 | 匹配资金用途，关注再融资定价窗口分布 | 一般3Y为主流，5Y、7Y也有较多发行 |
| 票面利率类型 | 固定/浮动（SHIBOR/LPR/国债为锚） | 中国信用债几乎全是固定利率 |
| 付息频率 | 按年/半年/按季 | 以按年为主（大型央企有按月案例） |
| 是否含权 | 回售权(put)、赎回权(call)、调整票面利率 | 含权债给予投资者灵活性 |
| 赎回条款 | 发行人是否有权提前赎回，赎回价格和时间 | 含Call的债券一般溢价赎回(如105→100) |
| 回售条款 | 投资者是否有权提前回售，回售时间窗口 | 一般3Y+2结构在3年末设回售权 |
| 利率调整选择权 | 发行人可在约定时点调整后续票面利率 | 常在回售日配套使用（如不行使上调利率） |
| 交叉违约条款 | 触发情形（如其他债务逾期≥5000万）、宽限期、救济措施 | 保护投资者的重要条款 |
| 事先承诺/财务指标承诺 | 资产负债率上限、EBITDA利息倍数下限 | 部分企业债/MTN有承诺 |
| 控制权变更条款 | 控股股东/实际控制人变更触发保护 | 上市公司债多用 |

### 7.2 "3+2"结构设计要点
最常见的含权结构。假设债券期限5年（3+2），在第3年末：
- 投资者持有**回售权（Put）**：可在3年末选择将债券按面值回售给发行人
- 发行人持有**调整票面利率权**：可在3年末调整剩余2年的票面利率（上调/下调）

**设计战术：**
- 若预期利率下行 → 建议设计为"利率可下调"，以降低融资成本
- 若预期利率上行或信用分化 → 应承诺不调降利率或仅在合理区间调整，避免投资者集中回售
- "3+2"结构下投资者按3年期估值定价，初始票面接近3年期中债估值

### 7.3 永续条款设计
- 初始期限一般5+N（即5年后可选续期）
- 利率跳升机制（Step-up）：如第5年末选择不赎回，票面利率上调200-300BPS
- 利息递延：在特定条件下可递延支付利息（但会触发票面利率跳升+限制分红）
- 会计处理考量：需达到"股性"条件才能计入权益（无固定付息义务+自主递延+无交叉违约联动）
