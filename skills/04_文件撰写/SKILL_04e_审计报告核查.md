# SKILL_04e：审计报告核查

## 一、概述

审计报告是IPO/重组项目的财务数据基础。券商作为保荐人/独立财务顾问，虽不直接出具审计报告，但必须对审计报告进行全面核查，确保其作为申报文件基础的可靠性。本技能指导券商如何核查审计报告，而非如何编制审计报告。

## 二、审计意见类型判断

### 2.1 三种基本类型

| 审计意见类型 | 判断标识 | 处理方式 |
|-------------|---------|---------|
| ✅ 标准无保留意见 | "我们认为……公允反映了……" | 直接采用，正常推进 |
| ⚠️ 带强调事项段的无保留意见 | 有强调事项段但无保留结论 | **评估强调事项的性质和影响**，通常需补充核查论证，确保不构成实质性障碍 |
| ❌ 保留意见/否定意见/无法表示意见 | 有保留/否定/无法表示 | **通常无法申报**，需解决导致非标意见的问题后重新审计 |

### 2.2 强调事项段常见内容及应对

| 强调事项 | 常见情景 | 券商应对 |
|---------|---------|---------|
| 持续经营能力存在重大不确定性 | 连续亏损、高负债率、现金流为负 | 重点核查持续经营能力，单独出具专项核查报告 |
| 重大诉讼或仲裁结果不确定 | 未决诉讼 | 评估最坏情况影响，判断是否构成实质性障碍 |
| 对比较财务报表审计意见不同于本期的说明 | 更换会计师事务所 | 确保前任审计无遗留问题 |

## 三、关键审计事项核查

### 3.1 核查思路

阅读审计报告中的"关键审计事项"段落，理解审计师认定高风险领域的理由、审计师应对程序、审计发现/结论。

### 3.2 券商应对

对于每项关键审计事项，券商需：
- 与审计师沟通，了解审计程序执行细节
- 评估该事项是否与尽职调查发现一致
- 如属于高风险事项，补充券商自身的核查程序
- 在招股书中对应章节充分披露

## 四、财务报表核心关注点

### 4.1 必备条件核查

| 核查项目 | 达标标准 | 不达标处理 |
|---------|---------|-----------|
| 最近一年末净资产 | ≥3000万元（主板/中小板）/ ≥2000万元（创业板/科创版） | 不满足条件，暂不能申报 |
| 最近两年净利润（扣非孰低） | 主板：≥3000万累计 / 创业板：≥5000万累计 / 科创版：最近一年≥盈利或符合市值+营收指标 | 逐一核实 |
| 最近一年净利润 | 主板：≥3000万 / 创业板：≥盈利 / 科创版：视市值标准 | 逐一核实 |
| 最近一年经营活动现金流净额 | 主板：≥3000万 或 最近三年营收累计≥3亿 / 创业板/科创版：无硬性要求 | 如不符合第一项则核查第二项 |
| 最近一年营业收入 | 主板：≥3000万 / 创业板：≥1亿 / 科创版：视市值标准 | 逐一核实 |
| 无形资产占净资产比例 | ≤20%（主板）/ 科创版无此限制 | 测算，计算口径：扣除土地使用权、采矿权等后 |

### 4.2 资产重组影响核查

如果报告期内存在同一控制下企业合并（业务重组），需核查：
- 是否触发主营业务变更认定（重组前后主营业务是否相同）
- 重组前后运行时间是否满足要求（>24个月=不触发，否则需运行满24个月或更长时间）
- 被重组方资产/收入/利润占比及运行时间矩阵

## 五、会计政策一致性核查

### 5.1 重点关注的会计政策

| 会计政策 | 关键核查点 |
|---------|-----------|
| **收入确认** | 各业务模式收入确认时点是否一致，与业务合同条款是否匹配，是否与同行业可比 |
| **应收账款坏账准备** | 计提比例是否与同行业可比，单独计提合理性，期后回收情况 |
| **存货跌价准备** | 库龄结构，可变现净值测算依据，同行业计提比例对比 |
| **固定资产折旧** | 折旧年限、残值率是否与同行业可比 |
| **无形资产摊销** | 摊销年限合理性 |
| **股份支付** | 公允价值确定依据，费用分摊期间，是否应确认而漏确认 |
| **研发支出资本化** | 资本化条件是否满足（5条条件逐一比对），资本化时点是否清晰 |
| **政府补助** | 划分标准（与资产相关/与收益相关），摊销期间合理性 |

### 5.2 跨期一致性

- 会计政策在三年一期是否保持一贯性
- 如有变更，变更原因是否合理，影响是否定量说明
- 是否存在利用会计政策变更调节利润的嫌疑

## 六、关联交易披露完整性核查

### 6.1 关联交易完整性验证

比对审计报告关联交易披露与券商尽调底稿的关联方清单：
- 是否存在审计报告有但券商未识别的关联方
- 是否存在券商识别但审计报告未涵盖的关联交易
- 关联交易定价是否已审计验证（穿行测试/函证）

### 6.2 关联资金往来核查

- 关联方资金拆借的发生额、余额、利率、还款情况
- 是否已全部清理（申报基准日原则上不得存在关联方资金占用）
- 资金占用费的收取情况

## 七、期后事项核查

### 7.1 核查期间

- 审计报告日（资产负债表日后事项审计截止日）至申报前的期间
- 重点关注：重大交易、新增重大合同、诉讼进展、资产重组、分红等

### 7.2 核查清单

- [ ] 期后是否有重大采购/销售/投资
- [ ] 期后是否有权益分派（分红/转增）
- [ ] 期后是否有重大诉讼或仲裁的新进展
- [ ] 期后是否有重大资产变化
- [ ] 期后经营数据是否与预期/上年同期有重大偏差
- [ ] 期后财务报表是否已获得（如申报时最新一期已到期后）

## 八、与尽调发现的交叉核对

### 8.1 尽调→审计报告数据印证

| 尽调发现 | 审计报告对应验证 |
|---------|----------------|
| 大额银行存款 | 银行函证结果 |
| 应收账款金额 | 应收账款函证结果 |
| 存货金额 | 存货监盘结果 |
| 销售收入 | 收入穿行测试 |
| 采购额 | 供应商函证 |
| 银行借款 | 银行函证或银行对账单 |
| 关联交易 | 关联交易审计程序 |

### 8.2 差异处理

如果尽调发现与审计报告数据存在差异：
1. 与审计师沟通差异原因
2. 若为审计师遗漏，要求补充审计程序
3. 若为尽调错误，修正券商底稿
4. 差异无法消除的，评估影响并决定是否暂缓申报
