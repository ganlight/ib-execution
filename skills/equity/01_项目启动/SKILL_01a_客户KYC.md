# SKILL_01a：客户KYC与反洗钱

## 概述

KYC（Know Your Customer）与反洗钱（AML）是投资银行业务的前置合规要求。根据中国人民银行《金融机构客户身份识别和客户身份资料及交易记录保存管理办法》、证监会《证券公司反洗钱工作指引》，投行须在建立业务关系时完成客户身份识别、风险等级划分，并持续监控。

---

## 一、受益所有人识别

### 1.1 穿透识别原则

必须**穿透至自然人**或**国资委**级别，不得以持股公司/基金/信托等作为最终受益所有人。

```
层级示例：
  自然人A（实际控制人）          ← 最终受益所有人
        │ 51%
  控股公司B（BVI/开曼）
        │ 100%
  香港公司C
        │ 100%
  境内外商独资企业D（WFOE）
        │ 100%
  发行人（拟上市公司）          ← KYC对象
```

### 1.2 受益所有人判定标准

| 类别 | 判定标准 | 识别依据 |
|------|----------|----------|
| 直接或间接持股≥25% | 穿透后合计持股达到25%的自然人 | 股东名册、合伙协议、信托契约 |
| 通过其他方式控制 | 表决权、协议控制(VIE)、人事任免权 | 公司章程、一致行动协议、VIE协议 |
| 高级管理人员 | 无≥25%股东时，识别法定代表人/实际控制人/董事/高管 | 工商登记、公司决议 |

### 1.3 特殊结构识别

| 结构类型 | 要点 | 额外文件要求 |
|----------|------|-------------|
| VIE架构 | 识别WFOE股东、VIE协议签署方 | VIE全套协议、开曼股东名册 |
| 信托持股 | 识别委托人/受益人/保护人 | 信托契约、信托声明书 |
| 合伙企业 | 识别GP（无限责任）、LP（≥25%） | 合伙协议、合伙人名录 |
| 国资企业 | 穿透至国资委/财政厅 | 国有产权登记证 |
| 员工持股平台 | 识别管理人+≥25%间接持有人 | ESOP计划文件、持股名单 |

## 二、负面清单筛查

### 2.1 制裁名单筛查

必须对照以下清单进行逐一核查：

| 清单名称 | 发布机构 | 覆盖范围 |
|----------|----------|----------|
| 联合国安理会制裁名单 | UN Security Council | 全球 |
| OFAC SDN List | 美国财政部 | 美国管辖范围 |
| 中国公安部反恐名单 | 公安部 | 中国 |
| 外交部制裁名单 | 外交部 | 中国 |
| EU Sanctions List | 欧盟 | 欧盟管辖范围 |
| 世界银行处罚名单 | World Bank | 全球 |

### 2.2 PEP（政治公众人物）筛查

- **国内PEP**：现任/曾任副处级以上党政领导干部、国企负责人的本人及近亲属
- **国际PEP**：外国政要、国际组织高级管理人员
- **PEP离职后敏感期**：国内离职后3年，国际离职后1年（参考FATF建议）
- **近亲属范围**：配偶、父母、子女、兄弟姐妹

### 2.3 负面媒体筛查 (Adverse Media)

| 筛查类别 | 关键词示例 | 风险等级 |
|----------|------------|----------|
| 洗钱/恐怖融资 | 洗钱、非法集资、地下钱庄 | 🔴 高 |
| 腐败/贿赂 | 行贿、贪污、利益输送 | 🔴 高 |
| 欺诈/违规 | 财务造假、信息披露违规 | 🔴 高 |
| 环境/社会 | 重大环境污染、安全事故 | 🟡 中 |
| 税务违规 | 逃税、虚开发票 | 🟡 中 |
| 诉讼/争议 | 重大合同纠纷、知识产权侵权 | 🟢 低（视情况升级） |

## 三、资金来源核查

### 3.1 核查要点

| 资金来源类型 | 核查要求 | 风险提示 |
|--------------|----------|----------|
| 经营积累 | 审计报告验证收入/利润匹配性 | 远高于行业平均利润率需解释 |
| 股权投资 | 追溯投资者资金来源 | 循环出资、委托代持 |
| 银行借款 | 核查贷款用途合规性 | 流动资金贷款用于股权投资 |
| 资产转让 | 核查资产来源及定价公允性 | 关联方非公允价格转让 |
| 遗产/赠与 | 核查赠与方资金来源+关系 | 无亲属关系的巨额赠与 |

### 3.2 出资能力交叉验证

- 注册资本实缴：银行流水/验资报告 + 出资人纳税证明
- 股权受让款：股权转让协议 + 付款凭证 + 转让方完税证明
- 增资款：增资协议 + 进账单 + 评估报告（非货币出资）

## 四、风险等级划分

### 4.1 评分维度

| 评分维度 | 权重 | 低风险(1分) | 中风险(3分) | 高风险(5分) |
|----------|------|-------------|-------------|-------------|
| 客户类型 | 20% | 国有控股上市公司 | 民营非上市 | 海外避税地注册 |
| 股权结构 | 20% | 简单清晰、自然人持股 | 多层持股、存在境外层 | VIE/信托/代持 |
| 业务性质 | 15% | 传统制造业 | 一般服务业 | 现金密集型/跨境 |
| 地域因素 | 15% | 一线城市 | 二三线城市 | FATF高风险地区 |
| 资金来源 | 15% | 经营积累为主 | 含外部融资 | 来源不清/境外资金 |
| PEP关联 | 15% | 无关联 | 间接关联 | 直接关联 |

### 4.2 风险等级与措施

| 风险等级 | 总分 | 持续尽调周期 | 审批层级 | 额外措施 |
|----------|------|-------------|----------|----------|
| 低风险 | <1.8 | 3年更新一次 | 合规部门 | 标准DD |
| 中风险 | 1.8-3.0 | 2年更新一次 | 合规总监 | 强化客户背景调查 |
| 高风险 | >3.0 | 每年更新 | 高级管理层 | EDD+持续交易监控 |

## 五、增强尽调（EDD）触发条件

以下任一条件触发EDD：

1. 风险评级为"高风险"
2. 客户或受益所有人出现在负面媒体（Adverse Media）
3. 客户来自FATF认定的高风险/不合作司法管辖区
4. 交易涉及现金密集型行业（餐饮、零售、博彩）
5. 受益所有人身份存疑或拒绝配合识别
6. 资金来源无法合理解释
7. 客户被列入制裁名单（→直接拒绝业务关系）
8. 客户拒绝提供KYC必要资料

## 七、红旗警示指标（Red Flags）

### 🔴 一票否决类
- [ ] 客户/实控人在联合国/OFAC制裁名单
- [ ] 客户/实控人涉及恐怖融资
- [ ] 存在明确的洗钱犯罪记录
- [ ] 客户拒绝配合KYC/AML审查
- [ ] 资金来源与毒品/走私/腐败明确相关

### 🟡 需EDD及高管审批类
- [ ] 实控人为PEP或其近亲属
- [ ] 股权结构异常复杂（>5层），且有避税地实体
- [ ] 注册资本与经营规模明显不匹配
- [ ] 存在匿名/代持股东，且无法穿透识别
- [ ] 大额现金交易占营业收入比例>20%
- [ ] 实控人/主要股东有重大诉讼或行政处罚
- [ ] 频繁变更股东/实际控制人（3年内>2次）
- [ ] 注册地址与实际经营地严重不符

### 🟢 低风险但需记录类
- [ ] 资金来源涉及境外（但可追溯）
- [ ] 有历史股东纠纷但已解决
- [ ] 关联交易占比较高但价格公允

## 八、审批与记录保存

### 8.1 审批流程

```
项目组提交KYC材料 → 合规部初步审查 → 负面清单系统筛查 
    ↓ (通过)                                  ↓ (命中)
客户风险评级        ←──────────────── 合规部人工复核
    ↓                                      ↓
中低风险: 合规部审批             高风险/PEP: 高管层审批
    ↓                                      ↓
进入项目       ←────────────────    批准/拒绝
```

### 8.2 记录保存要求

- KYC档案保存期限：业务关系终止后**至少5年**
- 客户身份资料保存期限：业务关系终止后**至少5年**
- 大额交易记录保存期限：交易发生后**至少5年**
- 可疑交易报告保存期限：报告后**至少5年**
