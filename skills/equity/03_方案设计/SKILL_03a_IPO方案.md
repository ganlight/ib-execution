# SKILL_03a：IPO方案设计

## 概述

IPO方案设计是投行项目中最复杂的结构化工作之一，需要在监管合规框架内，为发行人设计最优的上市板块、发行结构、重组方案、募集资金用途和推进时间表。

---

## 一、板块选择：全板块财务标准对照

### 1.1 主板 — 3套标准

| 标准 | 条件 | 说明 |
|------|------|------|
| 标准一 | 最近3年净利润均为正 + 累计≥1.5亿元 + 最近1年≥6000万元 | 净利润 = 扣非前后孰低 |
| 标准二 | 最近3年经营活动现金流累计≥1亿元 或 营业收入累计≥10亿元 | 现金流/营收二选一 |
| 标准三 | 预计市值≥80亿元 + 最近1年营收≥8亿元 | 市值驱动型 |

### 1.2 科创板 — 5套标准

| 标准 | 条件 | 说明 |
|------|------|------|
| 标准一 | 预计市值≥10亿元 + 近2年净利润均为正+累计≥5000万元；或预计市值≥10亿元+近1年净利润为正+营收≥1亿元 | 盈利企业 |
| 标准二 | 预计市值≥15亿元 + 近1年营收≥2亿元 + 近3年研发投入占比≥15% | 研发驱动型 |
| 标准三 | 预计市值≥20亿元 + 近1年营收≥3亿元 + 近3年经营活动现金流累计≥1亿元 | 现金流驱动型 |
| 标准四 | 预计市值≥30亿元 + 近1年营收≥3亿元 | 营收驱动型 |
| 标准五 | 预计市值≥40亿元 + 主要产品经批准+市场空间大 + 医药企业至少1项核心产品完成II期临床 | 医药/科技突破型 |

### 1.3 创业板 — 3套标准

| 标准 | 条件 | 说明 |
|------|------|------|
| 标准一 | 最近2年净利润均为正 + 累计≥5000万元 | 稳定盈利 |
| 标准二 | 预计市值≥10亿元 + 最近1年净利润为正 + 营收≥1亿元 | 成长中盈利企业 |
| 标准三 | 预计市值≥50亿元 + 最近1年营收≥3亿元 | 市值驱动型（可未盈利） |

### 1.4 北交所 — 4套标准

| 标准 | 条件 | 说明 |
|------|------|------|
| 标准一 | 预计市值≥2亿元 + 近2年净利润均≥1500万元 且加权ROE≥8% 或 近1年净利润≥2500万元 且加权ROE≥8% | 盈利型 |
| 标准二 | 预计市值≥4亿元 + 近2年营收平均≥1亿元 + 营收增长率≥30% + 近1年经营活动现金流为正 | 成长型 |
| 标准三 | 预计市值≥8亿元 + 近1年营收≥2亿元 + 近2年研发投入合计占营收≥8% | 研发型 |
| 标准四 | 预计市值≥15亿元 + 近2年研发投入合计≥5000万元 | 市值+研发型 |

### 1.5 板块条件快速比对表

| 条件/板块 | 主板 | 科创板 | 创业板 | 北交所 |
|-----------|------|--------|--------|--------|
| 最低市值 | 80亿(标准三) | 10亿 | 10亿 | 2亿 |
| 最高利润标准 | 1.5亿/3年 | 5000万/2年 | 5000万/2年 | 1500万/年 |
| 允许未盈利 | ❌ | ✅ | ✅(标准三) | ❌ |
| 行业限制 | 无 | 科创6大领域 | 三创四新 | 创新型中小企业 |
| 科创属性指标 | 无 | ✅ 必须 | 无 | 无 |

**科创板科创属性指标（必须满足）**：
- 近3年研发投入占比≥5%（或累计≥6000万元）
- 研发人员占比≥10%
- 发明专利≥5项（形成主营业务收入）
- 近3年营收复合增长率≥20%（或近1年营收≥3亿元）

## 二、发行结构设计

### 2.1 发行规模计算

```
新增股份数量 = 发行后总股本 × 公开发行比例

公开发行比例要求：
- 主板/科创板/创业板：
  - 总股本<4亿股 → 公开发行≥25%
  - 总股本≥4亿股 → 公开发行≥10%
- 北交所：公开发行≥25%（总股本<4亿）或 ≥10%（总股本≥4亿）
```

### 2.2 定价方式

| 定价方式 | 适用板块 | 特点 |
|----------|----------|------|
| 询价发行 | 主板/科创/创业 | 向网下投资者询价确定，市场定价 |
| 直接定价 | 所有板块 | 发行人和主承销商直接确定（一般<2000万股） |
| 竞价发行 | 均可但不常用 | 投资者竞价 |
| 累计投标询价 | 港股 | 投资者报量报价结合 |

### 2.3 战略配售与超额配售

| 机制 | 说明 | 数量限制 |
|------|------|----------|
| 战略配售 | 锁定≥12个月的战略投资者配售 | 首次≤30%（>4亿股≤50%） |
| 超额配售（绿鞋）| 主承销商超额配售机制，稳定股价 | ≤初始发行数量15% |
| 高管/员工参与 | 高管和核心员工资管计划参与战略配售 | ≤首次发行数量10% |

### 2.4 战略投资者选择标准

| 类型 | 选择偏好 | 注意事项 |
|------|----------|----------|
| 产业投资者 | 同行业或产业链上下游龙头企业 | 可能与发行人存在竞争/关联 |
| 财务投资者 | 知名PE/VC、国资基金、保险资金 | 关注锁定期和退出路径 |
| 上下游伙伴 | 有助于业务协同 | 可能构成关联交易或依赖 |
| 基石投资者（港股） | 知名机构/主权基金/产业龙头 | 锁定期6个月 |

## 三、Pre-IPO重组

### 3.1 股权结构清理

| 清理事项 | 具体要求 |
|----------|----------|
| 股权代持 | 全部清理还原 ➔ 必须穿透至自然人 |
| 股权纠纷 | 全部解决 ➔ 取得各方书面确认 |
| 股东人数穿透 | 穿透后≤200人（>200人需纳入非上市公众公司监管） |
| 三类股东 | 契约型基金/信托计划/资管计划需符合监管要求 |
| 对赌协议 | 原则上清理（科创板/创业板可保留部分非业绩对赌） |

### 3.2 ESOP方案设计

| 要素 | 设计要点 |
|------|----------|
| 总量 | ≤上市前总股本的10%-15% |
| 个量 | 单人≤总量的25%或上市前总股本的1% |
| 授予价格 | ≥最近1年经审计的每股净资产 |
| 锁定期 | ≥上市后12个月（离职需加速到期） |
| 激励对象 | 核心员工、技术人员、管理人员 |
| 平台形式 | 有限合伙企业（GP通常为实控人或管理人） |

### 3.3 业务重组

- 剥离非主营业务（如房地产业务、类金融业务）
- 注入同一控制下的同业资产（消除同业竞争）
- 清理不必要的子公司/分公司
- 确认重组是否适用同一控制下企业合并

## 四、募集资金用途设计

### 4.1 募投项目筛选标准

| 标准 | 要求 |
|------|------|
| 政策合规 | 符合国家产业政策、环保政策 |
| 业务相关 | 与主营业务紧密相关 |
| 证照齐全 | 已取得用地/环评/能评等前置审批 |
| 可行性 | 市场、技术、团队、资金均有保障 |
| 效益可测 | 有合理的效益测算（IRR/回收期等） |
| 金额合理 | 不超实际需求（避免过度融资） |

### 4.2 募集资金分配（参考比例）

| 用途 | 占比参考 |
|------|----------|
| 产能扩建/新建项目 | 50-70% |
| 研发中心建设 | 10-20% |
| 补充流动资金 | ≤30% |
| 偿还银行贷款 | ≤20% |
| 收购资产/股权 | 交易对价的100% |

## 五、IPO时间表

### 5.1 各板块IPO时间轴

| 阶段 | 主板 | 科创板 | 创业板 | 北交所 |
|------|------|--------|--------|--------|
| 尽职调查 | 3-4月 | 2-3月 | 2-4月 | 1-2月 |
| 辅导备案及验收 | 3-6月 | 3-6月（或豁免） | 3-6月 | 无需辅导 |
| 申报材料制作 | 1-2月 | 1-2月 | 1-2月 | 1月 |
| 在审/问询 | 6-12月 | 3-6月 | 6-12月 | 2-3月 |
| 注册/核准 | 1-2月 | 1-2月 | 1-2月 | 15-20工作日 |
| 发行上市 | 1-2月 | 1-2月 | 1-2月 | 1月 |
| **总计** | **12-24月** | **8-14月** | **12-22月** | **5-8月** |

### 5.2 关键里程碑时间逻辑

| 里程碑 | 时间逻辑 |
|--------|----------|
| 审计基准日 | 通常选12月31日或6月30日 |
| 申报受理 | 审计报告有效期6个月（可加期1个月） |
| 首轮问询回复 | 受理后20个工作日内 |
| 上市委审议 | 问询回复满意后安排 |
| 注册生效 | 批文有效期12个月 |
| 发行完成 | 注册生效后12个月内 |
