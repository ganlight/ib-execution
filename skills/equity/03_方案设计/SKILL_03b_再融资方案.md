# SKILL_03b：再融资方案设计

> **适用阶段**：项目承揽 → 方案设计 → 发行执行全流程  
> **核心功能**：帮助投行团队在融资品种确定后，完成产品结构设计、财务测算、募投项目包装及监管沟通策略制定  
> **前置依赖**：已完成融资主体筛选及尽职调查（参考 SKILL_02 相关内容）

## 一、产品选择决策框架

### 1.1 A股再融资产品谱系总览

A股再融资市场经过2020年再融资新政、2023年827新政等制度演进，已形成以定向增发、可转债、配股为核心的三大品种格局，辅以公开增发和优先股等特殊工具。不同品种在审核效率、盈利门槛、稀释程度、股东保护机制等维度存在显著差异，方案设计的第一步在于精准匹配融资需求与产品特性。

**定向增发**是最主流的再融资工具，2023年全市场发行规模超过6000亿元（按发行预案公告计），占全市场再融资规模的65%以上。其核心优势在于灵活性——无硬性财务门槛（主板要求最近3年盈利、创业板无要求、科创板无要求），募资规模可达市值的20%（并购配套融资可达100%），发行价格可在市价或底价之间博弈。定增分为锁价发行和市价发行两种模式，前者以董事会决议公告日为定价基准日，投资者通常获得折扣，但锁定期较长（控股东/实控人36个月，其他投资者18个月）；后者以发行期首日为定价基准日，价格随行就市，锁定期仅6个月。

**可转债**兼具债性保护和股性弹性的特点，是追求"稳中求进"的融资方首选。2023年可转债市场发行规模约2000亿元，其票面利率通常低于1%，大幅降低即期财务费用；转股条款赋予投资者未来以固定价格转为股东的权利，形成隐性看涨期权。可转债的审核周期与定增相当（约3-6个月），但财务门槛明显更高——要求最近3年加权平均ROE不低于6%、现金分红比例不低于30%、累计现金分红不低于年均可分配利润的30%。这一门槛将大量周期性强、波动性大的发行人挡在门外。

**配股**是老股东全员参与的再融资工具，理论上对原股东最为公平（按持股比例认购），但实践中"包销风险"是最大隐患——若老股东认购不足，承销商须全额包销剩余股份。2021-2023年，配股市场明显萎缩，全年发行案例不超过20单，主因是包销风险和窗口指导趋严。配股的财务门槛为最近3年盈利，募资规模上限为股本的30%。

**公开增发**在A股市场已极度边缘化，2020年以来全年案例不超过5单，主要原因在于门槛高（ROE≥6%+现金分红≥30%）、稀释大（无锁定期）、市场反应敏感。仅有少数具备强基本面支撑的蓝筹公司能够成功实施。

```
│       │   │
│       │   └── 否（纯融资需求，希望快速完成）

│           │   └── → 【可考虑】配股（对老股东最公平，但包销风险需预案）
│           │             前提：认购比例需达70%以上，否则承销压力巨大
│           │             注意：配股停牌周期长（5-10个交易日），流动性风险
│           │

│               └── → 优先考虑可转债（市价定增备选）

    ├── 市场情绪高涨 + 估值处于历史中位线以上

    │
    └── 市场情绪低迷 + 估值处于历史低位

            警示：股价过低时，可转债转股价格下修压力增大
```

### 1.4 2023-2025年监管政策动态（方案设计必须关注）

**827新政（2023年8月）** 是本轮再融资监管收紧的标志性文件，核心要点包括：

1. **破发率约束**：连续破发（即发行价格低于上市首日开盘价）的定增项目，监管将启动专项检查；
2. **补流比例窗口指导**：纯补流类定增（无实质募投项目）窗口指导从严，募资中用于"补充流动资金"的比例不得超过拟投资总额的30%；
3. **募投项目真实性**：要求发行人提供募投项目的备案证明、环评批复（必要时），项目需与主业高度相关；
4. **可转债门槛提高**：对近3年累计现金分红低于年均可分配利润30%的企业，窗口指导直接劝退。

**2024年补充要求**：监管层对定增的发行窗口进行动态指导，以下情形将面临审核放缓：
- 发行对象为境外投资者且涉及国家敏感行业；
- 募投项目涉及新增产能且发行人为高能耗行业；
- 发行人最近一期末持有的交易性金融资产占总资产比例超过40%。

## 二、财务条件测算

### 2.1 各品种财务门槛逐项拆解

#### 2.1.1 主板（含中小板）定增

```
财务条件（须同时满足）：
① 最近3个会计年度连续盈利（扣除非经常性损益前后孰低）
② 最近3个会计年度加权平均净资产收益率平均不低于6%
   （扣除非经常性损益后归属于普通股股东的净利润÷加权平均净资产）
③ 最近3年以现金方式累计分配的利润不少于最近3年年均可分配利润的30%
④ 发行价格不低于定价基准日前20个交易日均价的80%

窗口指导（实际掌握标准）：
- ROE计算时，非经常性损益中的"一次性损益"（政府补助、资产处置收益等）
  须从分子分母同时扣除，以确保指标的真实性
- 若最近一年ROE为负，但近3年平均ROE≥6%，理论上可通过，实际上窗口指导会关注
- 存在重大资产重组的企业，备考财务数据也可纳入计算（需经审计）
```

#### 2.1.2 科创板/创业板定增（注册制差异化安排）

```
科创板定增财务条件（简化版）：
① 上市公司最近2年盈利（扣除非经常性损益前后孰低）
② 无ROE硬性要求（但审核中会关注最近2年的加权平均ROE水平）
③ 最近2年现金分红符合公司章程规定

创业板定增财务条件（参照主板，但门槛略低）：
① 最近2年盈利（创业板原要求），注册制改革后与科创板趋同
② 无ROE硬性门槛，但窗口指导意见：最近2年加权平均ROE不宜低于4%
③ 现金分红要求与主板一致（近3年累计30%）

【重要区分】科创板与创业板的差异：
- 科创板：强调"科创属性"，募投项目须与"硬科技"相关
- 创业板：强调"成长性"，对传统行业转型开放度更高
- 两者均适用注册制，审核周期实质相同（3-6个月）
```

#### 2.1.3 可转债（沪深两市统一标准）

```
财务门槛（2024年最新窗口指导口径）：

① 盈利能力：最近3个会计年度连续盈利
② ROE要求：最近3个会计年度加权平均净资产收益率（加权）不低于6%
   计算公式：ROE = 归属于普通股股东的净利润÷加权平均净资产 × 100%
   其中：加权平均净资产 = 年初净资产 + 年末净资产 × 50%（简化算法）
         精确算法需使用365天加权
③ 分红要求：最近3年以现金方式累计分配的利润 ≥ 最近3年年均可分配利润的30%
   特别注意：年均可分配利润 = 近3年归属于母公司净利润之和 ÷ 3
            累计分配利润 = 近3年实际现金分红总额（含税）
④ 发行规模上限 = 最近一期经审计净资产 × 40%
   （金融类企业（如券商、银行）可提高至60%，须经监管部门个案审批）
⑤ 评级要求（窗口指导）：主体评级不低于AA-，债项评级不低于AA
   实务中，AA以下评级的可转债市场接受度极低，发行困难

【实质性障碍】以下情形将导致可转债方案难以通过审核：
- 最近一期末对外担保余额（合并报表口径）超过净资产的50%
- 最近一期末可转债余额已超过净资产的40%（不得重复发行）
- 最近12个月内受到过交易所公开谴责
```

#### 2.1.4 配股

```
财务门槛（相对宽松）：
① 最近3个会计年度盈利
② 最近3年现金分红比例符合公司章程规定（无统一最低要求）
③ 最近3年财务报告无虚假记载

特殊限制（区别于定增/可转债）：
- 配股拟配售股份数量不超过本次配股前股本总额的30%（即30%上限）
- 控股股东须在股东大会召开前公开承诺认配数量（或书面确认认购上限）
- 若控股股东不参与认购或认购不足，承销商须包销余额
  → 包销风险是配股失败的最主要因素
```

### 2.2 募资规模精准测算

#### 2.2.1 定增募资规模计算模型

```
标准定增（非配套融资）上限公式：

┌─────────────────────────────────────────────────────────┐
│                                                         │
│   MAX_定增 = MIN ( 拟投资总额 [A] ， 基准股本 × 20% [B] )  │
│                                                         │
│   其中：                                                │
│   [A] 拟投资总额 = 募投项目投资额 + 铺底流动资金 + 发行费用   │
│   [B] 基准股本 = 董事会前1交易日总股本（或经董事会确认的股本） │
│                                                         │
└─────────────────────────────────────────────────────────┘

【实战举例】
某制造业上市公司，总股本8亿股，前20日均价12元/股，市值96亿元。
拟投资3个募投项目，总额18亿元，另有补流3亿元，发行费用0.5亿元。

Step 1：拟投资总额 = 18 + 3 + 0.5 = 21.5亿元
Step 2：基准股本上限 = 8亿 × 20% = 1.6亿股
Step 3：以基准股本计算的募资上限 = 1.6亿 × 12元 = 19.2亿元
Step 4：实际可申报规模 = MIN(21.5, 19.2) = 19.2亿元

→ 该公司最大募资规模为19.2亿元，无法覆盖全部21.5亿元投资需求
→ 解决方案：调整募投结构（减少补流比例）或寻求配套融资例外

【配套融资（并购重组配套定增）特殊规则】

并购配套融资不受20%上限约束：
MAX_配套定增 = 标的资产交易价格（100%）

前提条件：
- 须经证监会并购重组委审核（重大资产重组程序）
- 标的资产须满足相关估值及盈利预测要求
- 发行股份购买资产与配套融资可同步或分步实施
- 分步实施时，配套融资规模受届时市值的20%限制
```

#### 2.2.2 可转债募资规模计算模型

```
可转债发行规模上限公式：

┌─────────────────────────────────────────────────────────┐
│                                                         │
│   MAX_可转债 = 最近一期经审计净资产（合并口径）× 40%         │
│                                                         │
│   其中：                                                │
│   [净资产认定] 以募集说明书公告前最近一期经审计报告为准        │
│   [合并口径] 包括母公司及全部纳入合并报表的子公司             │
│   [金融企业特例] 证券公司、银行等金融机构上限为60%           │
│                                                         │
└─────────────────────────────────────────────────────────┘

【实战举例】
某主板上市公司，最近一期经审计净资产（合并）为45亿元。
非金融企业，可转债上限 = 45 × 40% = 18亿元

若拟融资总额为20亿元，则：
方案一：分期发行（先发18亿元可转债，剩余2亿元通过银行贷款解决）
方案二：调整为首轮定增（定增无40%净资产限制）
方案三：降低可转债申报规模至18亿元，差额通过银团贷款补足

【净资产的认定细节】
以下项目须从净资产中扣除（不可计入40%计算基数）：
① 其他权益工具（如永续债、优先股）的期末余额
② 少数股东权益（合并报表净资产 ≠ 归母净资产，须特别注意）
③ 计入其他综合收益的累计浮亏（如套期会计下的浮亏）

实务中，建议以内核前最后一次审计报告的归母净资产为计算基准，
并预留5-10%的安全边际（考虑审计调整可能性）。
```

#### 2.2.3 配股募资规模计算模型

```
配股发行规模公式：

┌─────────────────────────────────────────────────────────┐
│                                                         │
│   MAX_配股 = 配股前总股本 × 30% × 配股价格                │
│                                                         │
│   其中：                                                │
│   [配股价格] 通常为股权登记日前收盘价的50%-80%             │
│   [定价原则] 价格须经董事会、股东大会审议，并经律师核查        │
│   [除权计算] 配股完成后，股价将除权，除权参考价计算方式：       │
│             除权参考价 = (股权登记日收盘价 + 配股价格×0.3)    │
│                           ÷ (1 + 0.3)                  │
│                                                         │
└─────────────────────────────────────────────────────────┘

【实战举例】
某上市公司总股本5亿股，股权登记日前收盘价10元/股。
大股东承诺按持股比例60%全额认购，其余股东认购意愿不确定。

方案设计：
假设配股比例30%，配股价格7元/股（收盘价10元的70%）。
可募资总额 = 5亿 × 30% × 7 = 10.5亿元

若所有股东按比例认购（理想情况）：
新增股份 = 5亿 × 30% = 1.5亿股
募集资金 = 10.5亿元
老股东持股比例不变

若认购率仅60%（最坏情况）：
包销风险 = 5亿 × 30% × 40% × 7 = 4.2亿元
承销商须在30日内将包销股份出售或持有为长期股权

【除权效应分析】
配股除权对投资者（非老股东）存在理论上的"损失"：
除权参考价 = (10 + 7×0.3) ÷ 1.3 = 8.54元
若投资者不参与配股（且无法卖出配股权），将承受14.6%的理论损失。
→ 这正是配股"强制老股东认购"机制的设计原理
```

### 2.3 摊薄效应精密测算

#### 2.3.1 摊薄比例核心公式体系

```
【基本摊薄模型】

① 新增股份数 = 募资总额 ÷ 发行价格

② 摊薄比例 = 新增股份数 ÷ (原股本 + 新增股份数)
            = 募资总额 ÷ (原股本 × 发行价格 + 募资总额)

③ 摊薄后ROE（简化计算，未考虑募集资金效益时滞）：
   ROE摊薄后 = ROE原 × (1 - 摊薄比例)
   
④ 更精确的摊薄后ROE：
   ROE摊薄后 = 归属于普通股股东净利润(募后) ÷ 归属于普通股股东净资产(募后)

   其中：
   归属于普通股股东净利润(募后) = 原净利润 + 募集资金年均效益 - 新增利息费用(如有)
   归属于普通股股东净资产(募后) = 原净资产 + 募资净额
```

#### 2.3.2 定增摊薄测算案例

```
【案例：制造业定增摊薄测算】

基本情况：
- 原总股本：10亿股
- 前20日均价：8元/股
- 拟募资总额：16亿元
- 发行价格：6.4元/股（即前20日均价的80%，最低折扣）
- 原净利润：8亿元（假设不增长）
- 原净资产：60亿元
- 原ROE：8÷60 = 13.3%

测算过程：

Step 1：计算新增股份
新增股份 = 16亿 ÷ 6.4元 = 2.5亿股

Step 2：计算摊薄比例
摊薄比例 = 2.5 ÷ (10 + 2.5) = 20.0%

Step 3：计算摊薄后ROE（静态，假设净利润不变）
ROE摊薄后 = 13.3% × (1 - 20%) = 10.67%

Step 4：考虑募集资金效益（假设募投项目IRR=12%，年均新增效益=1.92亿元）
ROE摊薄后(动态) = (8 + 1.92) ÷ (60 + 16 - 0.5) = 9.92÷75.5 = 13.1%
→ 募投项目效益覆盖稀释效应，动态ROE优于原ROE

Step 5：EPS摊薄（假设净利润不变）
原EPS = 8亿 ÷ 10亿股 = 0.80元/股
摊薄后EPS = 8亿 ÷ 12.5亿股 = 0.64元/股
EPS摊薄幅度 = (0.80 - 0.64)÷0.80 = 20.0%（与股份稀释比例一致）

【监管关注阈值】
根据窗口指导，定增完成后摊薄后ROE不宜低于6%（与发行门槛对应），
否则可能被问询"是否损害中小股东利益"。
本案例动态ROE=13.1%，明显超过阈值，审核风险较低。
```

#### 2.3.3 可转债摊薄测算特点

```
可转债的摊薄具有"渐进性"和"或有性"双重特征：

【渐进性】
可转债在转股前不增加股份数，对EPS/ROE的摊薄为零。
转股后，每1张可转债按初始转股价转换为股份：
  转换股份数 = 可转债面值(100元) ÷ 初始转股价格

【或有性】
若股价长期低于转股价（触发下修条款：连续30个交易日中有15个交易日
正股收盘价低于转股价的85%），发行人有权（但非义务）下修转股价。
下修将增加潜在稀释股份数。

【摊薄测算场景】

情景A：可转债未转股（最保守）
- 摊薄比例 = 0
- EPS/ROE不受影响
- 但账面上存在还本付息压力（若可转债规模10亿元，票息1%/年，
  每年财务费用约1000万元，对净利润影响极小）

情景B：可转债在3年后一次性转股（假设转股率100%）
- 假设转股价格 = 募集说明书公告日前20日均价 = 10元/股
- 可转债规模 = 10亿元
- 转股新增股份 = 10亿 ÷ 10元 = 1亿股
- 假设原股本10亿股，摊薄比例 = 1÷(10+1) = 9.1%
- 注意：此时净利润已包含可转债利息税后节省（约800万元/年，3年约2400万元）
  实际EPS稀释程度低于股份比例

情景C：可转债部分转股（最常见，实际转股率约70-80%）
- 测算方法同情景B，但以实际转股比例加权计算摊薄
```

## 三、募投项目设计

### 3.1 募投项目分类体系

#### 3.1.1 五大项目类型详解

**扩产类项目**是再融资募投项目中最常见的类型，适用于行业景气上行、现有产能饱和或即将饱和的企业。审核重点关注：市场需求是否可持续（需第三方可研报告）、现有产能利用率（一般要求70%以上）、募投新增产能的消化路径（新增客户开拓计划、现有客户增量订单）。合规性要求：项目备案（发改委）、环评批复（生态环境厅）、土地使用权证/租赁协议，缺一不可。2024年起，对于高能耗行业（电解铝、钢铁、水泥等）的新增产能募投项目，监管额外要求提供能效评估报告和碳排放测算。

**研发类项目**是科创板和创业板再融资的重要载体，体现"科创属性"和"技术先进性"。审核关注点：研发方向与主业的关联性（反对为研发而研发）、拟购入设备的先进性（须与可比公司对标）、研发成果转化路径（形成什么产品/专利/收入）。常见形式：研发中心建设（场地+设备+人员）、关键核心技术攻关（材料费+设备费+合作费）、中试生产线（介于研发和量产之间）。合规要求：项目备案、科技部门意见（重大专项需科技部意见）。特别注意：研发类项目的经济效益测算相对困难，审核中容易被追问"如何衡量研发投入的回报"。

**补流类项目**（补充流动资金）是最敏感的项目类型，827新政后监管明显趋严。核心逻辑：若发行人账上货币资金充裕（通常以"剔除受限货币资金后的可动用资金"计算），则补流必要性存疑。审核关注点：发行人资金缺口测算依据（需提供详细测算模型）、货币资金真实用途（是否存在资金被关联方占用、大额定期存款、理财产品等受限情形）、补流后对资产负债率的改善效果。合规要求：须在募集说明书中单独披露补流测算依据。

**偿债类项目**（归还银行贷款/有息负债）本质上是一种"以股权融资替代债权融资"的行为，其必要性取决于贷款利率、再融资必要性及发行人的财务结构目标。审核关注点：提前还款是否需要银行出具同意函（须在申报前取得）、还款前后资产负债率变化、是否存在"低息贷款置换为高息股权融资"的逻辑悖论。实务中，若发行人综合资金成本（贷款加权平均利率）明显低于募资成本（定增市盈率倒数的倒数），则偿债类募投的必要性通常被质疑。

**收购类项目**是定增配套融资的核心应用场景，通过发行股份购买资产同时配套融资，实现并购整合。审核关注点：标的资产的评估方法（收益法通常溢价较高，需充分论证）、业绩承诺的合理性和可实现性（通常是3年利润对赌）、协同效应量化分析（收入协同+成本协同，须有测算依据）。合规要求：须完成资产评估备案、审计报告（有效期6个月内）、法律意见书。重大资产重组须经并购重组委审核，审核周期（不含恢复期）约1-2个月。

### 3.2 募投项目设计六项原则

```
【原则一：与主业高度相关】

坚决反对：跨行业扩张、与现有业务无关的"热门概念"募投
监管逻辑：再融资的初心是服务实体经济，不是帮发行人"转型"或"讲故事"
实操标准：募投项目产品/技术与现有主营收入来源的产品/技术大类一致
          即：募投产品类别与现有产品类别重合度>60%

【原则二：具备市场可行性】

必需文件：
① 第三方可行性研究报告（具有工程咨询资质单位出具）
② 市场需求论证（行业协会数据、第三方市场研究报告）
③ 目标客户验证（意向性合同、客户访谈纪要，审核中可能被调取）
④ 同业可比分析（募投单位投资成本/产能对比，须优于或持平行业水平）

【原则三：备案/环评/土地齐全，申报前必须取得】

备案层级：
- 非审批制项目：企业投资项目备案（省/市发改委）
- 审批制项目（涉及政府资金、重大外资等）：可行性研究报告批复
环评批复：
- 编制环评影响报告书：须生态环境厅批复（报告表：市级生态环境局批复）
- 涉及新增污染物排放的，申报前须取得主要污染物排放总量指标
土地使用权：
- 自建厂房须自有土地使用权（证载用途须为工业/仓储用地）
- 租赁场地须提供租赁协议（剩余租赁期须覆盖项目建设期+5年）

【原则四：经济效益可测算】

审核要求：募投项目须提供以下财务指标测算：
① 总投资额（按建筑工程费、设备购置费、安装工程费、其他费用、预备费分项列示）
② 项目建设期（通常不超过24个月，再融资募投项目适用）
③ 达产年收入（须有市场基础，不可随意假设）
④ 达产年毛利率、净利率
⑤ 项目IRR（内部收益率，须大于发行人加权平均资金成本WACC+3%以上）
⑥ 项目回收期（静态回收期通常不超过8年）

【原则五：投资规模合理（与公司体量匹配）】

常见质疑：募投投资额过大，与公司现有资产规模、人员规模、管理能力不匹配
测算标准（经验值）：
- 募投投资额 / 公司总资产 < 50%（超过则需专项论证）
- 募投新增产能 / 公司现有产能 < 100%（翻倍扩产须充分理由）
- 募投投资额 / 公司年收入 < 80%（超过80%须特别关注）

【原则六：不超过24个月建设期】

政策依据：《发行监管问答——关于引导规范上市公司融资行为的监管要求》
          （2020年修订版）
核心要求：募集资金用于补流的部分限18个月内使用，
          用于铺底流动资金的部分在募投项目投产后使用
监管关注：超过24个月建设期的项目，审核中会被要求说明建设期延长的原因
          及是否影响募集资金使用效率
```

### 3.3 补流比例限制深度解析

```
【定增补流限制】
上限：30%（含偿债）
计算基数：拟投资总额 = 募投项目投资总额 + 补充流动资金
即：补流上限 = 拟投资总额 × 30% / 1.3（简化算法）

实战案例：
某公司拟募资10亿元，其中募投项目8亿元，补流2亿元
补流比例 = 2÷10 = 20% → 合规
若补流2.5亿元，比例 = 25% → 合规
若补流3.5亿元，比例 = 35% → 违规，须调整

【可转债补流限制】
上限：30%（含偿债），与定增一致
特别说明：可转债"补充流动资金"项目中，若包含"归还银行借款"，
需在补流总额中合并计算，不能单独享受偿债豁免

【配股补流】
无硬性比例限制
实务中，配股若100%用于补流，审核问询概率较高（需充分论证融资必要性）

【公开增发补流】
无硬性比例限制
但由于公开增发审核难度最高，无真实募投项目的"纯补流"方案几乎无法通过
```

### 3.4 募投项目组合设计范式

```
【范式一：核心扩产 + 研发升级 + 补流】（最常见，制造业企业）
典型结构（募资10亿元）：
- 扩产项目A（主产品技改扩产）：5亿元（占比50%）
- 研发中心项目（智能化升级）：2亿元（占比20%）
- 补充流动资金：2亿元（占比20%）
- 发行费用：1亿元（占比10%，含承销费、律师费、审计费、评估费等）
合规性：补流占比20% < 30%，满足要求

【范式二：并购配套 + 现金支付 + 补流】（定增专用）
典型结构（募资8亿元）：
- 发行股份购买资产对价：5亿元（通过发行股份支付，不占用募资规模）
- 配套融资（现金对价）：4亿元
- 补充流动资金：0.8亿元（占比10%）
- 发行费用：0.2亿元
合规性：配套融资4亿元 + 补流0.8亿元 < 总募资4.8亿元，补流占比16.7%

【范式三：可转债专属——募投+债底保护设计】
可转债的募投项目设计需额外考虑转股可能性下的股权稀释，
通常选择回报周期较长、与主业绑定深的长期项目：
典型结构（可转债10亿元）：
- 产能扩建项目A（3年建设期，IRR=14%）：6亿元
- 研发大楼项目（4年建设期，无直接收入，但提升技术壁垒）：2亿元
- 补充流动资金：1.2亿元（占比12%，低于30%上限）
- 发行费用：0.8亿元
注意：可转债募投项目IRR须高于同评级企业债利率，
否则转股吸引力不足，可能导致转股失败、触发回售条款
```

## 四、定价机制

### 4.1 定向增发定价机制

#### 4.1.1 定价基准日的法律框架

```
定价基准日的三个可选时点及其法律效力：

┌─────────────────────────────────────────────────────────┐
│ 定价基准日选项          │ 发行价格下限          │ 锁定期 │
├─────────────────────────────────────────────────────────┤
│ 董事会决议公告日        │ 基准前20日均价×80%    │ 18/36月│
│ 股东大会决议公告日      │ 基准前20日均价×80%    │ 18/36月│
│ 发行期首日（市价发行） │ 不低于基准前20日均价   │ 6个月  │
└─────────────────────────────────────────────────────────┘

【锁价发行（适用战略投资者）】

定价基准日选择策略：
- 优选董事会决议公告日（可锁定价格，发行窗口可预期）
- 次选股东大会决议公告日（适用于董事会预案后市场下跌的情形）
- 回避发行期首日（不确定性高，不适合锁价发行）

定价折扣博弈：
- 理论上：发行价格 ≥ 基准前20日均价的80%（下限）
- 实务中：发行价格通常在基准前20日均价的80%-95%之间
- 窗口指导（2024年）：锁价发行若折扣低于85%，须额外论证定价合理性
- 定价基准日前20日均价计算公式：
  基准均价 = Σ(第i日收盘价 × 20日总交易量) / Σ(第i日交易量)
  注意：须使用交易量加权，而非简单算术平均

【市价发行（无锁定期定增）】

定价基准日：发行期首日（申购报价日前的第3个交易日）
发行价格确定方式：
- 询价定价（主流）：向符合条件的投资者（不少于35家）询价
- 协商定价（例外）：向特定投资者定向增发，投资者数量不超过10名

市价发行的价格博弈：
- 发行价格的下限 = 定价基准日前20日均价的100%（不允许打折）
- 实务中：发行价格通常在基准前20日均价的100%-110%之间
- 溢价发行常见原因：市场情绪好、发行规模较小、战投背书等

【特殊情形：分红除权对定价的影响】
若在定价基准日到发行日之间公司实施现金分红：
- 发行价格须相应调减（现金分红不归投资者独有）
- 调减公式：调整后价格 = 原价格 - 每股现金分红金额
- 实务影响：现金分红金额较小时，影响可忽略；大比例分红（如特别股息）
  可能导致发行价格调减幅度较大
```

#### 4.1.2 发行窗口期测算

```
【从预案公告到资金到账的全流程】

T + 0      董事会决议公告日（定价基准日①）
  ↓        公告预案、募集资金可行性研究报告
T + 20-30  股东大会决议公告日（定价基准日②，如选择）
  ↓        审议预案、战投协议等
T + 30     申报材料制作完成（含律师鉴证意见、会计师意见）
  ↓
T + 31     向交易所/证监会提交申报材料
  ↓
T + 31-35  受理（5个工作日内发出受理通知书）
  ↓
T + 55     交易所审核（20个工作日，发出审核问询或要求落实函）
  ↓        发行人及中介机构回复问询（通常15-30工作日）
  ↓
T + 80     上市委审议（10个工作日内）
  ↓
T + 80-100 证监会注册（10个工作日内，决定注册/不予注册）
  ↓
T + 100+   发行阶段（开始询价到资金到账，通常2-4周）
  ↓
T + 120+   上市流通

【锁价定增的发行窗口风险】
锁价发行最大的风险在于"价格锁定"后股价大幅下跌，导致：
① 发行价格（基准价×80%）高于市价，倒挂
② 投资者弃购或要求重新定价基准日（须重新上会）
③ 极端情况：发行失败

风险对冲机制：
- 在发行方案中加入"价格调整机制"（触发条件：基准价跌幅>20%）
- 明确"重新定价基准日"触发条件和程序
- 设置发行有效期（通常12个月内，逾期须重新上会）
```

### 4.2 可转债转股价格机制

#### 4.2.1 初始转股价格设定规则

```
初始转股价格的法定要求：

┌─────────────────────────────────────────────────────────┐
│                                                         │
│  初始转股价格 ≥ 募集说明书公告日前20日均价 × 98%           │
│                                                         │
│  即：允许的初始折价空间仅为2%（近似值）                   │
│                                                         │
└─────────────────────────────────────────────────────────┘

【定价基准日】
募集说明书公告日（T-3至T-1之间的某个工作日）

【实战解读】
- 若募集说明书公告日前20日均价为10元/股
- 最低初始转股价格 = 10 × 98% = 9.80元/股
- 转股溢价率 = (10 - 9.8)/10 = 2% （初始溢价2%是监管允许的最低门槛）
- 实务中，初始转股价通常等于基准均价的100%（无折价）

【下修条款（最重要的条款博弈点）】

触发条件（标准表述，须逐字核对）：
"在本次发行的可转债存续期间，当公司股票在任意连续30个交易日中
有15个交易日的收盘价低于当期转股价格的85%时..."

下修空间：
- 普通下修：不低于前20日均价（须经股东大会审议，出席股东所持表决权2/3通过）
- 特别下修（深圳交易所2023年新增规则）：经股东大会出席股东所持表决权
  3/4以上通过，可突破"不低于前20日均价"的限制（理论上可下修至任意价格）
- 下修幅度通常一次不超过20%（如10元→8元），但无强制限制

【转股价格博弈矩阵】

情景A：正股股价低迷（<转股价85%），触发下修条件
- 发行人选择下修 → 转股价格降低 → 转股数量增加 → 稀释加大
- 发行人选择不下修 → 投资者等待，债底保护托底

情景B：正股股价持续低迷（<转股价70%），触发回售条款
- 投资者有权将可转债按面值+应计利息回售给发行人
- 发行人面临巨大的即时偿债压力

情景C：正股股价持续上涨（>转股价130%，持续30交易日）
- 触发强制赎回条款
- 发行人有权以略高于面值的赎回价（通常103元）强制赎回
- 迫使投资者转股（否则被低价赎回，转股价值远超赎回价）
→ 强制赎回是促进转股的核心机制
```

### 4.3 可转债债底保护与纯债价值测算

```
可转债的纯债价值（债底）是其下限保护，公式如下：

┌─────────────────────────────────────────────────────────┐
│                                                         │
│   纯债价值 = Σ (票面利息 / (1+YTM)^t ) + 面值/(1+YTM)^n │
│                                                         │
│   其中：                                                │
│   YTM = 到期收益率（以同评级企业债收益率替代）            │
│   t   = 第t期（年）                                     │
│   n   = 剩余期限（年）                                  │
│   面值 = 100元/张                                       │
│                                                         │
└─────────────────────────────────────────────────────────┘

【实战举例】
某可转债：面值100元，票息0.3%/0.5%/1.0%/1.5%/2.0%/到期赎回108元
同评级企业债YTM = 3.2%，剩余期限5年

纯债价值计算：
t=1: 0.3/(1.032)^1 = 0.291
t=2: 0.5/(1.032)^2 = 0.469
t=3: 1.0/(1.032)^3 = 0.911
t=4: 1.5/(1.032)^4 = 1.286
t=5: (2.0+108)/(1.032)^5 = 110/1.171 = 93.93
合计 ≈ 96.89元

→ 该可转债的债底约为97元，提供了明确的下限保护
→ 若可转债二级市场价格跌至97元以下，出现明显的"错误定价"机会
```

## 五、审核问询深度解析

### 5.1 问询问题模式分类与应答策略

#### 5.1.1 融资必要性问询

```
【经典问询模式】

"发行人最近一期持有货币资金X亿元（包括理财产品Y亿元），
 同时计划再融资Z亿元。请发行人说明：
  (1) 持有大额货币资金的合理性，是否存在资金被关联方占用情形？
  (2) 本次融资的必要性，是否存在'圈钱'嫌疑？
  (3) 融资规模的测算依据，与现有资金用途的匹配关系？"

【标准应答框架】

第一步：详细披露货币资金构成
- 现金及银行存款（区分使用受限和不受限）
- 理财产品（区分浮动收益/固定收益，到期时间）
- 其他货币资金（银行承兑汇票保证金、信用证保证金等）

第二步：说明资金用途规划
- 未来12个月刚性支出（银行贷款到期还款、资本支出承诺、股利分配）
- 日常运营资金最低保有量（通常为1-3个月营业成本）
- 可动用资金 = 货币资金 - 受限资金 - 刚性支出 - 最低保有量

第三步：量化融资必要性
- 本次募资用途与资金缺口一一对应
- 融资规模与项目投资额的匹配关系
- 不融资而通过自有资金/银行贷款解决的资金成本差异对比

第四步：如有资金充裕情形，主动说明
- 资金用途承诺（承诺不用于委托理财、房地产投资等）
- 或者主动调减募资规模（实务中最常见的处理方式）
```

#### 5.1.2 募投项目可行性问询

```
【经典问询模式】

"发行人现有产能利用率仅X%，本次募投新增产能Y万吨（是现有产能的Z%）。
 请发行人说明：
  (1) 现有产能利用率偏低的原因？
  (2) 新增产能的市场消化措施？
  (3) 是否存在重复建设/过度融资？"

【标准应答框架】

产能利用率问题应答：
- 区分产能利用率低的原因（季节性停工/设备检修/下游需求不足/新投产线爬坡）
- 提供产能利用率低的持续性判断（是短期因素还是长期趋势）
- 展示产能扩张的渐进性（首批产能爬坡后逐步释放，非一次性投放）

市场消化论证：
- 需求侧：目标市场容量及增速（引用权威行业数据）
- 供给侧：募投产品在目标市场的竞争优势（技术壁垒、成本优势、客户认证）
- 客户验证：提供已签订的意向性合同（可注明"尚未正式合同"）
- 量化测算：新增销量 ÷ 目标市场容量 = 市场占有率目标（通常<10%为合理）

重复建设质疑应答：
- 与前次募投（若有）进行区分，说明本轮募投与前轮募投的差异
- 强调本轮募投的"升级改造"属性（技术迭代）而非"简单复制"
- 提供项目建设用地的区分证明（不同地块/不同产线）
```

#### 5.1.3 摊薄效应问询

```
【经典问询模式】

"请发行人详细测算本次再融资对每股收益（EPS）和加权平均净资产收益率（ROE）
 的摊薄影响，并说明是否会对中小股东利益造成重大不利影响。
 请提供：(1) 摊薄前后的EPS/ROE对比表；(2) 填补摊薄的具体措施；"

【标准应答框架】

定量测算（须逐项列示）：
- 摊薄前EPS = 最近一年归母净利润 ÷ 发行前股本
- 摊薄后EPS = 最近一年归母净利润 ÷ (发行前股本 + 新增股本)
- 须区分"静态测算"（不考虑募集资金效益）和"动态测算"（考虑募投项目效益）

填补摊薄措施（实务中常用的回应策略）：
① 募投项目效益承诺：说明募投项目预期IRR和达产年份净利润增量
② 股东回报规划：披露未来3年现金分红政策（不低于年均可分配利润30%）
③ 股价稳定机制（若有）：大股东/董监高增持计划
④ 缩短锁定期（如定增）：通过承诺更短锁定期展示信心
⑤ ROE改善路径：若募资用于偿债，降低财务费用→提升净利润→改善ROE

监管接受标准：
- 摊薄后ROE ≥ 6%（主板门槛），或优于同行业可比公司
- EPS摊薄幅度 < 30%（经验值，超过则须充分论证合理性）
- 若ROE摊薄不可避免，须明确说明"股东回报规划"作为对冲措施
```

#### 5.1.4 同业竞争与关联交易问询

```
【经典问询模式】

"募投项目实施后，是否会新增与控股股东、实际控制人及其关联方的同业竞争？
 募投项目所需原材料/设备的采购及产品的销售是否会新增关联交易？
 关联交易定价是否公允？"

【标准应答框架】

同业竞争论证：
- 明确募投项目的产品类别和应用领域
- 与控股股东/实控人控制的企业的业务进行逐项对比
- 提供律师意见：募投项目实施主体与关联方之间不存在同业竞争
- 必要时：控股股东出具避免同业竞争的承诺函

关联交易分析：
- 识别募投项目实施过程中可能发生的关联交易类型（采购/销售/提供劳务/租赁等）
- 说明关联交易金额占同类交易的比例（通常要求<30%）
- 提供关联交易定价依据（比照市场第三方价格）
- 说明关联交易的必要性（产业链协同）和公允性（审计师核查意见）
```

### 5.2 审核周期管理

```
【注册制审核流程详解（科创板/创业板）】

T + 0       提交申报（申报材料齐备性核查）
  ↓
T + 1-5     受理阶段（发出受理函/补正通知）
  ↓
T + 5-25    审核问询阶段（首轮问询，约20个工作日）
  ↓         问题数量：通常5-15个，复杂项目可达30+个
  ↓
T + 25-45   回复阶段（发行人+中介机构落实问询，约15-30工作日）
  ↓
T + 45-65   审核中心意见落实（如有第二轮问询，重复T+25-45流程）
  ↓
T + 65-75   上市委审议（10个工作日，审议会议当场意见）
  ↓
T + 75-85   报送证监会注册（10个工作日决定是否注册）
  ↓
T + 85-95   注册结果（领取注册批文 / 不予注册 / 意见函）
  ↓
T + 95-115  发行阶段（询价/定价/申购/缴款，约2-4周）
  ↓
T + 120+    上市流通

【审核周期影响因素】

加速因素：
- 问询问题少（5个以下），一轮问询即可通过
- 募投项目成熟（已取得全部合规文件）
- 财务数据无异常（无业绩大幅下滑、审计非标意见等）
- 发行方案简单（单一募投项目，无复杂条款）

减速因素：
- 多轮问询（通常不超过三轮，超过三轮将被重点关注）
- 募投项目合规文件不全（环评/备案缺失）
- 财务数据异常（业绩下滑>30%、应收账款激增、存货大幅增长）
- 涉及敏感行业（房地产、类金融、教育等受限制行业）
- 可转债条款复杂（转股价下修机制、赎回/回售条款博弈）
- 举报信或媒体质疑（审核期间出现负面舆情）
```

```
□ 15. 发行费用：承销费、律师费、审计费、评估费的收费依据合理
□ 16. 申报文件签字：所有对外文件经发行人法定代表人、董秘、财务总监签字
□ 17. 审计报告有效期：最近一期审计报告在审核中持续有效（6个月）
□ 18. 重大诉讼/仲裁：披露募投项目相关的重大法律风险
```

## 七、典型案例深度复盘

### 7.1 案例A：芯片封装龙头引入战投（定增锁价）

```
【基本情况】
发行人：某科创板半导体封装测试公司（股票代码：688XXX）
总股本：3.2亿股，前20日均价：28元/股，市值：约90亿元
融资动因：引入全球第二大封装测试厂作为战投，共建先进封装研发中心
拟融资规模：10亿元

【方案设计】
发行价格：28元 × 90% = 25.2元/股（战投折扣9折，窗口指导下限附近）
新增股份：10亿 ÷ 25.2元 = 3968万股
摊薄比例：3968万 ÷ (3.2亿 + 3968万) = 11.0%
锁定期：36个月（战略投资者，适用《上市公司证券发行注册管理办法》）

【问询关注点及应对】
1. 战投认定：要求提供双方战略合作协议及联合研发计划书
   → 应对：提供双方盖章的技术合作框架协议，列明具体研发方向和知识产权归属

2. 定价合理性：为何给战投9折（而非更高的折扣）？
   → 应对：引用可比交易案例（同类战投定增项目），说明9折属于市场惯例

3. 摊薄效应：ROE从15%降至12%，是否损害中小股东利益？
   → 应对：提供研发中心未来5年知识产权转化计划（专利产出、新增收入测算）

【结果】
审核周期4.5个月，一轮问询后通过注册
发行结果：战投全额认购，融资10亿元，资金于注册批文取得后2周到账
```

### 7.2 案例B：制造业扩产可转债（被问询后调整）

```
【基本情况】
发行人：某深市民营制造业龙头，主营精密机械零部件
总股本：6亿股，净资产（归母）：28亿元
财务指标：最近3年加权平均ROE=7.2%（踩线6%），现金分红比例35%
融资动因：扩产+补充流动资金，拟融资10亿元

【初始方案】
可转债10亿元，其中：
- 扩产项目A（投资8亿元，IRR=13%）
- 补充流动资金3亿元（占比30%，踩线）
- 发行费用0.5亿元

【审核问询】
"发行人ROE刚过6%门槛，2019年ROE=5.8%（低于6%），
 请说明本次融资是否可能导致ROE持续下滑，对中小股东利益的影响？"
"补流比例30%，请说明资金缺口的测算依据，是否存在融资必要性不足？"

【方案调整】
1. 补流比例从30%降至22%（2.2亿元），新增扩产项目B（投资0.8亿元）
2. 补充披露：未来3年资金缺口测算模型（分项列示：运营资金需求、
   刚性支出、贷款到期等），证明2.2亿元补流具有必要性
3. 追加承诺：若ROE持续低于6%，将触发可转债转股价格下修

【结果】
二轮问询后通过注册，审核周期6个月
发行结果：原股东优先配售率68%，社会公众认购率32%，转股价格=28.5元/股
```

### 7.3 案例C：配股包销风险处置

```
【基本情况】
发行人：某上证主板上市公司，主营商业物业运营
总股本：4亿股，股权登记日前收盘价：9.5元/股
大股东持股比例：38%，承诺认购上限：38%（即理论上最多认购38%）
拟配股方案：10配3，配股价格6.5元/股（收盘价的68%），融资上限7.8亿元

【风险识别】
38% × 30% = 11.4%的股份由大股东认购
剩余88.6% × 30% = 26.6%的股份由其他股东认购
若其他股东认购率仅50%，承销商须包销股份比例 = 26.6% × 50% = 13.3%
包销金额 = 13.3% × 4亿 × 6.5元 = 3.46亿元

【预案措施】
1. 大股东额外承诺：若其他股东认购不足50%，大股东承诺追加认购至70%，
   追加认购上限不超过总股本的5%（即追加认购最多2000万股，1.3亿元）
2. 承销商风险对冲：与做市商提前沟通，若包销股份超过预期，
   做市商将在二级市场进行流动性管理
3. 引入基石投资者：引入2家机构投资者作为"托底认购方"，
   合计承诺认购约10%的配股股份

【实际结果】
配股认购率82%，承销商包销约6%的股份（3600万股，2.34亿元）
大股东追加认购1500万股（0.975亿元）
配股除权后首日开盘参考价8.2元，高于配股价格6.5元
包销股份于6个月后上市流通，承销商实现小幅盈利

【教训总结】
- 配股方案设计时，必须提前做好"包销风险预案"
- 大股东认购承诺须在配股说明书披露前锁定
- 引入托底机构投资者是降低包销风险的有效手段
```

## 八、方案设计常用公式速查表

```
【定增规模公式】
MAX_定增 = MIN(拟投资总额, 股本×20%)

【可转债规模公式】
MAX_可转债 = 归母净资产×40%（金融企业60%）

【配股规模公式】
MAX_配股 = 总股本×30%×配股价格

【摊薄比例公式】
摊薄比例 = 新增股份÷(原股本+新增股份)
         = 募资总额÷(发行价格×原股本+募资总额)

【摊薄后ROE公式】
ROE_摊薄后 = 原净利润÷(原净资产+募资净额-新增折旧)

【除权参考价公式（配股）】
除权参考价 = (股权登记日收盘价 + 配股价格×配股比例)
             ÷ (1 + 配股比例)

【可转债纯债价值公式】
纯债价值 = Σ(票息t÷(1+YTM)^t) + 面值÷(1+YTM)^n

【转股数量公式】
转股新增股份 = 可转债面值总额÷初始转股价格

【项目IRR近似算法】
IRR ≈ 税后利润年均值÷总投资额 × 100%（简化算法）
精确IRR须使用现金流折现法（ExcelIRR函数或专业财务软件）

【资金缺口公式】
资金缺口 = 必要运营资金 + 到期贷款 + 资本支出承诺 + 分红支出
          - 可动用货币资金 - 预计经营现金流入
```
