# SKILL_03e：财务建模

## 概述

财务建模是投行方案设计中估值定价的核心技术环节。本模块覆盖五种核心模型：**DCF模型、可比公司分析、可比交易分析、并购模型（accretion/dilution）、IPO估值**。建模的核心原则：假设透明、逻辑可查、敏感可控。

---

## 一、DCF模型（现金流折现法）

### 1.1 模型框架

```
企业自由现金流(FCFF) 预测（通常5-10年）
        ↓
折现率(WACC) 确定
        ↓
终值(Terminal Value) 计算
        ↓
企业价值(Enterprise Value)
        ↓
- 净债务
+ 非核心资产
        ↓
股权价值(Equity Value)
        ↓
每股价值
```

### 1.2 FCF预测要点

```
FCFF = EBIT × (1 - 税率) + 折旧摊销 - 资本支出 - 营运资本增加

各项目预测方法：
- 收入：自下而上（量×价）或自上而下（市场规模×市占率） — 必须有业务逻辑支撑
- 毛利率：基于历史趋势+行业对标±合理调整
- 费用率：区分固定费用和变动费用
- 折旧摊销：基于现有固定资产+计划资本支出
- 资本支出：与产能扩张计划匹配
- 营运资本：应收账款周转/存货周转/应付账款周转假设
```

### 1.3 WACC计算

```
WACC = E/V × Re + D/V × Rd × (1 - t)

其中：
E = 股权市值        D = 债务市值
V = E + D           t = 所得税率
Re = 股权成本(CAPM)  Rd = 债务成本

CAPM: Re = Rf + β × (Rm - Rf) + 规模溢价 + 公司特定风险溢价

参数选取指南：
- Rf（无风险利率）：10年期国债收益率（中国约2.5%-3.5%）
- ERP（Rm - Rf）：中国市场风险溢价通常5%-8%
- β（贝塔）：可比公司平均β去杠杆后再加目标资本结构杠杆
- 规模溢价：小市值公司通常1%-3%
```

### 1.4 终端价值

```
Gordon增长模型：TV = FCFF(n+1) ÷ (WACC - g)

其中：
g = 永续增长率（中国通常2%-3%，不超过GDP长期增速）
FCFF(n+1) = 最后一预测年FCFF × (1 + g)
```

**终值通常占DCF估值的60%-80%，因此对g极为敏感——必须做敏感性分析。**

### 1.5 敏感性分析（必做）

**二维敏感表**：WACC × 永续增长率

| | g=1.5% | g=2.0% | g=2.5% | g=3.0% |
|------|--------|--------|--------|--------|
| WACC=8% | | | | |
| WACC=9% | | | | |
| WACC=10% | | | | |
| WACC=11% | | | | |
| WACC=12% | | | | |

## 二、可比公司分析

### 2.1 可比公司筛选标准

| 筛选维度 | 标准 |
|----------|------|
| 业务模式 | 主营业务相似度>70% |
| 市场区域 | 相同/相近市场 |
| 规模体量 | 收入/市值在同一数量级 |
| 成长阶段 | 相似的生命周期阶段 |
| 上市地点 | 同一市场（估值基准一致） |
| 数据可得 | 财务数据完整、市场数据可靠 |

### 2.2 常用估值倍数

| 倍数 | 适用场景 | 不适用场景 |
|------|----------|------------|
| P/E（市盈率） | 盈利稳定成熟企业 | 亏损/微利企业 |
| P/B（市净率） | 金融（银行/保险/券商） | 轻资产服务业 |
| P/S（市销率） | 初创/高增长但未盈利 | 毛利率差异大的行业 |
| EV/EBITDA | 重资产/高杠杆行业 | 金融业（不适用） |
| EV/Revenue | SaaS/平台型企业 | 与P/S类似 |
| PEG | 成长性差异大的同行 | 极度依赖盈利预测 |

### 2.3 流动性折扣

A股IPO前估值通常比可比上市公司估值打折扣：

| 折扣因素 | 参考折扣 |
|----------|----------|
| 非上市流动性折价 | 20%-40% |
| 规模折价（小市值） | 5%-15% |
| 控股折价/溢价 | ±5%-20% |

### 2.4 常见错误

- 选取不合理的可比公司（业务差异大/规模悬殊）
- 使用不经过调整的公开市场数据（如包含重大非经常性损益的P/E）
- 忽略增长率和ROE的差异直接套用倍数
- 在可比公司数量极少（<3家）时仍强行做可比分析

## 三、可比交易分析

### 3.1 交易筛选

| 筛选维度 | 标准 |
|----------|------|
| 交易类型 | 同类型（收购/合并/资产购买） |
| 标的行业 | 同一/相近行业 |
| 交易规模 | 在同一数量级 |
| 交易时间 | 最近3-5年 |
| 可比性 | 控制权交易优先（含控制权溢价信息） |

### 3.2 控制权溢价

```
控制权溢价通常为 20%-30%
（基于全球并购历史统计，中国市场通常偏高端）

含控制权溢价估值 = 可比公司估值 × (1 + 控制权溢价)

应用于：
- 收购>50%股权 → 含控制权溢价
- 收购<20%少数股权 → 通常不含控制权溢价
- 收购20%-50% → 视情况部分溢价
```

## 四、并购模型（Accretion/Dilution）

### 4.1 核心分析

```
增厚/稀释分析：

交易后EPS vs 交易前EPS

交易后EPS > 交易前EPS → Accretive（增厚）
交易后EPS < 交易前EPS → Dilutive（稀释）
```

### 4.2 模拟合并利润表

| 项目 | 买方 | 标的 | 调整 | 合并后 |
|------|------|------|------|--------|
| 收入 | A | B | 协同收入 | A+B+协同 |
| 毛利 | | | | |
| EBITDA | | | 协同成本节约 | |
| D&A | | | +新增折旧(PPA) | |
| EBIT | | | | |
| 利息费用 | | | +新增融资利息 | |
| EBT | | | | |
| 税金 | | | | |
| 净利润 | | | | |
| 少数股东 | | | | |
| 归母净利润 | | | | |
| 股本 | | | +新增股份 | |
| **EPS** | | | | |

### 4.3 PPA（Purchase Price Allocation）影响

| 项目 | 处理 |
|------|------|
| 可辨认资产增值 | 按剩余使用年限折旧摊销（影响未来利润） |
| 商誉 | 不摊销，每年减值测试 |
| 核心无形资产 | 按预期收益期摊销（通常3-15年） |
| 递延税 | PPA产生的应纳税暂时性差异 |

### 4.4 融资方式对EPS的影响

| 融资方式 | EPS影响 | 杠杆影响 |
|----------|---------|----------|
| 全现金（自有） | 利息收入减少（失去现金的利息） | 无新增负债 |
| 全现金（债务融资） | 新增利息费用 | 资产负债率上升 |
| 全股份 | 股本增加 → EPS摊薄 | 无新增负债 |
| 混合 | 综合上述两种影响 | |

## 五、IPO估值

### 5.1 A股/H股/美股估值差异

| 维度 | A股 | H股 | 美股 |
|------|-----|-----|------|
| PEPE估值基准 | 20-40x（行业差异大） | 10-25x | 15-30x |
| 估值方法偏好 | PE + PB | PE + DCF | PE + EV/EBITDA |
| 散户影响 | 大（打新热情高） | 中 | 小 |
| 流动性折价 | 小 | 中 | 小（中概股折价） |
| 行业偏好 | 消费/科技/医药高估值 | 金融/地产高权重 | 科技/互联网高估值 |

### 5.2 IPO定价流程

```
Step 1: 估值分析（DCF + 可比 + 可比交易）
    ↓
Step 2: 确定估值区间
    ↓
Step 3: 路演（管理层路演 → 分析师预路演 → 正式路演）
    ↓
Step 4: 簿记建档（收集投资者订单和价格）
    ↓
Step 5: 定价（发行人和承销商协商确定最终发行价）
    ↓
Step 6: 上市后表现（首日涨跌幅/30日稳定期）
```

### 5.3 IPO折价

正常情况下IPO发行价较理论估值留有10%-20%的折价（为投资者留出首日盈利空间）。

| 折价因素 | 原因 |
|----------|------|
| 新发溢价 | 吸引投资者参与认购 |
| 信息不对称 | 投资者对公司的了解少于管理层 |
| 承销商信誉 | 知名承销商折价更小 |

## 六、关键参数范围速查

| 参数 | A股参考范围 | 说明 |
|------|------------|------|
| Rf（无风险利率） | 2.5%-3.5% | 10年国债收益率 |
| ERP（市场风险溢价） | 5%-8% | 沪深300长期收益率-无风险利率 |
| β（贝塔系数） | 0.6-1.5 | 科技>1.2，公用事业<0.8 |
| g（永续增长率） | 2%-3% | 不超过长期GDP增速 |
| 规模溢价 | 1%-3% | 小市值公司加 |
| 公司特定风险 | 1%-5% | 视公司成熟度/客户集中度等 |
| 非上市流动性折价 | 20%-40% | IPO估价vs二级市场价 |
| 控制权溢价 | 20%-30% | 控股收购vs少数股权投资 |
| IPO折价 | 10%-20% | 发行价vs理论估值 |
| 债务成本(Rd) | 3%-8% | AAA约3%-4%、AA+约4%-5% |

## 七、模型常见陷阱

| 陷阱 | 问题 | 修正 |
|------|------|------|
| 收入增速过于乐观 | 预测期收入CAGR>30% | 对比行业增速+公司历史增速 |
| 毛利率假设不合理 | 假设毛利率持续上升 | 考虑竞争加剧降价压力 |
| WACC偏低 | 低估股权成本 | β/E/Rf每个参数交叉验证 |
| 终值占比过高 | 终值>85%的估值 | 延长预测期或降低永续增长率 |
| 不考虑资本支出 | FCF高估 | 增长需要匹配的资本支出 |
| 环比跳跃 | 各项指标缺乏平滑过渡 | 用过渡期建立趋势 |
| 模型硬编码 | 参数写在公式里 | 所有假设放在独立的假设页 |
| 只做单点估值 | "估值是20.5亿" | 必须提供区间（如18-23亿） |
| 不做交叉验证 | 只用一种方法 | DCF+可比+可比交易三者交叉 |

## 八、模型文档与审计追踪

### 8.1 模型结构规范

```
Sheet 1: 封面
Sheet 2: 假设页（所有可调参数）
Sheet 3: 收入预测
Sheet 4: 利润表
Sheet 5: 资产负债表
Sheet 6: 现金流量表
Sheet 7: FCF计算
Sheet 8: WACC计算
Sheet 9: DCF估值
Sheet 10: 可比公司分析
Sheet 11: 可比交易分析
Sheet 12: 敏感性分析
Sheet 13: 图表（估值桥/瀑布图/敏感图）
```

### 8.2 关键原则

- 蓝色字体 = 手动输入值（假设）
- 黑色字体 = 公式计算值
- 绿色字体 = 引用其他Sheet的值
- **绝不在公式中硬编码**
- 所有假设集中管理，方便修改和审计
- 模型版本号+更新日志
