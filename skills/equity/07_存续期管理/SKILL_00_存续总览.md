# SKILL_07：存续期管理总览

## 一、概述

存续期管理是投行项目生命周期中最长的阶段。IPO上市后需履行持续督导义务（通常2-3年），债券存续期内需进行受托管理，并购重组完成后需跟踪业绩承诺和整合进展。合规的存续期管理不仅是监管要求，也是维护市场声誉和保护投资者利益的重要保障。

---

## 二、三层存续期管理职责

存续期管理的三层架构覆盖了从日常信息披露到退市/到期的全维度管理职责：

### 第一层：持续信息披露

```
职责范围：
  ├── 年度报告审阅及披露（4个月内）
  ├── 半年度报告审阅及披露（2个月内）
  ├── 季度报告审阅及披露（1个月内，如适用）
  ├── 重大事项临时公告（发生后2个交易日内）
  ├── 业绩预告/业绩快报（按规定时点）
  └── 信息披露质量持续审核

责任人：保荐人（IPO）/ 受托管理人（债券）/ 独立财务顾问（并购）
监管处罚风险：信息披露不及时/不准确/不完整 → 监管警示函/罚款
```

### 第二层：持续督导 / 受托管理

```
职责范围：
  ├── 财务指标跟踪（收入、利润、现金流、资产负债率的异常变化）
  ├── 募集资金使用核查
  │    ├── 月度资金使用台账核对
  │    ├── 募集资金专户余额与银行对账单核对
  │    ├── 募集资金用途变更的合规审查
  │    └── 募集资金暂时补充流动资金的期限监控
  ├── 关联交易监控
  │    ├── 是否超出年度预计额度
  │    ├── 定价是否公允
  │    └── 决策程序是否合规
  ├── 对外担保监控（占净资产比例/是否履行审批程序）
  ├── 股份减持/限售股解禁监控
  ├── 现场检查（至少每半年一次）
  └── 年度督导报告 / 受托管理报告

责任人：保荐人（IPO）/ 受托管理人（债券）/ 独立财务顾问（并购）
核心风险：募集资金挪用、关联方资金占用、违规担保
```

### 第三层：退市/到期管理

```
IPO端：
  ├── 退市风险评估（连续亏损/重大违法/面值退市等）
  ├── 退市整理期督导
  └── 主动退市方案督导

债券端：
  ├── 回售权行使管理（投资者回售选择权到期前30日公告）
  ├── 赎回权行使管理（发行人赎回选择权到期前30日公告）
  ├── 到期兑付管理（本息兑付日前15日公告）
  ├── 持有人会议召集（如触发召开条件）
  └── 违约处置（如触发违约条款）

并购端：
  ├── 业绩承诺期到期核查
  ├── 或有对价（Earnout）到期结算
  ├── 股份解锁/限售期到期管理
  └── 商誉减值终测（业绩承诺期满当年）
```

## 三、IPO督导 vs 债券托管 vs 并购整合 三条线的差异化职责

### 3.1 全维度对比

| 维度 | IPO 持续督导 | 债券 受托管理 | 并购 整合跟踪 |
|:---|:---|:---|:---|
| **法律依据** | 《证券发行上市保荐业务管理办法》 | 《公司债券发行与交易管理办法》 | 《上市公司重大资产重组管理办法》 |
| **存续期时长** | 上市当年+2个完整会计年度（创业板/科创板：3个） | 债券存续期间+债券到期/清偿完毕 | 业绩承诺期（通常3年） |
| **核心义务主体** | 保荐人/保荐代表人 | 受托管理人（券商/银行） | 独立财务顾问 |
| **定期报告审阅** | 是（年报+半年报，必要时季报） | 是（年报+半年报） | 是（年报，关注业绩承诺完成情况） |
| **募集资金监控** | 是（每半年出具专项核查意见） | 是（按募集说明书约定用途） | 不适用 |
| **现场检查** | 每年至少1次 | 按约定（通常每年1次） | 视需要 |
| **年度报告** | 年度督导报告 | 受托管理事务年度报告 | 业绩承诺实现情况专项核查意见 |
| **关联交易监控** | 是 | 不适用 | 是（防止利益输送） |
| **财务指标监控** | 持续经营能力+财务异常 | 偿债能力指标（利息保障倍数/资产负债率） | 业绩承诺达成率+商誉减值迹象 |
| **重大事项关注** | 10类重大事项 | 债券持有人会议触发事项 | MAC事件/减值迹象 |
| **担保核查** | 对外担保>净资产10% | 增信措施持续有效 | 不直接适用 |
| **持有人会议** | 不适用 | 是（触发条件发生时召集） | 不适用 |
| **业绩承诺跟踪** | 不适用 | 不适用 | 是（核心职责） |
| **商誉减值测试** | 不适用 | 不适用 | 是（每年+随时） |
| **整合监控** | 不适用 | 不适用 | 是 |
| **信息披露违规后果** | 保荐人执业评价降级 | 受托管理人责任 | 独立财务顾问责任 |
| **监管报告周期** | 年报后15个工作日内 | 每年6月30日前 | 年报后（上市公司发布后） |
| **底稿保存期限** | ≥10年（重要底稿≥20年） | ≥债券到期后5年 | ≥业绩承诺期满后5年 |

### 3.2 三条线的关键风险差异

| 风险维度 | IPO督导 | 债券受托管理 | 并购整合 |
|:---|:---|:---|:---|
| **头号风险** | 募集资金挪用 | 债券违约 | 业绩承诺落空 |
| **常见风险** | 关联方资金占用、违规担保、信息披露不及时 | 交叉违约、增信措施失效、偿债能力恶化 | 商誉大幅减值、整合失败、对赌纠纷 |
| **监管敏感度** | 极高 | 高（违约事件挂钩） | 高（商誉"爆雷"为市场焦点） |
| **投资者关注度** | 高（股价直接相关） | 中（机构投资者为主） | 高（业绩对股价影响大） |

## 四、信息披露时间轴

### 4.1 定期报告法定期限

| 报告类型 | 截止时间 | 宽限期 | 逾期后果 |
|:---|:---|:---|:---|
| **年度报告** | 会计年度结束后4个月内（即次年4月30日前） | 无 | 停牌→ST/退市风险警示 |
| **半年度报告** | 上半年结束后2个月内（即8月31日前） | 无 | 停牌→监管问询 |
| **第一季度报告** | 第一季度结束后1个月内（即4月30日前） | 不得晚于年报披露 | 监管关注 |
| **第三季度报告** | 第三季度结束后1个月内（即10月31日前） | 无 | 监管关注 |
| **业绩预告** | 预计净利润为负/扭亏为盈/变动>50%：年度结束后1月31日前 | — | 监管处罚 |
| **业绩快报** | 如计划在3-4月披露年报，可不披露快报 | — | — |

### 4.2 临时公告时限

| 重大事项类型 | 披露时限 | 法律依据 |
|:---|:---|:---|
| 重大诉讼/仲裁（>净资产10%） | 知悉后2个交易日内 | 《上市规则》 |
| 重大资产重组 | 首次董事会决议后 | 《重组办法》 |
| 重大关联交易（>净资产5%或>3000万） | 董事会决议后2个交易日内 | 《上市规则》 |
| 对外担保（>净资产10%） | 董事会/股东大会决议后2个交易日内 | 《上市规则》 |
| 控股股东/实控人变更 | 知悉后立即公告 | 《上市规则》 |
| 董监高被调查/处罚 | 知悉后立即公告 | 《上市规则》 |
| 募集资金投向变更 | 董事会/股东大会决议后 | 《上市规则》 |
| 债券违约/交叉违约 | 知悉后2个交易日内 | 《债券管理办法》 |
| 信用评级下调 | 知悉后2个交易日内 | 《债券管理办法》 |

### 4.3 年度信息披露日历（IPO项目参考）

```
1月：
  □ 1月31日前：业绩预告（如适用）
  □ 募集资金年度使用计划更新

2月-4月：
  □ 年报审计启动
  □ 4月30日前：年报 + 第一季度报告披露
  □ 年报后15个工作日内：年度督导报告出具

5月-6月：
  □ 年度股东大会
  □ 利润分配实施
  □ 债券受托管理事务年度报告（6月30日前）

7月-8月：
  □ 半年度报告审阅
  □ 8月31日前：半年度报告披露
  □ 募集资金半年度专项核查报告

9月-10月：
  □ 第三季度报告审阅
  □ 10月31日前：第三季度报告披露

11月-12月：
  □ 现场检查（通常在年报审计前）
  □ 募集资金年度使用情况盘点
  □ 次年工作计划制定
```

## 五、重大事项判断标准

### 5.1 量化判断标准（通用）

| 指标 | 披露线 | 股东大会审议线 | 重大资产重组线 |
|:---|:---:|:---:|:---:|
| **总资产** | ≥10% | ≥30% | ≥50% |
| **营业收入** | ≥10% | ≥30% | ≥50% |
| **净利润** | ≥10% | ≥30% | ≥50% |
| **净资产** | ≥10%且绝对额>1000万 | ≥30% | ≥50%且绝对额>5000万 |
| **成交金额** | ≥10% | ≥30% | ≥50% |

> 注：上述比例均相对上市公司最近一个会计年度经审计的对应指标而言。达到"披露线"须及时披露；达到"股东大会审议线"须提交股东大会审议；达到"重大资产重组线"须按重大资产重组程序办理。

### 5.2 IPO持续督导须关注的重大事项

| 序号 | 事项 | 量化标准（任一满足） | 行动 |
|:---:|:---|:---|:---|
| 1 | 经营业绩大幅下滑 | 营收/净利润下降>50% | 查明原因→评估持续性→必要时出具核查意见 |
| 2 | 重大资产重组/再融资 | — | 审阅方案→出具核查意见 |
| 3 | 关联交易超出预计额度 | 超出部分>预计的20% | 督促补充审批→核查定价公允性 |
| 4 | 对外担保 | 担保总额>净资产50% 或 单笔>净资产10% | 评估担保风险→核查审批程序 |
| 5 | 重大诉讼/仲裁 | 涉案金额>净资产10% 或 绝对额>1000万 | 评估影响→核查披露充分性 |
| 6 | 董监高被立案调查 | — | 评估影响→核查披露充分性 |
| 7 | 重大亏损或重大损失 | 损失>净资产10% | 查明原因→评估影响 |
| 8 | 募集资金投向变更 | — | 督促履行程序→出具核查意见 |
| 9 | 产业政策重大变化 | — | 评估实质影响→必要时出具核查意见 |
| 10 | 核心技术人员/高管离职 | — | 评估影响→关注持续经营能力 |

### 5.3 债券受托管理须关注的重大事项

| 序号 | 事项 | 触发标准 | 行动 |
|:---:|:---|:---|:---|
| 1 | 发行人经营亏损 | >净资产10% | 评估偿债能力影响 |
| 2 | 发行人债券违约 | 任何金额的违约 | 立即召集持有人会议 |
| 3 | 信用评级下调 | 任何评级下调 | 评估影响→信息披露 |
| 4 | 增信措施变化 | 担保物价值下降>30% / 担保人信用恶化 | 督促补充增信 |
| 5 | 发行人出售/转移重大资产 | 转让资产>净资产30% | 评估→持有人会议 |
| 6 | 控制权变更 | 实际控制人发生变化 | 评估→持有人会议（如触发提前赎回条款） |

## 六、募集资金使用监督

### 6.1 月度核查清单

```
月度核查项目：

□ 募集资金专项账户余额 vs 银行对账单余额（必须一致）
□ 本月募集资金支出是否经过规定审批程序
□ 支出用途是否与招股书/募集说明书承诺一致
□ 是否存在"大额资金转入后短期内等额转出"的可疑交易
□ 募集资金暂时补充流动资金是否在12个月内归还
□ 募集资金用于现金管理的产品（如结构性存款）是否到期回款
□ 募集资金置换预先投入资金是否有审计师鉴证
```

### 6.2 募集资金用途变更的披露要求

| 变更类型 | 审批程序 | 披露要求 | 中介核查 |
|:---|:---|:---|:---|
| 取消/终止原项目，实施新项目 | 董事会→独董/监事会意见→股东大会 | 决议后2个交易日内 | 保荐人出具核查意见 |
| 变更实施主体 | 董事会→独董/监事会意见 | 决议后2个交易日内 | 保荐人出具核查意见 |
| 变更实施方式 | 董事会→独董/监事会意见（如重大变更须股东大会） | 决议后2个交易日内 | 保荐人出具核查意见 |
| 用途变更超净额30% | 董事会→独董/监事会意见→股东大会 | 决议后2个交易日内 | 保荐人出具核查意见 |
| 节余资金永久补流 | 董事会→独董/监事会意见→股东大会（如>净额10%） | 决议后2个交易日内 | 保荐人出具核查意见 |

### 6.3 募集资金违规使用的常见模式

| 违规模式 | 表现 | 识别方法 |
|:---|:---|:---|
| **挪用于非募投项目** | 以"工程款"名义支付但实际用于其他项目 | 追踪资金下游去向 |
| **大额存单质押** | 募集资金以存单形式存放，但被用于质押 | 核查银行函证中的受限资金 |
| **变相资金占用** | 通过募投项目供应商向关联方输送资金 | 核查供应商是否与关联方有关联关系 |
| **超期临时补流** | 募集资金临时补充流动资金超过12个月未还 | 逐笔核对补流资金的借出和归还时间 |
| **现金管理越线** | 以现金管理名义购买高风险理财产品或非保本产品 | 核查理财产品合同 |

## 八、存续期监管报告日历

### IPO项目年度报告日历

| 时间 | 事项 | 完成标志 |
|-----|------|---------|
| 1月-4月 | 审核上市公司年度报告，出具年度督导报告 | 年度督导报告提交交易所 |
| 3月-4月 | 募集资金存放与使用专项核查（年度） | 专项核查意见公告 |
| 7月-8月 | 审核上市公司半年度报告 | 审阅记录归档 |
| 不定期 | 关联交易、担保、重大交易审查 | 核查意见留档 |
| 不定期 | 现场检查 | 现场检查报告 |

### 并购项目年度报告日历

| 时间 | 事项 | 完成标志 |
|-----|------|---------|
| 1月-4月 | 审核年度报告中的业绩承诺完成情况 | 核查意见 |
| 1月-4月 | 商誉减值测试结果审核 | 核查意见 |
| 3月-4月 | 出具业绩承诺实现情况专项核查意见 | 专项核查公告 |
| 不定期 | 评估整合进展 | 整合报告 |

### 债券受托管理报告日历

| 时间 | 事项 | 完成标志 |
|-----|------|---------|
| 每年6月30日前 | 受托管理事务年度报告 | 向债券持有人公告 |
| 付息/兑付前 | 督促发行人按时足额还本付息 | 提醒函+资金确认 |
| 不定期 | 持有人会议召集（如触发条件） | 持有人会议决议公告 |

## 九、常见合规风险

### 9.1 高发风险点

| 风险 | 触发场景 | 后果 |
|-----|---------|------|
| 募集资金使用不合规 | 挪用募集资金、未按承诺进度使用 | 监管警示函、保荐代表人处罚 |
| 关联交易越线 | 未审议/未披露的关联交易 | 监管问询、处罚 |
| 业绩承诺未兑现 | 实际业绩低于承诺 | 市场质疑、商誉减值、补偿纠纷 |
| 信息披露不及时 | 重大事项未在2个交易日内披露 | 监管处罚 |
| 督导工作底稿缺失 | 未保存督导工作记录 | 执业质量评价降级 |
| 现场检查流于形式 | 现场检查未发现已存在的重大问题 | 保荐人责任追究 |
| 债券受托管理失职 | 未及时发现发行人偿债能力恶化 | 投资者索赔 |
| 商誉减值提示缺失 | 减值迹象已出现但年度报告中未充分提示 | 监管问询、投资者质疑 |

### 9.2 风险防范措施

- 建立定期提醒机制（日历+系统）
- 督导工作底稿标准化、流程化
- 重大事项识别清单随身携带
- 双人复核各项报告的出具
- 设置关键财务指标预警线，超过预警线自动触发核查
- 定期轮换督导项目团队（避免"疲劳性合规"）
- 存续期管理SOP定期更新（至少每季度更新一次以反映最新监管要求）

### 9.3 监管检查应对准备

```
准备清单：
  □ 所有持续督导工作底稿已归档，可按检查要求即时调取
  □ 年度督导报告/受托管理报告/专项核查意见齐备
  □ 现场检查报告和整改跟踪记录完整
  □ 募集资金使用台账与银行对账单一一对应
  □ 关联交易审查记录完整
  □ 重大事项核查底稿和决策记录保存完整
  □ 人员变更的交接记录（保荐代表人/受托管理负责人）
  □ 培训记录（如有持续督导培训要求）
```
