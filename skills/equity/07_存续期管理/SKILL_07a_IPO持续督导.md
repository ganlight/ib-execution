# SKILL_07a：IPO持续督导

## 一、概述

IPO持续督导是保荐人在发行人上市后需履行的持续监督和指导义务。根据《证券发行上市保荐业务管理办法》，持续督导期间为上市当年剩余时间及其后2个完整会计年度（创业板/科创版为3个完整会计年度）。持续督导工作的质量直接影响保荐人的执业评价和后续项目承揽。

---

## 二、督导期的法律要求

### 2.1 督导期限

| 上市板块 | 督导期 | 说明 |
|:---|:---|:---|
| 主板 | 上市当年+2个完整会计年度 | 含上市当年剩余部分 |
| 科创板/创业板 | 上市当年+3个完整会计年度 | 含上市当年剩余部分 |
| 北交所 | 上市当年+2个完整会计年度 | 含上市当年剩余部分 |

**督导期起算**：上市之日（T日）

**督导期届满判断**：
- 示例：2025年7月1日上市 → 督导期至2027年12月31日（主板）届满
- 示例：2025年7月1日上市（科创板）→ 督导期至2028年12月31日届满

### 2.2 督导期届满的工作移交

```
督导期届满前30日：
  □ 确认是否需要申请延期（如发行人仍存在重大未了事项）
  □ 如无特殊情况，向发行人出具《督导期届满工作总结》
  □ 与发行人签署《持续督导期届满交接备忘录》
  □ 将督导工作底稿归档保存（保存期限≥10年）
  □ 通知交易所督导期届满

延长督导的情形（需向监管报告）：
  - 发行人在督导期内存在重大违规行为
  - 募集资金使用严重不合规
  - 存在尚未完结的监管调查/处罚
  - 保荐代表人发生变更，新任代表人需重新计算督导期
```

### 2.3 保荐代表人的持续履职要求

```
督导期内保荐代表人的核心义务：
  ├── 在督导工作文件上签字（年度督导报告、专项核查意见等）
  ├── 出席发行人年度股东大会（通常须现场出席）
  ├── 对重大事项出具核查意见并签字
  ├── 每年接受保荐机构内部培训（≥15学时）
  └── 如离职须提前通知发行人并完成交接，不得在未完成交接前擅自离职
```

## 三、定期督导内容

### 3.1 年度报告审阅

**时限要求**：年报公告后15个工作日内出具年度督导报告

**审阅重点**：
- 财务数据的合理性（同比、环比分析）
- 与招股书预测数据的对比
- 审计意见类型（是否仍为标准无保留）
- 关联交易及定价公允性
- 募集资金使用情况
- 内部控制情况（内控鉴证报告是否继续出具）
- 重大承诺事项的履行进展
- 是否存在应披露未披露的重大事项

**审阅记录要求**：
- 对年报逐项审阅，制作审阅工作底稿
- 记录审阅发现和分析判断
- 对异常事项形成专项核查底稿

### 3.2 半年度报告审阅

**时限要求**：半年报公告前协助审阅

**审阅重点**：
- 中期财务数据的合理性
- 与上年同期的比较分析
- 募集资金半年度使用进展
- 半年度期间发生的重大事项

## 四、募集资金使用监控

### 4.1 募集资金专项账户管理

- 确认募集资金已存入专项账户
- 募集资金的支出经过规定的审批程序
- 不存在违规挪用募集资金的情况
- 临时补充流动资金是否符合规定（不得超过12个月，不得影响募投项目正常进行）

### 4.2 募投项目实施进度监控

**监控内容**：

| 监控维度 | 数据来源 | 异常判断标准 |
|---------|---------|-------------|
| 投资进度 | 募集资金使用台账 | 实际投入远低于计划进度 |
| 建设进度 | 项目现场考察 | 工期延误超过6个月 |
| 预期效益 | 投产后财务数据 | 实际效益低于可行性研究报告预期50%以上 |
| 项目变更 | 发行人公告/会议 | 募投项目变更或终止 |

### 4.3 预算执行vs实际使用的差异分析

```
月度对比分析模板：

| 募投项目 | 预算金额 | 累计计划投入 | 累计实际投入 | 差异金额 | 差异率 | 原因分析 |
|---------|---------|------------|------------|---------|-------|---------|
| 项目A | 30,000万 | 20,000万 | 16,500万 | -3,500万 | -17.5% | 设备采购延迟 |
| 项目B | 20,000万 | 10,000万 | 10,800万 | +800万 | +8.0% | 提前启动 |
| 合计 | 50,000万 | 30,000万 | 27,300万 | -2,700万 | -9.0% | |

差异分析：
  · 偏差率<5%：正常，无需特别关注
  · 偏差率5%-15%：关注，分析原因
  · 偏差率>15%：重大偏差，评估是否触发变更程序 + 是否影响预期效益
```

### 4.4 募集资金使用变更程序

如发生以下情形，需要履行变更程序：
- 取消或终止原募集资金投资项目，实施新项目
- 变更募集资金投资项目实施主体
- 变更募集资金投资项目实施方式
- 募集资金用途变更超过募集资金净额的一定比例

变更程序：
1. 董事会审议
2. 独立董事、监事会、保荐人发表意见
3. 股东大会审议通过
4. 公告相关决议和意见

### 4.5 结余资金使用决策步骤

```
步骤1：判断是否为节余资金
  · 节余资金 = 募集资金净额 - 已投入金额 - 永久补流金额 - 暂时补流未还金额
  · 是否>募集资金净额10%（部分板块要求>10%才可永久补流）

步骤2：制定使用方案
  · 方案A：永久补充流动资金（最常见）
  · 方案B：用于其他募投项目
  · 方案C：用于新项目投资

步骤3：履行审批程序
  · 董事会审议 + 独董意见 + 监事会意见
  · 保荐人核查意见
  · 如>净额10%须提交股东大会审议

步骤4：信息披露
  · 董事会决议后2个交易日内公告

步骤5：执行
  · 将节余资金从专项账户转入一般账户
  · 按方案使用资金
```

### 4.6 募集资金专项核查报告

- **频率**：每半年出具一次
- **时间**：中期报告/年度报告公告时同步披露
- **内容**：募集资金基本情况和存储情况、本年度/半年度使用情况、募集资金投资项目进展情况、变更情况、保荐人核查意见

## 五、关联交易监控

### 5.1 需关注的关联交易

| 交易类型 | 关注阈值 | 核查重点 |
|---------|---------|---------|
| 日常关联交易（采购/销售/服务） | 超出年度预计额度 | 是否重新履行审批程序 |
| 资产收购/出售 | >净资产5% 或 >3000万 | 评估公允性、必要性、决策程序 |
| 对外担保 | 任何关联担保 | 反担保安排、决策程序、是否损害上市公司利益 |
| 资金拆借 | 任何资金往来 | 利率公允性、是否构成资金占用 |
| 共同投资 | 与关联方共同设立公司 | 是否构成同业竞争、决策程序 |

### 5.2 关联交易审查程序

1. 审查交易的商业必要性（是否具备独立商业目的）
2. 价格公允性分析（与市场第三方价格对比）
3. 决策程序的合规性（回避表决、独董事前认可）
4. 信息披露的充分性

## 六、重大事项关注

### 6.1 应关注的重大事项类别

根据《证券发行上市保荐业务管理办法》，以下情形属于需要特别关注的重大事项：

| 序号 | 事项 | 关注要点 |
|-----|------|---------|
| 1 | 经营业绩大幅下滑（营收/净利润下降>50%） | 查明原因，判断是临时性还是持续性 |
| 2 | 重大资产重组/再融资 | 审阅方案，出具核查意见 |
| 3 | 关联交易超出预计额度 | 督促履行程序 |
| 4 | 为他人提供担保（>净资产10%） | 评估担保风险 |
| 5 | 重大诉讼/仲裁（>净资产10%） | 评估影响 |
| 6 | 控股股东/实际控制人/董监高被立案调查 | 评估影响及披露充分性 |
| 7 | 发生重大亏损或重大损失 | 评估原因及影响 |
| 8 | 募集资金投向变更 | 督促履行程序 |
| 9 | 产业政策重大变化 | 评估对公司经营的实质影响 |
| 10 | 核心技术人员/高管离职 | 评估是否影响持续经营 |

## 七、现场检查

### 7.1 基本要求

- **频率**：每年至少1次
- **时间**：通常在年度报告审计期间进行
- **内容**：
  - 查阅公司治理、内控制度运行记录
  - 查阅募集资金使用凭证和台账
  - 实地查看募投项目进度
  - 访谈管理层及相关人员
  - 查阅重大合同和关联交易审批文件

### 7.2 现场检查报告

现场检查后应出具报告，内容包括：
- 检查时间、检查人员、检查内容
- 检查发现的问题
- 发行人整改措施（如有问题）
- 核查结论

### 7.3 现场检查详细流程

```
检查前准备（T-7至T-3）：
  □ 与发行人协调检查时间和行程
  □ 准备检查清单（通用清单+针对本项目的个性化清单）
  □ 提前通知发行人需准备的资料
  □ 安排参与人员分工

现场实施（通常1-2天）：
  □ 首次会议：说明检查目的、范围、方式
  □ 查阅资料：
      - 治理文件（三会记录、决议文件）
      - 财务凭证（重点抽查募集资金使用凭证）
      - 合同档案（关联交易合同、对外担保合同）
      - 内控制度文件
      - 募集资金使用台账
  □ 实地走访：
      - 募投项目现场（工程进度、设备安装情况）
      - 办公场所（经营状况直观感受）
  □ 管理层访谈：
      - 董事长/总经理（经营情况、发展战略）
      - CFO（财务情况、募资使用）
      - 董事会秘书（信息披露合规性）
      - 核心部门负责人
  □ 末次会议：初步反馈发现

检查后工作（T+7内）：
  □ 撰写现场检查报告（5个工作日内）
  □ 通知发行人检查发现（如有）
  □ 督促整改（如有问题）
  □ 底稿归档
```

## 八、年度保荐工作报告的撰写标准

### 8.1 证监会模板要求（7个核心部分）

根据《证券发行上市保荐业务管理办法》及证监会相关指引，年度督导报告须包含以下内容：

```
第一部分：发行人基本情况及持续督导期内概况
  - 发行人基本信息（股票代码、简称、上市日期）
  - 持续督导期（起止日期）
  - 督导期内发行人重大事项概述

第二部分：保荐人对上市公司信息披露审阅的情况
  - 年度报告/半年度报告审阅情况
  - 重大事项临时公告审阅情况
  - 信息披露的真实性、准确性、完整性判断

第三部分：保荐人对募集资金使用情况的核查意见
  - 募集资金存储情况
  - 本年度使用情况
  - 项目实施进度
  - 是否存在违规使用情形

第四部分：保荐人对关联交易和对外担保的核查意见
  - 关联交易情况（是否公允、是否履行程序）
  - 对外担保情况（是否合规、是否存在风险）

第五部分：保荐人对上市公司收购兼并（如有）的核查意见
  - 督导期内是否发生重大收购兼并事项
  - 相关决策程序和披露情况

第六部分：保荐人对承诺事项履行情况的核查意见
  - 发行人及控股股东/实际控制人的承诺
  - 承诺履行情况
  - 未履行承诺的处理

第七部分：保荐结论
  - 本年度持续督导工作结论
  - 关注的主要风险事项
  - 下一年度持续督导重点
```

### 8.2 撰写时间管理

```
年报披露前：
  □ 与发行人建立年报审阅计划
  □ 提前了解审计进度和可能出具非标意见的风险

年报披露后（T+15个工作日内）：
  □ 立即启动年度督导报告撰写
  □ 协调各中介核查意见（律师意见、会计师意见等）
  □ 发行人确认事实性内容
  □ 保荐人内核审核
  □ 提交交易所
```

## 九、减持/限售股解禁安排

### 9.1 减持相关法规体系

```
主要法规：
  - 《上市公司股东、董监高减持股份的若干规定》（证监会公告〔2017〕9号）
  - 《上市公司创业投资基金股东减持股份的特别规定》（2020年修订）
  - 各交易所《上市公司股东及董事、监事、高级管理人员减持股份实施细则》
```

### 9.2 大股东减持预披露要求

```
预披露时点：计划减持前15个交易日（集中竞价）/ 3个交易日（大宗交易）

预披露内容：
  · 减持股份来源
  · 减持原因
  · 减持期间（不超过6个月，集中竞价）
  · 减持方式（集中竞价/大宗交易/协议转让）
  · 减持价格区间
  · 减持数量上限

保荐人核查职责：
  □ 核查是否处于限售期内
  □ 核查是否符合减持比例限制
  □ 核查预披露公告是否合规
  □ 如存在违规减持情形，及时向监管报告
```

### 9.3 限售期与减持限制

| 股东类型 | 限售期 | 减持限制 | 备注 |
|:---|:---|:---|:---|
| 控股股东/实控人 | 上市后36个月 | 解除限售后任意12个月≤2% | 锁定期满后两年内减持须提前3个工作日披露 |
| IPO前其他股东 | 12个月 | 解除限售后任意12个月≤2% | — |
| IPO战略投资者 | 12个月 | 按协议约定 | — |
| 董监高 | 任职期间每年≤25% | 任期内离职须继续遵守 | 离职后6个月内不得转让 |
| 科创板/创业板特定股东 | 按各板块规定 | 较主板更严格 | 询价转让/配股等方式 |

### 9.4 保荐人对减持的监管配合

```
减持期间，保荐人须：
  □ 提前提醒发行人/大股东减持相关规则
  □ 核查减持是否符合预披露计划
  □ 如实控人/大股东拟通过大宗交易减持，提前与交易所沟通
  □ 对是否存在"绕道减持"（如协议转让给关联方后通过二级市场减持）保持警惕
  □ 如发现违规减持，及时向证监会/交易所报告
```

### 10.2 问题分类追踪

**A类：募集资金使用问题**

```
| 子项 | 描述 | 整改措施 | 完成标准 | 期限 |
|:---|:---|:---|:---|:---|

```

```

|:---|:---|:---|:---|:---|
| 三会记录缺失 | 某次董事会会议记录不完整 | 补充完善会议记录 | 记录完整存档 | 5个工作日 |
| 内控缺陷 | 某项内控制度未有效执行 | 修订制度+培训+内审跟踪 | 整改报告 | 30个工作日 |
```

### 10.3 整改验收标准

```
每项问题的整改验收须满足以下条件：
  ① 整改措施已全部执行（附执行证明文件）
  ② 整改效果已通过验证（如：台账已与银行流水核对无误）
  ③ 同一问题未再发生（观察期：至少3个月）
  ④ 相关制度已完善（如涉及制度修订）
  ⑤ 整改结论经督导项目组负责人确认
```

## 十二、保荐代表人变更

### 12.1 变更情形

- 保荐代表人离职
- 保荐代表人因违规被处罚，不能继续履职
- 保荐代表人岗位调整
- 发行人要求变更（需合理理由）

### 12.2 变更程序

1. 保荐人内部确认新任保荐代表人
2. 原保荐代表人与新任保荐代表人做好工作交接
3. 保荐人出具《变更保荐代表人说明》
4. 通知发行人并公告
5. 新任保荐代表人签署新的持续督导文件

### 12.3 交接清单

```
交接清单：
  □ 项目基本信息（发行人、股票代码、上市日期、督导到期日）
  □ 督导进展概述（已完成事项、待办事项）
  □ 底稿档案位置和查阅方式
  □ 重大未了事项（问题追踪表、待核查事项）
  □ 联系人信息（发行人董秘/财务总监联系方式）
  □ 重要文件（近一年的督导报告、核查意见、底稿目录索引）
  □ 特殊注意事项（已知风险点、特殊安排）
```
