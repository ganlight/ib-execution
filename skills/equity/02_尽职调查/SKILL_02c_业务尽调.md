# SKILL_02c：业务尽调

## 概述

业务尽职调查回答"公司是否具有持续经营能力和成长性"这一核心问题。业务DD与财务DD、法律DD相互印证——财务发现必须从业务层面找到原因，业务判断也需要财务数据支撑。

---

## 一、行业分析

### 1.1 分析框架：PEST + 波特五力

| 分析维度 | 核心问题 | 数据来源 |
|----------|----------|----------|
| **政策(P)** | 行业是否受鼓励/限制？核心政策变化趋势？ | 国家发改委产业目录/各部委政策文件 |
| **经济(E)** | 行业与经济周期关系？下游需求趋势？ | 国家统计局/行业协会 |
| **社会(S)** | 人口/消费习惯/社会观念对行业的影响？ | 行业研究报告 |
| **技术(T)** | 技术替代风险？行业技术迭代周期？ | 专利数据库/技术论文 |
| **竞争格局** | 市场集中度(CR3/CR5)？公司在行业中位置？ | 行业报告/上市公司年报 |
| **进入壁垒** | 资金/技术/牌照/品牌壁垒高度？ | 行业对比分析 |

### 1.2 行业生命周期阶段判断

| 阶段 | 特征 | 典型行业示例 |
|------|------|-------------|
| 导入期 | 增速高但基数小、技术路线不确定、亏损普遍 | 量子计算、固态电池 |
| 成长期 | 增速>20%、竞争加剧、模式快速迭代 | AI应用、创新药 |
| 成熟期 | 增速5-15%、格局稳定、龙头市占率提升 | 白电、水泥、银行 |
| 衰退期 | 增速<5%或负增长、产能过剩、企业退出 | 报刊、传呼机 |

### 1.3 🔴 红旗指标

- 行业处于明显的衰退期且无转型升级路径
- 核心政策发生不利变化（如教育双减/游戏版号限制）
- 行业市场空间<100亿元且增速<5%
- 行业CR3>80%且公司在CR3之外（马太效应挤压）
- 替代技术/产品已出现且性能价格比接近

## 二、技术与研发

### 2.1 核心技术核查

| 核查维度 | 核查要点 |
|----------|----------|
| 技术来源 | 自主研发/合作研发/购买/许可？ |
| 技术独立性 | 是否依赖第三方技术授权？授权到期时间？ |
| 技术壁垒 | 壁垒高度（专利封锁/工艺秘密/先发优势） |
| 技术迭代 | 公司在下一代技术路线中是否有布局？ |
| 人才储备 | 核心技术人员履历、竞业/保密协议、离职率 |

### 2.2 专利组合分析

| 专利类型 | 核查内容 |
|----------|----------|
| 发明专利 | 含金量最高，关注授权数量、保护范围、剩余期限 |
| 实用新型 | 含金量中等，关注是否形成专利族 |
| 外观设计 | 含金量较低（科创属性评估中权重低） |
| PCT国际专利 | 体现国际竞争力 |
| 专利许可 | 许可费收入/支出、排他性、地域限制 |
| 专利诉讼 | 是否涉诉、诉讼结果对业务的影响 |

### 2.3 R&D团队评估

| 指标 | 参考标准 | 
|------|----------|
| 研发人员占比 | 科创板要求≥10%，高新企业≥10% |
| 核心技术人员 | 至少2-3名在行业中具有知名度 |
| 研发人员流失率 | <15% 为健康，>30% 需关注 |
| 研发薪酬竞争力 | 与同行业/同地区对比 |
| 研发组织架构 | 是否有清晰的研发决策和激励机制 |

### 2.4 🔴 红旗指标

- 核心技术依赖第三方（非自主）且许可即将到期
- 核心专利被宣告无效/存在权属争议
- 核心技术人员近1年大量流失（>30%）
- 研发投入占收入比<3%（科技型企业<5%）
- 研发成果转化率持续走低
- 在下一代技术路线中完全没有布局

## 三、采购与供应商

### 3.1 供应商分析

| 分析维度 | 要点 |
|----------|------|
| 前5大供应商 | 采购金额占比、合作年限、是否为关联方 |
| 单一依赖 | 是否存在单一供应商采购占比>50% |
| 原材料价格 | 主要原材料价格波动趋势及对毛利率影响 |
| 供应商稳定 | 前5大供应商近3年变化情况 |
| 供应商集中度 | 行业特征决定还是公司被动选择 |
| 进口依赖 | 是否存在关键原材料/零部件进口受限 |

### 3.2 🔴 红旗指标

- 单一供应商采购占比>50%且无替代方案
- 关键原材料/零部件严重依赖进口且存在贸易限制
- 主要供应商为关联方且采购价格不公允
- 前5大供应商近3年更换>3家
- 原材料占成本>70%而价格波动剧烈
- 供应商与客户重叠（既是供应商又是客户）

## 四、销售与客户

### 4.1 客户分析

| 分析维度 | 要点 |
|----------|------|
| 前5大客户 | 销售金额占比、合作年限、是否为关联方 |
| 单一依赖 | 单一客户收入占比>50% — 重大依赖 |
| 客户质量 | 客户方是否为知名企业/上市公司 |
| 客户稳定 | 前5大客户近3年变化情况 |
| 经销商体系 | 经销模式占比、经销商数量及分布、退换货 |
| 销售模式 | 直销/经销/ODM/OEM/代理 |

### 4.2 经销模式专项

| 核查要点 | 方法 |
|----------|------|
| 经销商是否买断 | 物权和风险是否转移 |
| 最终销售实现 | 走访经销商+终端客户 |
| 退换货政策 | 退换货比例是否异常 |
| 经销商关联关系 | 核查经销商股东/高管与发行人关系 |
| 经销商库存 | 是否存在压货（渠道库存过高） |
| 新增/退出经销商 | 大量新增或退出需关注 |

### 4.3 🔴 红旗指标

- 单一客户收入占比>50%
- 前5大客户中超过3家为当期新增
- 客户与公司存在关联关系未披露
- 主要客户经营困难/回款持续恶化
- 经销商退换货率>15%
- 经销模式收入确认存在重大判断空间
- 海外销售占比高且依赖单一区域

## 五、生产能力

### 5.1 产能分析

| 指标 | 计算公式 | 健康范围 |
|------|----------|----------|
| 产能利用率 | 实际产量 ÷ 设计产能 | 70%-90% |
| 产销率 | 销量 ÷ 产量 | >90% |
| 产能消化 | 在建产能 ÷ 现有产能 | 与行业增速匹配 |

### 5.2 🔴 红旗指标

- 产能利用率<60%却有大规模扩产计划
- 产销率<80%持续下滑
- 核心生产线老化（使用年限>10年）、维护成本高
- 制造费用占比持续上升
- 产品合格率持续下降
- 关键工序外包严重（独立生产）

## 六、商业模式分析

### 6.1 商业模式拆解

```
采购端 → 生产端 → 销售端 → 盈利模式
   ↓         ↓        ↓        ↓
原材料   工艺路线   定价策略  毛利率
供应商   产能利用   销售模式  净利率
议价力   质量控制   客户结构  现金流
```

### 6.2 商业模式可持续性评估

| 评估维度 | 可持续标志 | 不可持续信号 |
|----------|------------|-------------|
| 客户价值 | 解决真实痛点、客户粘性高 | 靠补贴/低价获客 |
| 盈利模式 | 单位经济模型为正 | 规模不经济、越增长越亏 |
| 护城河 | 品牌/网络效应/专利/规模 | 无差异化、易被替代 |
| 可复制性 | 成功经验可复制到新市场 | 每个客户高度定制 |

## 七、行业专项检查清单

### 7.1 半导体行业

- [ ] Fabless/IDM/Foundry模式判定及模式可持续性
- [ ] 制程节点及与同业对比
- [ ] EDA工具来源（国产化率/进口依赖）
- [ ] 流片代工厂依赖度及产能保障
- [ ] IP授权来源及到期时间
- [ ] 美国出口管制（EAR/BIS）影响评估
- [ ] 晶圆/封测产能锁定量及价格趋势
- [ ] 芯片设计工具/架构是否受EAR管辖

### 7.2 软件/SaaS行业

- [ ] 产品架构（单体/微服务/SaaS/PaaS）
- [ ] ARR/MRR趋势与NRR(Net Revenue Retention)
- [ ] LTV/CAC比值（健康>3x）
- [ ] 客户流失率(Churn Rate)趋势（SaaS年流失<10%为优）
- [ ] 收入构成（License/订阅/实施/维保）
- [ ] 底层技术依赖（数据库/中间件/OS国产化）
- [ ] 信创/国产化替代政策受益程度
- [ ] 开源组件合规性及License风险

### 7.3 医药/生物科技行业

- [ ] 在研管线进度（IND/临床I/II/III/NDA）
- [ ] 核心产品专利到期日及专利挑战
- [ ] 临床试验数据质量（试验设计/P值/不良事件）
- [ ] CMO/CDMO/CRO依赖度
- [ ] 药品注册路径（优先审评/突破性疗法/附条件批准）
- [ ] 集采风险（已集采/可能集采品种）
- [ ] 医保谈判进度及价格降幅预期
- [ ] 生产设施GMP合规认证状态

### 7.4 制造业（传统）

- [ ] 单位生产成本构成及趋势
- [ ] 产能利用率及产业供需格局
- [ ] 原材料价格波动对毛利率的敏感性
- [ ] 能源消耗量及能耗双控影响
- [ ] 环保排放达标及环评批复
- [ ] 安全生产合规及近3年事故记录
- [ ] 自动化/智能制造水平
- [ ] 劳动力成本及招工难度
