# SKILL_02a：法律尽调

## 概述

法律尽职调查围绕**发行人是否合法设立、合法存续、股权清晰、资产权属完整、无重大违法**五大核心展开。法律DD是所有产品类型（IPO/再融资/M&A）的必做项目。本文件按五大板块组织。

---

## 一、发行人主体资格

### 1.1 核查要点

| 核查事项 | 核查内容 | 核查方法 |
|----------|----------|----------|
| 依法设立 | 设立程序、审批、出资是否合规 | 工商档案+设立批文+验资报告 |
| 合法存续 | 是否持续经营、有无吊销/注销风险 | 工商机读档案+年检公示 |
| 股东出资 | 出资是否实缴、形式是否合规 | 验资报告+银行流水+评估报告 |
| 历次变更 | 历次股权变更/增资/改制是否合规 | 工商档案全册+内部决议 |
| 业务资质 | 是否取得经营所需全部许可/资质 | 资质证书原件+主管机关确认 |

### 1.2 文件清单

- [ ] 现行营业执照（正副本）
- [ ] 工商登记档案全册（含历次变更）
- [ ] 设立批文/核准文件
- [ ] 历次验资报告
- [ ] 非货币出资评估报告及过户证明
- [ ] 整体变更设立股份公司的审计/评估/验资报告（如有改制）
- [ ] 国有资产评估核准/备案文件（如涉及）
- [ ] 历次股东会/董事会决议
- [ ] 公司章程及全部修正案
- [ ] 外商投资企业批准证书（如适用）

### 1.3 🔴 Red Flag指标

- 出资不实且金额≥注册资本30%
- 存在抽逃出资情形
- 重要业务资质/许可证已过期或即将到期且续期困难
- 公司被列入经营异常名录≥3年未处理
- 改制程序存在重大瑕疵（如未评估、未职工安置）

## 二、股权结构与实际控制人

### 2.1 核查要点

| 核查事项 | 核查内容 | 核查方法 |
|----------|----------|----------|
| 实际控制人认定 | 穿透后谁真正控制公司 | 股权架构图+一致行动协议+章程 |
| 股权穿透 | 穿透至自然人/国资委 | 股东名册+工商档案+信托契约 |
| 控制权稳定性 | 近3年实际控制人是否变更 | 历次股权变动记录+董事会构成变化 |
| 股权清晰度 | 是否存在代持/纠纷/质押 | 股东访谈+股权代持声明+中登公司质押查询 |
| 特殊架构 | VIE/红筹/信托/A/B股 | 全套VIE协议+开曼股东名册 |

### 2.2 VIE架构专项核查

对于存在VIE架构的发行人（港股常见）：

| 核查文件 | 关注点 |
|----------|--------|
| 独家购买权协议 | 行权条件、对价是否合理 |
| 表决权代理协议 | 不可撤销性、POA期限 |
| 股权质押协议 | 质押登记是否完成 |
| 独家业务合作协议 | 服务费定价、转移定价合规 |
| VIE结构终止/解除条款 | 是否具备境内上市转换路径 |

### 2.3 文件清单

- [ ] 股权架构图（穿透至最终自然人/国资委）
- [ ] 股东名册（加盖公章的最新版本）
- [ ] 控股股东/实际控制人身份证明及简历
- [ ] 一致行动协议/表决权委托协议
- [ ] 股权代持确认函（全体股东签署：确认无代持或披露代持）
- [ ] 股权质押/冻结查询结果
- [ ] VIE全套协议（如适用）
- [ ] 员工持股平台合伙协议及份额明细

### 2.4 🔴 Red Flag指标

- 实际控制人认定不清（尤其是多名股东持股接近且无一致行动协议）
- 近3年实际控制人发生变更（IPO实质性障碍）
- 存在大额股权代持且无法还原
- 控股股东持股被大比例质押（≥50%）
- VIE协议存在被认定无效的风险
- 控制权争夺/未决股权纠纷诉讼

## 三、主要资产

### 3.1 核查领域

| 资产类别 | 核查内容 | 核心风险 |
|----------|----------|----------|
| 土地使用权 | 权证、性质(出让/划拨)、用途合规 | 权属不清/划拨地未出让/集体土地 |
| 房产 | 权证、建设审批、抵押情况 | 违建/权证不齐/被抵押 |
| 知识产权 | 专利/商标/软著/域名权属 | 职务发明纠纷/核心专利到期/侵权 |
| 机器设备 | 所有权、抵押、融资租赁 | 关键设备被抵押/租赁到期 |

### 3.2 知识产权专项

| 专利类型 | 核查要点 | 文件要求 |
|----------|----------|----------|
| 自有专利 | 权属证书、年费缴纳、剩余保护期 | 专利登记簿副本 |
| 受让专利 | 转让合同+登记公告+定价依据 | 转让合同+付款凭证 |
| 许可专利 | 许可合同+备案+范围限制 | 许可合同+备案证明 |
| 职务发明 | 发明人是否前雇主、竞业限制 | 发明人声明+劳动合同 |
| 共有专利 | 共有协议+处置权限 | 共有合同 |

### 3.3 文件清单

- [ ] 不动产权证书/土地使用权证/房屋所有权证
- [ ] 土地出让合同+出让金缴纳凭证
- [ ] 建设用地规划许可证+建设工程规划许可证
- [ ] 竣工验收备案文件
- [ ] 专利证书+最近1年年费缴纳凭证
- [ ] 专利登记簿副本（最近2个月内）
- [ ] 商标注册证
- [ ] 软件著作权登记证书
- [ ] 域名注册证书
- [ ] 知识产权许可/转让合同（如有）
- [ ] 主要固定资产盘点表

## 四、重大合同

### 4.1 合同审查清单

| 合同类别 | 审查内容 | 合同金额阈值 |
|----------|----------|-------------|
| 银行借款合同 | 金额/利率/期限/担保/限制性条款 | 全部 |
| 担保合同 | 担保方式/金额/期间/反担保 | 全部对外担保 |
| 采购框架合同 | 定价机制/排他性/期限/违约 | 年度采购额≥500万元 |
| 销售框架合同 | 定价/返利/排他性/回款 | 年度销售额≥500万元 |
| 建设工程合同 | 总价/工期/变更/质保 | 合同金额≥500万元 |
| 合资/合作协议 | 出资/治理/退出/不竞争 | 全部 |
| 技术许可合同 | 范围/排他性/期限/支付 | 年度费用≥100万元 |

### 4.2 文件清单

- [ ] 全部银行借款合同及补充协议
- [ ] 授信协议/最高额保证合同
- [ ] 对外担保合同（含子公司间担保）
- [ ] 前5大供应商采购合同
- [ ] 前5大客户销售合同
- [ ] 建设工程施工合同
- [ ] 重大资产购买/处置合同（≥净资产的10%）
- [ ] 合资/合作协议
- [ ] 股权转让/增资协议（历次）
- [ ] 资产/业务重组协议
- [ ] 董事/高管聘用合同

## 五、诉讼、仲裁与行政处罚

### 5.1 核查维度

| 类别 | 核查方法 | 关注重点 |
|------|----------|----------|
| 未决诉讼/仲裁 | 公司披露+裁判文书网+天眼查+信用中国 | 金额/性质/进展/不利结果概率 |
| 行政处罚 | 各部门官网+信用中国+公司披露 | 是否构成重大违法 |
| 被执行/失信 | 中国执行信息公开网+失信名单 | 金额+原因+履行能力 |
| 税务处罚 | 税务局查询+完税证明 | 虚开/偷税/抗税 — 一票否决 |

### 5.2 重大诉讼判定标准

| 判定维度 | 量化标准（参考） |
|----------|-----------------|
| 涉案金额 | ≥最近一期经审计净资产的5% 或 ≥500万元 |
| 案件性质 | 涉及核心专利/商标权属争议 |
| 案件影响 | 可能导致公司无法持续经营 |
| 处罚性质 | 责令停产停业/吊销许可证/大额罚款(≥100万) |

### 5.3 文件清单

- [ ] 未决诉讼/仲裁清单（含案件信息、金额、进展）
- [ ] 重大案件起诉状/答辩状/裁判文书
- [ ] 行政处罚决定书及缴款凭证
- [ ] 公司及其子公司信用报告
- [ ] 完税证明（最近3年+1期）
- [ ] 环保合规证明
- [ ] 安全生产合规证明
- [ ] 实际控制人/董监高无犯罪记录证明

### 5.4 🔴 Red Flag指标

- 核心专利/商标存在权属争议且败诉风险大
- 实际控制人/高管存在刑事犯罪记录
- 最近36个月内受刑事处罚或重大行政处罚
- 存在未决诉讼合计金额≥净资产的20%
- 税务等级为D级或被列入税收违法黑名单
- 存在环保/安全重大事故处罚

## 六、法律DD工作流程

```
Day 1-2:  发出文件清单 + 开设数据室
Day 3-7:  第一轮资料审阅 + 管理层访谈 (法务/董秘)
Day 5-10: 工商/诉讼/知识产权公开检索
Day 8-14: 起草资料缺失清单 + 回复客户问题
Day 12-18: 第二轮补充资料 + 重点问题专题研究
Day 15-21: 关联方核查 + 重大合同审阅
Day 18-25: 起草法律DD问题清单
Day 22-30: 问题讨论与整改方案
Day 28-35: 撰写法律DD报告初稿
Day 32-40: 报告内部审阅 + 定稿
```
