# SKILL_05a：IPO申报材料清单

## 一、概述

IPO申报文件是向交易所和证监会提交的全套法定文件。根据《首次公开发行股票注册管理办法》和各板块上市审核规则，申报文件涵盖招股说明书、发行人申请文件、保荐人文件、会计师文件、律师文件等多个大类，按交易所申报目录分为五大类（含35项具体文件）。

本清单基于**全面注册制**下的最新要求编制，每个文件标注**必须/可选**、**份数**、**电子版要求**和**易错点**，各板块差异处另行标注。

---

## 二、完整申报文件清单

---

### 第一类：申请文件（6项）

#### 1-1 招股说明书（申报稿） 🔴 必须

| 属性 | 内容 |
|------|------|
| **出具方** | 发行人（券商主办撰写） |
| **签字要求** | 发行人 + 全体董事、监事、高级管理人员签字 |
| **盖章要求** | 发行人公章 |
| **份数** | 电子版1份（通过系统上传）；纸质版3-5份（按交易所要求） |
| **电子版** | ✅ 必须提供PDF电子版，含书签目录导航 |
| **格式** | PDF + Word（系统上传需求） |

**📌 易错点**：
- ❌ 部分董事/监事未在签字页上签字（仅盖章代替）→ 必须全员手签
- ❌ 签字页日期早于内核通过日期或董事会决议日期 → 所有签字日期须在内部审批完毕后
- ❌ 电子版PDF与纸质版内容不一致（修订痕迹未清理）→ 申报前双人交叉比对
- ❌ 招股书中的募集资金总额与募投项目可研报告不一致 → 逐项核对
- ❌ 重大事项提示和风险因素章节未及时更新（沿用旧版本）→ 申报前专项审查该两章

---

#### 1-2 发行人关于本次发行的申请报告 🔴 必须

| 属性 | 内容 |
|------|------|
| **出具方** | 发行人 |
| **格式** | 红头文件（公司正式编号文件） |
| **签字要求** | 法定代表人签字 |
| **盖章要求** | 发行人公章 |
| **份数** | 纸质版3份 + 电子版1份 |
| **电子版** | ✅ PDF扫描件（含红头格式） |

**📌 易错点**：
- ❌ 红头文件编号跨年度重复 → 确认公司公文编号唯一性
- ❌ 申请报告中的发行方案与董事会决议不一致 → 三文件逐项比对（申请报告/董事会决议/招股书方案章节）
- ❌ 未注明申报板块 → 必须明确写明"拟在上海证券交易所主板上市"或对应板块

---

#### 1-3 保荐机构推荐文件 🔴 必须

保荐机构推荐文件实际包含3个子文件：

| 子文件 | 出具方 | 签字/盖章 | 份数 | 电子版 |
|--------|--------|----------|------|--------|
| 保荐协议 | 保荐人+发行人 | 双方公章+法定代表人 | 原件1份+副本2份 | ✅ PDF |
| 尽调报告/内核意见 | 保荐人 | 保荐代表人+内核负责人签字 | 3份 | ✅ PDF |
| 保荐代表人专项授权书 | 保荐人 | 法定代表人签字+公章 | 原件1份+副本2份 | ✅ PDF |

**📌 易错点**：
- ❌ 保荐协议签署日期早于立项日期 → 保荐协议签署须在立项之后
- ❌ 专项授权书上的保荐代表人姓名与申报系统登记不一致 → 提前核实中证协从业人员登记信息
- ❌ 尽调报告中提及的核查程序在实际底稿中未找到对应记录 → 每条核查结论必须有底稿支撑
- ❌ 内核意见中"必须落实事项"在申报文件中尚未落实 → 内核后须逐条核对落实，出具落实说明

---

#### 1-4 募投项目相关文件 🔴 必须

| 子文件 | 出具方 | 份数 | 电子版 | 备注 |
|--------|--------|------|--------|------|
| 项目备案/核准文件 | 发改/经信部门 | 每项目1份原件+2份副本 | ✅ PDF | 须在有效期内 |
| 环境影响评价批复/登记表 | 生态环境部门 | 每项目1份原件+2份副本 | ✅ PDF | 视项目环评分类等级 |
| 土地使用权证/用地预审意见 | 自然资源部门 | 每项目1份原件+2份副本 | ✅ PDF | 新增用地必须 |
| 募投项目可行性研究报告 | 发行人/第三方 | 3份 | ✅ PDF | 与招股书募投章节一致 |

**📌 易错点**：
- ❌ 项目备案证上的项目名称/投资总额与招股书不一致 → 备案变更后重新出具备案证
- ❌ 环评批复的项目建设地点与实际用地位置不符 → 逐一核查地理坐标
- ❌ 募投项目用地的规划用途与项目建设内容冲突（工业用地用于商业用途等）→ 核查用地规划许可证
- ❌ 境外募投项目未取得国家发改委ODI备案/发改委境外投资批复 → 境外项目需补充ODI文件

---

#### 1-5 审计报告（3年+1期）+ 审阅报告 🔴 必须

| 属性 | 内容 |
|------|------|
| **出具方** | 会计师事务所 |
| **签字要求** | 两名签字注册会计师签字盖章 |
| **盖章** | 事务所公章 |
| **份数** | 3份 |
| **电子版** | ✅ PDF |
| **时效性** | 审计报告基准日距申报日不得超过6个月 |

**📌 易错点**：
- ❌ 审计报告基准日超过6个月（最常见的退回原因之一）→ 申报前45天检查有效期
- ❌ 签字注册会计师与系统登记不一致 → 与中注协系统核对
- ❌ 审阅报告期间覆盖不完全（中期审阅未覆盖最近一期期末）→ 确认审阅期间与最近一期财务数据一致
- ❌ 审计报告中的关键审计事项与招股书风险披露不一致 → 确保互相呼应
- ❌ 审计报告中会计政策变更未在招股书"财务会计信息"章节同步披露 → 双重核对比对

---

#### 1-6 法律意见书 + 律师工作报告 🔴 必须

| 子文件 | 出具方 | 签字要求 | 份数 | 电子版 |
|--------|--------|----------|------|--------|
| 法律意见书 | 律所 | 两名签字律师+事务所公章 | 3份 | ✅ PDF |
| 律师工作报告 | 律所 | 两名签字律师+事务所公章 | 3份 | ✅ PDF |

**📌 易错点**：
- ❌ 法律意见书中网络核查底稿与意见书结论不一致 → 逐条核对核查底稿
- ❌ 法律意见书的出具日期与签字律师执业证有效期冲突 → 确认律师执业证未过期
- ❌ 律师工作报告中"未发现"与法律意见书中"不存在"表述不一致 → 统一法律意见表述口径

### 第二类：发行人信息（11项）

#### 2-1 公司设立及历次变更的政府批准文件 🔴 必须

| 属性 | 内容 |
|------|------|
| **包含** | 设立批文、历次增资扩股批文、历次股权转让批文、工商变更登记证明 |
| **份数** | 1套（原件备查，申报提交复印件） |
| **电子版** | ✅ PDF扫描件 |
| **备注** | 外商投资企业含商务部门批文；国有企业含国资监管部门批文 |

**📌 易错点**：
- ❌ 缺少某次股权变更的工商变更登记证明 → 工商档案调取需覆盖自设立至今全部变更记录
- ❌ 国有股东股权变动缺少国资评估备案文件 → 涉及国有资产须有评估备案/核准文件
- ❌ 外商投资企业批准证书已过期 → 确认《外商投资法》实施后的衔接程序

---

#### 2-2 营业执照及公司章程 🔴 必须

| 属性 | 内容 |
|------|------|
| **包含** | 现行有效营业执照（副本）+ 现行公司章程 + 章程修正案全集 |
| **份数** | 3份 |
| **电子版** | ✅ PDF扫描件 |

**📌 易错点**：
- ❌ 公司章程与工商备案版本不一致 → 提交工商备案的最新版本
- ❌ 营业执照经营范围与招股书描述的主营业务不完全匹配 → 经营范围须覆盖主营业务

---

#### 2-3 发起人协议/创立大会文件 🔴 必须

| 属性 | 内容 |
|------|------|
| **包含** | 发起人协议、创立大会通知/签到/表决/决议/记录全套文件 |
| **份数** | 3份 |
| **电子版** | ✅ PDF扫描件 |

**📌 易错点**：
- ❌ 创立大会到会发起人所持表决权比例不足 → 确认法律规定的有效表决基数
- ❌ 发起人协议签署主体与最终股东名册不一致 → 逐一核实签署主体

---

#### 2-4 股东大会/董事会/监事会议事规则及决议 🔴 必须

| 属性 | 内容 |
|------|------|
| **包含** | 三会议事规则 + 报告期内历次三会决议全套（会议通知/签到/表决票/决议/记录） |
| **份数** | 3份（可提交目录+关键节点决议，全套备查） |
| **电子版** | ✅ PDF扫描件 |

**📌 易错点**：
- ❌ 决议文件页码不连续，存在跳页 → 逐页编号并核页码
- ❌ 线上会议表决记录与纸质决议签字日期矛盾 → 线上线下记录须一致
- ❌ 内控鉴证报告中提及的董事会/股东大会程序瑕疵在决议文件中未做说明 → 瑕疵事项应有补充说明或重新决议记录

---

#### 2-5 内部控制鉴证报告 🔴 必须

| 属性 | 内容 |
|------|------|
| **出具方** | 会计师事务所 |
| **签字** | 两名签字注册会计师 |
| **基准日** | 必须与审计报告基准日一致 |
| **份数** | 3份 |
| **电子版** | ✅ PDF |

**📌 易错点**：
- ❌ 内控鉴证报告基准日与审计报告不一致 → 两者基准日必须同一天
- ❌ 内控鉴证结论中有"带强调事项段"未在招股风险因素中披露 → 强调事项必须同步披露

---

#### 2-6 业务资质/许可证 🔴 必须

| 属性 | 内容 |
|------|------|
| **包含** | 行业经营许可证/特许经营证书/生产经营许可证/资质证书等全部行政许可文件 |
| **份数** | 1套（每件原件备查 + 3份复印件） |
| **电子版** | ✅ PDF扫描件 |

**📌 易错点**：
- ❌ 核心业务资质已过期或临近到期（距到期不足6个月）→ 提前启动续期；评估续期风险
- ❌ 资质证书持证主体与发行人名称不一致（子公司持有）→ 明确资质归属和使用合规性
- ❌ 招股书中声称拥有某资质但证书未出现在文件中 → 逐项对号

---

#### 2-7 重要业务合同（含前5大客户/供应商合同） 🔴 必须

| 属性 | 内容 |
|------|------|
| **包含** | 报告期内前5大客户销售合同/订单/框架协议 + 前5大供应商采购合同 + 重大合作/战略协议 |
| **份数** | 每类3份（可提交代表性合同和合同摘要表） |
| **电子版** | ✅ PDF扫描件（可脱敏处理） |

**📌 易错点**：
- ❌ 提交的合同与招股书中披露的客户/供应商排名不一致 → 排名必须以合同金额/期间交易额为准
- ❌ 合同中的重要商业条款与招股书描述矛盾（赊销期限、定价条款等）→ 逐项比对
- ❌ 前5大客户含关联方但未充分披露关联关系 → 穿透核查最终客户

---

#### 2-8 知识产权证书/许可合同 🔴 必须

| 属性 | 内容 |
|------|------|
| **包含** | 专利证书/登记簿副本 + 商标注册证 + 计算机软件著作权登记证书 + 知识产权许可/转让合同 |
| **份数** | 每类1套（证书清单+关键证书副本） |
| **电子版** | ✅ PDF扫描件 |

**📌 易错点**：
- ❌ 专利年费未按时缴纳导致专利权失效 → 申报前查询专利状态并补缴欠费
- ❌ 核心专利存在质押/许可第三方未披露 → 核查专利登记簿副本
- ❌ 专利用于科创板科创属性论证但保护期不足 → 核心专利应有充分的剩余保护期

---

#### 2-9 土地使用权/房产证/租赁合同 🔴 必须

| 属性 | 内容 |
|------|------|
| **包含** | 土地使用权证/不动产权证 + 房屋所有权证 + 主要经营场所租赁合同 |
| **份数** | 1套（原件备查 + 3份副本） |
| **电子版** | ✅ PDF扫描件 |

**📌 易错点**：
- ❌ 核心生产经营用地存在权属瑕疵（集体土地/划拨地未转性等）→ 权属瑕疵必须在招股书中充分披露
- ❌ 租赁合同未在房屋租赁管理部门办理租赁备案 → 核查备案登记证明
- ❌ 房产/土地存在抵押未在招股书中披露 → 与征信报告交叉验证

---

#### 2-10 关联交易协议 🔴 必须

| 属性 | 内容 |
|------|------|
| **包含** | 报告期内全部关联交易协议（采购/销售/租赁/服务/资金拆借等） |
| **份数** | 1套 |
| **电子版** | ✅ PDF扫描件 |

**📌 易错点**：
- ❌ 招股书披露的关联交易金额与协议金额不一致 → 逐项比对
- ❌ 关联交易协议中未参照市场公允价值设定价格 → 附第三方比价证据
- ❌ 存在资金拆借但未签署资金拆借协议 → 存量资金往来必须有书面协议

---

#### 2-11 员工劳动合同与社会保险缴纳情况 🔴 必须

| 属性 | 内容 |
|------|------|
| **包含** | 代表性劳动合同样本 + 报告期各期末社会保险/住房公积金缴纳凭证 + 劳动监察部门合规证明 |
| **份数** | 1套（缴纳明细表/凭证1套 + 合规证明） |
| **电子版** | ✅ PDF扫描件 |

**📌 易错点**：
- ❌ 社保缴纳人数与年报员工人数明显不符（大量未缴纳）→ 差异须有充分合理解释
- ❌ 劳动合同签署率不足100% → 申报前全部签署完毕
- ❌ 劳务派遣用工比例超过10% → 降至10%以内或提供合理说明

### 第三类：保荐机构文件（8项）

#### 3-1 保荐协议 + 补充协议 🔴 必须

| 属性 | 内容 |
|------|------|
| **出具方** | 保荐人 + 发行人（双边签署） |
| **签字/盖章** | 双方法定代表人签字 + 公章 |
| **份数** | 原件1份 + 副本2份 |
| **电子版** | ✅ PDF扫描件 |

**📌 易错点**：
- ❌ 保荐协议签署日期早于保荐机构立项日期 → 必须在正式立项后签署
- ❌ 保荐费用结构与招股书披露不一致 → 费用须精确到元

---

#### 3-2 保荐机构尽调报告 + 内核会议纪要 🔴 必须

| 属性 | 内容 |
|------|------|
| **出具方** | 保荐人 |
| **签字** | 保荐代表人 + 内核负责人 |
| **包含** | 尽调过程详细记录、尽调结论、内核会议全过程纪要（含表决结果） |
| **份数** | 3份 |
| **电子版** | ✅ PDF |

**📌 易错点**：
- ❌ 内核会议纪要中表决票数与表决规则矛盾（5名委员必须有4票同意）→ 复核表决结果合法性
- ❌ 尽调结论与招股书风险披露口径不一致 → 双文件交叉比对
- ❌ 内核会上提出"必须落实事项"但无后续落实跟踪记录 → 附内核落实报告

---

#### 3-3 保荐代表人专项授权书 🔴 必须

| 属性 | 内容 |
|------|------|
| **出具方** | 保荐人 |
| **签字/盖章** | 法定代表人签字 + 公章 |
| **份数** | 原件1份 + 副本2份 |
| **电子版** | ✅ PDF扫描件 |

**📌 易错点**：
- ❌ 保荐代表人姓名/身份证号/执业编号任何一项错误 → 逐项与中证协系统核对
- ❌ 保荐代表人曾受处罚但未在授权书中声明 → 被处罚信息必须如实披露

---

#### 3-4 保荐机构对签字会计师/律师的独立核查 🔴 必须

| 属性 | 内容 |
|------|------|
| **出具方** | 保荐人 |
| **内容** | 对会计师事务所/律所及其签字人员的资质、独立性、执业质量的独立核查记录 |
| **份数** | 3份 |
| **电子版** | ✅ PDF |

**📌 易错点**：

- ❌ 会计师/律师变更但核查记录未更新 → 变更后重新核查

## 三、文件格式要求

### 3.1 电子版申报

- 格式：PDF文件（含书签/目录导航）
- 命名：与文件清单序号对应的规范命名
- 上传：通过交易所审核系统上传
- **特别注意**：科创板/创业板的板块定位文件须在系统中选择对应目录上传

### 3.2 签章要求

| 主体 | 签章要求 |
|------|---------|
| **发行人** | 公章 + 法定代表人签字或盖章 |
| **发行人董监高** | 招股说明书全体签字（含独立董事） |
| **保荐人** | 公章 + 法定代表人、保荐业务负责人、保荐代表人签字 |
| **会计师事务所** | 公章 + 两名签字注册会计师签字盖章 |
| **律师事务所** | 公章 + 两名签字律师签字盖章 |

### 3.3 纸质版要求

- 份数：按交易所要求报送（通常3-5份）
- 装订：A4纸张，胶装成册
- 封面：统一格式（含文件名称、发行人名称、保荐人名称、申报日期）
- 编码：每册独立编页码，签章页单列

## 五、申报系统提交流程

1. 通过交易所审核系统创建项目
2. 逐一上传申报文件（按系统目录结构，五大类对应五个上传目录）
3. 提交受理申请
4. 交易所形式审查（5个工作日内完成）
5. 形式审查通过 → 正式受理 → 披露预披露文件（招股书申报稿）
6. 形式审查补正 → 10个工作日内补正 → 重新提交

## 七、申报文件版本管理

### 7.1 版本命名规范

```
[文件简称]_[版本号]_[日期YYYYMMDD]_[状态]
示例：招股书_V0.5_20250115_内核稿
     招股书_V1.0_20250210_申报稿
     招股书_V1.1_20250601_首轮更新稿

状态标识：
- 初稿/讨论稿：V0.1~V0.4
- 内核稿：V0.5
- 申报稿：V1.0
- 更新稿（问询后）：V1.1, V1.2...
- 上会稿：V2.0
- 注册稿：V3.0
```

### 7.2 版本差异追踪

每次版本更新，须维护一份《版本差异说明》，记录：
- 修改了哪些章节
- 修改原因（内核意见/律师建议/发行人反馈/数据更新）
- 修改涉及的数据/陈述变化
- 是否影响其他文件的对应内容
