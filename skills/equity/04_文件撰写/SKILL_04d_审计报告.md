# SKILL_04d：审计报告核查

## 一、概述

审计报告是IPO/重组项目的财务数据基础。券商作为保荐人/独立财务顾问，虽不直接出具审计报告，但必须对审计报告进行全面核查，确保其作为申报文件基础的可靠性。本技能指导券商如何核查审计报告，而非如何编制审计报告。

## 一（附）、财务报告的法定构成

### A.1 四表一注体系

根据《企业会计准则第30号——财务报表列报》，完整的财务报告由以下部分构成：

| 组成部分 | 核心内容 | 券商核查关注点 |
|---------|---------|--------------|
| **资产负债表** | 资产、负债、所有者权益在某时点的状况 | 资产质量（应收账款可回收性、存货可变现净值、固定资产成新率）；负债完整性（是否存在表外负债） |
| **利润表** | 一定期间的收入、费用、利润 | 收入真实性（无虚增）；费用完整性（无推迟确认）；非经常性损益对利润的影响 |
| **现金流量表** | 经营/投资/筹资活动的现金流入流出 | 经营活动现金流与净利润的匹配度（长期背离警示盈利质量）；筹资活动对银行借款的依赖 |
| **所有者权益变动表** | 所有者权益各项目的期初期末变动 | 资本公积来源（股东入资 vs 评估增值）；未分配利润的积累与分红政策 |
| **附注** | 对四表的详细解释和补充披露 | 附注是勾稽校验的关键来源：会计政策说明、报表项目注释、关联方关系及交易、或有事项、承诺事项、期后事项 |

### A.2 附注的不可替代性

附注不是"补充信息"而是财务报告的**有机组成部分**。券商需核查：
- 附注是否完整覆盖所有报表项目
- 附注中的关联交易、质押担保等敏感信息是否与尽调发现一致
- 分部报告的划分是否合理（不同业务/地区的收入成本分解）
- 会计估计（如坏账计提比例、折旧年限）是否在附注中充分披露

## 二、审计意见类型判断

### 2.1 五种基本类型及其影响

| 审计意见类型 | 判断标识 | 严重程度 | 处理方式 |
|-------------|---------|---------|---------|
| ✅ 标准无保留意见 | "我们认为……公允反映了……" | 无问题 | 直接采用，正常推进 |
| ⚠️ 带强调事项段的无保留意见 | 有强调事项段但无保留结论 | 需审慎评估 | **评估强调事项的性质和影响**，通常需补充核查论证，确保不构成实质性障碍 |
| ❌ 保留意见 | "除……的影响外，公允反映了……" | 严重 | 审计范围受限或会计处理存在错报但非广泛；**需解决导致保留意见的问题后重新审计** |
| ❌ 否定意见 | "由于……未能公允反映……" | 最严重 | 财务报表存在重大且广泛的错报；**发行人根本不具备申报条件** |
| ❌ 无法表示意见 | "我们无法获取充分适当的审计证据……" | 极端严重 | 审计范围严重受限，无法形成审计结论；**不具备申报条件** |

### 2.1（续）各非标意见对IPO的影响深度分析

**保留意见的四种典型场景及处理：**

| 保留原因 | 具体情形 | 恢复路径 |
|---------|---------|---------|
| 审计范围受限 | 境外子公司无法审计、被投资单位不配合 | 消除限制因素后补充审计；或处置受限资产/子公司 |
| 会计政策分歧 | 审计师认为收入确认时点不当 | 按审计师意见调整会计处理，重新出具审计报告 |
| 重大不确定性未充分披露 | 诉讼结果无法估计且披露不充分 | 充实在附注中的披露内容 |
| 合并范围问题 | 应合并未合并或不应合并已合并 | 调整合并范围，重新编制合并财务报表 |

**否定意见与无法表示意见：**
- 这两种意见意味着审计师认为财务报表整体不可信赖
- 发行人必须从根本上解决导致该意见的问题（通常涉及会计基础薄弱、持续经营能力丧失等系统性问题）
- 并非短期技术性调整可解决，可能需要重大资产重组或根本性改善后重新审计

### 2.2 强调事项段常见内容及应对

| 强调事项 | 常见情景 | 券商应对 |
|---------|---------|---------|
| 持续经营能力存在重大不确定性 | 连续亏损、高负债率、现金流为负 | 重点核查持续经营能力，单独出具专项核查报告 |
| 重大诉讼或仲裁结果不确定 | 未决诉讼 | 评估最坏情况影响，判断是否构成实质性障碍 |
| 对比较财务报表审计意见不同于本期的说明 | 更换会计师事务所 | 确保前任审计无遗留问题 |

## 三、关键审计事项（KAM）识别与核查

### 3.1 KAM的基本框架

根据《中国注册会计师审计准则第1504号——在审计报告中沟通关键审计事项》，关键审计事项是指审计师根据职业判断认为对本期财务报表审计最为重要的事项，从与治理层沟通过的事项中选取。

### 3.2 IPO项目常见的KAM类别

| KAM类别 | 典型触发场景 | 券商延伸核查 |
|---------|------------|------------|
| **收入确认** | 多种业务模式、完工百分比法、经销/代销模式、电商销售 | 穿行测试样本抽取、IT系统审计、经销商终端核查 |
| **应收账款减值** | 逾期金额大、单一客户集中度高、行业景气下行 | 单独评估主要逾期客户信用状况、期后回款检查 |
| **商誉减值** | 报告期内有重大收购、被收购标的业绩未达预期 | 独立复核减值测试模型（增长率、折现率假设合理性） |
| **存货跌价** | 技术迭代快、产品生命周期短、库龄长 | 获取库龄表、盘点观察、可变现净值测试 |
| **研发支出资本化** | 研发投入大、存在资本化会计处理 | 逐一核对资本化五项条件满足情况 |
| **股份支付** | 报告期内实施了股权激励 | 公允价值评估方法合理性、费用分摊期间正确性 |
| **递延所得税资产** | 存在大额可抵扣亏损或暂时性差异 | 盈利预测合理性（未来是否可产生足够应纳税所得额） |
| **合并范围与结构化主体** | 复杂的股权架构、VIE协议控制、合伙企业并表 | 逐一分析控制权判断依据 |

### 3.3 KAM披露质量的判断标准

| 维度 | 良好披露 | 不足披露（监管关注） |
|-----|---------|------------------|
| 认定理由 | 结合发行人具体业务特征和金额重大性 | 仅引用准则条文，无发行人具体分析 |
| 审计应对 | 详细说明执行的审计程序和结果 | 仅列示通用审计程序，无针对性 |
| 量化信息 | 提供KAM涉及的金额、比例等量化数据 | 仅有定性描述 |

### 3.4 券商应对

对于每项关键审计事项，券商需：
- 与审计师沟通，了解审计程序执行细节
- 评估该事项是否与尽职调查发现一致
- 如属于高风险事项，补充券商自身的核查程序
- 在招股书中对应章节充分披露
- **关注KAM是否遗漏了尽调中发现的重大风险领域**（如尽调发现某客户存在回款困难但KAM未涵盖应收账款减值，须与审计师沟通）

## 四、财务报表核心关注点

### 4.1 必备条件核查

| 核查项目 | 达标标准 | 不达标处理 |
|---------|---------|-----------|
| 最近一年末净资产 | ≥3000万元（主板/中小板）/ ≥2000万元（创业板/科创版） | 不满足条件，暂不能申报 |
| 最近两年净利润（扣非孰低） | 主板：≥3000万累计 / 创业板：≥5000万累计 / 科创版：最近一年≥盈利或符合市值+营收指标 | 逐一核实 |
| 最近一年净利润 | 主板：≥3000万 / 创业板：≥盈利 / 科创版：视市值标准 | 逐一核实 |
| 最近一年经营活动现金流净额 | 主板：≥3000万 或 最近三年营收累计≥3亿 / 创业板/科创版：无硬性要求 | 如不符合第一项则核查第二项 |
| 最近一年营业收入 | 主板：≥3000万 / 创业板：≥1亿 / 科创版：视市值标准 | 逐一核实 |
| 无形资产占净资产比例 | ≤20%（主板）/ 科创版无此限制 | 测算，计算口径：扣除土地使用权、采矿权等后 |

### 4.2 资产重组影响核查

如果报告期内存在同一控制下企业合并（业务重组），需核查：
- 是否触发主营业务变更认定（重组前后主营业务是否相同）
- 重组前后运行时间是否满足要求（>24个月=不触发，否则需运行满24个月或更长时间）
- 被重组方资产/收入/利润占比及运行时间矩阵

## 五、会计政策一致性核查

### 5.1 重点关注的会计政策

| 会计政策 | 关键核查点 |
|---------|-----------|
| **收入确认** | 各业务模式收入确认时点是否一致，与业务合同条款是否匹配，是否与同行业可比 |
| **应收账款坏账准备** | 计提比例是否与同行业可比，单独计提合理性，期后回收情况 |
| **存货跌价准备** | 库龄结构，可变现净值测算依据，同行业计提比例对比 |
| **固定资产折旧** | 折旧年限、残值率是否与同行业可比 |
| **无形资产摊销** | 摊销年限合理性 |
| **股份支付** | 公允价值确定依据，费用分摊期间，是否应确认而漏确认 |
| **研发支出资本化** | 资本化条件是否满足（5条条件逐一比对），资本化时点是否清晰 |
| **政府补助** | 划分标准（与资产相关/与收益相关），摊销期间合理性 |

### 5.2 跨期一致性

- 会计政策在三年一期是否保持一贯性
- 如有变更，变更原因是否合理，影响是否定量说明
- 是否存在利用会计政策变更调节利润的嫌疑

## 六、关联交易披露完整性核查

### 6.1 关联交易完整性验证

比对审计报告关联交易披露与券商尽调底稿的关联方清单：
- 是否存在审计报告有但券商未识别的关联方
- 是否存在券商识别但审计报告未涵盖的关联交易
- 关联交易定价是否已审计验证（穿行测试/函证）

### 6.2 关联资金往来核查

- 关联方资金拆借的发生额、余额、利率、还款情况
- 是否已全部清理（申报基准日原则上不得存在关联方资金占用）
- 资金占用费的收取情况

## 七、期后事项核查

### 7.1 核查期间

- 审计报告日（资产负债表日后事项审计截止日）至申报前的期间
- 重点关注：重大交易、新增重大合同、诉讼进展、资产重组、分红等

### 7.2 核查清单

- [ ] 期后是否有重大采购/销售/投资
- [ ] 期后是否有权益分派（分红/转增）
- [ ] 期后是否有重大诉讼或仲裁的新进展
- [ ] 期后是否有重大资产变化
- [ ] 期后经营数据是否与预期/上年同期有重大偏差
- [ ] 期后财务报表是否已获得（如申报时最新一期已到期后）

## 八、与尽调发现的交叉核对

### 8.1 尽调→审计报告数据印证

| 尽调发现 | 审计报告对应验证 |
|---------|----------------|
| 大额银行存款 | 银行函证结果 |
| 应收账款金额 | 应收账款函证结果 |
| 存货金额 | 存货监盘结果 |
| 销售收入 | 收入穿行测试 |
| 采购额 | 供应商函证 |
| 银行借款 | 银行函证或银行对账单 |
| 关联交易 | 关联交易审计程序 |

### 8.2 差异处理

如果尽调发现与审计报告数据存在差异：
1. 与审计师沟通差异原因
2. 若为审计师遗漏，要求补充审计程序
3. 若为尽调错误，修正券商底稿
4. 差异无法消除的，评估影响并决定是否暂缓申报

## 八、审计报告与招股书数字勾稽校验

### 8.1 勾稽校验层次

审计报告的数字必须与招股说明书相关章节的各项数据一致。券商必须执行的勾稽校验包括：

#### 层次一：审计报告内部勾稽

| 校验项 | 校验方法 |
|-------|---------|
| 资产负债表平衡 | 资产总计 = 负债合计 + 所有者权益合计 |
| 利润表与资产负债表关联 | 净利润（利润表）= 资产负债表"未分配利润"期末-期初 + 分红 |
| 现金流量表与资产负债表关联 | 现金及现金等价物净增加额 = 资产负债表"货币资金"期末-期初 |
| 所有者权益变动表一致性 | "综合收益总额" = 利润表"净利润"+其他综合收益 |
| 附注合计数与主表 | 各报表项目注释的明细合计 = 报表项目金额 |

#### 层次二：审计报告与招股说明书勾稽

| 招股书章节 | 应勾稽的审计数据 | 常见不一致 |
|-----------|----------------|-----------|
| 财务会计信息 | 三年一期财务报表完整引用 | 口径不一致（如审计报告含非经常性损益但招股书无调整） |
| 业务与技术 | 各业务收入、成本、毛利率 | 业务分类标准不一致 |
| 管理层讨论与分析 | 各项财务指标的变动趋势分析 | 招股书分析口径与审计数据口径不同 |
| 募集资金运用 | 资产负债率、流动比率等与补流测算的关联 | 财务指标引用时点不一致 |
| 同业竞争与关联交易 | 关联交易金额 | 审计按会计准则、法律意见书按法规口径，可能不一致 |
| 董监高薪酬 | 董监高薪酬总额 | 审计为权责发生制、招股书可能按实发口径 |

#### 层次三：审计报告与反馈回复勾稽

- 监管问询中涉及财务数据的回复数据，必须可追溯至审计报告
- 反馈回复中的测算底稿应与审计报告基础数据吻合

### 8.2 勾稽校验工作底稿模板

| 数据项 | 审计报告数据 | 招股书引用数据 | 差异 | 差异说明 |
|-------|------------|-------------|------|---------|
| 营业收入 | | | | |
| 净利润 | | | | |
| 总资产 | | | | |
| 净资产 | | | | |
| 经营活动现金流 | | | | |
| 各业务收入 | | | | |

| 关联交易金额 | | | | |

## 九、会计差错更正处理流程

### 9.1 差错的识别来源

会计差错通常在以下环节被发现：
1. 审计师在审计过程中发现（最常见）
2. 券商尽调中发现与审计报告不一致
3. 监管审核/问询中被指出
4. 发行人自查发现

### 9.2 差错更正的处理层级

| 差错类型 | 定义 | 处理要求 |
|---------|-----|---------|
| 重大会计差错 | 足以影响财务报表使用者对企业财务状况、经营成果和现金流量作出正确判断 | **必须追溯重述**前期财务报表；重新出具更正后的审计报告 |
| 非重大会计差错 | 不影响报表使用者决策判断 | 可调整发现当期的相关项目 |

### 9.3 "重大"的定量与定性判断

**定量判断（任一触及即为重大）：**
- 更正金额占更正前对应项目金额的10%以上
- 更正金额占更正前净利润绝对值的5%以上
- 更正导致盈亏性质变化

**定性判断：**
- 涉及收入确认、资产减值等核心会计政策变更
- 更正掩盖了财务舞弊或违规行为
- 更正与关联方资金占用有关
- 更正涉及以前年度已申报的财务数据

### 9.4 差错更正的标准处理流程

```
发现差错 → 判断是否重大
              │
     ┌───────┴───────┐
  重大差错        非重大差错
     │                 │
  追溯重述          调整当期
     │                 │
  审计师出具专项说明   补充审计
     │                 │
  会计师出具内部控制  会计师发表意见
  鉴证报告            │
     │                 │
  更新招股书财务数据   更新招股书
     │                 │
  券商出具专项核查意见
     │
  申报（或补充更新）
```

### 9.5 重大会计差错更正后的影响

- **内控鉴证报告可能被出具否定意见**：如差错系因内控缺陷导致
- **运行时间要求**：视差错性质和原因决定是否需额外运行时间
- **申报节奏影响**：可能需要延期申报

## 十、持续经营假设评估

### 10.1 持续经营能力恶化的警示信号

审计师根据《中国注册会计师审计准则第1324号——持续经营》评估发行人持续经营能力。券商应独立复核以下警示信号：

| 财务信号 | 经营信号 | 其他信号 |
|---------|---------|---------|
| 净资产为负或持续下降 | 关键管理人员离职且无替代 | 未决诉讼中可能需承担巨额赔偿 |
| 连续经营亏损 | 丧失主要市场/客户/特许经营权 | 违反资本或其他法定要求 |
| 经营性现金流持续为负 | 关键原材料/技术供应中断 | 对发生损失的主体提供巨额担保 |
| 重大债务到期无法偿还 | 主要竞争对手出现突破性技术 | |
| 过度依赖短期借款为长期资产融资 | 核心专利/商标已到期或即将到期 | |
| 出现债务违约 | 行业出现系统性衰退 | |
| 主要财务指标恶化（速动比率<0.5、资产负债率>80%） | | |

### 10.2 券商核查追加程序

当审计报告包含"持续经营存在重大不确定性"的强调事项段时，券商须：
1. 独立复核管理层的持续经营评估（包括未来12个月现金流预测的合理性）
2. 核查融资安排的确定性和可行性（是否有约束力的银行授信或股东注资承诺）
3. 核查是否存在资产处置计划及其可行性
4. 独立出具《持续经营能力专项核查报告》
5. 评估是否已充分披露风险，投资者是否可做出知情决策

## 十一、关联方资金占用核查程序

### 11.1 资金占用的形态识别

| 形态 | 具体表现 | 审计识别方法 |
|-----|---------|------------|
| 直接拆借 | 借款协议/借据，非经营性往来 | 银行流水分析 + 其他应收/应付款明细 |
| 代垫代付 | 为关联方代垫费用/款项 | 其他应收款异常发生额 |
| 无商业实质的交易 | 虚假采购/销售形成资金流出/占用 | 穿行测试 + 函证 + 实地走访 |
| 通过第三方中转 | 资金经由表面无关第三方流向关联方 | 大额异常交易对手的背景穿透 |
| 存单质押/票据贴现 | 为关联方融资提供增信 | 受限资产披露 + 贷款卡信息查询 |

### 11.2 审计程序复核要点

| 审计程序 | 券商复核内容 |
|---------|------------|
| 银行函证 | 是否函证了发行人全部银行账户（含已销户和零余额账户）；函证是否包含借款、担保、票据等信息 |
| 关联方交易审计 | 关联方清单是否完整；关联交易审计样本是否充分 |
| 资金流水核查 | 是否覆盖全部银行账户；大额资金往来是否穿透核查对手方 |
| 期后测试 | 申报基准日后关联方资金往来是否继续发生 |

### 11.3 资金占用的清理标准

- 申报基准日**不得存在**关联方非经营性资金占用
- 如报告期内曾有占用：（1）说明形成原因与完整过程；（2）已归还并收取不低于同期贷款基准利率的资金占用费；（3）已建立并运行防范占用的内控制度；（4）内控鉴证报告须对相关内控发表无保留结论
- 对IPO申报，资金占用是"红线"级问题

## 十二、监管关注财务问题 Top 10

以下为证监会/交易所在IPO审核中最常关注的财务问题（按频次排列）：

### Top 1：收入真实性与确认时点
- **关注要点**：是否存在虚增收入、提前确认收入、利用经销商囤货、虚构交易、变相资金流转
- **核查要求**：穿行测试+函证+走访+银行流水+IT审计，多维交叉验证
- **高频问询**："请保荐机构和会计师说明对报告期内收入的真实性采取的核查程序、核查比例和核查结论"

### Top 2：毛利率异常及波动
- **关注要点**：毛利率显著高于/低于同行业可比公司
- **核查要求**：逐一分析价格、成本、产品结构、客户结构等因素
- **高频问询**："发行人毛利率持续上升，而同行业公司毛利率呈下降趋势，请说明原因及合理性"

### Top 3：应收账款质量与坏账准备充足性
- **关注要点**：逾期比例高、账龄长、集中度高
- **核查要求**：单独测试主要客户信用风险、期后回款检查
- **高频问询**："发行人应收账款坏账准备计提比例低于同行业公司，请说明合理性"

### Top 4：存货跌价准备充分性
- **关注要点**：库存高企、周转慢、技术迭代导致产品跌价
- **核查要求**：盘点+库龄分析+可变现净值测试

### Top 5：研发支出资本化合理性
- **关注要点**：资本化时点不清晰、资本化条件不满足、研发与生产阶段的划分模糊
- **核查要求**：逐项目核实五项资本化条件

### Top 6：股份支付确认完整性
- **关注要点**：应确认未确认、公允价值确定方法不当、费用分摊期间错误
- **核查要求**：梳理全部股权变动，判断是否构成股份支付

### Top 7：非经常性损益界定
- **关注要点**：将经常性损失归入非经常性、扣非前后净利润差异过大
- **核查要求**：逐项复核非经常性损益的认定依据

### Top 8：关联交易价格公允性
- **关注要点**：关联交易缺乏商业合理性、定价缺乏第三方可比性
- **核查要求**：获取可比价格资料

### Top 9：税收优惠依赖
- **关注要点**：净利润中税收优惠占比高（>20%）、优惠即将到期
- **核查要求**：模拟测算无优惠情况下的盈利

### Top 10：现金流量与利润长期背离
- **关注要点**：连续多年净利润为正但经营性现金流为负
- **核查要求**：分析应收账款、存货、应付账款的变化趋势
