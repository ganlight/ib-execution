# SKILL_01 计划说明书撰写

> ABS计划说明书（核心发行文件）撰写规范——12章完整结构

---

## 第1章：发行概况

### 撰写要点

| 要素 | 内容 | 示例/模板 |
|------|------|----------|
| 专项计划名称 | "{原始权益人简称}{基础资产类型}资产支持专项计划" | "XX保理供应链第1期资产支持专项计划" |
| 发行规模 | 各档证券合计本金规模 | 合计{XX}亿元 |
| 分层情况 | 优先A/优先B/次级各自规模、比例 | |
| 证券期限 | 各档预期到期日 | 优先A档：{YYYY-MM-DD} |
| 票面利率 | 固定利率/浮动利率类型 | 固定利率，具体以簿记建档结果为准 |
| 还本付息方式 | 过手型/计划摊还型/到期一次还本 | 按月/按季过手 |
| 初始起算日（封包日） | 基础资产池锁定基准日 | {YYYY-MM-DD} |
| 预期设立日 | 专项计划成立日 | {YYYY-MM-DD} |
| 发行方式 | 簿记建档/定向发行/公开募集 | 簿记建档 |
| 投资者范围 | 合格投资者/专业投资者 | 符合《证券期货投资者适当性管理办法》的合格投资者 |

**占位符模板**：

```markdown
## 发行概况

{原始权益人名称}作为原始权益人，以其合法拥有的{基础资产类型}为基础资产，
由{计划管理人名称}担任计划管理人，设立"{专项计划名称}"。

本期专项计划发行总规模为人民币{XX}亿元，其中：
- 优先A档资产支持证券：{XX}亿元，占比{XX}%，预期到期日{YYYY-MM-DD}，目标评级{AAA}
- 优先B档资产支持证券：{XX}亿元，占比{XX}%，预期到期日{YYYY-MM-DD}，目标评级{AA+}
- 次级档资产支持证券：{XX}亿元，占比{XX}%，预期到期日{YYYY-MM-DD}，无评级

各档证券均为固定利率，具体票面利率以簿记建档结果为准。
还本付息方式为{按季/按月}过手偿还。
```

## 第2章：风险因素

### 撰写要点

#### 2.1 基础资产风险（占本章至少40%）

| 风险类别 | 具体风险点 | 风险缓释措施 |
|---------|-----------|-------------|
| 债务人违约风险 | 入池基础资产对应的债务人可能发生违约 | 影子评级/分散度/违约率分析支撑 |
| 早偿风险 | 借款人提前还款导致资产池期限缩短 | 历史早偿率分析+CPR假设 |
| 资产池集中度风险 | 单一债务人/行业/地域过于集中 | 分散度分析+压力测试 |
| 资产真实性和合法性风险 | 基础资产对应的贸易背景/合同可能不真实 | 尽调覆盖率+确权文件 |

#### 2.2 交易结构风险

| 风险类别 | 具体风险点 |
|---------|-----------|
| 破产隔离风险 | 原始权益人破产时基础资产可能被纳入破产财产 |
| SPV管理风险 | 计划管理人能力不足或违规 |
| 混同风险 | 基础资产回款与原始权益人自有资金混合 |

#### 2.3 流动性风险

- 资产支持证券二级市场流动性有限
- 投资者可能无法以公允价值及时变现

#### 2.4 利率风险

- 固定利率证券在利率上升时市场价值下降
- 基础资产端与证券端的利率类型不匹配

#### 2.5 政策风险

- 监管政策变化（如资产证券化业务规则调整）
- 税收政策变化
- 行业政策（如房地产调控政策对CMBS/供应链ABS的影响）

## 第3章：交易结构

### 撰写要点

- 交易各方列表及简介（原始权益人、计划管理人、托管行、监管行、评级机构、律师、会计师）
- 资产购买安排（《资产买卖协议》核心条款）
- 现金流归集与分配机制（详细Waterfall，须精确到每个步骤）
- 增信安排概要（分层+超额抵押+差额支付承诺等）
- 资产服务机构安排（通常为原始权益人）
- 专项计划的设立/终止/清算安排

---

## 第4章：基础资产池概况

### 撰写要点

#### 4.1 基础资产池统计特征

| 统计指标 | 数值 | 说明 |
|---------|------|------|
| 入池资产笔数 | {XXX}笔 | |
| 资产池未偿本金余额 | {XX.X}亿元 | |
| 单笔资产平均余额 | {XXX}万元 | |
| 单笔资产最大余额 | {XXX}万元 | 占资产池 {X.X}% |
| 加权平均剩余期限 | {X.X}年 | |
| 加权平均利率 | {X.X}% | |
| 加权平均影子评级 | S{X.X} | |

#### 4.2 资产池分布

- 按债务人行业分布（饼图或表格）
- 按地域分布（省份/城市）
- 按债务人集中度（前5/10/20大及各自占比）
- 按剩余期限分布
- 按影子评级分布

#### 4.3 影子评级结果

- 影子评级方法论简述
- 资产池加权平均影子评级及评级分布

#### 4.4 历史表现数据

- 累计违约率时间序列
- 早偿率趋势
- 回收率历史

## 第5章：原始权益人基本情况

### 撰写要点

- 公司概况（工商登记基本信息、股东结构、关联企业）
- 主营业务及经营状况（含行业地位、竞争优势）
- 财务管理（最近三年主要财务指标）
- 资产服务能力（信息管理系统、资产管理制度、历史服务表现）
- 信用状况（外部评级、征信记录、重大诉讼/处罚）

---

## 第6章：计划管理人基本情况

### 撰写要点

- 公司概况
- ABS业务资质与历史业绩（已管理专项计划数量、规模、资产类型）
- 风控体系与内核制度
- 团队配置（本项目团队成员的从业经验和资质）

## 第8章：增信安排

### 撰写要点

- 优先/次级分层（各档规模、比例、偿付顺序）
- 超额抵押（覆盖率、补足机制）
- 超额利差（利差水平、锁定机制）
- 差额支付承诺（承诺人、承偌内容、触发条件、限额）
- 信用触发机制（加速清偿事件/违约事件的具体定义和后果）
- 增信措施的局限性声明

## 第10章：持有人会议规则

## 第12章：法律意见摘要

### 撰写要点

- 摘要律师对交易合法合规性的核心结论：
  - 基础资产是否合法有效
  - 专项计划设立是否合法合规
  - 基础资产转让是否合法有效、是否构成真实出售
  - 破产隔离安排是否有效
  - 增信措施是否合法有效
- 法律意见书的保留意见或例外（如有）必须完整列明
