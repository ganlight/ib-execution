# SKILL_01 ABS簿记定价

> ABS分层簿记建档与定价执行规范

---

## 一、ABS定价特殊性

### 1.1 分层定价的基本逻辑

普通债券为单一票面利率定价（一个品种一个利率），ABS因分层结构，各档证券的信用质量不同，须分别确定票面利率。

```
基础资产池
    │ 现金流
    ▼
证券端分层
    │
    ├── 优先A档（最高信用质量，最低收益率）  → 定价最低
    ├── 优先B档（中等信用质量，中等收益率）  → 定价中等
    └── 次级档（最低信用质量，最高收益率）   → 定价最高（或原始权益人自持）
```

### 1.2 基础资产现金流→证券端定价传导

| 传导链 | 影响机制 |
|--------|---------|
| 基础资产池加权平均利率 | → 确定证券端利率上限（否则产生负利差） |
| 各项费用率（管理费+托管费+其他） | → 从资产端利率中扣除，确定可分配给证券的部分 |
| 目标分层评级 | → 确定各档利差水平（评级越高，利差越低） |
| 超额利差留存 | → 影响发行利率确定（预留保护越厚，定价可稍低） |

## 二、簿记建档流程

### 2.1 三阶段簿记建档

```
阶段1：优先A档簿记
    │  大量银行/保险机构参与
    │  需求通常最为充裕
    ▼
阶段2：优先B档簿记
    │  基金/券商参与为主
    │  需求较优先A档少
    ▼
阶段3：次级档定向
    │  原始权益人自持为主（风险自留）
    │  部分情况有第三方投资者参与
```

### 2.2 簿记建档标准流程

```
T-1日：簿记建档公告发布
    │  ├── 公告定价区间（如优先A {2.80%-3.30%}，优先B {3.50%-4.20%}）
    │  └── 公告申购规则（最低申购金额/报价精度/截止时间等）

T日：簿记建档日
    │
    ├── 09:00-10:00  优先A档申购接收
    │      ├── 投资者提交申购利率和金额
    │      └── 簿记管理人实时汇总申购情况
    │
    ├── 10:00-11:00  优先B档申购接收
    │      ├── 部分投资者可能同时参与多档申购
    │      └── 簿记管理人实时调整各档需求
    │
    ├── 11:00-12:00  定价分析
    │      ├── 按申购利率从低到高排序
    │      ├── 在预定发行量处"划线"确定票面利率
    │      └── 超额认购倍数参考：优先A {2-4}倍，优先B {1.5-2.5}倍
    │
    └── 12:00后      发行结果公告
           ├── 公告各档票面利率
           └── 公告超额认购情况
```

### 2.3 簿记规则

| 规则项 | 内容 |
|--------|------|
| 簿记管理人 | 通常由计划管理人（证券公司）担任 |
| 申购方式 | 利率投标（投资者报价以申购利率为准） |
| 最低申购金额 | 通常 {100}万元或其整数倍 |
| 报价精度 | 0.01%（即1bp） |
| 申购截止后修改 | 截止后不接受新增申购或修改报价 |
| 配售规则 | 同一利率水平按"时间优先"或"比例配售" |

## 三、投资者沟通要点

### 3.1 不同投资者类型的差异化关注点

| 投资者类型 | 投资偏好 | ABS关注重点 | 沟通策略 |
|-----------|---------|-----------|---------|
| **商业银行** | 安全性最高、期限匹配、持有至到期 | 优先A档AAA评级、预期期限、法律风险 | 强调评级稳定性、破产隔离 |
| **保险机构** | 长久期、稳定现金流、AAA | 匹配负债久期、优先A档的期限 | 提供详细的预期到期日分布 |
| **公募基金** | 收益率敏感、流动性、相对价值 | 收益率 vs 同类ABS和AAA信用债、二级市场流动性 | 提供同类型利差对比分析 |
| **券商自营** | 弹性要求、可配置优先B档 | 优先B信用质量、收益率溢价 | 提供优先B的信用分析和收益预测 |
| **理财子公司** | 兼顾收益和安全、可配置优先B | 各档收益率、信用支持充分性 | 强调整体方案的稳定性 |

### 3.2 Pre-marketing关键信息清单

| 信息项 | 内容 |
|--------|------|
| 基础资产类型和资产池概况 | 资产类型/规模/分散度/加权平均特征 |
| 分层结构和信用支持 | 各档比例/信用支持量/目标评级 |
| 可比ABS利差数据 | 近期同类ABS的发行利率和利差中枢 |
| 原始权益人概况 | 主体评级/行业地位/历史ABS表现 |
| 增信安排 | 差额支付承诺/超额抵押/利差等 |
| 发行时间窗口 | 预计簿记建档日/缴款日/起息日 |

## 四、定价参考基准

### 4.1 定价参考维度

| 定价因子 | 参考基准 | 对票面利率的影响 |
|---------|---------|----------------|
| 基准利率 | 同期限国开债/国债收益率 | 定价的基础线（无风险利率） |
| ABS利差中枢 | 同类/同评级ABS近3个月发行的利差区间 | 核心参考（反映ABS市场整体定价水平） |
| 资产质量溢价 | 基础资产类型特殊风险（如消费类 vs 融资租赁的差异） | 不同资产类型的利差差异 |
| 原始权益人溢价 | 原始权益人品牌/历史表现/市场信任度 | 知名原始权益人利差偏低 |
| 期限溢价 | 证券期限越长利差越宽 | 3年AAAABS vs 1年AAAABS利差差异 |
| 结构复杂度溢价 | 含循环购买等复杂结构 | 结构越复杂利差越宽 |

### 4.2 利差锚定表（参考，须结合市况更新）

| 证券类型 | 期限 | 目标评级 | 参考利差范围（vs同期限国开债） |
|---------|------|---------|---------------------------|
| 供应链ABS优先档 | 1-2年 | AAA | {30-60}bp |
| 融资租赁ABS优先A档 | 2-4年 | AAA | {40-80}bp |
| 消费金融ABS优先A档 | 1-3年 | AAA | {50-100}bp |
| CMBS优先A档 | 3-5年 | AAA | {60-120}bp |
| 企业ABS优先B档 | 2-4年 | AA+ | {100-200}bp |
| ABN优先档 | 2-3年 | AAA | {40-70}bp |

### 4.3 定价策略

```
① 确定定价区间
   步骤1：基准利率锚定（T日同期限国开债收益率）
   步骤2：叠加ABS利差中枢（取{近1个月}同类产品利差中位数）
   步骤3：微调（±{5-15}bp，考虑本项目特殊因素）
   步骤4：确定申购利率区间（下限=预期最优定价，上限=预期最高可接受定价）

② 发行结果定价
   取超额认购倍数≥{X}倍时对应的申购利率最高点作为票面利率
```

## 五、发行失败的三种情景预案

### 情景1：认购不足

**表现**：簿记建档中任一档证券的有效申购总额低于发行量。

**触发标准**：超额认购倍数 < {1.0}倍（即申购量 < 发行量）

**预案**：
| 应对级别 | 措施 | 实施条件 |
|---------|------|---------|
| 初级应对 | 延长簿记时间（+{1-2}小时） | 申购覆盖率达 {80%}以上 |
| 中级应对 | 调低价格（提高票面利率） | 簿记建档允许价格调整 |
| 高级应对 | 缩减发行规模 | 满足"最低发行规模"条件 |
| 终止应对 | 取消本次发行 | 无法达到最低发行规模或不接受当前价格 |

### 情景2：申购价格远高于预期

**表现**：投资者申购利率集中在定价区间上沿甚至更高。

**预案**：
- 如仍在定价区间上沿：接受较高定价，但须向原始权益人说明成本影响
- 如突破定价区间：评估市场原因，决定是否延迟发行或接受更高定价
- 事后复盘：Pre-marketing的沟通是否存在偏差

### 情景3：市场大幅波动

**表现**：簿记建档当日市场利率急剧波动（如央行突然降息/加息、重大政策发布）。

**预案**：
- 如属突发性大幅波动：暂停簿记，重新评估定价区间
- 如波幅可控（±{20}bp内）：继续簿记，适当调整定价区间
- 确定是否属于"不可抗力"：参照《申购要约》中的不可抗力条款

## 六、发行后事项

### 6.1 缴款管理

| 事项 | 时限 | 操作 |
|------|------|------|
| 配售确认通知 | 簿记完成当日或次日上午 | 向获配投资者发送配售确认函 |
| 缴款截止日 | 配售后T+{3}个工作日 | 投资者缴付认购款项 |
| 缴款跟踪 | 缴款截止日前每日 | 确认各投资者的缴款进度 |
| 未缴款处理 | 缴款截止后 | 未缴款部分取消配售 + 违约责任追究 |

### 6.2 专项计划设立/信托成立

| 事项 | 操作 |
|------|------|
| 设立确认 | 计划管理人出具《专项计划设立公告》 / 受托机构出具《信托成立公告》 |
| 起息日 | 通常为缴款截止日+{1}个工作日 |
| 证券登记 | 向中央结算/中证登申请证券登记 |

### 6.3 挂牌

| 事项 | 操作 |
|------|------|
| 挂牌申请 | 向交易所/银行间提交挂牌申请 |
| 挂牌日 | 通常为起息日后{3-7}个工作日 |
| 证券代码 | 交易所/NAFMII统一分配 |
