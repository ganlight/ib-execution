# SKILL_00 ABS存续总览

> ABS存续期管理全景——资产管理、信息披露与风险应对

## 一、存续期管理四大职责

```
                    ┌──────────────────────────┐
                    │    ABS存续期管理四大职责   │
                    └──────────────────────────┘
                                    │
        ┌─────────────┬─────────────┼─────────────┬─────────────┐
        ▼             ▼             ▼             ▼             ▼
  ┌───────────┐ ┌───────────┐ ┌───────────┐ ┌───────────┐
  │  资产服务  │ │  现金流   │ │  信息     │ │  风险事件 │
  │  报告审核  │ │  监控     │ │  披露     │ │  应对     │
  └───────────┘ └───────────┘ └───────────┘ └───────────┘
  月度/季度/年   实时监控      定期/临时     处置/持有人会议/
  度报告          分配执行     公告          信用触发
```

### 1.1 职责1：资产服务报告审核

- 月度资产服务机构报告（资产服务机构提交）
- 季度托管报告（托管银行提交）
- 年度资产管理报告和审计报告
- 核查报告数据与预期的一致性

### 1.2 职责2：现金流监控

- 基础资产回收款归集情况监控
- 监管账户→信托/托管账户的转付频率
- 按Waterfall执行的分配监督
- 异常情况（如回款大幅下降）的预警和调查

### 1.3 职责3：信息披露

- 定期报告（月度/季度/年度）的发布和核查
- 临时重大事项的披露
- 向监管机构（交易所/NAFMII/银保监）的定期报告

### 1.4 职责4：风险事件应对

- 信用触发事件的监控和启动
- 加速清偿事件的处理
- 持有人会议召集和决议执行
- 资产置换/赎回/提前清算的评估和执行

## 二、年度工作日历（12月重点任务）

| 月份 | 常规工作 | 重点关注事项 |
|------|---------|------------|
| 1月 | - | 年度审计启动，收集上年度数据 |
| 2月 | - | 年度报告初稿准备 |
| 3月 | - | 年度资产管理报告发布；年度审计报告发布；跟踪评级启动 |
| 4月 | Q1托管报告发布 | 关注Q1资产表现 |
| 5月 | Q1资产服务机构报告 | 半年期压力测试更新 |
| 6月 | Q2托管报告发布 | 下半年市场/利率环境评估 |
| 7月 | Q2资产服务机构报告 | 压力测试更新初稿 |
| 8月 | - | 年度压力测试定稿 |
| 9月 | Q3托管报告发布 | 关注前9个月资产表现趋势 |
| 10月 | Q3资产服务机构报告 | 评级机构初步沟通下年度评级 |
| 11月 | 下一年度工作计划制定 | 下年度预算（管理费/托管费续期） |
| 12月 | 年度总结 | 排查下年度到期还款压力；准备年度重大事项回顾 |

## 三、ABS重大事项判断标准（10项）

| # | 事项类别 | 具体情形 | 披露时限 |
|---|---------|---------|---------|
| 1 | 资产服务机构变更 | 资产服务机构发生变更或预期变更 | 发生后{2}个工作日内 |
| 2 | 托管银行变更 | 托管银行发生变更或预期变更 | 发生后{2}个工作日内 |
| 3 | 基础资产重大违约 | 资产池单月违约率超过阈值（如{X}%） | 发生后{2}个工作日内 |
| 4 | 信用触发事件 | 加速清偿事件/违约事件被触发 | 触发当日 |
| 5 | 增信方信用降级 | 差额支付承诺人/担保人的主体评级下调 | 发生后{2}个工作日内 |
| 6 | 原始权益人重大变化 | 原始权益人发生重大经营/财务/法律问题 | 发生后{2}个工作日内 |
| 7 | 计划管理人重大变化 | 计划管理人发生变更或被监管处罚 | 发生后{2}个工作日内 |
| 8 | 法律诉讼 | 涉及专项计划或基础资产的重大诉讼 | 发生后{5}个工作日内 |
| 9 | 资产池置换/赎回 | 原始权益人按合同约定行使置换/赎回权 | 置换/赎回执行前{10}个工作日 |
| 10 | 现金流异常 | 单期回收款较预测值大幅下降（超过{30}%） | 发生后{2}个工作日内 |

## 四、存续期管理参与方职责

| 参与方 | 存续期职责 |
|--------|---------|
| **计划管理人** | 总体负责：报告审核+信息披露+风险应对+持有人会议召集 |
| **资产服务机构**（通常为原始权益人） | 提交月度资产服务报告+维护资产池+执行回收和转付 |
| **托管银行** | 出具季度托管报告+按Waterfall执行资金分配指令 |
| **监管银行** | 监控回收款归集账户+执行监管指令 |
| **评级机构** | 年度跟踪评级+触发降级审查 |
| **律师事务所** | 协助重大事项判断+持有人会议法律意见 |
| **会计师事务所** | 年度审计报告 |

## 五、存续期管理的特殊挑战

| 挑战 | 描述 | 应对措施 |
|------|------|---------|
| 时间跨度长 | ABS存续期通常3-10年，人员更迭、信息流失风险高 | 建立项目知识库+标准操作手册（SOP） |
| 数据连续性 | 须持续跟踪资产池表现数据，数据质量须保持一致 | 统一数据口径+IT系统自动化采集 |
| 监管政策变化 | 存续期内信息披露要求可能变化 | 持续关注监管更新+及时调整 |
| 资产服务机构风险 | 资产服务机构（通常为原始权益人）可能信用恶化 | 设置后备服务机构条款+定期评估 |
| 循环购买管理 | 循环期内需持续监督新增资产的质量 | 建立合格标准监控系统+定期抽检 |
