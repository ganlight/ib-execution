# SKILL_00 ABS申报总览

> ABS监管申报体系全景——三大市场审核路径

---

## 一、三大申报体系对比

| 维度 | 信贷ABS（银保监备案） | 企业ABS（交易所挂牌确认） | ABN（NAFMII注册） |
|------|---------------------|------------------------|------------------|
| **监管主体** | 银保监会+央行 | 证监会+沪深交易所 | 交易商协会（NAFMII） |
| **审核模式** | 备案制（首次备案+分期发行） | 挂牌条件确认（类审批制） | 注册制 |
| **审核文件** | 《资产支持证券发行备案报告》 | 《挂牌条件确认申请文件》 | 《注册申请书》 |
| **审核周期** | 首次备案：约{2-3}个月 | 审核周期：约{2-4}个月（含反馈） | 注册周期：约{3-6}个月 |
| **审核标准** | 指引+窗口指导 | 交易所挂牌条件确认指引 | NAFMII注册文件清单 |
| **后续发行** | 备案有效期内分期发行（每期发行前报送发行文件） | 储架发行（一次确认、分期发行，限期24个月） | 注册有效期内分期发行 |

## 三、申报流程时间轴

### 3.1 企业ABS（交易所）挂牌条件确认

```
预沟通（T-4周）
    │  选取申报时机、确认审核要点
    ▼
正式申报（T+0周）
    │  向交易所提交挂牌条件确认申请文件
    ▼
受理（T+1~2周）
    │  交易所出具受理通知书（或不予受理）
    ▼
反馈阶段（T+3~8周）
    │  一轮反馈：{10-15}个工作日
    │  二轮反馈（如需）：{5-10}个工作日
    ▼
挂牌条件确认通过（T+9~14周）
    │  交易所出具《资产支持专项计划挂牌条件确认书》
    ▼
发行前备案（T+15~16周）
    │  发行前向交易所备案
    ▼
发行 → 挂牌
```

### 3.2 ABN（NAFMII）注册

```
预沟通（T-4周）
    │  NAFMII预评/预沟通
    ▼
正式注册申请（T+0周）
    │  提交《注册申请书》等文件
    ▼
受理（T+1~2周）
    │  NAFMII出具受理通知书
    ▼
注册评议（T+3~16周）
    │  反馈阶段（可能2-3轮）
    ▼
接受注册（T+17~22周）
    │  NAFMII出具《接受注册通知书》
    ▼
分期发行（注册有效期内）
```

### 3.3 信贷ABS（银保监）备案

```
首次备案申请（T+0周）
    │  向银保监会提交备案申请
    ▼
备案通过（T+8~12周）
    │  取得备案资格
    ▼
首期发行前备案（T+12~14周）
    │  向银保监报送首期发行文件
    ▼
首期发行
    │
    ▼
后续各期发行（备案有效期内）每期发行前向银保监报送
```

## 四、申报前自查清单

| # | 自查项 | 企业ABS | ABN | 信贷ABS |
|---|--------|---------|-----|---------|
| 1 | 基础资产合规性是否100%确认 | ✅ | ✅ | ✅ |
| 2 | 法律意见书是否对"真实出售+破产隔离"发表明确正面结论 | ✅ | ✅ | ✅ |
| 3 | 现金流预测报告是否由独立第三方完成 | ✅ | ✅ | ✅ |
| 4 | 主要交易合同文本是否全部定稿 | ✅ | ✅ | ✅ |
| 5 | 评级报告是否达到目标评级 | ✅ | ✅ | ✅ |
| 6 | 原始权益人主体是否存在未披露的重大负面信息 | ✅ | ✅ | ✅ |
| 7 | 风险自留安排是否符合监管要求 | ✅ | ✅ | ✅ |
| 8 | 是否完成必要的监管预沟通 | ✅ | ✅ | ✅ |

## 五、跨市场申报策略建议

### 5.1 市场选择决策树

```
基础资产类型？
    │
    ├── 金融机构持有的资产（信贷类）
    │      └── → 信贷ABS（银保监备案）
    │           ├── 个人住房贷 → 必然信贷ABS
    │           ├── 车贷 → 必然信贷ABS
    │           └── 不良资产 → 信贷ABS
    │
    ├── 非金融企业资产
    │      └── → 企业ABS 或 ABN
    │           ├── 规模大（>20亿）、投资者范围广 → 企业ABS（短期更市场化）
    │           ├── 规模中等（5-20亿）、原始权益人注册经验多 → ABN
    │           └── 融资租赁、供应链 → 两者均可
    │
    └── 双市场挂牌（ABN）
           └── 同时在银行间+交易所挂牌
```

### 5.2 申报时机选择

| 月份 | 审核节奏 | 建议 |
|------|---------|------|
| 1-2月 | 春节前后审核较慢 | ❌ 不建议此期间首次申报 |
| 3-6月 | 审核正常 | ✅ 黄金申报窗口 |
| 7-8月 | 审核正常（暑期略有放缓） | ⚠️ 可申报但不建议复杂项目 |
| 9-11月 | 审核高峰期（赶年底前完成） | ✅ 对成熟项目是优质窗口 |
| 12月 | 临近年末审核趋紧 | ⚠️ 不建议新项目首次申报 |
