# SKILL_02 ABS法律尽调

> ABS法律尽职调查执行规范——SPV结构、资产转让、破产隔离与增信法律效力

---

## 一、SPV法律结构核查

### 1.1 三种SPV形式对比

| 法律特征 | 特定目的信托（SPT） | 资产支持专项计划 | 特定目的公司（SPC） |
|---------|-------------------|-----------------|-------------------|
| **法律依据** | 《信托法》、《信托公司管理办法》 | 《证券法》、证监会《资产证券化业务管理规定》 | 《公司法》 |
| **法律主体地位** | 无独立法人资格（信托财产独立于委托人+受托人） | 无独立法人资格（合同集合体） | 独立法人资格 |
| **资产持有方式** | 受托人（信托公司）名义持有 | 计划管理人（证券公司/基金子公司）名义持有 | SPC自身名义持有 |
| **适用ABS体系** | 信贷ABS（银保监） | 企业ABS（交易所） | 较少使用，主要用于类REITs |
| **设立程序** | 签订信托合同→信托登记 | 计划说明书+标准条款+认购协议 | 工商注册+章程 |

### 1.2 SPV设立合规性核查要点

| 核查项 | 核查内容 | 合规基准 |
|--------|---------|---------|
| 管理人/受托人资质 | 是否具备从事资产证券化业务的法定资格 | 证监会/银保监会业务许可批复文件 |
| 设立文件完整性 | 专项计划/信托的文件是否齐备 | 计划说明书+标准条款+资产买卖协议+托管协议 |
| 设立程序合规 | 是否履行必要的内部决策和备案程序 | 管理人内核委员会决议 |
| SPV财产独立性 | 专项计划资产/信托财产是否独立于管理人自有财产 | 账户独立、财务独立、风险隔离的明确规定 |

### 1.3 常见SPV法律缺陷

| 缺陷类型 | 风险等级 | 法律后果 | 修复措施 |
|---------|---------|---------|---------|
| 管理人无ABS业务资格 | 🔴致命 | 专项计划可能被认定为无效 | 更换合格管理人 |
| 信托登记未完成 | 🟡中等 | 信托财产独立性主张可能受质疑 | 补齐登记 |
| SPV账户与管理人自有账户混同 | 🔴致命 | 破产隔离失效 | 立即开立独立账户并分离 |

## 二、基础资产转让——真实出售判定

### 2.1 "真实出售"vs"担保融资"判定标准

基础资产转让须构成"真实出售"（True Sale），而非"担保融资"（Secured Financing），才能实现破产隔离。

| 判定维度 | 真实出售特征 | 担保融资特征 |
|---------|------------|------------|
| **追索权** | 买方对卖方无追索权或追索权极其有限（仅就资产质量瑕疵） | 买方保留对卖方全部追索权 |
| **资产风险与收益** | 资产的全部风险和收益已转移至SPV | 卖方实质上仍承担资产的信用风险 |
| **对价合理性** | 转让价格基于公允市场价值确定 | 转让价格等于融资本金，与资产价值无明确关系 |
| **剩余权益安排** | SPV对剩余资产的处置无向原始权益人返还的义务（超额利差除外） | 资产清算后剩余价值归卖方 |
| **会计处理** | 原始权益人财务报表中资产已"出表" | 资产仍保留在原始权益人资产负债表中 |

### 2.2 真实出售核查五步法

```
Step 1: 核查资产买卖协议的转让条款（是否明确约定"真实出售"意图）
Step 2: 核查转让对价支付方式（是否取得公允价值支持文件）
Step 3: 核查追索权安排（原始权益人是否提供资产质量保证以外的实质追索担保）
Step 4: 核查会计处理（原始权益人审计报告中是否已出表）
Step 5: 核查评级机构/律师意见（评级机构对其构成"真实出售"的认定）
```

### 2.3 债权转让通知

| 通知方式 | 适用情形 | 法律效果 |
|---------|---------|---------|
| 逐笔通知 | 债务人数量有限，逐一通知成本可控 | 转让对债务人生效，债务人应向SPV履行 |
| 公告通知 | 债务人数量众多，逐一通知不现实 | 法律上对债务人生效（《民法典》第546条） |
| 权利完善通知 | 仅在触发特定条件时（如原始权益人信用降级）才通知 | 触发前债务人继续向原始权益人履行，触发后须通知 |

## 三、破产隔离有效性分析

### 3.1 破产隔离法律论证框架

破产隔离的核心命题：当原始权益人破产时，基础资产不属于破产财产范围，资产支持证券持有人对基础资产享有优先于原始权益人普通债权人的受偿权。

**论证五要素**：

| 论证要素 | 法律依据 | 核查要点 |
|---------|---------|---------|
| 资产已真实出售 | 《民法典》合同编、财产转让相关条款 | 买卖协议+转让对价支付凭证+出表审计意见 |
| SPV财产独立性 | 《信托法》第15-16条（信托财产独立性） | 信托合同/专项计划设立文件 |
| 无撤销风险 | 《企业破产法》第31-32条（破产撤销权） | 转让时间距离破产申请≥1年，转让价格公允 |
| 无重新定性风险 | 相关司法实践 | 综合真实出售五维度无反向证据 |
| 担保安排有效 | 《民法典》担保制度 | 抵质押登记已依法完成 |

### 3.2 破产撤销权风险分析

《企业破产法》第31条规定，法院受理破产申请前一年内的下列行为，管理人有权请求法院撤销：

| 情形 | ABS资产转让风险分析 |
|------|-------------------|
| 无偿转让财产 | ABS转让为有偿转让（取得转让对价），不适用 |
| 以明显不合理的价格进行交易 | 转让价格须以公允价值为基础并取得估值支持文件 |
| 对未到期的债务提前清偿 | 非债务清偿行为，不适用 |
| 放弃债权 | 不适用 |

**风险缓释措施**：
- 在资产买卖协议中明确转让的公允价值
- 取得独立第三方估值报告
- 在计划说明书中披露破产撤销权风险及其缓释措施

### 3.3 实体合并风险（Substantive Consolidation）

当原始权益人与SPV的资产/业务/人员/财务高度混同时，法院可能将二者视为同一实体，从而实质否定破产隔离。

**防控措施**：
- 确保SPV与原始权益人在机构设置、人员、财务、资产上完全独立
- 避免SPV以原始权益人名义对外进行任何商业活动
- SPV账户独立，且与原始权益人往来账务清晰可查

## 四、基础资产权利负担核查

### 4.1 权利负担类型与核查方法

| 权利负担类型 | 核查渠道 | 登记系统 |
|------------|---------|---------|
| 应收账款质押 | 全国动产融资统一登记公示系统 | 中国人民银行征信中心 |
| 股权质押 | 国家企业信用信息公示系统 | 各地市场监管局 |
| 不动产抵押 | 不动产登记中心 | 自然资源部不动产登记系统 |
| 司法查封/冻结 | 中国执行信息公开网、地方人民法院 | 全国法院执行系统 |
| 账户冻结 | 开户行查询 | — |

### 4.2 权利负担清理要求

| 权利负担状态 | 处理方式 | 时间要求 |
|------------|---------|---------|
| 已登记质押/抵押 | 原始权益人须与质权人协商解除并办理注销登记 | 封包日前完成 |
| 司法查封/冻结 | 须取得执行法院解除查封/冻结裁定 | 封包日前完成 |
| 待登记质押（合同已签但未登记） | 原始权益人须书面确认放弃质押登记 | 封包日前取得确认函 |

**未能清理的处理**：直接剔除该笔资产，不可入池。

## 五、现金流归集路径法律安排

### 5.1 典型路径

```
[付款人/债务人] → [收款账户（原始权益人名下）] → [监管账户（原始权益人名下，监管行监管）] → [信托账户/托管账户（SPV名下，托管行托管）]
```

### 5.2 各环节法律核查要点

| 环节 | 法律关系 | 核查要点 | 主要风险 |
|------|---------|---------|---------|
| 债务人→收款账户 | 债权债务关系（债务人履行） | 债务人是否知悉并同意将款项付至指定账户 | 债务人未按通知变更付款账户 |
| 收款账户→监管账户 | 原始权益人内部划转 | 监管账户是否已约定专用且受托管行/监管行控制 | 原始权益人挪用回收款 |
| 监管账户→信托账户 | 原始权益人向SPV转付 | 转付机制及时效性（T+N日）、是否可被原始权益人单方终止 | 转付延迟或拒绝转付 |
| 信托账户→证券持有人 | SPV（通过托管行）向投资者分配 | 托管协议明确授权和分配指令流程 | 托管行执行错误或延迟 |

### 5.3 混同风险

当回收款在原始权益人账户中与原始权益人自有资金混合时，产生混同风险（Commingling Risk）。

**缓释措施**：
- 缩短回收款在原始权益人账户中的停留时间（缩短转付频率）
- 在原始权益人信用评级降至某档以下时{立即}调整转付频率（缩短至按周甚至按日）
- 设置后备资产服务机构

## 六、增信措施的法律效力

### 6.1 三类增信措施的法律约束力对比

| 增信措施 | 法律性质 | 约束力强度 | 强制执行难度 |
|---------|---------|-----------|-------------|
| 差额支付承诺 | 单方承诺（无担保合同） | ⚠️ 中等 | 须通过诉讼确认承诺效力 |
| 第三方担保（保证/抵押/质押） | 担保合同（《民法典》担保制度） | ✅ 高 | 可直接申请实现担保物权 |
| 流动性支持 | 协议安排（非担保性质） | ⚠️ 较中等 | 需通过合同诉讼强制执行 |
| 维好协议 | 道义性承诺（英文Keepwell Deed） | ❌ 低 | 法律强制执行难度大，通常仅具安慰作用 |

### 6.2 差额支付承诺法律效力强化措施

差额支付承诺虽为单方承诺，但可通过以下方式增强其法律约束力：

- 承诺函由承诺人公司章程授权的主体签署（需核查董事会/股东会决议）
- 承诺函中明确约定"不可撤销"+"无条件"
- 承诺函中约定仲裁条款以加速争议解决
- 在计划说明书中充分披露承诺的法律性质（非保证担保）

### 6.3 增信方主体资格审查

| 审查项 | 合规要求 |
|--------|---------|
| 增信方主体资格 | 具有独立法人地位，依法存续 |
| 增信方内部授权 | 增信措施的提供须经董事会/股东会审批通过 |
| 增信方信用能力 | 增信方主体评级≥{AA}，或提供等同于AAA评级的增信实力证明 |
| 增信方关联关系 | 增信方与原始权益人/管理人的关联关系须充分披露 |
