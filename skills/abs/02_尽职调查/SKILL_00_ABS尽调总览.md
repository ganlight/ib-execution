# SKILL_00 ABS尽调总览

> ABS尽职调查框架与执行规范

---

## 一、ABS尽职调查特殊性

### 1.1 与普通债券尽调的本质差异

| 维度 | 普通债券尽调 | ABS尽调 |
|------|------------|---------|
| **核心对象** | 发行人主体信用 | 基础资产池信用 + SPV结构有效性 |
| **尽调重点** | 财务状况、经营能力、偿债能力 | 资产池质量、历史表现、现金流预测 |
| **法律尽调重点** | 担保物权属、或有负债 | 资产转让合法有效、破产隔离、真实出售 |
| **财务尽调重点** | 财务报表审计、盈利能力分析 | 现金流模型验证、压力测试 |
| **信用分析重点** | 主体评级 → 债项评级 | 影子评级 + 结构化分层 → 各档证券评级 |

### 1.2 ABS尽调三大维度

```
        ┌──────────────────┐
        │   资产池尽调      │  ← 核心（权重约60%）
        │ (Asset Pool DD)  │
        └────────┬─────────┘
                 │
    ┌────────────┴────────────┐
    │                         │
┌───┴───┐               ┌─────┴─────┐
│法律尽调 │               │原始权益人 │
│(Legal │               │尽调      │
│ DD)   │               │(Originator│
└───────┘               │DD)       │
                        └──────────┘
```

## 二、资产池尽调（核心）

### 2.1 尽调目标

- 确认基础资产真实存在且权属清晰
- 评估资产池历史表现和未来现金流可预测性
- 测算资产池加权平均信用质量（影子评级）
- 分析资产池分散度和集中度风险
- 验证现金流预测模型合理性
- 执行压力测试

### 2.2 尽调方法

| 方法 | 适用场景 | 输出物 |
|------|---------|-------|
| 逐笔核查 | 对入池资产逐一核实合同、发票、付款凭证 | 《逐笔资产核查表》 |
| 统计抽样 | 资产笔数过多（如消费金融类）时按样本比例抽调 | 《抽样核查报告》 |
| 历史数据分析 | 分析累计违约率、早偿率、回收率、迁徙率 | 《资产池历史表现分析报告》 |
| 影子评级 | 对债务人/资产进行信用质量内部评估 | 《影子评级报告》 |
| 现金流建模 | 构建现金流预测模型并验证 | 《现金流预测模型》 |
| 压力测试 | 设计压力情景并测试现金流覆盖率 | 《压力测试报告》 |

### 2.3 尽调深度要求

| 资产类型 | 最低尽调覆盖比例 | 例外情形 |
|----------|-----------------|---------|
| 应收账款/供应链 | 100%逐笔核查 | 单笔金额较小且笔数≥1000笔的可采用抽样，覆盖率≥{X}% |
| 融资租赁 | 100%逐笔核查 | — |
| CMBS | 100%逐笔核查（每笔含抵押物评估） | — |
| 消费金融 | 抽样尽调，覆盖率≥{5-10}% | 须验证历史数据的统计有效性 |
| 收费收益权 | 核心收费权的权属文件+历史数据验证 | — |

## 三、法律尽调

### 3.1 尽调目标

- 确认SPV设立合法有效
- 确认基础资产转让合法有效（真实出售）
- 确认破产隔离安排有效
- 核查基础资产权利负担（质押/冻结/查封）
- 验证现金流归集路径法律安排
- 评估增信措施法律效力

### 3.2 尽调重点

| 尽调项 | 核查文件 | 法律风险点 |
|--------|---------|-----------|
| SPV设立 | 专项计划设立文件/信托合同/营业执照 | SPV主体资格、管理人资质 |
| 资产转让 | 资产买卖协议、债权转让通知 | 转让效力、通知义务履行 |
| 破产隔离 | 资产转让协议、律师法律意见 | 是否构成真实出售 vs 担保融资 |
| 权利负担 | 不动产登记查询、动产担保登记 | 抵押/质押是否解除 |
| 现金流归集 | 监管协议、托管协议 | 账户安排是否形成混同风险 |
| 增信措施 | 差额支付承诺函、担保合同 | 承诺人资质、承诺可执行性 |

## 四、原始权益人尽调

### 4.1 尽调目标

- 评估原始权益人持续经营能力（资产服务机构能力）
- 核查原始权益人主体信用（虽非证券偿付依赖，但影响资产服务）
- 确认无重大负面信息

### 4.2 尽调范围

- 基本工商信息核查
- 最近三年财务报告审计（重点关注资产负债率和流动性指标）
- 征信报告核查
- 重大诉讼/仲裁/行政处罚核查
- 资产服务机构能力和内控制度评估
- IT系统能力核查（如涉及资产持续管理）

### 4.3 尽调深度差异

| ABS类型 | 原始权益人尽调深度 | 原因 |
|---------|------------------|------|
| 供应链ABS | 较轻（聚焦核心企业信用） | 基础资产依赖核心债务人，原始权益人仅作为保理商 |
| 消费金融ABS | 较轻（聚焦资产池表现） | 资产池高度分散，原始权益人主体信用影响较小 |
| CMBS/类REITs | 较重（需穿透至物业持有人） | 物业运营依赖原始权益人/实际控制人 |
| 收费收益权ABS | 较重 | 基础资产未来现金流直接依赖原始权益人运营能力 |

## 六、尽调质量控制

### 6.1 三级复核机制

| 复核层级 | 复核人 | 复核重点 |
|---------|-------|---------|
| 初步复核 | 尽调执行人 | 完整性、形式合规性 |
| 二级复核 | 项目负责人 | 实质性内容准确性、风险识别充分性 |
| 最终复核 | 内核委员会 | 重大风险事项、结论合理性 |

### 6.2 尽调留痕要求

- 所有尽调材料须存档备查
- 访谈记录须有被访谈人签字确认
- 实地踏勘须有照片或视频记录
- 尽调底稿保存期限≥项目结束后5年

## 七、尽调常见问题与应对

| 常见问题 | 风险等级 | 应对措施 |
|---------|---------|---------|
| 资产权属不清 | 🔴高 | 剔除该笔资产或补充权属证明文件 |
| 历史数据不足12个月 | 🟡中 | 延长观察期或采用替代数据源 |
| 影子评级下调可能性大 | 🟡中 | 预留评级下调缓冲或提高增信强度 |
| 原始权益人财务恶化 | 🟡中 | 评估是否触发后备服务机构启动条款 |
| 合同条款不利于转让 | 🔴高 | 取得合同相对方书面豁免 |
