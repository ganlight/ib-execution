## 一、逐笔资产核查

### 1.1 15项核查清单

对于入池基础资产的每笔/样本，须逐一核查以下15项指标：

| # | 核查项 | 核查内容 | 验证依据 | 合格标准 |
|---|--------|---------|---------|---------|
| 1 | 基础合同 | 合同原件/扫描件存在且完整 | 《{XX}合同》原件/扫描件 | 合同签署合法有效，所有签署方已盖章 |
| 2 | 付款凭证 | 资金已实际支付或交付 | 银行转账记录/银行流水 | 金额、收款方、日期与合同匹配 |
| 3 | 发票/收据 | 税务凭证完整合规 | 增值税发票/普通发票/收据 | 发票真实且与合同金额匹配 |
| 4 | 确权文件 | 债务人/付款义务人确认债务 | 确权函/应收账款确认函/收货确认单 | 确权方盖章确认，金额与合同一致 |
| 5 | 履约进度 | 对照合同核查实际履约状态 | 交付单/验收报告/进度确认书 | 履约完成比例≥{95}%或有清晰进度确认 |
| 6 | 到期日 | 合同约定的付款到期日 | 合同条款原文 | 到期日明确且为确定日期（非浮动） |
| 7 | 利率/费率 | 合同约定的利率或费率 | 合同条款原文 | 利率/费率明确、无歧义 |
| 8 | 还本付息安排 | 还款计划（每期金额与日期） | 还款计划表/等额本息表 | 还款安排清晰且可量化为现金流 |
| 9 | 担保/保证（如适用） | 基础资产的附属担保权益 | 保证合同/抵押合同/质押合同 | 担保手续完整，已办理登记 |
| 10 | 保险（如适用） | 基础资产/抵押物的保险 | 保险单 | 保险覆盖基础资产风险，到期日覆盖证券期限 |
| 11 | 债务人信用信息 | 债务人工商/信用状况 | 工商信息/征信报告 | 债务人无破产/注销/严重负面信息 |
| 12 | 禁止转让条款 | 合同是否含债权转让限制 | 合同条款搜索"转让"关键词 | 无禁止转让条款，或已取得书面豁免 |
| 13 | 权利负担查询 | 基础资产是否已质押/冻结 | 动产融资统一登记公示系统/不动产登记系统 | 无未解除的权利负担 |
| 14 | 关联交易 | 原始权益人与债务人是否存在关联关系 | 工商股权穿透 | 关联交易占比≤{X}%，超出部分须单独披露 |
| 15 | 封包日状态 | 截至封包日该笔资产是否正常 | 封包日时点还款状态 | 无逾期超过{30}天的情况 |

### 1.2 核查样本要求

- 仲裁/诉讼资产：100%逐笔核查
- 单笔金额占资产池权重≥{1}%的资产：100%逐笔核查
- 其余资产：随机抽样，覆盖率≥{X}%
- 消费金融/零售类（小额分散）资产：分层抽样，覆盖率≥{5-10}%

### 1.3 核查不合格资产处理

| 不合格情形 | 处理方式 |
|-----------|---------|
| 缺少核心文件（合同/付款凭证） | 直接剔除 |
| 权属存在瑕疵 | 剔除或补充确权 |
| 权利负担未清理 | 限期清理，否则剔除 |
| 关联交易占比过高 | 超出限额部分剔除 |
| 债务人信用严重恶化 | 直接剔除 |

## 二、历史表现数据分析

### 2.1 四大核心指标定义与计算方法

#### 指标1：累计违约率（Cumulative Default Rate, CDR）

**定义**：在一定观察期内，发生违约（通常定义为逾期{90}天以上或核销）的资产金额（或笔数）占期初资产池金额（或笔数）的比例。

**计算公式**：

```
累计违约率 = Σ(违约资产本金余额) / 期初资产池本金总余额 × 100%
```

**分时间窗口计算**：

| 观察窗口 | 计算公式 |
|---------|---------|
| T=3个月 | CDR_3 = (违约金额(前3个月)) / 期初资产池 |
| T=6个月 | CDR_6 = (违约金额(前6个月)) / 期初资产池 |
| T=12个月 | CDR_12 = (违约金额(前12个月)) / 期初资产池 |
| T=24个月 | CDR_24 = (违约金额(前24个月)) / 期初资产池 |

**合格阈值**（仅供参考，具体以评级要求为准）：
- 正常类资产 CDR_12 ≤ {3}%
- 关注类资产 CDR_12 ≤ {5}%

#### 指标2：早偿率（Conditional Prepayment Rate, CPR / SMM）

**定义**：借款人在约定还款日之前提前偿还全部或部分本金的比例。

**SMM（单月提前还款率）**：

```
SMM = 当月提前还款金额 / (当月期初本金余额 - 当月计划偿还本金)
```

**CPR（年化提前还款率）**：

```
CPR = 1 - (1 - SMM)^12
```

**合格阈值**：CPR年化控制在 {5-20}% 范围内（极端偏离须纳入压力测试）

#### 指标3：回收率（Recovery Rate）

**定义**：违约资产经过处置后实际回收的金额占违约本息的比例。

**计算公式**：

```
回收率 = 累计回收金额 / 累计违约本息 × 100%
```

**合格阈值**：
- 有抵质押担保的资产：回收率≥{40-60}%
- 无抵质押担保的资产：回收率≥{10-30}%

#### 指标4：迁徙率（Migration Rate / Transition Matrix）

**定义**：不同信用质量分类之间的迁移概率，用于衡量信用质量恶化的速度和程度。

**五级分类迁徙矩阵**：

| 期初→期末 | 正常 | 关注 | 次级 | 可疑 | 损失 |
|----------|------|------|------|------|------|
| 正常 | {XX}% | {XX}% | {X}% | {X}% | {X}% |
| 关注 | {XX}% | {XX}% | {XX}% | {X}% | {X}% |
| 次级 | {XX}% | {XX}% | {XX}% | {XX}% | {X}% |
| 可疑 | 0% | {X}% | {XX}% | {XX}% | {XX}% |
| 损失 | 0% | 0% | 0% | 0% | 100% |

**分析方法**：基于历史时间序列数据，分时段计算五级分类迁移矩阵，观察恶化趋势。

### 2.2 历史数据分析方法论

| 分析步骤 | 方法 | 输出 |
|---------|------|------|
| 1. 数据清洗 | 剔除异常值、补全缺失值、统一口径 | 清洗后数据集 |
| 2. 时间序列分解 | 分离趋势、季节性和随机波动 | 图表+分析报告 |
| 3. 统计分布分析 | 绘制违约/早偿/回收的分布直方图 | 统计特征参数 |
| 4. 相关性分析 | 分析违约与宏观经济指标的相关性 | 相关系数矩阵 |
| 5. 基准比较 | 与同类资产的行业基准进行比较 | 偏离度分析 |

## 三、影子评级方法论

### 3.1 定义

影子评级（Shadow Rating）是指管理人或评级机构对基础资产债务人信用质量进行的内部评级，用于替代/补充公开的外部主体评级，支撑资产池整体信用质量评估。

### 3.2 单笔资产影子评级

**评级要素（五大维度）**：

| 维度 | 权重 | 评估内容 |
|------|------|---------|
| 债务人财务实力 | {25-35}% | 资产负债率、现金流状况、盈利能力 |
| 行业地位 | {15-20}% | 市场份额、竞争格局、行业准入壁垒 |
| 管理能力 | {10-15}% | 管理层经验、治理结构 |
| 信用记录 | {15-20}% | 过往违约/逾期记录、外部评级 |
| 交易结构 | {10-15}% | 担保/保证/抵质押方案的保障强度 |

**评分映射表**：

| 影子评分 | 对应外部评级（标普/中诚信） | 预期违约率（1年） |
|---------|--------------------------|-------------------|
| S1 | AAA | ≤{0.01}% |
| S2 | AA+ ~ AA | ≤{0.05}% |
| S3 | AA- ~ A+ | ≤{0.10}% |
| S4 | A ~ A- | ≤{0.25}% |
| S5 | BBB+ ~ BBB | ≤{0.50}% |
| S6 | BBB- ~ BB+ | ≤{1.00}% |
| S7 | BB ~ BB- | ≤{2.50}% |
| S8 | B+及以下 | >{2.50}% |

### 3.3 资产池加权平均信用等级

**计算方法**：

```
资产池加权平均影子评级 = Σ(单笔资产本金余额 × 对应影子评分权重) / Σ(单笔资产本金余额)
```

**资产池级别目标映射**：

| 资产池WA影子评级 | 优先A档可达到评级（经分层后） |
|-----------------|---------------------------|
| S2 ~ S3 | AAA |
| S3 ~ S4 | AA+ |
| S4 ~ S5 | AA |
| S5 ~ S6 | AA- |
| ≥S7 | 需显著增信后方能达到投资级 |

### 3.4 影子评级主要风险点

- 债务人无公开财务信息 → 须依赖替代数据源（税务信息、水电煤数据、支付行为）
- 样本覆盖不全 → 须在报告中明确尽调覆盖率和剩余部分的风险推断逻辑
- 定性判断比例过高 → 须尽可能量化和标准化以减少主观偏差

## 四、现金流预测模型

### 4.1 四因素模型

```
资产池净现金流 = 本金回收 + 利息回收 - 违约损失 + 回收款净额
```

#### 因素1：本金回收

```
本金回收(t) = 计划本金偿还(t) + 提前还款本金(t)
```

**关键假设**：
- 计划本金偿还：依据每笔资产的合同还款计划
- 提前还款本金：依据资产类型设定CPR假设（如融资租赁CPR={X}%）

#### 因素2：利息回收

```
利息回收(t) = 生息本金(t) × 加权平均利率 / {12}（按月）或按合同计息频率
```

**关键假设**：考虑逾期对利息回收的影响比例

#### 因素3：违约损失

```
违约损失(t) = 违约本金(t) × (1 - 回收率)
```

**关键假设**：
- 违约率：依据历史CDR和影子评级
- 回收率：依据历史数据和担保类型

#### 因素4：提前还款

```
提前还款金额(t) = 期初本金余额 × SMM(t)
```

### 4.2 模型构建步骤

| 步骤 | 操作 | 注意事项 |
|------|------|---------|
| 1 | 逐笔建立资产还款时间表 | 区分等额本息、等额本金、到期一次还本 |
| 2 | 叠加违约/回收假设 | 按时间段分档设定违约率（如第1年低、第2-3年升高） |
| 3 | 叠加提前还款假设 | 消费类建议CPR={X}%起，对公类建议CPR={Y}%起 |
| 4 | 汇总得到资产池端总现金流 | 时间粒度通常为月/季 |
| 5 | 将资产池端现金流分配至证券端 | 按Waterfall规则逐层分配 |

### 4.3 模型验证

- 独立第三方（会计师）对模型逻辑和参数设定进行复核
- 与被评级机构模型输出进行对比验证
- 对关键假设参数进行敏感性分析

## 五、压力测试情景设定

### 5.1 三等级×三维度（9情景）

#### 情景等级定义

| 情景等级 | 定义 | 发生概率预估 |
|---------|------|-------------|
| 轻度 | 基准情景参数小幅恶化 | 大概率事件（如经济小幅下行） |
| 中度 | 基准情景参数显著恶化 | 可能事件（如区域性经济衰退） |
| 重度 | 基准情景参数极端恶化 | 小概率事件（如系统性金融危机，对应评级AAA标准） |

#### 三维度参数加压

| 维度 | 轻度加压系数 | 中度加压系数 | 重度加压系数 |
|------|-----------|-----------|-----------|
| 利率变动 | +{50-100}bp | +{150-200}bp | +{300}bp及以上 |
| 违约率 | ×{1.5}倍 | ×{2.5}倍 | ×{5}倍 |
| 早偿率 | ×{1.3}倍 | ×{2}倍 | ×{3}倍 |

### 5.2 9情景矩阵

| 情景编号 | 利率 | 违约 | 早偿 | 情景描述 |
|---------|------|------|------|---------|
| S1 | 基准 | 基准 | 基准 | 正常情况 |
| S2 | +{X}bp | 基准 | 基准 | 利率上升 |
| S3 | 基准 | ×1.5 | 基准 | 轻度违约恶化 |
| S4 | 基准 | 基准 | ×1.3 | 轻度早偿增加 |
| S5 | +{Y}bp | ×2.5 | ×2 | 中度综合恶化 |
| S6 | +{Y}bp | ×2.5 | 基准 | 中度利率+违约 |
| S7 | 基准 | ×2.5 | ×2 | 中度违约+早偿 |
| S8 | +{Z}bp | ×5 | ×3 | 重度综合恶化 |
| S9 | +{Z}bp | ×5 | 基准 | 重度利率+违约 |

### 5.3 压力测试通过标准

- **AAA级优先档**：所有9情景下（含重度），本金和利息均按时足额偿付
- **AA+级优先档**：轻度+中度情景全部通过，重度情景下本金可能在极端条件下略有延期（通常≤18个月）
- **次级档**：不要求全部情景通过，但须揭示极端情景下的最大可能损失

## 六、资产池分散度分析

### 6.1 三维度阈值标准

#### 行业集中度

| 集中度区间 | 评定等级 | 处理建议 |
|-----------|---------|---------|
| 前1大行业 ≤ {20}% | ✅ 良好 | 无需额外关注 |
| 前1大行业 {20-35}% | ⚠️ 关注 | 需专项分析该行业风险 |
| 前1大行业 ≥ {35}% | ❌ 高风险 | 须提升增信或限制该行业入池比例 |

#### 地域集中度

| 集中度区间 | 评定等级 | 处理建议 |
|-----------|---------|---------|
| 前1大省 ≤ {25}% | ✅ 良好 | 无需额外关注 |
| 前1大省 {25-40}% | ⚠️ 关注 | 须结合该地区经济基本面分析 |
| 前1大省 ≥ {40}% | ❌ 高风险 | 须专项论证 |

#### 单一债务人集中度

| 集中度区间 | 评定等级 | 处理建议 |
|-----------|---------|---------|
| 前1大 ≤ {5}% | ✅ 良好 | 无需额外关注 |
| 前1大 {5-10}% | ⚠️ 关注 | 须逐笔核查债务人信用 |
| 前1大 {10-15}% | ⚠️ 警惕 | 评级机构可能额外加压 |
| 前1大 ≥ {15}% | ❌ 高风险 | 该笔资产建议剔除或核减至阈值以下 |

### 6.2 分散度量化指标：赫芬达尔指数（HHI）

```
HHI = Σ(每笔资产占资产池比重²)
```

| HHI区间 | 分散度评定 |
|---------|-----------|
| < {500} | 高度分散 |
| {500-1500} | 中度分散 |
| {1500-2500} | 中度集中 |
| > {2500} | 高度集中（须增信强化） |

## 七、封包期管理

### 7.1 定义

封包日（Cut-off Date）为基准资产池的确定时点。封包日至发行日（或信托设立日）之间为封包期。

### 7.2 封包期内资产替换规则

| 资产状态 | 处理规则 |
|---------|---------|
| 正常回款 | 回款金额计入信托账户，不替换（直接对应偿付） |
| 提前还款 | 回收款不用于替换，直接用于偿付或按循环购买规则处理 |
| 资产违约/逾期 | 须由原始权益人以同等质量资产替换 |
| 新增资产 | 可选入池（如为储架分期发行则自动纳入下期） |

### 7.3 封包期管理要点

- 封包期不宜过长，建议 ≤{3}个月（过长增加资产质量不确定性）
- 封包期结束须重新出具资产池概况更新报告
- 封包期内如发生重大资产质量恶化，须立即通告评级机构和监管
