# SKILL_06a：IPO路演与定价

## 一、概述

IPO路演与定价是承销工作的最后冲刺阶段。路演的目的是向潜在投资者传播发行人投资价值故事，定价是通过市场询价发现发行价格的过程。本技能涵盖投资故事撰写、路演材料制作、Q&A准备、簿记建档和定价配售全流程。

## 二、投资故事撰写

### 2.1 投资故事的定义

投资故事是一个1-2页的叙事性摘要，向投资者简洁有力地传达"为什么这家公司值得投资"。非招股书的缩水版，而是一个独立的说服性叙事。

### 2.2 投资故事"一页纸"结构

```
【一句话定位】
[公司名称]是[细分领域]的[领导者/先行者/专家]，[做什么]，
受益于[市场驱动力]的持续增长。

【市场机会】（市场规模+成长性）
  - 目标市场规模：TAM XX亿 → SAM XX亿 → SOM XX亿
  - 市场增长率：CAGR XX%（YYYY-YYYY）
  - 增长驱动因素：
    1. ...
    2. ...

【竞争优势】（护城河/差异化）
  - 竞争壁垒1：[技术/规模/品牌/网络效应]
  - 竞争壁垒2：[客户黏性/数据积累/政策壁垒]
  - 竞争壁垒3：[成本优势/供应链优势]

【增长策略】（未来3-5年规划）
  - 增长引擎1：...
  - 增长引擎2：...
  - 增长引擎3：...

【财务亮点】（过去+未来预期）
  - 收入：XXXX年XX亿 → XXXX年XX亿（CAGR XX%）
  - 毛利率：XX% → XX%
  - 净利率：XX% → XX%
  - ROE：XX%

【发行概况】
  - 发行规模：不超过XX%股权，募集约XX亿元
  - 募集资金用途：XX%用于扩张、XX%研发、XX%补流
  - 发行后市值区间：XX - XX亿元
```

### 2.3 投资故事撰写原则

1. **以投资者视角**：从"我为什么应该投"出发，而非"我们公司做了什么"
2. **数据驱动**：每个论点有可量化支撑
3. **对比突出**：与竞争对手的差异化必须鲜明
4. **故事有弧线**：从冲突（行业痛点）到解决方案（公司价值）到回报（增长前景）
5. **避免招股书语言**：不用"合规""规范"等招股书用词，用投资级语言

## 三、路演材料

### 3.1 路演PPT标准结构（建议30-40页）

| 页码 | 内容 | 时长建议 |
|-----|------|---------|
| 1-2 | 封面+发行概览 | 1分钟 |
| 3-4 | 投资亮点摘要（TOP 3-5个） | 2-3分钟 |
| 5-8 | 公司概况 | 3分钟 |
| 9-16 | 市场与行业分析 | 5-8分钟 |
| 17-24 | 业务与竞争优势分析 | 8-10分钟 |
| 25-30 | 财务分析 | 5-8分钟 |
| 31-35 | 增长策略与募投项目 | 5分钟 |
| 36-38 | 发行条款摘要 | 2分钟 |
| 39-40 | 附录（可比公司估值、风险因素） | 备查 |

总时长：35-45分钟演讲 + 15-20分钟Q&A

### 3.2 管理层演讲脚本要点

- 董事长/CEO开场：公司愿景和战略定位（3-5分钟）
- CFO主讲：财务亮点和展望（5-8分钟）
- 保荐人补充：发行条款和可比估值（5分钟）
- Q&A环节：管理层主答，保荐人辅助

### 3.3 路演PPT设计原则

- 每页不超过5个要点
- 图表为主，文字为辅
- 数据可视化（柱状图/趋势线/对比图）
- 投资者关注的指标突出显示
- 避免信息过载（每页一个核心观点）

## 四、Q&A准备

### 4.1 路演常见问题50问框架

**公司业务类（约15题）**
1. 请简要介绍公司是做什么的？
2. 公司的目标客户是谁？
3. 主要竞争对手有哪些？相比他们的差异化优势是什么？
4. 公司的核心竞争力体现在哪些方面？
5. 公司的技术壁垒有多高？
6. 公司所在行业的进入门槛是什么？
7. 公司未来3-5年的增长驱动力是什么？
8. 海外业务的占比和增长趋势？
9. 公司是否会拓展新的业务领域？

**市场行业类（约10题）**
10. 公司所在行业的市场规模及增速？
11. 行业竞争格局是怎样的？
12. 行业的政策环境如何？是否存在政策风险？
13. 国产替代/进口替代的空间有多大？
14. 下游行业需求的前景如何？

**财务类（约15题）**
15. 过去3年收入增长的驱动因素是什么？（量vs价）
16. 毛利率趋势及未来展望？
17. 研发投入占收入比重偏低/偏高的原因？
18. 应收账款周转天数长的原因及回款风险？
19. 经营活动现金流波动大的原因？
20. 未来利润率提升的空间在哪里？
21. 资本支出计划和资金来源？
22. 分红政策？
23. 税收优惠的可持续性？

**募集资金类（约5题）**
24. 募集资金的具体用途和预期回报？
25. 募投项目达产后的产能增量？
26. 是否存在产能过剩风险？
27. 补充流动资金比例是否偏高？

**估值与发行类（约5题）**
28. 可比公司选择有哪些？
29. 如何看待自身估值水平？
30. 发行后锁定期安排？
31. 大股东是否参与配售？

### 4.2 Q&A应对原则

1. **管理层主答**，保荐人补充但不能代替
2. **保持一致口径**，管理层不能出现前后矛盾
3. **诚实面对问题**，不知道的说不清楚比瞎编强
4. **桥接技巧**：被问敏感问题时，过渡到已知信息的阐述
5. **练习练习再练习**：至少进行3次模拟路演

## 五、簿记建档与定价

### 5.1 簿记建档流程

```
1. 发布发行公告（含价格区间）
2. 路演开始，接收投资者订单（价格+数量）
3. 簿记截止
4. 分析订单簿（投资者质量、价格分布、需求热图）
5. 定价会议（发行人与保荐人协商确定最终发行价）
6. 发布定价公告
```

### 5.2 订单簿分析维度

| 维度 | 分析要点 |
|-----|---------|
| 总量 | 总申购倍数（覆盖倍数），通常3-5倍为健康水平 |
| 价格分布 | 投资者出价在各价位的分布 |
| 投资者构成 | 公募基金、保险、QFII、私募、个人投资者的比例 |
| 投资者质量 | 知名长线投资者占比、是否有"热钱"（短期交易者） |
| 限价单vs市价单 | 限价单（limit order）和市价单（market order）的比例 |
| 价格上限占比 | 按区间上限出价的比例，反映投资者价格接受度 |

### 5.3 定价决策

**定价考虑因素**：
- 订单簿覆盖倍数和价格分布
- 可比公司最新估值水平（PE/PB/PS/EVEBITDA倍数）
- 市场环境和板块近期走势
- 投资者反馈和需求质量
- 发行人预期和接受度
- IPO后表现（首日涨幅预期）

**定价策略**：
- "留一线"定价：略低于市场可接受上限，为后市留空间
- 首日涨幅：通常控制在10%-44%之间，过高或过低都不理想
- 绝对价格：注意股价"视觉效应"，配合适当拆股或股本设计

## 六、配售原则

### 6.1 配售对象分类

| 类别 | 配售比例 | 锁定要求 | 配售原则 |
|-----|---------|---------|---------|
| 战略投资者 | 视情况（需单独协议） | 12个月+ | 产业协同/长期合作 |
| 网下机构投资者 | 70%-80% | 无 | 价格优先→时间优先→按比配售 |
| 网上公众投资者 | 20%-30% | 无 | 摇号抽签 |

### 6.2 配售优化原则

- **质量优先于价格**：优质长线投资者优先于出价高但短期投资者
- **分散化**：避免单一投资者持股集中
- **历史关系**：回馈在路演中提供价值的投资者
- **后市稳定**：配售给可能长期持有的投资者
- **不鼓励"打新专业户"**：减少纯打新、上市即卖出的投资者

## 七、发行失败应对预案

### 7.1 认购不足处理

- 如遇大幅认购不足，可终止本次发行
- 调整发行方案（缩小发行规模、降低价格区间）重新发行
- 延期发行，等待更好市场窗口
- 大股东/关联方参与认购补充

### 7.2 极端市场波动

- 建立市场波动熔断机制（大盘单日跌超X%暂停簿记）
- 预留发行时间弹性（批文有效期内有缓冲）
