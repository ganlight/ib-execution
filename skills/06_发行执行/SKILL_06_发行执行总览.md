# SKILL_06：发行执行总览

## 一、概述

发行执行是获得监管批文后，将证券推向市场完成资金募集的最后一公里。IPO发行涉及路演、簿记建档、定价和配售全链条；债券发行由债券资本市场部（DCM）主导；并购重组发行涉及交割条件的满足和资产过户。本技能提供发行执行的总体框架和各产品线指引。

---

## 二、发行执行全流程图

### 2.1 总体时间线总览

```
阶段0：批文取得
  ├── 取得证监会/交易所注册批文（IPO）/ 核准文件（债券）/ 重组批文（并购）
  ├── 确认批文有效期（IPO：12个月 | 债券：24个月 | 并购：12个月）
  └── 发行窗口初步判断（结合二级市场走势、行业表现、资金面）

阶段1：路演准备（T-21 至 T-14）
  ├── 投资故事定稿
  ├── 路演PPT及Q&A手册制作
  ├── 分析师预路演（Pre-deal Analyst Education）
  └── 确定承销团分工及路演行程

阶段2：投资者教育/预路演（T-14 至 T-7）
  ├── 预路演（Pre-marketing）：向核心机构投资者非正式推介
  ├── 收集投资者初步反馈（估值预期、关注问题）
  ├── 根据反馈调整定价区间和投资故事重点
  └── 确定正式路演价格区间

阶段3：正式路演（T-7 至 T-1）
  ├── 管理层路演（一对一/小组会议/午餐会）
  ├── 分析师路演（面向买方分析师）
  ├── 投资者订单收集（簿记建档/累计投标）
  └── 每日簿记更新和策略调整

阶段4：定价与配售（T日）
  ├── 簿记结束，汇总订单簿
  ├── 定价会议（发行人与承销团协商最终发行价）
  ├── 配售方案确定（战略/机构/零售分配比例）
  └── 签署承销协议/配售协议

阶段5：发行结果公告（T+1 至 T+3）
  ├── T+1：发布发行结果公告（发行价/中签率/冻结资金）
  ├── T+2：网上中签缴款（IPO）/ 投资者缴款确认（债券）
  └── T+3：募集资金到账，验资

阶段6：挂牌上市（T+7 至 T+14）
  ├── 股份登记/债券登记托管
  ├── 上市公告书/债券上市公告发布
  ├── 上市仪式（敲钟/敲锣）
  └── 首日交易监控（重点关注破发风险）

阶段7：募集资金到账与后续管理
  ├── 募集资金存入专项账户
  ├── 验资报告出具
  ├── 募集资金使用计划启动
  └── 转入持续督导/受托管理阶段
```

### 2.2 发行执行关键角色分工

| 角色 | IPO职责 | 债券职责 | 并购职责 |
|-----|---------|---------|---------|
| **保荐人/主承销商** | 路演统筹、定价建议、配售执行 | 簿记建档管理、利率区间建议 | 交割条件核查、交割执行 |
| **发行人管理层** | 路演主讲、投资者问答 | 投资者沟通 | 资产/股权交割配合 |
| **承销团成员** | 投资者覆盖、订单收集 | 分销 | — |
| **律师** | 发行文件法律审查 | 法律意见书（发行阶段） | 交割法律意见书 |
| **会计师** | 安慰函/告慰函 | 财务数据更新 | 交割审计 |
| **登记结算机构** | 股份登记 | 债券托管登记 | 股份过户登记 |
| **交易所** | 上市审核 | 挂牌审核 | 信息披露审核 |

## 三、IPO vs 债券 vs 并购 三条线发行执行对比表

### 3.1 全维度对比

| 维度 | IPO | 债券 | 并购重组 |
|-----|-----|------|---------|
| **批文类型** | 注册批文（证监会/交易所） | 注册批文/核准文件 | 重组核准文件 |
| **批文有效期** | 12个月 | 24个月 | 12个月（可申请延期） |
| **路演周期** | T-14至T-1（约2周） | T-7至T-1（约1周） | 不适用（直接进入交割） |
| **路演形式** | 管理层一对一 + 小组会 + 午餐会 | DCM团队一对一 + 电话会 | — |
| **定价机制** | 累计投标询价/直接定价 | 簿记建档（利率区间投标） | 按交易协议约定对价 |
| **配售对象** | 战略配售 + 网下机构 + 网上公众 | 合格机构投资者（银行间/交易所） | 交易对方 |
| **网上申购** | 有（公众投资者参与） | 无（仅机构投资者） | 无 |
| **回拨机制** | 网上/网下回拨 | 无 | 无 |
| **绿鞋机制** | 有（超额配售选择权） | 无 | 无 |
| **资金到账时间** | 申购后T+2至T+3 | 缴款日（T+1至T+3） | 交割日 |
| **上市/挂牌审核** | 交易所上市委员会审核 | 交易所挂牌（流程更简） | 新增股份上市审核 |
| **上市仪式** | 敲钟/敲锣仪式 | 一般不举行 | 一般不举行 |
| **登记机构** | 中国结算（股票） | 中债登/上清所（债券） | 中国结算（新增股份）/ 工商局（股权） |
| **核心风险** | 发行失败 / 破发 / 认购不足 | 发行失败 / 利率上行 / 包销风险 | 交割条件未满足 / 资产瑕疵 |
| **关键文件** | 招股意向书、发行公告、投资价值分析报告 | 募集说明书、发行公告、簿记建档结果公告 | 交割确认函、交割法律意见书、资产过户公告 |
| **持续义务起点** | 上市之日起 | 起息日起 | 交割完成日起 |

```

T-1：簿记截止（通常15:00）

T-1 晚间：配售方案确定
T日：发行结果公告

D+14：新增股份上市
```

## 四、发行前的10项最终核验清单

> **使用时机**：在发行公告发布前（通常T-3至T-1），由保荐人牵头逐项核验。如发现任何一项未满足，须立即召开专项会议决定是否延期发行。

### 核验清单

| 序号 | 核验事项 | 核验标准 | 责任人 | 核验方法 | ✅ |
|:---:|:---|:---|:---|:---|:---:|
| 1 | **招股书/募集说明书终稿核对** | 所有信息为最新、无重大遗漏、数据一致性校验通过（财务数据与审计报告一致、业务数据与尽调底稿一致、签字页齐全） | 保荐人 + 发行人 | 逐页核对 + 数据交叉验证 | [ ] |
| 2 | **律师出具发行阶段法律意见书** | 法律意见书涵盖：发行批准程序合规、发行人主体资格有效、无重大不利变化、发行文件法律合规 | 发行人律师 | 审查法律意见书正本 | [ ] |
| 3 | **会计师出具安慰函/告慰函** | Comfort Letter覆盖范围：财务数据无重大变化、募集资金用途与招股书一致、负资产保证（Negative Assurance）范围恰当 | 审计师 | 审查安慰函原件 | [ ] |
| 4 | **承销协议/承销团协议签署** | 协议条款完整：承销方式（包销/代销）、发行价格区间、超额配售权条款、终止条款、违约责任 | 保荐人 + 发行人 | 审查签字盖章完整 | [ ] |
| 5 | **募集资金到账安排确认** | 募集资金专项账户已开立、银行收款路径确认、跨行资金调拨方案已测试、外汇结算安排（如涉及境外投资者）已就绪 | 发行人 + 保荐人 | 银行确认函 + 测试 | [ ] |
| 6 | **媒体关系预案** | 舆情监测机制已启动、媒体采访应答手册已准备、敏感问题Q&A已演练、危机公关联系人已明确、社交媒体监控已部署 | 财经公关 + 发行人 | 预案文件审查 + 演练 | [ ] |
| 7 | **投资人沟通记录回顾** | 路演期间投资者关切问题已汇总、核心投资者意向已确认、可能影响定价的重大反馈已传达至发行人管理层 | 保荐人 + 承销团 | 投资者反馈汇总表 | [ ] |
| 8 | **发行公告/上市公告书准备** | 发行公告（含发行价/发行规模/申购方式/时间表）已定稿，上市公告书初稿已完成 | 保荐人 + 发行人 | 文件定稿审查 | [ ] |
| 9 | **技术系统准备** | 交易所发行系统接入测试通过、网上申购平台就绪、簿记建档系统验证、登记结算系统接口测试通过 | 保荐人 + IT | 系统测试报告 | [ ] |
| 10 | **信息披露一致性最终检查** | 招股书、发行公告、路演材料、分析师报告（如有）之间的信息一致性；不存在选择性披露（Selective Disclosure）风险 | 保荐人 + 律师 | 交叉审查 | [ ] |

### 核验不通过的处理原则

| 不通过项 | 处理方案 |
|:---|:---|
| 任何一项"未通过" | 暂停发行时间表，召开专项会议评估影响 |
| 涉及重大信息披露（第1、2项） | 立即修正文件，重新履行内部审批流程 |
| 涉及法律/会计意见（第2、3项） | 待中介机构出具更新意见后再启动 |
| 涉及市场/技术层面（第6-9项） | 制定补救时间表，并行推进其他事项 |

## 五、各产品线发行执行对比

### 5.1 IPO发行执行全景

```
获得注册批文 → 
投资故事撰写 → 
路演准备（PPT+Q&A+行程）→ 
预路演（Pre-marketing）→ 
正式路演（Roadshow）→ 
簿记建档/累计投标 → 
定价 → 
配售（战略+机构+零售）→ 
发行结果公告 → 
募集资金到账 → 
股份登记 → 
上市交易
```

- 发行时间窗口：注册批文有效期内（12个月）
- 最优窗口：路演→定价→上市通常4-6周
- 关键决策：发行时机（市场窗口）、发行价格区间、发行规模

详细内容见 **SKILL_06a_IPO路演与定价.md**。

### 5.2 债券发行执行

> **→ 委托给 bond-dcm-core 技能处理。**

债券发行涉及簿记建档、利率区间确定、投资者分配等DCM专业领域工作，由债券资本市场部主导完成。具体流程参见 bond-dcm-core SKILL.md。

### 5.3 并购重组执行

```
获得批文 → 
满足交割前提条件 → 
股权过户/资产交割 → 
对价支付 → 
工商变更登记 → 
新增股份登记上市（如发股）→ 
过渡期损益结算 → 
中期/年度合并报表编制
```

详细内容见 **SKILL_06c_并购交割.md**。

## 六、发行成功的关键因素

### 6.1 市场窗口判断

| 因素 | 有利窗口特征 | 不利窗口特征 |
|-----|-------------|-------------|
| 二级市场走势 | 趋势向上，波动率适中 | 大幅下跌，波动率攀升 |
| 行业板块表现 | 板块强势，估值合理 | 板块遭受政策冲击或利空 |
| 资金面 | 流动性充裕，打新积极 | 资金紧张，打新意愿低 |
| IPO节奏 | 同期IPO项目少，供需有利 | 扎堆发行，资金分流 |
| 政策面 | 监管态度积极 | 监管收紧/暂缓 |

### 6.2 估值定价艺术

- 合理估值区间（不过高不缺需求，不过低不亏待老股东）
- 可比公司选择恰当（业务相近、规模可比、上市时间相似）
- 充分考虑发行时的市场情绪

## 七、常见发行执行陷阱

### 7.1 发行失败风险

| 陷阱 | 表现 | 预防 |
|-----|------|------|
| 定价过高 | 机构认购不足，发行失败 | 充分的路演定价发现，合理定价 |
| 发行时机错误 | 市场大跌期间发行 | 密切跟踪市场，灵活调整窗口 |
| 投资故事薄弱 | 路演反响平平，估值上不去 | 提前打磨投资故事 |
| 承销团协调不力 | 各承销商口径不一 | 统一话术，每日沟通会 |
| 信息泄露 | 定价前敏感信息外泄 | 严格信息隔离制度 |

### 7.2 批文过期风险

- 批文有效期12个月，到期需重新申请
- 如在有效期内无法完成发行，需重新走注册流程
- 避免"先拿批文再等窗口"的策略，应先选窗口再走最后注册环节

## 八、风险预案

### 8.1 发行失败风险及补救措施

#### 情景A：认购不足

| 认购倍数 | 状态 | 应对策略 |
|:---:|:---|:---|
| >2.0x | ✅ 充分覆盖 | — |
| 1.5x-2.0x | ⚠️ 适度覆盖 | 评估是否调整配售比例，优先保证战略投资者和核心机构 |
| 1.0x-1.5x | 🟡 勉强覆盖 | 立即启动以下措施：①扩大承销团分销力度；②与基石投资者沟通追加订单；③考虑下调价格（区间内调整） |
| <1.0x | 🔴 认购不足 | **紧急预案**：①商请包销承销商履行余额包销义务；②定价下调（若在区间内）；③推迟发行（若大幅不足）；④评估是否缩减发行规模 |

**簿记策略调整三步法**：

```
Step 1（T-3）：发现认购不足信号 → 通知承销团加大营销力度
Step 2（T-2）：认购不足趋势确认 → 管理层电话会，讨论降价/缩量方案
Step 3（T-1）：最终评估 → 如覆盖倍数<1.0x → 决定延期（保留批文）还是包销兜底
```

#### 情景B：破发

**首日破发应急预案**：

```
破发10%以内：
  → 正常监控，不主动介入
  → 准备投资者说明材料（强调基本面价值）
  
破发10%-20%：
  → 发行人与保荐人联合声明（强调长期价值）
  → 启动绿鞋机制（如有）稳定股价
  → 与核心机构投资者一对一沟通
  
破发>20%：
  → 启动绿鞋全额行使 + 大股东自愿增持（如有）
  → 召开投资者电话会议
  → 评估是否触发"股价维稳预案"
```

#### 情景C：绿鞋机制（超额配售选择权）

```
绿鞋机制运作流程：

1. 发行时：主承销商超额配售不超过发行数量15%的股份
2. 上市后30日内：
   ├── 股价≥发行价 → 主承销商行使超额配售权（发行人增发）
   │     → 募集资金增加，绿鞋全额行使
   └── 股价<发行价 → 主承销商从二级市场买入股票
         → 买入部分不增发，稳定股价
3. 30日后：超额配售权到期，主承销商确定最终超额配售数量

绿鞋行使判断：
  · 上市后5日平均价 vs 发行价
  · 上市后10日平均价 vs 发行价
  · 市场整体走势和行业表现
```

### 8.2 重大突发事件应急预案

| 突发事件 | 影响 | 应对方案 |
|:---|:---|:---|
| 路演期间发行人出现重大负面舆情 | 🔴 发行暂停/终止 | 立即启动内部核查 → 评估是否构成重大信息披露违规 → 向监管报告 → 暂停路演至澄清完毕 |
| 发行期间行业重大政策变化 | 🟡 影响估值和投资者情绪 | 补充披露政策影响分析 → 调整路演话术 → 评估是否需要更新风险因素章节 |
| 簿记期间市场暴跌（>5%） | 🟡 投资者撤单或观望 | 延长簿记截止时间（如可行）→ 与核心投资者直接沟通确认意向 → 评估推迟发行 |
| 网上/网下技术系统故障 | 🟡 影响申购流程 | 启动灾备系统 → 通知交易所 → 评估是否需要推迟申购截止时间 |
| 承销团成员临时退出 | 🟡 承销能力下降 | 评估剩余承销团包销能力是否满足最低发行规模要求 → 协调其他承销商补充 |
| 锁定期内大股东违规减持 | 🔴 监管处罚 + 市场信任危机 | 立即报告监管 → 督促承诺方纠正 → 发布补充公告 |

## 十、发行执行的质量控制

- [ ] 发行文件（招股意向书/发行公告）已更新至最新
- [ ] 路演材料经发行人、保荐人双重审核
- [ ] 所有对外披露信息经信息披露委员会审批
- [ ] 定价分析模型的假设和输入数据经过验证
- [ ] 承销协议/配售协议已签署
- [ ] 资金到账路径和账户已确认
- [ ] 发行前10项最终核验全部通过
- [ ] 风险预案（认购不足/破发/突发事件）已确认并通知全体承销团成员
- [ ] 绿鞋机制（如适用）触发条件及执行流程已确认
- [ ] 上市仪式安排（时间/地点/参与人员）已确认
- [ ] 舆情监测系统启动（至少覆盖发行周+上市首周）
- [ ] 投行内核/风控对发行定价方案已出具意见

## 十一、发行执行常用术语对照

| 中文 | 英文 | 说明 |
|:---|:---|:---|
| 簿记建档 | Bookbuilding | 投资者提交申购订单，承销商汇总形成订单簿 |
| 累计投标 | Cumulative Bidding | 投资者在价格区间内投标，累计确定最终发行价 |
| 超额配售选择权 | Over-allotment Option / Greenshoe | 主承销商在上市后30日内可超额配售≤15% |
| 回拨机制 | Clawback Mechanism | 网上/网下申购比例调整机制 |
| 战略配售 | Strategic Placement | 向战略投资者定向配售 |
| 基石投资者 | Cornerstone Investor | 在IPO前承诺认购一定股份的投资者 |
| 安慰函 | Comfort Letter | 会计师对发行文件财务数据的确认函 |
| 认购倍数 | Subscription Ratio / Oversubscription | 申购总额÷发行规模 |
| 包销 | Firm Commitment / Underwriting | 承销商保证全部发行 |
| 代销 | Best Efforts | 承销商尽力销售但不承诺包销 |
