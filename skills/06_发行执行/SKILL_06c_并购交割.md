# SKILL_06c：并购交割

## 一、概述

并购交割是重组获批后，按照交易协议完成股权/资产转移、对价支付和相关法律手续的最后环节。交割是风险最高的阶段之一——任何一件交割前提条件未满足，都可能导致交易无法完成甚至引发纠纷。本技能提供交割全程管理指南。

## 二、交割前提条件核查

### 2.1 交割前提条件分类

交割前提条件（Conditions Precedent, CP）是交易协议中约定的、交割前必须满足的各项条件。分为以下几类：

#### A类：监管审批类

```
□ 证监会/交易所（若需）对本次重组的核准/注册决定已出具，且未被撤销
□ 商务部反垄断审查（若触发申报门槛）已通过或确认不适用
□ 国资监管部门批复（若涉及国有资产）已出具
□ 外资准入审查（若涉及外资）已完成
□ 其他行业主管部门审批（视行业要求）
□ 所有审批均在有效期内，未被暂停或撤销
```

#### B类：交易双方内部审批类

```
□ 上市公司董事会/股东大会已审议通过
□ 交易对方已履行内部决策程序（董事会/股东会）
□ 标的公司已履行内部决策程序
□ 债权人同意（若交易触发借款合同中的控制权变更条款）
```

#### C类：第三方同意类

```
□ 重大合同的交易对方出具同意函（若交易触发合同中的控制权变更条款）
□ 合资合作伙伴同意（若适用）
□ 银行/金融机构同意（若涉及）
□ 核心客户/供应商确认不因重组变更合作条款
```

#### D类：状态确认类

```
□ 交易双方陈述与保证（Representations & Warranties）在交割日仍真实准确
□ 未发生重大不利变化（Material Adverse Change, MAC）
□ 交易协议未终止且完全有效
□ 标的公司资产/业务/人员无重大变化
□ 不存在限制或禁止交割的法院禁令或政府命令
```

| CP类别 | 主核查方 | 协助方 | 证明文件 |
|--------|---------|--------|---------|

## 三、交割流程

### 3.1 交割日流程

```
交割日前一周：
  - 召开交割准备会，确认所有CP状态
  - 准备交割文件包
  - 资金到位确认（收购方）

交割日前一日：
  - 最后一轮CP逐条确认
  - 签署交割确认函各方完成内部签批
  - 股权过户/资产过户材料准备完毕

交割日当日：
  Step 1: 各方签署交割确认函（确认CP全部满足）
  Step 2: 标的公司办理股东名册变更 / 工商变更登记申请
  Step 3: 收购方按约定支付首期对价
  Step 4: 获取工商变更受理回执
  Step 5: 标的公司印章、证照、财务资料的移交

交割日后1-5个工作日：
  - 工商变更登记完成
  - 支付余款（如有）
  - 新董事会/管理层到位
```

### 3.2 股权过户

**有限责任公司**：工商变更登记（股东变更登记）
- 准备材料：股权转让协议、股东会决议、章程修正案、新公司章程、股权转让完税证明
- 办理机构：标的公司注册地工商行政管理局/市场监督管理局
- 完成标志：取得变更后的《营业执照》

**股份公司股份**：
- 变更股东名册（公司内部）
- 中国证券登记结算公司过户登记（如涉及上市公司股份）

### 3.3 资产过户

**土地使用权**：不动产登记中心办理变更登记
**房屋所有权**：不动产登记中心办理变更登记
**知识产权**：专利局/商标局/版权中心办理权利人变更
**车辆**：车管所办理所有人变更
**一般设备/存货**：签署资产交接确认书

### 3.4 对价支付

| 支付时间点 | 支付比例 | 前提条件 |
|-----------|---------|---------|
| 交割日 | 50%-70% | CP全部满足+交割确认 |
| 工商变更完成 | 20%-30% | 工商变更登记受理证明 |
| 过渡期审计后 | 10%-20% | 过渡期审计报告出具+损益结算 |
| 业绩承诺期满 | 视协议 | 业绩承诺完成 |

> 建议留存一定比例的尾款（≥10%），以应对交割后可能出现的资产瑕疵和其他风险。

## 四、交割后事项

### 4.1 公告和披露

- [ ] 上市公司发布《关于重大资产重组完成资产过户的公告》
- [ ] 披露：交易对价、股权过户情况、标的公司基本信息
- [ ] 独立财务顾问出具《重大资产重组实施情况之独立财务顾问核查意见》
- [ ] 律师出具《重大资产重组实施情况之法律意见书》
- [ ] 披露工商变更后的新股权结构图

### 4.2 后续登记（如涉及发股）

- [ ] 新增股份登记：向中国结算提交登记申请
- [ ] 新增股份上市：向交易所提交上市申请

### 4.3 公司治理变更

- [ ] 修改上市公司章程
- [ ] 改选/补选董事会成员
- [ ] 改选/补选监事会成员
- [ ] 聘任/调整高级管理人员
- [ ] 调整内部组织架构
- [ ] 标的公司证照印鉴更新

## 五、交割风险管理

### 5.1 常见交割延迟原因

| 延迟原因 | 发生概率 | 建议 |
|---------|---------|------|
| 工商变更审核排队 | ⭐⭐⭐⭐ | 提前预约+加急通道 |
| 资金调配未及时到位 | ⭐⭐⭐ | 提前3个工作日确认资金路径 |
| 第三方同意尚未取得 | ⭐⭐⭐ | 申报阶段即启动沟通 |
| MAC事件发生 | ⭐⭐ | 密切关注行业和对手方动态 |
| CP文件签字体例争议 | ⭐⭐ | 提前草拟，减少最后时刻改字 |

### 5.2 交割失败预案

- 协议中应约定：如交割在约定截止日（Long Stop Date）前未完成，任一方可终止协议
- 约定违约责任（违约金/分手费）
- 设定交割延长期（Extension）条件和程序

## 六、交割完成标志

| 序号 | 完成标志 | 确认文件 |
|-----|---------|---------|

| 5 | 法律意见书出具 | 法律意见书 |
