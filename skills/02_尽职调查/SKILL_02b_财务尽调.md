# SKILL_02b：财务尽调

## 概述

财务尽职调查的核心目标是验证财务信息的**真实性、完整性和规范性**。重点关注收入确认、资产质量、负债完整性、关联交易公允性和盈利可持续性五大方面。

---

## 一、银行流水核查

### 1.1 全账户覆盖

核查范围：发行人及其子公司**所有银行账户**（含已注销账户，报告期内）

| 核查步骤 | 内容 |
|----------|------|
| Step 1 | 获取《已开立银行结算账户清单》（人行打印） |
| Step 2 | 逐户获取报告期内全部银行对账单 |
| Step 3 | 银行对账单与公司银行日记账双向核对 |
| Step 4 | 大额交易抽查（≥50万元） |
| Step 5 | 关联方资金往来专项核查 |
| Step 6 | 个人卡核查（实控人/董监高/关键岗位人员） |

### 1.2 大额交易核查（≥50万元）

| 核查维度 | 核查方法 |
|----------|----------|
| 交易对手 | 是否为关联方/实控人控制企业 |
| 交易性质 | 是否与公司业务相关 |
| 交易时间 | 是否存在期末突击交易 |
| 交易匹配 | 是否有对应合同/发票/验收单 |
| 资金来源 | 同一对手大额进出是否循环交易 |

### 1.3 🔴 红旗指标

- 发现未披露的银行账户
- 大额资金在关联方之间频繁划转
- 期末大额资金流入、期初流出（粉饰货币资金）
- 银行日记账与银行对账单存在重大未达账项（≥100万元）
- 实控人/高管个人卡与公司存在大额资金往来
- 银行账户存在冻结/止付情况

## 二、应收账款

### 2.1 账龄分析

| 账龄 | 正常比例范围 | 警示线 |
|------|-------------|--------|
| 1年以内 | 70-90% | <60% 需解释 |
| 1-2年 | 5-20% | >30% 需解释 |
| 2-3年 | 2-10% | >15% ⚠️ Amber |
| 3年以上 | <5% | >10% 🔴 Red |

### 2.2 前5大应收账款客户分析

对前5大应收账款客户逐一核查：

| 核查内容 | 方法 |
|----------|------|
| 客户基本信息 | 工商信息核查、是否为关联方 |
| 交易背景 | 对应合同、验收单、收入确认 |
| 账期 | 与合同约定是否一致、是否逾期 |
| 回款历史 | 历史回款周期、坏账记录 |
| 期后回款 | 资产负债表日后的回款情况 |
| 信用风险 | 客户经营状况、是否存在失信/破产 |

### 2.3 🔴 红旗指标

- 应收账款增速远超收入增速（>2倍）
- 1年以上账龄应收占比>30%
- 前5大应收客户中有多个为当年新增客户
- 前5大客户与公司存在关联关系
- 期后回款率<50%（截至DD日）
- 大额应收客户存在失信/破产/重大诉讼
- 坏账准备计提比例远低于同行业

## 三、存货

### 3.1 账龄分析

| 原材料 | 在产品 | 库存商品 |
|--------|--------|----------|
| 1年以内 = 正常 | 按生产周期判断 | 1年以内 = 正常 |
| 1-2年 = 关注慢动 | — | 1-2年 = 滞销关注 |
| >2年 = 减值风险 | — | >2年 = 大概率减值 |

### 3.2 跌价准备测试

围绕**可变现净值**判断：

```
可变现净值 = 预计售价 - 至完工尚需成本 - 销售费用 - 相关税费

若 可变现净值 < 账面成本 → 计提跌价准备
```

### 3.3 🔴 红旗指标

- 存货金额快速增长而收入增长缓慢
- 1年以上库存商品占比>20%
- 跌价准备计提比例显著低于同行业
- 已过保质期/淘汰型号未计提减值
- 盘点时发现大量残次/陈旧存货
- 产成品单位成本高于当前售价
- 存货存放于第三方但无书面协议

## 四、关联交易与资金占用

### 4.1 关联方识别

| 关联方类型 | 识别范围 |
|------------|----------|
| 控股股东/实际控制人 | 本人+控制的其他企业 |
| 持股5%以上股东 | 本人+控制的企业 |
| 董监高 | 本人+近亲属+控制的企业 |
| 子公司 | 发行人合并范围内所有子公司 |
| 其他 | 过去12个月曾为上述关联方的 |

### 4.2 关联交易核查

| 交易类型 | 核查要点 |
|----------|----------|
| 采购商品/服务 | 定价是否公允、是否商业必要 |
| 销售商品/服务 | 是否依赖关联方、是否合理 |
| 资金拆借 | 是否计息、利率是否公允 |
| 资产转让 | 评估作价是否合理 |
| 担保 | 是否必要、是否有反担保 |
| 共同投资 | 是否产生同业竞争 |
| 关键管理人员薪酬 | 是否合理、程序是否合规 |

### 4.3 资金占用专项核查

| 占用形式 | 核查方法 |
|----------|----------|
| 直接借款 | 其他应收款-关联方明细账 |
| 代垫款项 | 费用报销凭证、银行流水 |
| 无商业理由的预付款 | 预付账款-关联方 |
| 非经营性资金往来 | 全部关联方银行流水核查 |

### 4.4 🔴 红旗指标

- 关联交易占收入/采购>30%且无法说明商业合理性
- 关联交易定价与非关联方同类交易差异>20%
- 存在非经营性资金占用（IPO红线）
- 关联交易未履行内部审批程序
- 通过无关联第三方间接进行资金往来
- 与实控人控制企业存在同业竞争

## 五、收入确认

### 5.1 收入确认政策核查

| 收入类型 | 确认时点 | 常见问题 |
|----------|----------|----------|
| 商品销售 | 控制权转移（签收/验收） | 提前确认（发出商品即确认） |
| 服务收入 | 履约进度/服务完成 | 完工百分比估计不准确 |
| 建造合同 | 投入法/产出法 | 预计总成本低估 |
| 软件授权 | 许可控制权转移 | 混淆使用权/所有权转移 |
| SaaS订阅 | 按期间均匀确认 | 将年订确认在首月 |

### 5.2 截止性测试

| 方向 | 测试目的 | 方法 |
|------|----------|------|
| 期末前 | 防止提前确认 | 资产负债表日前5天发货记录→检查签收日期 |
| 期末后 | 防止延后确认 | 资产负债表日后5天发货记录→检查是否应在上期确认 |
| 期后退货 | 验证收入真实性 | 资产负债表日后大额退货→判断是否属于资产负债表日前虚增 |

### 5.3 🔴 红旗指标

- 收入确认时点与同行业显著不同
- 收入增速远高于行业平均水平（>2倍）
- 毛利率远高于行业平均水平
- 客户集中度突然变化
- 大额收入发生在最后一月/季度
- 销售回款与收入确认严重脱节
- 存在大额期后退货
- 经销商/客户为当年新设/关联方

## 六、毛利率分析

### 6.1 分析维度

| 维度 | 分析内容 |
|------|----------|
| 分产品 | 各类产品毛利率变化趋势 |
| 分客户 | 不同客户毛利率差异（是否合理） |
| 分区域 | 不同区域毛利率差异 |
| 分季度 | 季节性波动是否合理 |
| 与同行对比 | 与可比公司毛利率对比及差异分析 |

### 6.2 🔴 红旗指标

- 毛利率大幅波动（同比±10%以上）且无合理解释
- 毛利率远高于同行业且无法解释
- 主要产品毛利率持续下降（趋势性恶化）
- 新增客户毛利率异常高（可能是关联方）
- 不同客户间同类产品毛利率差异>20%

## 七、负债结构

### 7.1 债务分析清单

| 分析项目 | 内容 |
|----------|------|
| 债务结构 | 短期/长期借款比例、银行/非银行比例 |
| 利率分析 | 加权平均利率、与市场利率比较 |
| 担保方式 | 信用/保证/抵押/质押、担保方 |
| 授信情况 | 总授信额度、已用额度、剩余额度 |
| 限制性条款 | 财务指标承诺、资产处置限制、控制权变更条款 |
| 偿债能力 | 流动比率、速动比率、利息保障倍数 |
| 还款计划 | 未来12/24个月到期债务金额 |

### 7.2 🔴 红旗指标

- 资产负债率>70%且持续上升
- 短期借款占比>80%
- 存在已触发/即将触发的贷款限制性条款
- 银行已冻结授信或压缩额度
- 非银行机构借款利率远高于银行

## 八、对外担保与或有负债

| 核查内容 | 方法 |
|----------|------|
| 对外担保清单 | 公司本部+子公司全部担保台账 |
| 担保对象 | 是否为关联方、经营状况如何 |
| 担保金额 | 累计对外担保是否超过净资产的50% |
| 反担保 | 是否取得反担保措施 |
| 诉讼担保 | 是否有因诉讼产生的担保 |
| 隐性担保 | 承诺函/安慰函/流动性支持函 |
| 票据 | 已贴现/背书未到期票据 |
| 未决诉讼 | 预期可能产生负债的诉讼 |

### 🔴 红旗指标

- 对外担保总额>净资产的50%
- 为经营困难/资不抵债的企业提供担保
- 存在"互保圈"现象
- 对外担保未经内部审批
- 为实控人个人债务提供担保
