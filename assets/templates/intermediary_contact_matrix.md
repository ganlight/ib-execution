## 九、沟通协议（Communication Protocol）

### 9.1 常规沟通规则

| 事项类型 | 谁联系谁 | 渠道 | 频率 | 抄送 |
|:---|:---|:---|:---|:---|
| **日常工作协调** | 保荐人 Associate ↔ 发行人IPO协调人 ↔ 各中介日常联络人 | 微信/邮件 | 按需 | 各机构执行负责人 |
| **项目周报** | 保荐人执行负责人 → 全体中介 + 发行人协调人 | 邮件 | 每周五 | 各机构主管 |
| **DD Issue追踪** | 保荐人执行负责人 → 全体中介 + 发行人 | 邮件+共享文档 | 每周更新 | — |
| **资料清单跟进** | 保荐人 Associate → 发行人对口部门 | 微信/邮件 | 按需 | 发行人IPO协调人 |
| **重大事项通报** | 保荐人执行负责人 ↔ 保荐人项目总负责人 → 发行人董秘/CFO | 电话+邮件 | 24h内 | 涉及方 |
| **监管沟通** | 保荐人 MD → 交易所/证监会 | 电话/书面 | 按需 | 发行人、律师 |
| **合同/费用变更** | 保荐人 MD → 发行人 CEO/CFO | 面对面/电话→书面确认 | 按需 | 法务 |

### 9.2 紧急事项升级路径

```
日常问题
  └─ 保荐人 Associate 联系发行人对口部门解决
       └─ 若24h未解决 → 保荐人执行负责人介入
            └─ 若涉及法律判断 → 同步通知发行人和保荐人律师
                 └─ 若构成上市障碍 → 保荐人 MD + 发行人 CEO + 各方主管合伙人联席会议
```

### 9.3 文档流转规则

| 文档类型 | 流转路径 | 保密级别 |
|:---|:---|:---|
| **尽调资料（含保密商业数据）** | 发行人 → 虚拟数据室（VDR），授权查阅 | 🔴 最高 |
| **招股书草稿** | 保荐人发起 → 全体中介审阅 → 发行人确认 → 定稿 | 🟡 高 |
| **审计报告/财务数据** | 审计师 → 保荐人+发行人（锁定后发送各方） | 🔴 最高 |
| **法律意见书** | 律师 → 保荐人+发行人 | 🔴 最高 |
| **项目周报/会议纪要** | 保荐人 → 全体中介+发行人 | 🟢 内部 |
