# 债券承销协议审核清单

> **模板功能**: 为债券项目组提供系统化的承销协议审核检查清单，覆盖从签约到交割的全流程。
> **使用方式**: 逐项审查勾选。遇到异常项在备注栏记录并升级到项目负责人。
> **适用品种**: 公司债券、企业债券、中期票据、短期融资券、定向工具（PPN）
> **维护**: ib-execution 技能体系 | **版本**: v1.0 | **最后更新**: 2026-05-09

## 二、包销 vs 代销——核心条款对比审核

| 条款维度 | 包销（余额包销） | 代销（尽力销售） |
|---------|----------------|-----------------|
| **定义** | 承销期届满，未售出债券由承销商按约定价格自行认购 | 承销期届满，未售出债券退还发行人，承销商无认购义务 |
| **承销商风险** | 余额认购风险（市场利率上行/投资者需求不足） | 无认购风险（仅声誉风险） |
| **典型费率** | 0.3%-1.5%/年（按发行规模） | 0.2%-0.5%/年 |
| **适用场景** | 新主体首次发债/低评级/大规模/市场环境较差 | 强主体/高评级/小规模/经常发债的老主体 |
| **包销触发条件** | 通常为"发行结果确定日/缴款截止日仍未获得足额认购" | 不适用 |
| **包销价格** | 包销认购价格=发行利率对应的面值/按约定折扣？——需明确 | 不适用 |
| **资金划付时限** | 包销部分缴款截至日（通常为T+N工作日） | 不适用 |
| **协议审查要点** | 包销触发→包销认购→包销后的持券及后续处置 | 代销到期后的未售出部分处理（退还/展期） |

### 包销协议特别审查

| # | 检查项 | 状态 | 备注 |
|---|--------|------|------|
| P1 | 包销是否触发承销商的内部信用额度/风险审批限制？需确认已获得内部包销审批 | ☐ | |
| P2 | 包销价格：①按面值认购还是打折？②是否包含应计利息？ | ☐ | |
| P3 | 包销后的持券：是否可以二级市场卖出？是否有锁定期？ | ☐ | |
| P4 | 包销金额上限是否在协议中明确规定（如"不超过XX亿元"） | ☐ | |
| P5 | 若为银团承销，各承销商的包销份额和连带/按份责任是否清晰 | ☐ | |

## 三、发行失败处理

| # | 检查项 | 状态 | 备注 |
|---|--------|------|------|
| D1 | **失败定义**：是否清晰列出触发发行失败的全部情形——①认购不足（未达到最低发行规模）；②簿记建档利率超出约定区间上限；③利率未达成一致；④监管批文未在有效期内获得；⑤市场环境发生重大不利变化（"不可抗力/重大不利变化条款"） | ☐ | |
| D2 | **费用承担**：发行失败后，已发生的各项费用（评级费/律师费/会计师费/前期尽调差旅费/印刷费/路演费等）如何分摊？——发行人承担全部？承销商自行承担？按比例？ | ☐ | 行业惯例：通常发行人承担第三方费用，承销商承担自身人力成本 |
| D3 | 发行失败是否影响承销商收取前期费用？是否有不可退还的前期费用？ | ☐ | |
| D4 | **投资者资金退还**：缴款后发行失败（如因监管原因），投资者已缴款项退还时限（通常为T+2个工作日）及利息处理 | ☐ | |
| D5 | 重新发行的最短间隔/失败后的排他期设定 | ☐ | |
| D6 | 承销商是否可以独断宣布"市场环境发生重大不利变化"并退出？——双方协商机制还是单方权利？ | ☐ | 警惕：单方退出条款对发行人不利 |

| 联席承销费 | 按份额比例分配 | 同上 | 各自承销份额 | | ☐ |
| 财务顾问费 | 固定50-200万元 | 签约/完成各50% | — | | ☐ |
| 受托管理费 | 10-30万元/年 | 按年支付 | — | | ☐ |
| 簿记建档服务费 | 可并入承销费或单列 | 发行完成 | — | | ☐ |
| 超额奖励（如有） | 超额部分的0.2%-0.5% | 发行完成 | 超额认购且利率低于上限的部分 | | ☐ |

| 律师费 | 20-50万 | 尽调开始 | — | | ☐ |
| 会计师费 | 15-40万 | 尽调开始 | — | 是否需出具会计意见书？ | ☐ |

| # | 检查项 | 状态 | 备注 |
|---|--------|------|------|

## 五、责任条款精细审核

### 5.1 发行人陈述与保证（Representations & Warranties）

| # | 发行人陈述 | 核查方式 | 承销商可依赖度 | 状态 |
|---|----------|---------|-------------|------|
| R1 | **信息披露**：募集说明书及其引用的第三方报告在所有重大方面真实、准确、完整 | 交叉比对审计报告/评级报告/法律意见书/发行人原始资料 | 部分依赖 | ☐ |
| R2 | **无重大不利变化**：自最近审计报告日/财务报表日以来，发行人财务、经营、资产无重大不利变化 | 审阅最近一期财务报表+与管理层访谈 | 需独立性判断 | ☐ |
| R3 | **合法存续**：依法设立、有效存续、具备发行主体资格 | 查验营业执照+国家企业信用信息公示系统 | 完全可依赖 | ☐ |
| R4 | **有权发行**：内部决策程序完备（股东会/董事会决议/国资委批复），外部监管批文已取得或在正常流程中 | 核验决议原件+监管批文 | 完全可依赖 | ☐ |
| R5 | **募集资金用途合规**：不用于负面清单、不新增地方政府隐性债务、不用于非生产性支出（如涉及房地产） | 资金用途合规性分析+律师意见 | 部分依赖 | ☐ |
| R6 | **无未披露的重大诉讼/仲裁** | 查询中国执行信息公开网/天眼查/全国法院被执行人信息+律师尽调 | 部分依赖 | ☐ |
| R7 | **资产权属**：主要经营性资产权属清晰，无重大权属争议 | 查验核心资产的权属证书 | 部分依赖 | ☐ |
| R8 | **无限制性条款冲突**：本次发行不会违反发行人的任何既有的重大合同/银行借款中的限制性条款 | 审阅重大合同+律师尽调 | 部分依赖 | ☐ |
| R9 | **反贿赂/反腐败/反洗钱合规** | 管理层访谈+合规制度审查 | 部分依赖 | ☐ |

### 5.2 承销商责任限制

| # | 检查项 | 状态 | 备注 |
|---|--------|------|------|
| L1 | **免责条款**：承销商依赖发行人提供的信息——若发行人提供的信息不实致承销商被索赔，发行人应无条件补偿（Indemnify）承销商 | ☐ | 这是承销商的核心保护条款 |
| L2 | **不可抗力**：自然灾害/战争/恐怖袭击/重大监管规则变化/金融系统故障等导致发行无法进行的退出条款——范围是否合理？ | ☐ | |
| L3 | **损失赔偿上限**：承销商的总赔偿责任是否限于"实际收到的承销费"？不可为"无上限" | ☐ | 行业标准：以承销费为上限 |
| L4 | **无"结果保证"**：承销协议不应包含承销商"保证发行成功""保证最低认购倍数""保证利率水平"等措辞 | ☐ | |
| L5 | **无"连带责任"**：如为多个承销商，是否将各方责任界定为"按份责任"而非"连带责任" | ☐ | |

### 5.3 违约条款

| # | 检查项 | 状态 | 备注 |
|---|--------|------|------|
| B1 | **发行人违约事件**：①未按时支付承销费/费用；②陈述与保证在重大方面不实；③偿债能力显著恶化（可量化的财务比率触发）；④其他交易文件的交叉违约 | ☐ | |
| B2 | **承销商违约事件**：①未按约定履行承销义务；②重大失职导致发行失败 | ☐ | |
| B3 | **违约赔偿**：直接损失 vs 间接损失（Consequential Damages/利润损失通常排除） | ☐ | |
| B4 | **违约补救期**：违约后有X个工作日的补救期（Cure Period） | ☐ | |
| B5 | **争议解决方式**：①仲裁 vs 诉讼；②仲裁机构（中国国际经济贸易仲裁委员会/北仲/上仲等）；③管辖法院（发行人所在地/承销商所在地/交易场所所在地） | ☐ | 行业惯例：发行人所在地法院 |
| B6 | **适用法律**：中国法律（涉及境外投资者时需特别注意是否会产生管辖争议） | ☐ | |

## 六、交割先决条件（Conditions Precedent, CP）

|---|---------|---------|--------|------|
| C1 | 监管注册/核准/备案文件已取得并在有效期内 | 原件扫描/复印件加盖公章 | 发行人和承销商 | ☐ |
| C2 | 募集说明书等发行文件已定稿并完成披露 | 定稿版募集说明书 | 承销商牵头发行人配合 | ☐ |
| C3 | 主体及债项评级报告已出具（非公开发行除外） | 评级报告定稿 | 评级公司 | ☐ |
| C4 | 法律意见书已出具且无保留意见 | 法律意见书定稿 | 发行人律师 | ☐ |

| C10 | 资金监管协议（如有）已签署 | | | ☐ |
| C11 | 投资者路演/簿记建档完成 | — | 承销商 | ☐ |
| C12 | 承销团协议已签署（如有银团） | | | ☐ |

### 6.2 特殊项目交割条件

| 项目类型 | 额外交割条件 |
|---------|------------|
| 绿色债券 | ①绿色认证报告（由第三方认证机构出具）；②绿色债券支持项目目录符合性说明 |
| 可续期债券 | ①计入权益/负债的会计处理意见（审计师出具）；②递延付息条款与利率跳升机制的合规性 |
| 项目收益债 | ①项目收益测算报告（更新至最新）；②差额补偿协议；③项目资产/收益权质押登记 |
| 资产支持证券 | ①基础资产转让交割确认；②资产服务机构报告（初始报告） |
| 担保债券 | ①担保人内部决议；②担保人最近一期财务报表+近一年度审计报告；③担保人对外担保统计 |
| 熊猫债（境外发行人在境内发债） | ①境外发行人主体资格公证+使馆认证；②中国律师出具的跨境法律适用意见 |
| 金融债 | ①银保监会/人民银行行政许可决定书（适用具体品种详查） |

### 7.2 监管批文有效期检查

| 品种 | 批文有效期 | 检查 |
|------|----------|------|
| 公开发行公司债券（注册制） | 注册之日起2年内发行完毕，首期发行在注册之日起1年内完成 | ☐ |
| 非公开发行公司债券 | 取得无异议函后12个月内完成发行（或按函件具体规定） | ☐ |
| 银行间债务融资工具（注册制） | 注册通知书有效期2年，首期应在注册后6个月内完成发行（按各期通知书） | ☐ |
| 金融债券 | 人民银行行政许可决定书明确的有效期 | ☐ |

> ⚠️ **批文到期风险**: 距批文到期日<3个月应启动加速发行流程；<1个月须设立专案推进。

### 7.3 簿记建档流程核对

| # | 步骤 | 检查要点 |
|---|------|---------|
| B1 | 簿记前投资者沟通（预销售） | 意向订单≥发行规模的1.0×，否则需评估是否推迟发行 |
| B2 | 簿记建档日（簿记区间公告已发） | 簿记管理人技术系统+人员就位，备有应急方案 |
| B3 | 利率区间调整 | 簿记中如需调整区间，须依据《簿记建档发行指引》规定的流程和幅度 |
| B4 | 边际利率确定+配售 | 配售规则已事前确定（时间优先/比例配售/荷兰式/美式） |
| B5 | 发行结果公告 | 公告在X小时内发出（交易所时效要求） |

## 八、协议合规额外审查

### 8.1 关联交易审查

- [ ] 发行人是否持有承销商的股份（"发行人=承销商的股东"）——须披露
- [ ] 承销商是否持有发行人的股份——如持有超1%须披露
- [ ] 本次发行是否构成关联交易？是否已取得适当的审批？
- [ ] 利益冲突声明条款是否包含在协议中？

### 8.2 保密与信息使用

- [ ] 保密条款保护范围（发行人的哪些信息属于保密信息）
- [ ] 保密义务的存续期限（发行后/协议终止后仍有效）
- [ ] 保密例外（法律要求披露、向监管披露、向必要的分包方披露等）
- [ ] 承销商是否可以在"墓碑广告"（Tombstone）中使用发行人LOGO和交易信息
- [ ] 若为私募债/PPN——是否明文禁止承销商向无关第三方披露"发行人+发行"的事实

### 8.3 反洗钱/制裁合规（境外投资者参与时尤为重要）

- [ ] 募集资金是否涉及制裁地区/受制裁方
- [ ] 发行人及其实际控制人是否在制裁名单上
- [ ] 协议中是否有合规声明条款（双方均声明遵循适用反洗钱/制裁法规）

### 8.4 协议终止条件

- [ ] 协议可在什么情况下终止（双方合意 / 一方违约 / 不可抗力 / 发行失败 / MAC事件 / 监管不予注册）
- [ ] 终止后的权利义务如何处理（费用结算/保密/已发生的责任/争议解决条款继续有效）
- [ ] 排他期（Exclusivity Period）条款：在协议有效期内+终止后X个月内（如6个月）发行人是否不能另聘其他承销商？

## 九、常见"陷阱条款"警示

| 陷阱条款 | 风险 | 修改建议 |
|---------|------|---------|
| "承销商保证本次债券成功发行" | 使承销商承担结果保证责任（过高风险） | 改为"承销商将尽商业上合理的努力（Commercially Reasonable Efforts）完成本次发行" |
| "承销商对所有文件内容的真实性负责" | 将发行人的披露责任转嫁给承销商 | 明确承销商仅对自己撰写的章节负责，不对发行人提供的信息的真实性承担责任 |
| 无上限的赔偿保障（Indemnity） | 承销商可能承担远超其承销费收入的赔偿责任 | 加上赔偿上限≤承销费（或加合理区间） |
| "'市场环境发生重大不利变化'由承销商单方认定" | 承销商可能在任何不利情形下借机退出 | 增加双方协商机制/客观触发标准（如利率变动>Xbp的量化指标） |
| 费率全部后端支付（发行完成才支付） | 承销商付出大量工作但发行失败时颗粒无收 | 争取签约时支付一定比例（如30%）作为前期费用，不退还 |
| "代销"协议中隐含包销义务措辞（如"确保""承诺"等模糊用语） | 法律上可能被认定为包销 | 严格审核措辞，删除任何暗示包销的字眼 |
| 违约条款仅约束发行人、无承销商的违约条款 | 权利义务失衡 | 增加承销商的违约情形与责任 |
| 打包费用中实际包含了中介费由承销商代付但未明确 | 税务/责任不清 | 逐项分拆，明确"发行人直接支付中介机构"还是"承销商代付，发行人再支付给承销商" |
