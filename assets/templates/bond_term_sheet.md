## 三、含权条款（Embedded Options）

### 3.1 发行人权利

| 条款 | 内容 | 适用条件 |
|------|------|---------|
| 赎回选择权（Call Option） | 【发行第X年末，发行人可以面值（+应计利息）提前赎回全部/部分债券】 | 需提前公告+经持有人会议同意（视条款） |
| 利率调整选择权（Rate Reset） | 【发行第X年末，发行人有权调整后续期限的票面利率，调整幅度不超过±Xbps】 | 通常配合回售条款 |
| 延期选择权（Extendible） | 【每X年末发行人有权选择延长X年】 | 永续债常见 |

### 3.2 投资者权利

| 条款 | 内容 | 适用条件 |
|------|------|---------|
| 回售选择权（Put Option） | 【发行第X年末，投资者有权按面值（+应计利息）将全部/部分债券回售给发行人】 | 配合利率调整条款 |
| 交叉违约（Cross Default） | 【发行人/合并范围内子公司任何债务违约金额累计超过【X万元】，即构成对本债券的交叉违约】 | 触发后可要求提前兑付 |
| 控制权变更回售（CoC Put） | 【发行人实际控制人发生变更，投资者有权要求发行人按面值+应计利息回购全部债券】 | 关键在于"控制权变更"定义范围 |
| 加速清偿（Acceleration） | 【触发违约事件后，经持有人会议决议，可宣布本债券立即到期】 | 触发条件需明确 |

### 资金用途合规声明

- [ ] 募集资金不用于弥补亏损和非生产性支出
- [ ] 募集资金不用于房地产项目开发（除白名单企业外）
- [ ] 募集资金不用于金融投资（除金融类企业外）
- [ ] 募集资金不用于新增地方政府隐性债务（城投企业必选）
- [ ] 用于借新还旧的，已确认被偿还债券不存在违约或展期
- [ ] 用于项目建设的，已取得必要的政府批复文件

### 负面承诺（Negative Pledge）

| 承诺事项 | 具体限制 |
|---------|---------|
| 资产抵押限制 | 【未经持有人会议同意，不得将主要资产进行抵押/质押，除非同时为本债券提供同等担保】 |
| 次级债务限制 | 【不得发行清偿顺序优先或等同于本债券的次级债务，除非满足特定财务比率】 |
| 关联交易限制 | 【与关联方的交易应遵循市场化定价，超过XX万元需独立审批】 |
| 合并/分立/重组 | 【重大资产重组前需通知持有人，涉及控制权变更的需提供CoC回售选择权】 |

## 七、风险因素提示

### 与本期债券相关的风险

| 风险类别 | 风险描述 |
|---------|---------|
| 利率风险 | 【受宏观经济和货币政策影响，市场利率可能波动，债券价格存在波动风险】 |
| 流动性风险 | 【债券上市后可能因市场环境或投资者结构原因，无法以公允价格及时变现】 |
| 偿付风险 | 【若发行人经营/财务状况恶化，可能存在偿付能力不足的风险】 |
| 信用增进风险 | 【担保方/增信方自身信用恶化可能导致增信效力下降】 |
| 回售/赎回风险 | 【含权条款可能对投资者的持有期限和收益产生不确定性】 |

### 与发行人相关的风险

| 风险类别 | 风险描述 |
|---------|---------|

| 经营风险 | 【业务模式、核心资源、供应链、客户集中度等】 |
| 财务风险 | 【杠杆水平、流动性、资产质量、盈利能力波动等】 |
| 管理风险 | 【实控人风险、公司治理、重大诉讼、合规等】 |
| 政策风险 | 【产业政策/环保/土地/税收政策变化等】 |
