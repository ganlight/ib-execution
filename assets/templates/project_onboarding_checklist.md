# Project Onboarding Checklist / 新项目启动清单

> **使用说明**：本清单用于新IPO/并购/债券项目正式启动前的准备工作。共60项核查，分为6大类。由项目执行负责人逐项勾选，完成后存档。BD端交接材料须全部接收完毕方可进入执行阶段。
