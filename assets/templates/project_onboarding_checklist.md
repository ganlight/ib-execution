# Project Onboarding Checklist / 新项目启动清单

> **使用说明**：本清单用于新IPO/并购项目正式启动前的准备工作。由项目执行负责人逐项勾选，完成后存档。BD端交接材料须全部接收完毕方可进入执行阶段。

---

## 一、BD → 执行团队交接（Mandatory Handover）

> ⚠️ **交接要求**：BD团队须在项目启动会召开前，将以下材料打包移交执行团队，执行负责人签字确认接收。

### 1.1 签约相关

- [ ] 已签署的**委任书/聘书**（Engagement Letter）全套扫描件及原件归档位置
- [ ] 费率表/报价方案（须与委任书条款一致）
- [ ] 客户签署的保密协议（NDA）扫描件
- [ ] 客户背景调查结果（如是否涉及制裁、负面舆情等）

### 1.2 客户沟通记录

- [ ] BD阶段与客户沟通的**邮件往来关键摘要**（含客户的核心关注点、Pitch阶段承诺的特殊条款）
- [ ] Pitch/建议书终稿电子版
- [ ] 客户公司组织架构简图（含实际控制人链条）
- [ ] 客户关键决策人联系信息（董事长/CEO/CFO/董秘）
- [ ] 客户选聘投行的决策考虑因素（陪标/一对一/比选中标条件）

### 1.3 BD口头Brief

- [ ] 项目核心卖点/投资故事（3句话）
- [ ] 已知的硬伤/风险点（"地雷清单"）
- [ ] 客户风格与沟通偏好（频率/形式/汇报对象层级）
- [ ] 客户最看重的"一票否决"事项（如估值底线、上市地点偏好、时间要求）

### 1.4 执行准备

- [ ] 初步项目时间表（BD版，供执行团队细化）
- [ ] 已知的市场窗口/监管窗口预判
- [ ] 项目代号确认（如适用）

## 二、客户资料收集清单（Client Document Collection）

### 2.1 主体资格文件

- [ ] 营业执照（正副本）及工商登记信息截图
- [ ] 公司章程（最新版及历次修订版）

- [ ] 外商投资企业批准证书（如适用）
- [ ] 行业经营许可证/资质证书清单及复印件

### 2.2 股权与治理

- [ ] 股东名册（完整版，含持股比例、出资金额、出资方式、出资时间）

- [ ] 实际控制人认定材料（亲属关系证明、一致行动协议、控制链条图）
- [ ] 董事会/监事会成员名单及简历

### 2.3 财务资料（3年一期）

- [ ] FY【20XX】-FY【20XX】经审计财务报表（合并+母公司）
- [ ] 最新一期未经审计管理报表
- [ ] 审计报告正文及附注
- [ ] 主要会计政策文件
- [ ] 前五大客户/供应商名单及交易金额（按年度）
- [ ] 关联交易清单（按年度）
- [ ] 银行授信合同、贷款合同、担保合同

### 2.4 业务资料

- [ ] 主要产品/服务介绍材料

- [ ] 客户/供应商集中度分析
- [ ] 员工花名册（按部门、按职级）及薪酬结构
- [ ] 社保公积金缴纳记录及统计表

### 2.5 知识产权

- [ ] 专利清单（含申请号、授权号、有效期、权利归属）

- [ ] 域名清单
- [ ] 核心技术来源说明（自主研发/委托开发/受让/许可）
- [ ] 核心技术人员行业履历及竞业限制情况

### 2.6 法律合规

- [ ] 重要资产清单（土地、房产、重大设备）及权属证书

- [ ] 涉诉/仲裁/行政处罚清单及文件

## 三、监管合规预检（Regulatory Pre-Check）

- [ ] 行业准入要求核查——发行人主营业务是否属于外商投资负面清单限制/禁止类
- [ ] 实际控制人国籍/境外居留权审查——是否触发外资准入限制
- [ ] 发行人/实控人/董监高**涉诉/失信/行政处罚**检索（信用中国、中国执行信息公开网、裁判文书网、证监会/交易所网站）
- [ ] VIE架构合规性初步判断（如涉及）
- [ ] 数据安全/个人信息保护——是否涉及《数据安全法》《个人信息保护法》，是否需网络安全审查
- [ ] 反垄断审查——是否触发经营者集中申报义务
- [ ] 行业政策风向——近6个月监管政策是否对行业有重大不利影响
- [ ] 上市板块定位匹配——发行人是否符合拟上市板块的行业定位（如科创板"硬科技"、"专精特新"等定位）
