# M&A Restructuring Standards — Complete Reference

> **Purpose:** Reference for M&A deal structuring, regulatory review, and backdoor listing analysis.
> **Last Updated:** 2026-05

---

## 1. Three-Metric Test (重大资产重组判定)

Whether a transaction constitutes a "material asset restructuring" (重大资产重组) is determined by comparing the target's ratios to the acquirer's most recent audited financial year.

### The Three Metrics

```
Metric 1: Target Total Assets / Acquirer Total Assets ≥ 50%
Metric 2: Target Sales Revenue / Acquirer Sales Revenue ≥ 50%
Metric 3: Target Net Assets / Acquirer Net Assets ≥ 50%
(or: Transaction Consideration / Acquirer Net Assets ≥ 50%)
```

**Any ONE metric ≥ 50% → Material Asset Restructuring → subject to review.**

### Calculation Frequency Rule

When multiple transactions over 12 months in the same or related industry involve the same or related assets, their metrics are CUMULATED.

### Metric Waivers

| Waiver | Condition |
|---|---|
| Operational waiver | If target PB ratio ≤ 1.0, net asset metric can use "net assets × PB" — may reduce ratio below 50% |
| Industry integration waiver | Same-industry horizontal/vertical integration may apply for simplified procedure |

## 2. Backdoor Listing (借壳上市) — Triggers

### Trigger Conditions (ALL must be met simultaneously)
1. **Any metric ≥ 100%** (total assets / revenue / net assets / shares issued) — compared to acquirer's pre-transaction figures
2. **Control change** (actual controller of the listed company changes)
3. **Control change within 60 months** of the asset injection

### Detailed Triggers

| Trigger Type | Metric Calculation |
|---|---|
| Total assets | Target total assets ≥ 100% of acquirer total assets |
| Revenue | Target revenue ≥ 100% of acquirer revenue |
| Net assets | Target net assets / consideration ≥ 100% of acquirer net assets |
| Shares issued | New shares issued to acquire target ≥ 100% of pre-transaction shares |
| Primary business | Primary business fundamentally changes |
| **Time window** | Any of the above + control change within 60 months |

### Backdoor Listing Consequences
- Review standard = equivalent to IPO (首发上市条件)
- IPO profit requirements apply
- IPO board conditions apply
- **No simplified procedure available**

## 3. Review Process by Deal Type

### Cash-Only Acquisition (现金收购)
| Aspect | Detail |
|---|---|
| Review body | Exchange (交易所) |
| CSRC involvement | No (registration-based) |
| Timeline | 3–6 months |
| Shareholder meeting | Required (if material) |
| No new share issuance | Faster review |

**Condition:** Cash acquisition of non-material assets may only require board approval and disclosure — no regulatory review.

### Share-Issuance Acquisition (发行股份购买资产)
| Aspect | Detail |
|---|---|
| Review body | M&A Committee of Exchange (交易所并购重组委) |
| CSRC involvement | Registration review |
| Timeline | 6–8 months |
| Key documents | Restructuring report, legal opinion, audit, appraisal |
| Shareholder meeting | ≥ 2/3 vote required |

### Backdoor Listing (借壳上市)
| Aspect | Detail |
|---|---|
| Review body | M&A Committee |
| Review standard | **Equivalent to IPO** |
| CSRC involvement | Full review |
| Timeline | 12–18 months |
| Profit requirement | IPO-standard profit history |

### Simplified Procedure (小额快速)
| Condition | Detail |
|---|---|
| Transaction value | ≤ RMB 50 million OR ≤ 5% of total assets |
| No share issuance | Cash payment only |
| No control change | Same actual controller |
| Review | Board-level only |

## 4. Industry Restrictions on Backdoor Listings

### Prohibited Sectors (Cannot Backdoor Into / Be Backdoored Out Of)
- Real estate development
- "Two-highs and one-surplus" (高耗能、高排放、产能过剩)
- Financial services (banking, securities, insurance — special CSRC license required)

### Discretionary Restrictions (取决于窗口指导)
- Cultural entertainment
- Gaming
- Education
- VC/PE investment platforms

## 5. Post-M&A Lockup Rules

| Shareholder | Lockup | Trigger |
|---|---|---|
| Controlling shareholder (original) | 36 months | If control changes |
| Asset seller (controlling shareholder of target) | 36 months | If actual controller of acquirer changes |
| Asset seller (non-controlling) | 12 months | By default |
| Asset seller | 24 months | If lockup agreement specifies longer |

## 10. Quick-Reference M&A Decision Tree

```
Is any 3-metric ≥50%?
  ├── NO → No mandatory review (board approval + disclosure)
  └── YES → Material restructuring
        Is any metric ≥100% + control change within 60 months?
          ├── YES → Backdoor: equivalent to IPO review
          └── NO → M&A committee review
                Cash only?
                  ├── YES + ≤5% of assets → Simplified (board only)
                  ├── YES → Exchange review
                  └── NO (share issuance) → M&A committee review
```
