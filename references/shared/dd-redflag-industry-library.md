# 各行业尽职调查红旗库

> **定位**：投行实操参考手册，按行业分类整理尽调中见过的真实红旗模式。
> **适用场景**：IPO审计、重大资产重组、公开发行债券、尽职调查全流程
> **使用方式**：项目启动时通读行业章节，进场后对照核查清单逐项排查

---

## 一、半导体/集成电路

### 1.1 行业特点
- **资本密集 + 技术迭代快 + 国产替代政策加持**：行业估值溢价高，但周期波动大
- **轻资产设计公司 vs 重资产 IDM/Foundry 模式**：核查逻辑截然不同，不可混为一谈
- **出口管制（BIS/实体清单）是最大政策风险**：直接影响供应链安全和业绩可持续性

### 1.2 核心技术指标

| 指标 | 科创板标准 | 核查重点 |
|------|-----------|---------|
| 先进制程 | 14nm 以下 = 行业头部 | 是否真正进入量产，还是仅流片验证 |
| 产能利用率 | 80%–95% 健康区间 | 是否刻意报高？与同产线其他客户数据交叉验证 |
| 良率 | >90% 行业平均 | 良率定义是否统一（芯片级别/晶圆级别/封装级别差异大） |
| 研发投入比 | >15% 头部设计公司 | 资本化比例是否异常高于可比公司 |
| 客户验证周期 | 6–18 个月 | 是否存在收入确认时点过早（样品阶段即确认） |
| 专利数量 | 科创板门槛参考 | 专利结构（自主申请 vs 购买 vs 受让）与业务实质是否匹配 |

### 1.3 红旗模式库

| 模式 | 典型特征 | 核查方法 |
|------|---------|---------|
| **虚假订单填充** | 签署无约束力 MOU/意向书即计入收入预测，实际无 PO | 核查正式 Purchase Order + Invoice + 出库记录 + 客户验收单，四者缺一不可 |
| **代理商压货** | 代理商年末集中提货冲量，Q1 退货或信用调整 | 月度发货记录（逐月比对）+ 代理商库存盘点 + 期后退货率分析 |
| **技术夸大** | 声称"已完成 5nm 验证"实际仅完成 MPW（多项目晶圆）流片 | 要求提供晶圆代工厂产能证明 + 量产批次记录，流片≠量产 |
| **政府购买依赖** | 收入 >50% 来自政府/国企，价格不遵循市场化定价 | 穿透至最终用户，核查实际使用场景，判断是否属于政府采购/补贴变体 |
| **研发资本化过度** | 将应费用化的研发支出资本化（人员工资/材料虚增） | 对比资本化时点（进入开发阶段节点）+ 可比公司资本化率区间 |
| **技术依赖造假** | 声称"自主可控"但底层 IP 依赖 ARM/Synopsys 授权 | 获取完整 IP 授权清单 + 授权协议到期时间 + 续期确定性评估 |
| **产能数据造假** | 产能利用率填报数据与实际用电量/用水量/设备运行时间不符 | 调取园区水电账单 + 设备运行日志（部分 Foundry 可获取） |
| **核心人员挂名** | 核心技术团队含实控人亲属或已离职人员 | 社保缴纳记录 + 个税申报记录 + 竞业协议核查 |

### 1.4 核查要点清单

- [ ] **EDA 工具来源**：使用 Synopsys/Cadence/Mentor 是否已取得正式授权，是否在有效期内，出口管制合规条款是否已满足（BIS 新规重点关注）
- [ ] **晶圆代工厂**：主要代工厂（SMIC/TSMC/三星/格芯）产能证明 + 过去 12 个月价格趋势，代工厂若在境外需关注出口管制传导风险
- [ ] **流片结果**（设计公司）：提供 Tape-out 报告 + 量产认证记录，区分工程样片与量产片
- [ ] **封装测试厂**：委外封装是否存在单一代工厂依赖，封装环节是否涉及关键设备/材料进口
- [ ] **出口管制清单**：逐一核查 EAR/ BIS 实体清单、UVL（未经核实名单）、军事最终用户名单，触线则业务存续性存疑
- [ ] **核心技术人员**：核查是否在美实体清单、是否签署竞业禁止、是否存在潜在竞业竞争
- [ ] **专利布局**：是否在中美欧韩日等目标市场均有有效专利，专利剩余有效期，侵权风险排查（FTO）
- [ ] **政府补助性质**：与资产相关 vs 与收益相关的政府补助划分是否正确，补贴是否具备持续性

## 二、软件 / SaaS / 信息技术

### 2.1 行业特点
- **订阅制 ARR/MRR 是估值核心**：ARR（年度经常性收入）= 金标准，非订阅收入占比过高需警惕
- **人力成本高 + 研发资本化空间大**：人员工资虚增是研发资本化造假的主要手段
- **底层技术依赖**：OS/数据库/中间件依赖是政策风险点（尤其信创替代浪潮下）
- **信创/国产化替代**：政策红利期，但也成为部分企业包装"自主可控"的温床

### 2.2 核心指标

| 指标 | 健康标准 | 警示线 | 备注 |
|------|---------|-------|------|
| NRR（净收入留存率） | >110% | <100% | NRR < 100% 意味着存量客户流失 > 新增 |
| CAC 回收期 | <12 个月 | >24 个月 | 反映销售效率 |
| LTV/CAC | >3x | <1x | 客户生命周期价值 / 获客成本 |
| ARR 占收比 | >70% | <50% | 非订阅收入占比过高则估值逻辑不成立 |
| Churn（年流失率） | <10% | >20% | 流失率 >20% 说明产品粘性存疑 |
| Gross Margin | >70% | <60% | 订阅型业务毛利率通常较高，过低说明成本异常 |

### 2.3 红旗模式库

| 模式 | 典型特征 | 核查方法 |
|------|---------|---------|
| **合同拆单** | 将大合同拆成多个小于披露门槛的小合同，规避审议程序或信息披露 | 核查合同时间关联性（同一批人同一天签署多份）+ 金额分布规律 |
| **渠道填充（SaaS）** | 经销商/代理商年末集中采购，期后退货，实际为库存转移 | 经销商库存盘点 + 期后退货率分析（重点关注 Q1 收入是否大幅下滑） |
| **一次性充注** | 将一次性的系统集成/实施/定制开发收入包装为"订阅"收入 | 收入分类口径是否一致，历史数据是否可比；合同中是否存在一次性交付条款 |
| **开源组件商业化** | 将开源代码（尤其 GPL 许可）包装为"自主可控"产品，隐藏合规风险 | 代码溯源审计（Git history）+ License 扫描，GPL 传染性问题可能导致产品无法商业化 |
| **收入跨期** | 将应属于当期的收入延后至下期确认（Q4 故意少收） | 月度收入分布曲线 + 季末最后两周发货/服务交付记录，核查是否有异常平滑 |
| **研发资本化** | 将应费用化的研发支出资本化（人员工资虚增、工时记录造假） | 研发人员名册 + 工时填报系统记录 + 资本化时点证据（评审/立项文件） |
| **IP 侵权** | 核心产品存在未披露的知识产权纠纷，上市后暴雷 | 专利自由实施分析（FTO）+ 版权登记检索 + 涉诉情况核查 |
| **收入季节性造假** | 季度末最后一天确认大额收入，实际未完成交付 | 核查交付证明文件日期 + 银行回款时间匹配性 |
| **关联交易虚构** | 通过关联方虚构技术服务收入或成本 | 穿透最终受益所有人 + 银行流水逐笔核对 + 合同文本异常条款 |

### 2.4 核查要点清单

- [ ] **ARR/MRR 计算口径**：是否仅包含有正式合同约束的确定性订阅收入，不含无约束力意向或框架协议
- [ ] **合同负债余额**：余额增长快 = 新签多，余额持续下降 = 流失加速，结合来看判断业务健康度
- [ ] **前 10 大客户**：客户集中度（CR10 >50% 为高风险）+ 近 3 年客户变动原因
- [ ] **开源 License 审计**：使用 GPL 组件需关注传染性，采购开源组件需确认商业许可
- [ ] **信创适配证明**：与国产 OS（麒麟/统信）、国产数据库（达梦/人大金仓）、国产芯片（鲲鹏/飞腾）的兼容性认证
- [ ] **数据安全 / 等保认证**：等保 2.0 证书 + 数据跨境合规 + 个人信息保护合规（个保法）
- [ ] **员工股权激励**：股份支付费用是否足额计提，激励对象是否与服务期挂钩

## 三、医药 / 生物科技

### 3.1 行业特点
- **研发周期极长（10 年+）**：成功率本质上不可预测，管线价值评估主观性极强
- **政策高度敏感**：集采、医保谈判、优先审评审批、DRG/DIP 支付改革均直接影响商业化价值
- **商业贿赂是重大法律风险**：学术推广费是商业贿赂的经典变体，核查难度大
- **销售费用率是核心关注**：异常高企的销售费用率往往是财务操纵的信号

### 3.2 核心指标（按研发阶段）

| 阶段 | 关键指标 | 核查重点 |
|------|---------|---------|
| 临床前 | 核心化合物专利布局 | 专利族规模、剩余有效期、目标市场覆盖；是否为引进专利（License-in） |
| 临床 I 期 | 安全性数据（耐受性） | 受试者入组速度 + 不良事件（SAE）报告质量 |
| 临床 II 期 | 有效性数据（概念验证） | 对照组设计 + 统计假设是否合理 + 样本量 |
| 临床 III 期 | 注册临床（确证性） | 试验质量（GCP 核查）+ 数据完整性（溯源）+ 统计分析计划 |
| 商业化 | 市占率 + 进院价格 | 医保谈判价格 + 集中采购中标省份 + 医院覆盖数量 |

### 3.3 红旗模式库

| 模式 | 典型特征 | 核查方法 |
|------|---------|---------|
| **核心管线造假** | 将临床前阶段描述为"即将进入临床"，管线进度严重注水 | 核查 CDE（药品审评中心）临床试验登记 + PI（主要研究者）确认 + 伦理批件时间线 |
| **销售费用虚增** | 学术推广费/市场开发费→实质为商业贿赂，回扣通过 CSO 套现 | 费用结构横向对比（销售费用率 vs 可比公司）+ CSO 穿透 + 资金最终流向 |
| **CSO 供应商/客户重叠** | CSO（销售外包）公司由实控人近亲属或关联方设立，虚构销售服务 | 穿透 CSO 股东背景 + 实控人银行流水 + 服务成果交付文件 |
| **经销商压货** | 经销商年末大量进货，Q1 退货，实际销售给最终终端存疑 | 经销商库存盘点 + 期后退货率 + 最终医疗机构销售数据（部分省份有数据） |
| **集采影响低估** | 招股书/重组报告低估集采对存量市场的冲击，选择性披露 | 已中标省份实际价格 + 量的变化（医药魔方/药智数据可查）+ 竞争格局变化 |
| **生产场地合规** | GMP 证书与实际生产场地不符，批次记录造假 | 药监局官网 GMP 证书查询 + 飞行检查记录 + 现场走访生产车间 |
| **专利挑战** | 核心专利被仿制药或创新药发起专利挑战/无效宣告 | 专利复审委审查记录 + 专利到期时间 + 仿制药上市时间预测 |
| **临床数据操纵** | 临床试验数据存在逻辑不一致（入组时间/不良事件/合并用药） | 原始病历溯源（Source Data Verification）+ 统计分析异常值 |
| **两票制套利** | 两票制下通过高开套取利润，实际控制人个人账户收取 | 核查银行流水 + 发票与出货记录匹配性 + CSO 合同实质 |

### 3.4 核查要点清单

- [ ] **临床试验方案**：是否与 CDE 达成 EOP2（II 期临床结束）沟通意见，方案变更是否重新报批
- [ ] **试验数据完整性**：CRO 数据管理体系是否通过监管核查，原始病历溯源比例是否合规（通常 100%）
- [ ] **商业贿赂历史**：检索裁判文书网 + 药监局行政处罚记录 + 税务稽查报告
- [ ] **供应商重合**：原料药供应商是否与实控人存在股权/任职关联
- [ ] **医保目录 + 中标价格**：已集采品种的中标省份、中标价格变化趋势，市场份额真实变化
- [ ] **GMP 合规**：最近一次药监局 GMP 飞行检查结果，是否存在严重缺陷（Critical/Major）
- [ ] **环保合规**：API（原料药）生产需关注排污许可证 + 危险废物处理记录 + 环评批复
- [ ] **药品注册批件**：药品批准文号 vs 在研管线的区分，同一通用名不同剂型的注册批件归属

## 四、制造业（通用）

### 4.1 行业特点
- **重资产 + 产能周期**：固定资产折旧对利润影响大，产能利用率是核心变量
- **原材料价格敏感**：大宗商品价格波动直接传导至毛利率，套保策略是否有效
- **化工/钢铁/建材与经济周期强相关**：宏观需求下行时，财务造假动机最强（粉饰报表）
- **环保/安全生产**：重大合规风险，停产整改可能导致持续经营假设不成立

### 4.2 红旗模式库

| 模式 | 典型特征 | 核查方法 |
|------|---------|---------|
| **收入跨期** | 期末大单提前确认收入（次年退货或不开票） | 月度收入分布 + 季末前后各 10 天发货记录 + 开票时间与确认时间对比 |
| **关联方虚构** | 构造假的供应商/客户走账，虚增收入或虚减成本 | 银行流水逐笔核查 + 穿透最终受益所有人（UBO）+ 工商查询关联方链 |
| **存货虚构** | 账面存货实际不存在（仓库存货盘点为 0 或大幅缩水） | 实地全面盘点 + 复盘记录（注意是否由被审计单位人员陪同） + 存货周转天数异常 |
| **成本少计** | 将应计入营业成本的支出挂在在建工程/预付款项，偷逃成本 | 成本明细账与在建工程明细交叉比对 + 银行流水对应预付款项的最终用途 |
| **政府补贴虚报** | 将与经营无关的政府补贴计为经常性损益，夸大盈利能力 | 补贴文件逐项核查（补贴名目/依据文件）+ 资金到账凭证 + 是否与资产相关 |
| **应收坏账少提** | 宽限大客户账龄不计提坏账，虚增利润 | 应收账龄分析 + 客户经营状况恶化信号（被执行信息/失信/欠薪） |
| **固定资产虚增** | 在建工程转固推迟 / 虚增工程量 / 设备采购价格虚高 | 在建工程进度文件（监理报告）+ 设备采购合同/发票/付款记录三单匹配 |
| **税务造假联动** | 虚开增值税发票配合收入造假，税务系统与账面数据不一致 | 纳税申报表与账面数据差异分析 + 税局系统导出数据比对 |

### 4.3 监盘程序要点

| 资产类型 | 盘点方式 | 抽查比例 | 特别注意事项 |
|----------|---------|---------|-------------|
| 原材料 | 实地盘点 + 标签核验 + 账实核对 | ≥20%，高价值材料提高至 ≥50% | 注意在途材料（单到货未到 vs 货到单未到） |
| 在产品 | 工艺路线 + 在制数量推导 + 现场抽盘 | ≥15% | 注意工序进度与账面在产品成本是否匹配 |
| 产成品 | 实地盘点 + 入库单/出库单匹配 | ≥30% | 关注寄存于第三方的产成品（仓库/客户处） |
| 在建工程 | 现场查看 + 进度文件 + 发票付款核对 | 重大工程 100% | 警惕长期挂账在建工程（延迟转固） |
| 固定资产（异地） | 委托当地审计师盘点 + 实物拍照确认 | ≥10%，高价值设备提高比例 | 设备铭牌/序列号与账面记录一致性核对 |

## 五、城投 / 基础设施

### 5.1 行业特点
- **收入依赖地方政府**：可支配财力弱，财政纪律约束不足
- **土地财政依赖**：土地出让金返还作为重要偿债来源，土地市场下行时风险暴露
- **隐性债务是核心政策红线**：43 号文后城投举债规范逐步建立，但隐性债务化解仍在路上
- **信用分析逻辑与产业债完全不同**：城投看政府支持意愿和能力，而非自身现金流

### 5.2 城投特殊红旗

| 红旗类型 | 具体表现 | 判断依据 |
|----------|---------|---------|
| **隐性债务新增** | 以企业债务形式承载政府支出责任（BT 回购/政府购买服务纳入预算） | 审计署审计报告 + 银保监会监测数据 + 访谈地方财政 |
| **政府兜底承诺** | 安慰函/承诺函/流动性支持函等模糊增信文件 | 合同文本逐条审查，注意"不承担连带责任"等免责条款 |
| **公益性资产注入** | 将无收益的土地/管网/道路注入平台，资产虚胖 | 资产收益性分析：投资性现金流 / 总资产 < 5% 为高风险 |
| **收入结构失真** | 土地整理/代建收入占比 >60%，实质为政府转移支付 | 收入分类+实质判断，政府回款周期是否与土地出让节奏挂钩 |
| **空壳平台** | 无实际业务运营，仅负责融资（"融资平台"本质） | 业务实质分析：人员规模 + 业务条线 + 实际经营决策权归属 |
| **土地财政依赖** | 土地出让金返还作为主要偿债来源，依赖卖地收入偿还债务 | 土地市场分析：当地土地成交量价走势 + 平台公司土地抵押率 |
| **流动性危机信号** | 非标融资占比过高（信托/租赁/定融）+ 到期债务集中 | 融资结构分析 + 一年内到期债务 / 货币资金 > 3 倍为警戒 |
| **担保互保链** | 对其他平台公司提供大量担保，风险传导 | 对外担保明细 + 被担保方财务状况 + 代偿风险评估 |

### 5.3 城投尽调核查清单

- [ ] **名单分类**：是否在银保监会融资平台名单（"名单内"vs"名单外"），名单内平台银行贷款受管控
- [ ] **隐性债务化解**：是否纳入隐债系统，化解方案是否已制定并执行（6 种化解方式：财政资金/资产盘活/项目收益/破产清算/债务重组/展期降息）
- [ ] **政府注资合规性**：近 3 年政府注资是否合规（不得以土地注入新增隐债，财综[2018]号文）
- [ ] **公益性资产占比**：公益性资产占总资产比例 >50% 为高风险，收益性资产不足
- [ ] **土地资产**：是否已缴纳土地出让金（划拨土地不能抵押）+ 是否可处置变现
- [ ] **对外担保**：担保对象是否包含其他平台公司，互保链风险
- [ ] **融资结构**：银行贷款 : 债券 : 非标比例，高成本非标占比过高是流动性预警信号
- [ ] **现金流覆盖**：利息保障倍数 <1 倍意味着付息依赖借新还旧
- [ ] **主体信用评级**：外部评级 vs 隐含评级差异，评级公司调级记录

## 六、房地产

### 6.1 行业特点
- **政策强周期**：宏观调控（限购/限贷/限价）+ 融资三条红线 + 预售资金监管
- **收入确认特殊**：交付为确认收入前提，期房制度下交付风险影响财务真实性
- **土地增值税**：特殊税种，土地增值税清算影响利润显著
- **关联往来复杂**：集团内关联方往来频繁，隐性担保多，资金链断裂风险高

### 6.2 红旗模式库

| 模式 | 典型特征 | 核查方法 |
|------|---------|---------|
| **收入确认提前** | 将未交付房产确认为收入，实际未达交付标准 | 对照合同约定交付条件 vs 竣工备案 vs 交房通知书三张表，交付确认早于竣工备案为异常 |
| **土地款虚增** | 土地增值税/增值税差额操控，虚增土地成本减少税负 | 土地成本明细 + 土地增值税清算报告 + 税局口径，核查土地出让金发票真实性 |
| **关联方应收** | 与实控人控制公司的大额关联方往来，实质为资金占用 | 穿透资金流向 + 外部债务（明股实债/土地抵押）+ 中登房产抵押登记查询 |
| **担保链风险** | 上市公司对集团内公司的大量担保，对外担保总额超净资产 | 担保明细 + 被担保方财务状况 + 代偿风险敞口 |
| **明股实债** | 表面为股权投资（入资产负债表），实为债权融资（固定收益+回购） | 抽屉协议 + 回购条款 + 投资方账务处理（权益 vs 金融工具分类） |
| **虚假销售** | 员工购房/关联方购房冲量，实际无真实交付意图 | 抽查合同 + 付款记录 + 交房记录 + 更名记录，员工购房尤其关注是否限定转售 |
| **土地增值税少计** | 通过虚增土地成本或混淆普通/非普通住宅标准少预征 | 清算报告与账面预提对比，关注地方税局清算口径差异 |
| **存货高估** | 已售未交付房源成本未结转，或土地成本分摊不合理 | 已售未交付面积与成本结转匹配性，在建项目成本归集合理性 |
| **预售资金挪用** | 预售资金未进入监管账户，或监管账户资金被违规提取 | 核查监管账户银行流水 + 住建部门公示的入账金额 |

### 6.3 核查要点清单

- [ ] **五证核查**：土地证/用地规划/工程规划/施工证/预售证，证照是否齐全有效
- [ ] **三线指标**：剔除预收款资产负债率 / 净负债率 / 现金短债比，红线数量直接影响融资能力
- [ ] **预售资金监管**：监管账户余额 + 提取条件 + 是否存在超额提取
- [ ] **土地储备**：储备规模（可售货值）+ 土地增值税清算义务触发条件
- [ ] **合作开发**：并表范围（是否通过合作方调节杠杆）+ 少数股东权益占比
- [ ] **受限资产**：土地/房产抵押情况，在手货币资金是否受限
- [ ] **诉讼仲裁**：涉及项目转让/交付纠纷/施工合同纠纷的未决诉讼

## 附：通用尽调工作方法论

### A. 财务造假三角理论
财务造假通常需要三个条件同时满足：**动机（Pressure）+ 机会（Opportunity）+ 合理化（Rationalization）**
- **动机识别**：业绩对赌/股权激励/IPO 对赌条款触发 → 造假动机最强
- **机会识别**：内部控制薄弱 + 管理层凌驾 + 外部审计独立性受限
- **合理化识别**：利用行业特殊性构建"合理"解释（技术门槛高、信息不对称）

### B. 流水核查优先级
1. **法人银行流水**（基本户 + 一般户全量导出）
2. **实控人个人银行流水**（所有账户，近 3–5 年）
3. **支付宝/微信支付流水**（实控人及核心管理层）
4. **信用卡账单**（辅助核查大额消费与资金去向）
5. **银承/商承贴现记录**（配合应收票据核查）
