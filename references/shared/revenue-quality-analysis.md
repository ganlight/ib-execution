## 一、收入质量综合评估体系

### 1.1 四象限质量矩阵

收入质量由"现金质量"（是否收到钱）和"持续性"（明年还有吗）两个维度决定，二者交叉形成四个象限：

|  | **现金质量高**（收现比>0.85） | **现金质量低**（收现比<0.70） |
|---|---|---|
| **持续性高** | ⭐ **理想状态**<br>订阅制SaaS、长期稳定大客户<br>典型特征：高续费率、稳定回款、可预测 | ⚠️ **风险状态**<br>大客户赊销为主、战略客户账期长<br>典型特征：客户稳定但回款差，存在流动性风险 |
| **持续性低** | ✓ **稳定状态**<br>项目制但回款好、政府项目验收后结算<br>典型特征：单次项目质量高，但订单持续性不确定 | 🚨 **高风险状态**<br>一次性大单+宽付+新客户<br>典型特征：增长不可持续+现金流差=双重危险 |
| **中等持续性** | ✅ **良好状态**<br>品牌消费品、优质经销商网络<br>典型特征：渠道掌控力强，账期管理严格 | 🔴 **警示状态**<br>放宽信用换增长、新客户铺货<br>典型特征：应收账款快速累积，增长质量存疑 |

**矩阵使用说明：**
1. 将目标公司定位到对应象限
2. 象限决定了尽调重点：理想→验证持续性；风险→重点查应收质量；稳定→关注订单持续性；高风险→全面尽调
3. 同象限公司横向对比，识别相对优劣

### 1.2 收入质量综合评分（10分制）

综合评分是QOE分析的数字化输出，为估值和投资决策提供客观参考。

| 评估维度 | 权重 | 评分标准（1–10分） | 数据来源 |
|---------|------|-------------------|---------|
| **收现比** | 25% | >0.90 = 10分；>0.85 = 8分；>0.80 = 7分；>0.70 = 5分；<0.70 = 3分 | 现金流量表÷营业收入 |
| **应收占收比** | 20% | <20% = 10分；<25% = 8分；<30% = 6分；<40% = 4分；>40% = 2分 | 应收账款÷营业收入 |
| **客户集中度** | 20% | CR1<10% = 10分；CR3<20% = 8分；CR3<30% = 6分；CR3<40% = 4分；CR3>40% = 2分 | 前十大客户收入占比 |
| **收入增速合理性** | 15% | 与行业增速差异<5% = 10分；<10% = 8分；<20% = 6分；>20% = 3分 | 行业数据+公司披露 |
| **毛利率稳定性** | 10% | 波动<3% = 10分；<5% = 8分；<8% = 6分；<10% = 4分；>10% = 2分 | 三年毛利率标准差 |
| **渠道健康度** | 10% | 无填充风险 = 10分；轻微填充 = 7分；明显填充 = 4分；严重填充 = 1分 | 月度收入分布分析 |

**评分等级定义：**
| 综合评分 | 质量等级 | 投资建议 | 典型特征 |
|---------|---------|---------|---------|
| ≥8.0分 | ⭐ 高质量 | 可接受 | 各项指标均衡，无重大红旗 |
| 6.0–7.9分 | ✓ 中等质量 | 需关注弱势项 | 1–2个维度有短板，需深入核查 |
| 5.0–5.9分 | ⚠️ 低质量 | 谨慎，需重新谈判 | 存在系统性收入质量问题 |
| <5.0分 | 🚨 质量存疑 | 重新考虑或放弃 | 存在重大造假嫌疑或商业模式根本缺陷 |

**计算示例：**
```
某公司数据：
- 收现比=0.78 → 6分（权重25%）
- 应收占收比=32% → 6分（权重20%）
- CR3=38% → 4分（权重20%）
- 收入增速高于行业15% → 6分（权重15%）
- 毛利率波动5% → 8分（权重10%）
- 渠道轻微填充 → 7分（权重10%）

综合评分 = 6×25% + 6×20% + 4×20% + 6×15% + 8×10% + 7×10%
          = 1.5 + 1.2 + 0.8 + 0.9 + 0.8 + 0.7
          = 5.9分 → 低质量，建议谨慎
```

## 二、收入变动归因与增长质量评估

### 2.1 收入变动分解模型

分析收入增长质量，必须将增长拆解到最底层：量增贡献 vs 价增贡献 vs 结构贡献。

**杜邦收入分解法：**

```
收入变动 = 销量变动贡献 + 价格变动贡献 + 产品结构变动贡献

其中：
销量变动贡献 = Σ(各产品销量变化量 × 基期单价)
价格变动贡献 = Σ(各产品单价变化量 × 报告期销量)
结构变动贡献 = Σ[(报告期产品结构占比 - 基期占比) × 全产品平均单价 × 总销量]

交叉影响项（通常计入结构项）：
= Σ(销量变化量 × 价格变化量) × 产品结构权重
```

**实战应用：**

取得目标公司产品销售明细（分产品分月度），计算各因素贡献占比：

| 增长来源 | FY2贡献（万元） | FY3贡献（万元） | 趋势判断 |
|---------|---------------|---------------|---------|
| 销量增长贡献 | +8,500 | +6,200 | 增速放缓，需关注原因 |
| 价格增长贡献 | +3,200 | +8,500 | 价格驱动增强（好事or价格战？） |
| 产品结构升级贡献 | +1,800 | +2,100 | 结构优化，健康 |
| 合计 | +13,500 | +16,800 | FY3增速更高 |

**判断逻辑：**
- 销量↑+价格稳 → 内生增长，健康
- 销量↑+价格↓ → 以价换量，红旗
- 价格↑为主 → 分析提价能力来源（品牌力？垄断？稀缺性？）
- 若提价来自原材料涨价转嫁 → 无护城河，判断中性

### 2.2 增长质量信号灯系统

**客户维度：**

| 信号 | 健康（绿灯） | 警示（黄灯） | 危险（红灯） |
|------|------------|------------|------------|
| 新增客户质量 | 新客户毛利率与老客户相当 | 新客户毛利率略高（<10%） | 新客户毛利率高>15ppt（关联方？） |
| 新客户占比 | <20%年新增 | 20–30%年新增 | >30%年新增（增长依赖新客户） |
| 存量客户粘性 | 续费率>90% | 续费率70–90% | 续费率<70% |
| 客户流失原因 | 正常（客户倒闭/业务调整） | 价格原因流失 | 产品/服务问题导致流失 |
| 大客户依赖 | CR3<40% | CR3=40–55% | CR3>55% |

**区域维度：**

| 信号 | 健康（绿灯） | 警示（黄灯） | 危险（红灯） |
|------|------------|------------|------------|
| 区域扩张模式 | 复制已有成功模式（标准产品+标准打法） | 区域定制化（需要投入研发） | 每个客户高度定制（无规模效应） |
| 新区域盈亏平衡 | <18个月 | 18–36个月 | >36个月或持续亏损 |
| 跨区域毛利率 | 新区域毛利率与老区域相当 | 新区域略低（<5ppt） | 新区域毛利率低>10ppt |
| 外地收入占比 | 外地占比>30%（分散风险） | 外地占比20–30% | 本地占比>70%（集中风险） |

### 2.3 季节性调节后分析方法

对于强季节性行业（如白酒、教育、建筑），必须将季节性剥离后再做同比分析，否则会严重误判增长质量。

**季节指数法：**

```
步骤1：计算36个月各月平均值
步骤2：季节指数 = 各月平均 ÷ 全年平均月收入 × 12
步骤3：季调后收入 = 实际收入 ÷ 季节指数
步骤4：同比（在季调后数据上做）
```

**季节性异常识别：**

| 月份 | 历史季节指数 | 报告期季节指数 | 差异 | 判断 |
|------|------------|--------------|------|------|
| 1月 | 0.8 | 0.6 | -0.2 | 异常偏弱，春节因素 or 收入缺口？ |
| 12月 | 1.2 | 1.8 | +0.6 | 异常偏高，渠道填充 or 结算加速？ |
| 6月 | 1.0 | 0.7 | -0.3 | 异常偏弱，半年度缺口？ |

**行业季节性参考基准：**

| 行业 | 旺季季节指数 | 淡季季节指数 | 正常波动区间 |
|------|------------|------------|------------|
| 白酒 | 1.3–1.5（春节前） | 0.6–0.7（Q2） | 0.5–1.5 |
| 空调 | 1.2–1.4（夏季） | 0.4–0.6（冬季） | 0.4–1.5 |
| 建筑 | 1.1–1.3（年末） | 0.5–0.7（春节前） | 0.5–1.3 |
| 教育 | 1.3–1.4（开学季） | 0.5–0.7（暑期） | 0.5–1.4 |

## 三、关联交易收入识别与定价

### 3.1 隐性关联交易识别体系

关联交易披露是A股最常见的造假温床之一。识别隐性关联交易是收入质量分析的重要组成部分。

**四大识别路径：**

**路径一：工商信息穿透**
- 核查客户/供应商的股东结构（与目标公司是否共享股东）
- 追溯实际控制人（个人→家族→一致行动人）
- 重点工具：天眼查、企查查、国家企业信用信息公示系统
- 触发条件：任何持股>5%的共同股东，或董监高重叠

**路径二：关键人员穿透**
- 取得目标公司董监高名单
- 核查客户/供应商董监高名单
- 比对：同名同姓（可同一自然人）、近亲属关系
- 关注：独立董事在客户/供应商任职（独董往往在关联企业兼职）

**路径三：商业逻辑合理性**
- 交易频率异常（如每天都大额采购）
- 交易金额规律性（如每月正好500万）
- 交易时间规律（如每月末最后一天集中确认）
- 物流与资金流不匹配（如货物不发往客户注册地）

**路径四：地理信息分析**
- 注册地或办公地在同一园区/同一栋楼
- 联系方式相同（电话、邮箱相同注册人）
- 历史上曾存在股权或人员关联（历史关联方未完全脱钩）

**关联交易识别检查表：**
- [ ] 是否存在工商穿透可识别的关联关系（持股>5%或董监高重叠）
- [ ] 新增客户/供应商成立时间<2年
- [ ] 新增客户/供应商与公司注册地/办公地在5公里以内
- [ ] 交易金额占客户/供应商收入>30%（对方的收入依赖度极高）
- [ ] 毛利率显著高于/低于其他同类客户
- [ ] 无实际物流单据（纯资金走账）
- [ ] 期末集中确认大额收入，期后退货

### 3.2 关联交易定价合理性评估

确认关联交易存在后，需要评估定价是否市场化。定价不公允的关联交易本身就是一个盈利质量红旗。

**各类关联交易定价方法：**

| 关联交易类型 | 推荐定价方法 | 可比性要求 | 参考标准 |
|------------|------------|----------|---------|
| 商品销售 | 市场法（同类产品价格） | 至少3个可比交易或市场报价 | 出厂价/批发价 vs 市场价差异<10% |
| 采购 | 市场法（询价比较） | 至少3个供应商报价 | 采购价 vs 市场价差异<5% |
| 资金拆借 | 同期银行贷款基准利率 | 不低于央行基准利率的90% | 可参照同期AAA企业债利率 |
| 担保/抵押 | 通常免费（问题本身） | 应有市场公允的担保费 | 担保费率参照同期限贷款1–3% |
| 租赁 | 市场租金法 | 区域+用途+面积+楼层可比 | 租金 vs 市场均价差异<15% |
| 服务（管理/IT/后勤） | 成本加成法 | 成本核算需规范透明 | 成本加成率5–15%为常见区间 |
| 许可/商标使用 | 入门费+分成法 | 参照行业 licensing deals | 分成率通常为收入0.5–5% |
| 研发/技术许可 | 行业通行费率 | 技术先进性评估 | 提成率通常为收入2–10% |

**非公允关联交易对收入质量的影响量化：**

```
情形一：低价向关联方销售（转移利润出公司）
= 正常价格收入 - 关联交易价格收入
= [1000万 × (1.0元 - 0.7元)] ÷ 1.0元 = 300万收入虚减
= 影响：当期收入虚减300万（需加回）

情形二：高价从关联方采购（利益输送）
= 关联交易采购价 - 市场采购价 = 200万
= 影响：当期成本虚增200万（需加回）

情形三：关联方免费使用资产（隐性补贴）
= 市场租金 - 实际租金 = 150万
= 影响：当期利润虚增150万（需调整）
```

## 四、客户与合同质量深度评估

### 4.1 客户质量评估矩阵

| 客户类型 | 付款能力 | 付款意愿 | 粘性 | 质量评级 |
|---------|---------|---------|------|---------|
| 央企/国企（直接客户） | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ 极优 |

| 大型外企 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ | ⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ 优质 |
| 优质民企（细分龙头） | ⭐⭐⭐⭐ | ⭐⭐⭐ | ⭐⭐⭐⭐ | ⭐⭐⭐⭐ 良好 |
| 中型民企（无担保） | ⭐⭐⭐ | ⭐⭐⭐ | ⭐⭐⭐ | ⭐⭐⭐ 中等 |
| 小型民企 | ⭐⭐ | ⭐⭐ | ⭐⭐ | ⭐⭐ 较差 |
| 关联公司 | ⭐⭐⭐ | ⭐⭐ | ⭐⭐ | ⭐ 存疑 |

### 4.2 合同质量评估六维度

| 评估维度 | 优质合同特征 | 劣质合同特征 | 风险点 |
|---------|------------|------------|-------|
| **合同金额** | 金额明确，条款清晰 | 框架合同无金额上限 | 无法预计收入规模 |
| **履约义务** | 交付物明确，节点清晰 | 模糊表述，存在争议空间 | 收入确认分歧 |
| **验收条款** | 明确验收标准+验收期 | 无标准或验收期过长（>6个月） | 履约完成但无法确认收入 |
| **付款条件** | 里程碑付款，验收即付 | 验收后留10%质保金，账期长 | 资金占用大 |
| **变更条款** | 有变更签证流程 | 无变更管理，隐性变更 | 工程量追加无法确权 |
| **退出条款** | 违约责任明确，可执行 | 责任模糊，诉讼困难 | 纠纷无保障 |

### 4.3 收入确认时点核查

收入确认时点错误（提前或推迟）是IPO最常见的财务造假手段之一。

**不同商业模式下的正确确认时点：**

| 商业模式 | 正确确认时点 | 常见错误 | 核查方法 |
|---------|------------|---------|---------|
| 商品销售（买断式） | 发货并取得签收单时 | 提前开票确认、期末集中发货 | 发货记录+签收单日期 |
| 商品销售（代销） | 收到代销清单时 | 发出商品即确认 | 代销系统数据+清单日期 |
| 服务收入（直线法） | 服务期间均匀确认 | 一次性确认或严重不均匀 | 服务进度与发票匹配 |
| 服务收入（工作量法） | 完成工作量占合同比例 | 虚报工作量 | 工时记录+客户确认 |
| 软件授权（永久许可） | 交付安装序列号/授权码 | 签合同即确认 | 授权码发放记录 |
| 软件SaaS（月度订阅） | 每月服务完成，提供服务时 | 提前确认全年限额收入 | 月度服务日志 |
| 工程建设（完工百分比） | 按履约进度确认，参照监理报告 | 进度虚报 | 监理报告+业主确认函 |
| EPC总包 | 单项履约义务完成并验收 | 混淆EPC各部分确认时点 | 分部验收报告 |

## 五、收入质量尽调现场检查程序

### 5.1 现场核查三板斧

**板斧一：银行流水与收入的匹配**

在现场获取目标公司所有银行账户（报告期+报告前一年）：

```
核查步骤：
1. 取得全部银行账户清单（与账面账户数对比，核查是否有账外账户）
2. 取得全部银行对账单（电子版+纸质版，双向核对）
3. 核对收入回款：销售收款 vs 银行收款（金额匹配性）
4. 识别"可疑"流水：
   - 整数金额（100万、500万等整倍数）
   - 规律性付款（每月同一天付同样金额）
   - 大额整数次日转出（不留余额）
   - 收付款时间高度集中（月末/年末最后3天）
5. 追踪主要客户资金来源（是否来自关联方、实控人）
```

**板斧二：截止性测试（Cut-off Test）**

截止性测试是识别期末突击确认和跨期调节的核心手段：

```
核查步骤：
1. 取得报告期最后4周和次年1月的前4周销售记录
2. 标记所有大额交易（>100万）
3. 核查以下关键日期文件：
   - 发票日期（是否在报告期末前开具）
   - 出库单/发货单日期（货物是否真的在报告期末前发出）
   - 签收单日期（客户是否在报告期末前收到）
   - 银行回单日期（款项是否在报告期末前到账）
4. 判断：若发票/出库在报告期末前，但签收在次年 → 提前确认
5. 延伸：核查期后30天退货记录，若有大额退货 → 期末突击确认
```

**板斧三：函证与访谈**

```
应收账款函证：
- 发函率：年末余额>10万的客户应全部发函
- 回函率目标：>80%
- 函证内容：期末余额+报告期交易额+合同条款
- 替代程序：若不回函，执行替代测试（检查期后回款+合同+发票）

客户访谈（核心客户）：
- 访谈比例：前10大客户中至少5家，金额覆盖>40%
- 访谈方式：视频/电话+书面确认
- 必问问题：
  1. 采购量与目标公司披露是否一致？
  2. 付款周期是否如合同约定？
  3. 是否存在未披露的返利/折扣/退货安排？
  4. 是否与目标公司存在关联关系？
  5. 未来采购计划如何（判断收入持续性）？
```

## 六、收入质量快速判断模型（现场版）

### 6.1 10分钟快速扫描

| 检查项 | 操作方法 | 判断标准 | 发现异常后下一步 |
|-------|---------|---------|---------------|
| 1. 收现比 | 经营现金流净额÷营业收入 | <0.70立即启动应收专项 | 列出应收>1年客户清单 |
| 2. 12月占比 | 12月收入÷全年收入 | >20%开始警惕 | 取得12月大单明细 |
| 3. 毛利率突变 | 与可比公司对比 | 差异>8ppt | 要求管理层书面解释 |
| 4. CR3集中度 | 前三客户收入占比 | >50%高度集中 | 核查大客户合同+访谈 |
| 5. 应收增速 | 应收增速 vs 收入增速 | 应收增速>收入增速30ppt+ | 核查信用政策是否放宽 |
| 6. 期末大单 | 报告期末500万+订单数 | 显著高于历史均值 | 核查期后退货情况 |
| 7. 新客户占比 | 新增客户收入÷总收入 | >25%且毛利率异常高 | 工商穿透新客户 |

### 6.2 收入质量最终判断决策树

```
开始判断
    │
    ▼
收现比 < 0.70 ？
    │
    ├─ 是 → 应收专项核查 → 列出重点应收问题客户
    │
    └─ 否 → 继续
            │
            ▼
        12月收入占比 > 20% ？
            │
            ├─ 是 → 渠道填充核查 → 期后退货分析
            │
            └─ 否 → 继续
                    │
                    ▼
                CR3 > 50% ？
                    │
                    ├─ 是 → 大客户专项尽调 → 客户访谈
                    │
                    └─ 否 → 继续
                            │
                            ▼
                        综合评分 ≥ 6.0 ？
                            │
                            ├─ 是 → 结论：收入质量可接受（有条件）
                            │
                            └─ 否 → 结论：需深入尽调或重新估值
```
