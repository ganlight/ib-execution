## 第1章：公募公司债（含大公募/小公募）

### 1.1 证监会审核关注点

#### 审核一部 vs 审核二部差异

| 维度 | 审核一部（大公募） | 审核二部（小公募） |
|------|------------------|------------------|
| 适用范围 | AAA级超级主体，面向公众 | AA级以上，面向合格投资者 |
| 交易所预审深度 | 仅合规性审查 | 实质性审核 |
| 反馈轮次 | 通常0-1轮 | 1-3轮典型 |
| 财务关注 | 简式报告即可 | 要求完整三年一期 |
| 募集资金 | 用途概括性披露即可 | 须逐笔列明 |
| 平均审核周期 | 30-45日 | 45-90日 |
| 存续期管理要求 | 简化年报+重大事项公告 | 完整的受托管理 |

#### 核心审核要点（证监会审核指引）

1. **发行人主体资格**：股份公司/有限公司、存续满3年（以36个月计）、无重大违法违规
2. **财务条件**：最近三年平均可分配利润≥债券一年利息的1.5倍（大公募）/1倍（小公募）
3. **40%净资产余额限制**：累计债券余额≤最近一期净资产的40%
4. **评级要求**：大公募需AAA、小公募需AA及以上
5. **募集资金用途**：不得用于非生产性支出、不得用于弥补亏损
6. **债券持有人权益保护**：受托管理人制度、债券持有人会议规则
7. **信息披露合规性**：是否存在虚假记载、误导性陈述、重大遗漏

### 1.2 章节结构详解（14章标准结构）

#### 第一章：封面及声明
- **撰写要点**：主承销商名称、发行人名称、发行总额、票面利率、期限、评级、担保情况（含担保人）
- **常见错误**：遗漏担保人名称、利率区间与最终簿记结果不一致、Bond Connect标识遗漏
- **声明文字标准条款**：发行人董事会声明、主承销商声明、受托管理人声明、评级机构声明

#### 第二章：重大事项提示（⚠️ 核心章节）
- **撰写要点**：至少列出5-8条，按重要程度排序；财务风险→经营风险→合规风险→增信风险
- **常见错误**：重大事项与后文风险因素简单复制（应提纲挈领）、遗漏评级调整风险、控制权变更条款未在重大事项提示中突出
- **结构性要求**：须在开篇以醒目方式提示投资者关注（加粗/加框）

#### 第三章：释义
- **撰写要点**：按字母拼音排序；至少包含30-50条核心术语
- **常见错误**：释义与正文用词不一致（如定义"本次发行"但正文改为"本期发行"）、遗漏增信安排中专用术语
- **必含术语**：发行人、控股股东、实际控制人、本次债券、本次发行、基准利率、信用评级下调、交叉违约

#### 第四章：发行概况
- **撰写要点**：债券名称、发行总额、票面金额和发行价格、债券期限、票面利率及其确定方式、发行对象、发行方式、付息兑付方式
- **常见错误**：含权条款（回售/赎回/调整票面利率）在此章未充分披露、发行日程表与实际情况不符
- **含权债特别标记**：如含回售权/赎回权/利率调整选择权，须在本章专设"特殊发行条款"小节

#### 第五章：风险因素（⚠️ 审核重点）
- **撰写要点**：按风险类型分节——与发行人相关的风险、与本次债券相关的风险、市场风险、政策风险
- **常见错误**：定性描述过多缺少定量数据支撑、毛利率/资产负债率等财务指标变化不纳入风险分析、行业风险与发行人个体风险混淆
- **信息披露标准**：须遵循"重大性"原则，面向一般投资者时须用简单语言说明

#### 第六章：发行人基本情况
- **撰写要点**：历史沿革、控股股东及实际控制人、组织结构、对其他企业的重要权益投资
- **常见错误**：股权结构图未标注各层级持股比例、未穿透至实质控制人、董事/监事/高管兼职情况未充分披露
- **关联方认定**：须严格按照《公司法》《企业会计准则第36号》认定关联方

#### 第七章：财务会计信息（⚠️ 数据一致性要求高）
- **撰写要点**：三年一期完整财务报表、主要财务指标（流动比率/速动比率/资产负债率/利息保障倍数/EBITDA等）
- **常见错误**：与审计报告数据不一致、财务指标计算公式未统一、模拟合并报表口径不清
- **特殊处理**：近一年一期有重大资产重组须编制备考合并报表

#### 第八章：募集资金运用（⚠️ 证监审核重点）
- **撰写要点**：逐项列明资金用途、每项用款总额及占比、实施主体、资金使用进度安排
- **常见错误**："补充流动资金"占比过高（建议≤30%）、偿还债务未注明被偿债项明细、用于项目投资但缺少项目批复文件
- **资金专户管理**：须明确专户银行、账户号、资金使用审批流程

#### 第九章：偿债计划及保障措施
- **撰写要点**：偿债资金来源（经营活动现金流/处置资产/再融资）、偿债应急保障方案、偿债专户安排
- **常见错误**：偿债资金来源过于笼统（"日常经营收入"缺乏量化支持）、未体现偿债时间与资金来源的匹配度
- **压力测试**：建议补充偿债压力测试情景（收入下滑20%/毛利率下降5个百分点等）

#### 第十章：发行人信用状况
- **撰写要点**：主体信用评级及评级报告主要结论、银行授信额度及使用情况、历史债务履约记录
- **常见错误**：仅列评级级别未附评级观点、银行授信额度含已到期/已暂停额度、未披露最近三年违约/逾期记录

#### 第十一章：增信机制（如有）
- **撰写要点**：担保类型（保证/抵押/质押）、担保人基本情况及财务数据、担保物评估价值及登记情况
- **常见错误**：担保人未出具有效决议、担保物存在在先顺位的他项权利、未披露担保人与发行人关联关系
- **差额补偿/维好协议**：须明确法律效力不同于保证担保

#### 第十二章：持有人会议规则
- **撰写要点**：召集情形、召集程序、表决机制、决议效力
- **常见错误**：触发情形过于宽泛或过于严格、表决比例设置不合理（50%或2/3须明确）
- **特殊情形**：控制权变更条款触发后是否需召开持有人会议须对应

#### 第十三章：受托管理人
- **撰写要点**：受托管理人名称及资质、受托管理职责（信息披露/违约追索/诉讼代表）
- **常见错误**：受托管理人与主承销商为同一机构但未说明独立性保障措施
- **费用安排**：受托管理费收取标准及支付方式

#### 第十四章：其他事项
- **撰写要点**：信息查阅场所、备查文件目录、投资者保护机制联系方式
- **常见错误**：备查文件目录不完整，遗漏相关协议全文

### 1.3 反馈意见高频问题（20条）

1. **【财务一致性问题】** 请发行人说明最近一年经营活动现金流净额大幅变动的原因及与净利润的差异原因
2. **【应收账款问题】** 请说明应收账款前五大客户明细、账龄结构、坏账准备计提政策及充分性
3. **【关联交易问题】** 请补充披露报告期内关联交易定价原则、公允性分析及对发行人独立性的影响
4. **【募集资金问题】** 请细化募集资金用途，补充披露偿还债务的被偿债项发行时间/利率/到期日
5. **【重大未决诉讼】** 请说明未决诉讼/仲裁的最新进展及对发行人的潜在影响
6. **【对外担保】** 请补充披露对外担保明细、被担保人经营情况及代偿风险分析
7. **【受限资产】** 请说明资产抵押/质押/冻结情况，分析对发行人偿债能力的影响
8. **【投资性房地产】** 公允价值变动对利润影响的敏感性分析
9. **【政府补贴依赖】** 请分析报告期内政府补贴占利润比例及可持续性
10. **【客户/供应商集中度】** 前五大客户/供应商集中度分析及替代风险
11. **【收购形成的商誉】** 商誉减值测试方法、关键假设及减值风险
12. **【资产负债率持续上行】** 资产负债率变动原因及控制措施
13. **【控股股东股权质押】** 控股股东股权质押比例、平仓线及对控制权的影响
14. **【业务模式可持续性】** 主营业务的成长性、可持续性分析
15. **【环保合规】** 发行人及主要子公司是否存在环保违法违规及整改情况
16. **【安全生产】** 报告期内是否发生重大安全生产事故
17. **【资金被占用】** 是否存在控股股东/实控人非经营性占用发行人资金的情形
18. **【表外负债】** 是否存在未纳入合并报表的明股实债/结构化融资安排
19. **【交叉违约风险】** 本次发行是否触发现有债务合同中的交叉违约条款
20. **【债券持有人权益保护】** 投资者保护条款设计的合理性及充分性

### 1.4 特殊条款披露

#### 控制权变更条款
- **定义**：须量化——"（1）发行人控股股东发生变更；或（2）实际控制人发生变更；或（3）发行人被收购且收购完成后原股东不再持有控制权"
- **触发后果**：持有人回售权或债券提前到期
- **披露位置**：重大事项提示 + 发行概况 + 持有人会议规则

#### 交叉违约条款
- **定义要素**：触发金额（建议≥最近一期经审计净资产5%或5,000万元孰低）、触发主体范围、豁免情形
- **常见错误**：触发金额过低造成"技术性违约"频发、未明确通知/宽限期安排

#### 投资者保护条款（核心条款体系）
```
投资者保护条款体系
├── 交叉保护条款（债权保护）
├── 事先约束条款（行为限制）
│   ├── 财务指标承诺（资产负债率/有息负债/EBITDA利息倍数）
│   ├── 资产出售/质押限制
│   ├── 对外担保/关联交易限制
│   └── 合并/分立/减资限制
└── 信用事件条款
    ├── 评级下调条款
    ├── 未按时披露定期财务信息
    └── 实质违约事件
```

## 第2章：企业债（含专项债）

### 2.1 发改委审核体系

#### 审核层级
```
国家发改委财金司（注册决策）
        ↑
中央结算公司（预审 + 受理）
        ↑
省级发改委（转报）
        ↑
主承销商 / 发行人（申报）
```

#### 审核周期
- 省级发改委转报：15-30个工作日
- 中央结算公司预审：30-45个工作日
- 发改委财金司注册：15-30个工作日
- **总计**：约3-6个月

### 2.2 与募投项目绑定的特殊披露要求

企业债募集说明书的独特之处在于与募投项目的强绑定关系。须逐项披露：

#### 项目基本信息
- 项目名称、建设地点、建设规模及内容
- 总投资及资金来源构成（资本金比例≥项目总投资的20%）
- 项目建设期及投产后预计效益
- 项目实施主体及与发行人的股权关系

#### 项目合规性论证框架（必须完备）

| 合规要件 | 审批机构 | 获取要求 | 替代文件 |
|---------|---------|---------|---------|
| 立项/核准/备案 | 发改委/经信委 | 须取得正式批复 | 无 |
| 环境影响评价 | 生态环境部门 | 环评批复或备案 | 环评登记表（特定小项目） |
| 用地审批 | 自然资源部门 | 用地预审意见/土地证/不动产权证 | 租赁合同+出租方权属证明 |
| 规划许可 | 自然资源/规划部门 | 建设用地规划许可证+建设工程规划许可证 | 选址意见书（特定阶段） |
| 节能审查 | 发改委/经信委 | 节能审查意见 | 能耗说明（低能耗项目） |
| 施工许可 | 住建部门 | 建筑工程施工许可证 | 可延期取得（部分项目） |
| 社会稳定风险评估 | 地方政府 | 稳评批复或备案 | 不适用特殊区域 |

#### 项目经济效益分析
- 内部收益率（IRR）测算——需高于同期贷款利率
- 投资回收期——静态与动态均须计算
- 盈亏平衡分析
- 敏感性分析（投资增加10%/收入减少10%）

### 2.3 发改委反馈高频问题（15条）

1. **【项目合规性】** 募投项目尚未取得全部合规手续，请说明后续取得的时间安排
2. **【资本金到位】** 募投项目资本金到位情况，资本金来源是否合规
3. **【债券存续期匹配】** 债券期限与项目建设/运营期的匹配性分析
4. **【募投项目效益测算】** 项目经济效益测算过程、关键假设合理性的说明
5. **【项目资金缺口】** 除本次债券外，项目建设其他资金来源是否已落实
6. **【偿债保障措施】** 项目自身收益对债券本息的覆盖倍数（要求≥1.0）
7. **【土地取得方式】** 募投项目用地的取得方式及权属清晰性
8. **【重复融资风险】** 是否存在同一项目在多只债券中作为募投项目的情形
9. **【主体经营状况】** 发行人在建项目总规模与自身经营体量的匹配性
10. **【应收账款】** 应收账款规模及账龄分析，是否与营业规模匹配
11. **【有息负债】** 有息负债结构、融资成本及偿债高峰分布
12. **【城投类发行人特殊问题】** 发行人是否涉及隐债化解、是否在财政部的隐债名单中
13. **【专项债品种特殊问题】** 专项企业债对应的专项领域政策符合性论证
14. **【评级下调风险】** 分析发行人信用评级是否有下调风险
15. **【中介机构专项核查】** 要求会计师对募投项目资金使用专项核查并出具意见

## 第3章：MTN（中期票据）

### 3.1 NAFMII表格体系

#### MB表（中期票据募集说明书表格）标准内容

```
MB表体系（2023版）
├── MB-1表：封面及目录格式
├── MB-2表：重要提示
├── MB-3表：释义
├── MB-4表：风险提示及说明
├── MB-5表：发行条款
├── MB-6表：募集资金运用
├── MB-7表：发行人基本情况
├── MB-8表：发行人主要财务情况（含近三年一期财务数据）
├── MB-9表：发行人资信状况
├── MB-10表：信用增进（如有）
├── MB-11表：税项
├── MB-12表：信息披露安排
├── MB-13表：持有人会议机制
└── MB-14表：备查文件
```

#### 关键差异化要求（vs 公司债）

| 维度 | MTN（NAFMII体系） | 公募公司债（证监会体系） |
|------|-----------------|---------------------|
| 财务披露期间 | 近三年及一期（两期报表） | 近三年及一期 |
| 财务报表要求 | 仅需近一年经审计（近一期可不审） | 三年均须审计 |
| 审计意见类型 | 标准无保留意见（近一年） | 三年均应为标准无保留意见 |
| 受托管理人 | 无强制要求 | 须聘任受托管理人 |
| 募集资金用途 | 可补充营运资金+偿还债务 | 用途详细披露+资金专户 |
| 分层注册 | DFI/TDFI可事后备案 | 无分层机制 |

### 3.2 DFI/TDFI框架下的披露简化规则

#### DFI（统一注册，Debt Financing Instruments）
- **资格条件**：同时满足境内AAA+境外投资级/市场认可度高/信息披露优秀/无违规记录
- **简化规则**：
  - 财务信息可使用已公开披露的年度报告/半年度报告
  - 募集资金用途可概括性披露（"用于公司营运资金及一般企业用途"）
  - 无需逐期编制完整募集说明书，可使用"发行方案+募集说明书补充信息"
  - 存续期信息披露可在年度报告中合并

#### TDFI（分层注册，Tiered DFI）
- **分两档**：第一档（成熟发行人）、第二档（常规发行人）
- **第一档简化程度**：较DFI稍严格，但仍可享受财务报表简式引用、募集资金用途概括披露
- **第二档完整程度**：接近普通注册，须提供完整募集说明书

### 3.3 注册反馈高频问题（10条）

1. 发行人是否符合DFI/TDFI发行条件
2. 最近一年经营活动现金流量净额大幅变动的原因及对偿债能力的影响
3. 在建工程及拟建工程投资规模与公司资金实力的匹配性
4. 对外担保规模较大的代偿风险分析
5. 受限资产比例较高的流动性影响分析
6. 关联交易的定价公允性
7. 发行人新增的或有事项/期后事项
8. 对单一客户/供应商的重大依赖风险
9. 报告期内主要财务指标波动原因（须附分业务板块分析表）
10. 本次注册额度使用计划及后续发行安排

### 3.4 债务融资工具特殊披露——持有人会议机制

- **召集情形**（NAFMII《持有人会议规程》明确）：
  1. 发行人未能按期足额兑付本息
  2. 发行人转移债务融资工具清偿义务
  3. 发行人减资、合并、分立、解散或申请破产
  4. 信用增进安排发生重大变化
  5. 持有人会议认为需召开的其他情形
- **表决机制**：须经出席会议的持有人所持表决权的2/3以上通过
- **与公司债对比**：NAFMII体系对持有人会议触发情形规定更细致、表决生效门槛更低

## 第4章：CP/SCP（短期融资券）

### 4.1 短期融资券特殊披露要求

#### 财务报表期间差异

| 项目 | CP（短期融资券） | SCP（超短期融资券） | MTN（中票） |
|------|-----------------|-------------------|-----------|
| 财务数据期间 | 近一年一期 | 近一年一期 | 近三年一期（两期完整报表） |
| 经审计报表 | 近一年经审计 | 近一年经审计 | 近三年经审计（近一期可不审） |
| 评级要求 | 主体评级A+及以上 | 主体评级AA及以上 | 主体+债项评级 |
| 信息披露详简度 | 简化 | 大幅简化 | 完整 |
| 发行期限 | ≤1年 | ≤270天 | 2-10年 |

### 4.2 SCP信息披露简化规则

- SCP可使用"发行披露材料简版"（SCP-1至SCP-6表），无需编制完整募集说明书
- 主要披露内容：发行条款 + 募集资金用途 + 发行人近一年财务概况 + 信用评级
- 无需披露：完整的历史沿革、行业分析、未来资本支出计划、风险因素的细化分析
- 注册额度内单次发行仅需报备发行方案，无需重新注册

### 4.3 信用评级强制要求

- CP信用评级报告有效期为1年（MTN为2年）
- CP须同时披露主体评级和债项评级
- SCP在DFI/TDFI框架下可仅披露主体评级
- 跟踪评级须按年度进行（不因期限短而豁免）

## 第5章：PPN（定向债务融资工具）

### 5.1 合格投资者制度下的披露豁免

#### 信息披露分层

| 层级 | 披露内容 | 对比公募 |
|------|---------|---------|
| 注册时 | 定向募集说明书（内部版） | vs 公开版募集说明书 |
| 发行时 | 发行方案+定向募集说明书（投资人版） | vs 公告版募集说明书 |
| 存续期 | 定向向持有人披露 | vs 公开信息披露 |

#### 核心豁免项目
- **豁免公开财务详细信息**：无需公开披露近三年详细财务数据（可向定向投资人定向提供）
- **豁免公开评级**：无强制公开评级要求
- **豁免媒体公告**：发行结果无需在指定媒体刊登公告
- **豁免公开担保信息**：担保安排可定向披露

### 5.2 定向发行协议的特殊约定

- **《定向发行协议》** 是PPN的核心法律文件，替代公募中的募集说明书
- 协议须明确：投资人资格认定标准、发行价格确定方式、信息披露内容和频率、转让限制
- **投资者适当性管理**：
  - 投资人须为NAFMII认可的定向投资人
  - 单只PPN持有人数不得超过200人
  - 发行人须建立投资人合格性审查制度

### 5.3 登记托管与转让限制

- PPN在上海清算所登记托管
- **转让限制**：
  - 只能在定向投资人之间协议转让
  - 受让人须符合定向投资人资格
  - 不得采用集中竞价方式交易
- **信息披露不对称性风险提示**：须在发行协议中明确提示"定向发行存在信息不对称的流动性风险"

## 第6章：永续中票/可续期公司债

### 6.1 续期条款触发条件

#### 条款设计要求
- **发行人的续期选择权**：在每个重定价周期（通常3年或5年）末，发行人可选择将债券期限延续一个重定价周期
- **明确约定要素**：
  - 初始重定价周期及票面利率
  - 每个重定价周期的利率重置机制（基准利率+初始利差+上调基点）
  - 利率跳升幅度（Step-up）——通常为300bps，须明确
  - 递延支付利息的利率惩罚——通常为当期票面利率上浮300bps

### 6.2 递延支付利息条款

#### 强制付息事件清单（必须触发即付息，不得递延）
1. 向普通股股东分红（现金分红触发，送股不触发）
2. 减少注册资本
3. 向偿付顺序劣后于本期永续债的其他金融工具支付利息

#### 利息递延的限制条件
- 每笔递延利息在递延期间应按当期票面利率累计计息
- 递延支付次数不限，但一旦递延，所有未付递延利息须在付息前全部结清
- 会计认定影响：若递延条款被视为"发行人可无条件避免支付"，可能影响权益工具认定

### 6.3 会计处理变更风险（⚠️ 须在募集中明确提示）

- **权益工具 vs 金融负债认定**：
  - 判定关键：发行人是否可"无条件地避免交付现金或其他金融资产"
  - 现行主流认定：永续中票含利率跳升机制+强制付息事件→仍可认定为权益工具
  - 潜在变更：若《企业会计准则第37号》修订或财政部出具新解释，可能导致已发行的永续中票在会计上重新划分为金融负债
- **披露要求**（证监/NAFMII均要求）：须在募集说明书"重大事项提示"及"风险因素"中专项提示"会计处理变更风险"

### 6.4 评级下调触发条款

- 若发行人主体评级下调至AA+（含）以下，利率重置时上调基点再增加100-300bps
- 若评级下调至A+（含）以下，触发持有人提前回售权（可选设计）
- 与普通债券的评级下调条款相比，永续债因存续期极长，评级下调影响更为显著

## 第7章：绿色债券

### 7.1 绿色项目认定标准（三套标准体系）

#### 标准一：NAFMII《绿色债务融资工具指引》
- 七大绿色产业领域：节能环保、清洁生产、清洁能源、生态环境、基础设施绿色升级、绿色服务
- 参考目录：《绿色债券支持项目目录（2021年版）》（人民银行/发改委/证监会联合发布）

#### 标准二：发改委《绿色企业债指引》
- 适用发改委主管的绿色企业债
- 六类重点支持：节能减排技术改造、绿色城镇化、能源清洁高效利用、新能源开发利用、循环经济发展、水资源节约和非常规水资源开发利用

#### 标准三：交易所《绿色公司债券指引》
- 适用交易所挂牌的绿色公司债
- 项目分类与2021年版目录保持一致，但审核口径偏保守
- 额外关注：绿色项目的可持续经营能力

### 7.2 环境效益量化指标披露

#### 七类指标+测算方法

| 绿色项目类型 | 关键量化指标 | 测算方法 |
|------------|------------|---------|
| 清洁能源（光伏） | 年发电量（万kWh）+CO₂减排量（吨） | 发电量×当地电网排放因子 |
| 清洁能源（风电） | 年上网电量（万kWh）+替代标煤量（吨标准煤） | 上网电量×平均供电煤耗 |
| 污水处理 | 年处理污水量（万吨）+COD削减量（吨） | 处理量×进水/出水COD浓度差 |
| 轨道交通 | 年客运人次（万人次）+替代私家车出行减排量 | 客运周转量×小汽车单公里排放因子 |
| 建筑节能 | 年节约标准煤（吨）+减排CO₂（吨） | 节能面积×单位面积节能量 |
| 绿色产业园区 | 年资源循环利用量（吨）+废弃物减排量（吨） | 循环利用量=输入量-输出量 |
| 生态修复 | 修复面积（公顷）+植被覆盖率提升 | 修复面积×单位面积碳汇增量 |

### 7.3 绿色认证机构要求

- **NAFMII/交易所均要求**：须聘请具有绿色债券评估认证业务能力的认证机构出具《绿色债券评估认证报告》
- **认可机构名单**：中诚信绿金、联合赤道、中节能衡准、绿融（北京）等（以NAFMII/交易所官网公布的最新名单为准）
- **认证内容包括**：项目绿色属性判定、环境效益测算复核、募集资金管理方案评估

### 7.4 募集资金专项管理

- 须开立募集资金专项账户，与一般资金账户独立
- 监管银行须出具资金监管承诺函
- 在债券存续期内，须按年度（或半年度）披露《募集资金使用情况专项说明》
- 闲置募集资金可投资于货币市场工具，但不得用于非绿色领域

### 7.5 绿债标签使用规范

- **"绿色债券"标签仅限**：募集资金100%用于绿色项目
- **"专项用于绿色（某领域）"标签**：募集资金≥70%用于对应绿色领域
- **"含绿色投向"标签**：部分资金用于绿色项目，但不足70%
- 发行人不可在未经过认证的情况下使用绿色债券标签

## 第8章：可交换债

### 8.1 预备用于交换的股票（换股标的合规性）

#### 标的股票条件
- 预备用于交换的股票须为其持有的上市公司A股流通股
- 不存在质押、冻结或其他权利限制（初始状态下）
- 换股标的上市公司须为规范运作、不存在重大违规
- 发行人持有标的公司股票的来源合法（原始持股/二级市场增持等）

### 8.2 换股价格条款

#### 初始换股价格
- 不低于募集说明书公告日前20个交易日标的股票均价的90%
- 不低于募集说明书公告日前1个交易日标的股票均价

#### 向下修正条款
- 触发条件：标的股票在连续N个交易日（通常30/20个）中有M个（通常15/10个）收盘价低于当期换股价的X%（通常85%/80%）
- 修正权限：发行人董事会有权提出修正，需经股东大会审议
- 修正底线：修正后不低于审议日前1日交易日均价和前20日交易日均价的较高者

#### 向上修正条款（较少见）
- 设置在标的股价大幅上涨为限制换股节奏
- 须与投资者博弈平衡

### 8.3 换股期安排

- 通常为发行日起6个月后至到期日（可交换债存续期1-6年）
- 换股期内可随时选择换股（或约定特定窗口期）
- 换股数量须为1手或其整数倍
- 换股后剩余可换股数量触发赎回条款

### 8.4 质押担保要求

- 预备用于交换的标的股票须办理质押登记，质权人为全体债券持有人（由受托管理人代持）
- 质押率设置：初始质押率不超过发行规模的120-140%
- 维持担保比例：如因股价下跌导致担保比例低于维持比例（通常110%），发行人须追加担保物
- 解除质押：仅在换股行使时按比例解除

### 8.5 赎回/回售的特殊情形

#### 发行人赎回权
- 换股期内，标的股票连续N个交易日收盘价≥当期换股价的130%（或120%），发行人有权赎回
- 到期赎回：未换股部分按票面+当期应计利息兑付

#### 持有人回售权
- 改变募集资金用途（须持有人会议通过）
- 标的公司发生重大不利情形（退市风险/重大违法等）
- 换股价格多次下修后仍远高于市价

## 第9章：ABS/ABN（资产支持证券）

### 9.1 资产池概况披露

#### 披露要素

| 要素 | 内容要求 | 数据精度 |
|------|---------|---------|
| 资产类型 | 按合同类别分类 | 精确到子类 |
| 入池资产笔数 | 逐笔统计 | 精确 |
| 资产池总规模 | 未偿本金余额合计 | 万元 |
| 单笔资产平均规模 | 均值+中位数 | 万元 |
| 期限结构 | 加权平均剩余期限（WAL）+最终到期日 | 精确到月 |
| 利率特征 | 固定利率/浮动利率比例+加权平均利率 | bps |
| 集中度 | 前五大借款人/地区/行业占比 | 百分比 |
| 信用质量 | 借款人信用评分/评级分布 | 分层统计 |

### 9.2 现金流归集与分配机制

```
现金流入（资产池回款）
  │
  ├── 回收款归集（原始权益人收款账户）
  ├── 转付至监管账户（通常按月/按季）
  └── 信托/专项计划账户
        │
        ├── 优先支付：税+托管费+服务报酬
        ├── 优先级A档：当期利息
        ├── 优先级B档：当期利息
        ├── 优先级A档：本金（按计划/过手）
        ├── 优先级B档：本金
        └── 次级档：剩余收益
```

- **现金流覆盖倍数**：须计算各档的本息覆盖倍数（DSCR）
- **压力测试**：须披露不同违约率/早偿率/利差变动下的现金流覆盖情况

### 9.3 优先/次级分层结构

- **典型结构**：优先级A档（AAA）→优先级B档（AA+）→次级档（无评级）
- **超额覆盖**：超额现金流覆盖（Overcollateralization，OC）比率
- **信用触发机制**：
  - 加速清偿事件（本金/利息支付加速）
  - 违约事件（改变现金流分配顺序，不再按计划支付）
- **次级占比**：通常5-15%，由原始权益人自持

### 9.4 外部增信措施披露

- **差额支付承诺**：原始权益人或关联方承诺在回收款不足时补足差额
- **流动性支持**：第三方在特定时点提供流动性借款
- **担保**：第三方对优先级本息提供保证担保
- **备用信用证**：银行出具Standby LC

### 9.5 计划说明书 vs 募集说明书差异

| 维度 | 计划说明书（ABS） | 募集说明书（ABN） |
|------|-----------------|-----------------|
| 产品名称 | 资产支持专项计划（SPP） | 资产支持票据信托/特定目的载体 |
| 监管体系 | 证监会/交易所 | NAFMII/银行间 |
| 发行载体 | 券商/基金子公司设立的专项计划 | 信托公司设立的SPT/SPV |
| 管理职责 | 计划管理人 | 主承销商+受托管理人 |
| 文件签署方 | 计划管理人+原始权益人+托管人 | 主承销商+发行人+受托人 |
| 投资者范围 | 合格投资者（非公开为主） | 银行间合格机构投资者 |

## 第10章：美元债券（境内主体境外发行）

### 10.1 144A/Reg S规则下的披露豁免

#### Rule 144A
- **私募配售**：仅向合格机构买方（QIB，Qualified Institutional Buyers）
- 较少的公开信息披露要求
- 不可在美国公开市场转售（转售限于QIB之间）
- **适用场景**：发行规模较大、需吸引美国机构投资者

#### Regulation S
- **离岸交易豁免**：发行和配售在美国境外进行
- 最简化的披露要求（信息披露主要依赖市场惯例，非美国SEC硬性规定）
- 仅在离岸市场交易（40天分销限制期内不得回流美国）
- **适用场景**：不追求美国投资人，主要面向亚洲/欧洲投资者

### 10.2 境内评级 vs 国际评级差异披露

| 维度 | 境内评级 | 国际评级（穆迪/标普/惠誉） |
|------|---------|--------------------------|
| 评级方法论 | 侧重政府支持+行业地位 | 侧重独立信用质量+国家上限 |
| 典型差异 | 中国企业境内评级通常高2-5个子级 | 受国家主权评级上限约束 |
| 披露要求 | 不强制披露国际评级 | 取决于发行方式 |
| 风险提示 | 须提示两种评级体系差异可能导致的误解 |

### 10.3 备用信用证（Standby LC）结构披露

- **结构安排**：银行向债券受托人出具Standby LC，承诺在发行人违约时向投资者兑付
- **常见开证行**：中资国有大行、股份制银行的境外分行
- **关键披露要素**：
  - 开证行名称及国际评级
  - Standby LC额度与本次发行本息的覆盖关系
  - 兑付条件及程序
  - Standby LC的法律适用及争议解决方式（通常为英国法/纽约法）
- **风险提示**：Standby LC并非直接保证，开证行可能因法律/合规原因拒绝兑付

### 10.4 外汇管理合规披露

- **发改委外债备案**：1年期以上须在发改委办理外债备案登记（《企业中长期外债审核登记管理办法》）
- **外管局登记**：在外管局办理外债签约登记
- **资金回流**：若募集资金调回境内使用，须遵守跨境资金管理规定
- **汇率风险**：发行人须提示人民币汇率波动对偿债成本的影响

## 第11章：申报文件交叉索引

### 11.1 审计报告、法律意见书、评级报告、担保合同之间的相互引用关系

#### 数据依赖DAG（有向无环图）
```
审计报告（近三年及一期）
  ├── 募集说明书 → 第七章"财务会计信息"
  ├── 评级报告 → "财务分析"章节
  └── 法律意见书 → "财务指标分析"（引述）

法律意见书
  ├── 募集说明书 → 第六章"发行人基本情况"（关联方/股权结构）
  ├── 募集说明书 → 第十一章"增信机制"（担保合法性合规意见）
  └── 受托管理协议 → 作为附件

信用评级报告
  ├── 募集说明书 → 第十章"发行人信用状况"
  └── 受托管理协议 → 评级下调触发条款引用

担保合同
  ├── 募集说明书 → 第十一章"增信机制"
  ├── 法律意见书 → 增信合规性意见
  └── 评级报告 → 增信效果分析
```

|-------|----------|---------|----------|---------|---------|
| 注册资本 | ✓ | — | ✓ | ✓ | — |
| 控股股东 | ✓ | — | ✓ | — | — |

| 对外担保额度 | ✓ | — | ✓ | ✓ | ✓ |
| 发行总额/期限/票面利率 | ✓ | — | ✓ | ✓ | ✓ |
| 担保人财务数据 | ✓ | — | — | ✓ | ✓ |
| 违约/逾期记录 | ✓ | — | ✓ | ✓ | — |

#### 数据一致性检查清单

1. **审计报告→募集说明书**：三年一期所有财务指标应完全对应，不得出现四舍五入差异
2. **法律意见书→募集说明书**：发行人基本情况描述（股权结构/历史沿革/重大诉讼）应一致
3. **评级报告→募集说明书**：信用等级、评级观点、财务分析结论应一致
4. **担保合同→募集说明书**：担保金额、担保方式、担保期限、登记情况应完全对应
5. **发行人内部决议→募集说明书**：发行规模、用途、增信安排应与董事会/股东会决议一致
