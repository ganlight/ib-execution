# 债券监管申报体系详解

> 最后更新：2026年5月 | 涵盖NAFMII、证监会/交易所、国家发改委三大监管系统

---

## 一、三大监管体系全景

```
中国信用债监管格局
│
├── 银行间市场（中国人民银行 + NAFMII）
│   │ 品种：MTN / CP / SCP / PPN / ABN / 金融债
│   │ 核心机制：注册制（注册会议评议）
│   │ 主承销商：商业银行为主
│   │ 投资者：银行间市场合格机构投资者
│   │
│   └── NAFMII注册体系层级
│       ├── 统一注册（DFI）：超级主体专用
│       ├── 分层注册（TDFI）：分两档
│       ├── 常规注册：普通企业
│       └── 定向注册：PPN
│
├── 交易所市场（证监会 + 沪深北交易所）
│   │ 品种：公募公司债 / 私募公司债 / ABS / 可转债 / 可交债
│   │ 核心机制：注册制（交易所审核+证监会注册）
│   │ 主承销商：证券公司
│   │ 投资者：合格机构投资者+公众（公募）
│   │
│   └── 证监会公司债注册流程
│       ├── 大公募：交易所审核→证监会注册
│       ├── 小公募：交易所审核（注册已下放）
│       └── 私募债：交易所挂牌条件确认+证券业协会事后备案
│
└── 发改委体系（国家发改委）
    │ 品种：企业债 / 项目收益债 / 专项企业债
    │ 核心机制：注册制（中债登预审+发改委注册）
    │ 主承销商：证券公司
    │ 投资者：银行间+交易所双向挂牌
    │
    └── 发改委企业债注册路径
        ├── 中央企业：直报国家发改委
        ├── 地方企业：省发改委转报→国家发改委
        └── 特殊品种：专项债绿色通道
```

### 三大体系核心差异速览

| 对比维度 | NAFMII体系 | 证监会/交易所体系 | 发改委体系 |
|---------|-----------|----------------|-----------|
| **审核哲学** | 注册制（市场自律为主） | 注册制（行政+自律结合） | 注册制（产业政策导向强） |
| **审核主体** | NAFMII注册会议 | 交易所审核+证监会注册 | 中债登/交易商协会预审+发改委注册 |
| **前置条件** | 无硬性盈利要求 | 近3年平均可分配利润覆盖利息 | 项目符合国家产业政策 |
| **承销商主导** | 商业银行 | 证券公司 | 证券公司 |
| **信息披露取向** | 市场化、标准化 | 投资者保护导向 | 产业合规导向 |
| **事后监管** | NAFMII + 交易商协会 | 证监会派出机构+交易所 | 发改委 + 省级发改委 |
| **周期弹性** | 最快（DFI 1个月） | 中等（2-4个月） | 较慢（3-6个月） |
| **品种灵活性** | 统一注册额度可多品种切换 | 按品种逐项申报 | 按项目申报 |

## 二、NAFMII体系详解

### 2.1 注册会议机制

NAFMII注册工作实行"注册会议"制度。注册会议由5名注册专家组成，由NAFMII从注册专家库中随机抽取。

**注册会议审议流程**：
```
材料受理 → 形式审查（5个工作日）→ 注册会议安排 → 
注册会议评议（投票表决）→ 出具注册结论（同意/有条件同意/推迟表决/不同意）

投票规则：≥4票同意 → "接受注册" | 3票同意 → "有条件接受" | ≤2票同意 → "推迟表决"
```

**专家回避规则**：注册专家与发行人、主承销商、会计师事务所、评级机构存在利害关系的，应当回避。

### 2.2 DFI / TDFI 分层管理体系

| 层级 | 全称 | 适用条件 | 注册额度 | 有效期 | 优势 |
|------|------|---------|---------|--------|------|
| **DFI** | 债务融资工具统一注册 | ①资产≥1000亿且AAA；或②中国500强；或③全球500强境内子公司 | 不设具体额度，按"待偿还余额"管理 | 2年 | 注册后MTN/CP/SCP/PPN等可任意品种切换发行，仅需发行前备案 |
| **TDFI-成熟层** | 第一类企业（成熟层） | AAA评级+资产≥500亿+市场认可度高 | 统一注册额度 | 2年 | 类似DFI但额度受限，品种切换灵活 |
| **TDFI-基础层** | 第二类企业（基础层） | 不符合成熟层条件的一般企业 | 单品种注册额度或统一额度（较小） | 2年 | 注册制基本流程 |

**分层转换**：基础层企业符合条件后可申请升级至成熟层。

### 2.3 注册额度与有效期管理

| 管理维度 | 规则内容 |
|---------|---------|
| **注册额度** | 由NAFMII根据企业净资产、信用评级、融资计划综合核定 |
| **有效期** | 注册通知书出具日起2年内有效 |
| **分次发行** | 注册有效期内可分期滚动发行，首期发行需在注册后6个月内完成 |
| **额度恢复** | 到期兑付后的债券本金可恢复为可用注册额度 |
| **重大事项** | 发行期间如发生重大事项变更，需重新提交注册或补充说明 |

### 2.4 发行前备案/公告要求

各品种发行前需在NAFMII进行备案或公告：

| 时间节点 | 事项 | 要求 |
|---------|------|------|
| T-5日以上 | 发行公告（含申购说明） | 通过NAFMII系统发布 |
| T-2日 | 发行方案备案/申购说明终稿 | 通过NAFMII系统提交 |
| T日 | 簿记建档/缴款 | 按申购说明操作 |
| T+1日 | 发行结果公告 | 公告发行利率、规模等 |
| T+2-5日 | 债务融资工具初始登记托管 | 银行间市场清算所（上清所） |

> **短期限品种简化**：SCP发行公告可于发行前1个工作日披露（较常规缩短1天）。

## 三、证监会/交易所公司债体系详解

### 3.1 审核权限变迁（2023-2026）

| 时间 | 关键变化 |
|------|---------|
| 2020年3月 | 新《证券法》实施，公司债全面实行注册制 |
| 2023年10月 | 企业债审核职能由发改委划转证监会（过渡期启动） |
| 2023-2025年 | 小公募注册权逐步下放沪深北交易所，证监会仅保留形式审查 |
| 2025年 | 公司类信用债在交易所端实现"审核即注册"（证监会不再逐单注册） |
| 2026年（预计） | 企业债全面纳入证监会体系，取消发改委注册通道 |

### 3.2 公募公司债审核流程

**小公募（现行流程）**：
```
申报材料提交 → 交易所受理（2个工作日）→ 交易所反馈问询（1-2轮）
→ 交易所出具审核意见 → 证监会注册 → 启动发行
```

**大公募（更为审慎）**：
```
申报材料提交 → 交易所受理 → 交易所反馈+审核 → 出具审核意见 
→ 报送证监会发行注册 → 证监会反馈（必要时）→ 予以注册 → 启动发行
```

> **关键差异**：大公募面向公众投资者，审核标准严于小公募，证监会注册环节可能发起补充问询。

### 3.3 审核的重点关注领域

| 关注领域 | 审核要点 | 可能触发补充披露的情况 |
|---------|---------|---------------------|
| **财务真实性** | 收入确认、关联交易定价、资产减值 | 与同行业可比公司关键财务指标偏差>30% |
| **偿债能力** | 现金流覆盖、再融资能力、资产质量 | 短期偿债指标显著弱于基准 |
| **资金用途** | 是否用于限制性领域、是否存在挪用风险 | 募集资金用于借新还旧占比>80%、无具体用途 |
| **信息一致性** | 各中介文件之间是否有矛盾 | 审计报告与评级报告数据不一致 |
| **负面清单** | 是否触犯地方政府债务管控红线（城投）、"三道红线"（房企） | 平台类企业发债需穿透核查是否新增政府隐性债务 |
| **公司治理** | 实控人情况、重大诉讼、股权质押 | 实控人质押>80%、近2年有重大行政处罚 |

### 3.4 私募公司债挂牌确认制度

私募公司债实行"交易所挂牌条件确认"制度，区别于公募的"证监会注册"：

```
发行人+主承销商 → 向交易所提交非公开发行申请材料 →
交易所出具"挂牌条件确认意见"（或无异议函）→ 
发行完成后5个工作日内在证券业协会备案
```

**审核周期**：交易所挂牌确认一般1-1.5个月，是审核最快的通道。

## 四、发改委企业债体系详解

### 4.1 中央企业申报路径

```
中央企业（含子公司）
  → 直接向国家发改委提交申请
  → 国家发改委委托中债登（中央结算公司）进行技术评估
  → 国家发改委财金司审核
  → 国家发改委出具注册通知书
```

**提交材料清单**：
- 债券发行注册申请报告
- 发行人有权机构（董事会/股东会）决议
- 募集说明书
- 财务审计报告（近3年及1期）
- 信用评级报告
- 法律意见书
- 募投项目批复文件（包括立项/环评/能评/用地等）
- 担保文件（如有）
- 承销协议

### 4.2 地方企业申报路径

```
地方企业
  → 向所在省（自治区、直辖市）发改委提交申请
  → 省级发改委出具转报意见
  → 国家发改委受理
  → 委托中债登/交易商协会进行技术评估
  → 国家发改委财金司审核
  → 国家发改委出具注册通知书
```

**省级转报要点**：
- 省级发改委需审查：①是否符合国家产业政策和区域发展规划；②募投项目是否已落实审批程序；③发行人偿债保障措施是否充分。
- 转报时间通常2-4周（取决于省级发改委效率）。

### 4.3 专项企业债与支持方向

| 专项品种 | 支持方向 | 申报优势 |
|---------|---------|---------|
| 城市停车场建设专项债 | 城市公共停车场 | 审核较快，可配建商业设施增收 |
| 地下综合管廊专项债 | 城市地下管网综合建设 | 可配一定比例补流 |
| 养老产业专项债 | 养老服务设施建设 | 中央预算内投资可贴息20%利息 |
| 战略性新兴产业专项债 | 新一代信息技术/新能源/新材料等 | 允许将创业投资基金出资纳入资金用途 |
| 小微企业增信集合债 | 通过银行转贷支持小微企业 | 设置了风险缓释基金要求 |
| 县城新型城镇化专项债 | 县城补短板强弱项 | 2024-2026重点支持方向 |

### 4.4 企业债审核的"跨市场"特殊环节

由于企业债可在银行间市场+交易所市场双向挂牌交易，审核中涉及：
1. **发行前**：发改委需征求人民银行意见（确保银行间市场承接能力）
2. **上市交易**：发行人可选择银行间+交易所双挂牌，或单边挂牌
3. **信息披露**：双市场要求同时满足，信息披露平台包括中债登和交易所

## 五、跨市场协调机制

### 5.1 跨市场发行与交易框架

```
跨市场债券（企业债、部分专项债）
  ┌── 发行端 ──┐
  │            │
银行间市场   交易所市场
（上清所托管）（中证登托管）
  │            │
  └── 交易端 ──┘
       │
  投资者可跨市场转托管
```

### 5.2 执法协调机制

| 机制 | 内容 |
|------|------|
| **统一执法备忘录** | 人民银行、证监会、发改委签署《关于进一步加强债券市场执法工作的意见》 |
| **信息共享** | 三大监管机构建立发行人重大事项共享通报机制 |
| **联合惩戒** | 跨市场违法违规行为由最先发现机构牵头查处，其他机构配合 |

### 5.3 主承销商跨市场执业

| 指标 | 银行间 | 交易所 | 双市执业要求 |
|------|--------|--------|------------|
| 牌照 | NAFMII主承销商资质 | 证监会券商牌照 | 券商需同时取得NAFMII会员资格 |
| 商业银行 | 主承（主导） | 不可担任公司债主承销商 | 仅可做银行间品种 |
| 证券公司 | 可做主承（但需申请资质） | 主承（主导） | 双市均可执业 |
