# 债券二级市场交易手册

## 一、二级市场分类与架构

### 1.1 三大市场对比

| 市场 | 主管部门 | 主要交易品种 | 参与主体 | 托管体系 |
|-----|---------|-----------|---------|---------|
| **银行间市场** | 人民银行、NAFMII | 国债/政金债/信用债/ABS/同业存单 | 银行、券商、保险、基金、农村金融机构 | 中债登（上清所） |
| **交易所市场** | 证监会、交易所 | 公司债、企业债、可转债、可交换债、公募REITs | 机构投资者（部分个人可投） | 中证登 |
| **商业银行柜台市场** | 人民银行、银保监会 | 储蓄国债、记账式国债 | 个人投资者和机构 | 承销银行 |

### 1.2 银行间市场详解

**核心特征**：
- 中国债券市场主体，占全市场成交量80%以上
- 采用**询价驱动**（做市商报价）为主、**点击成交**（点击报价）為輔的交易机制
- 实名制账户（CFMMS系统），中央对手方清算
- 主要交易品种期限覆盖1天~50年

**参与机构类型**：
| 机构类型 | 市场角色 | 交易偏好 |
|---------|---------|---------|
| 商业银行 | 最大的配置盘 | 国债、政金债、地方政府债 |
| 证券公司 | 交易盘+做市商 | 信用债、可转债 |
| 保险公司 | 长期配置盘 | 超长期国债、金融债 |
| 基金公司 | 交易盘为主 | 信用债、债券ETF |
| 农村金融机构 | 区域配置 | 当地城投债为主 |

### 1.3 交易所市场详解

**两个交易所对比**：

| 要素 | 上海证券交易所（上交所） | 深圳证券交易所（深交所） |
|-----|----------------------|----------------------|
| 成立时间 | 1990年 | 1990年 |
| 债券品种 | 公司债、企业债、可转债等 | 公司债、可交换债、公募REITs |
| 主要参与者 | 机构投资者为主 | 机构投资者+部分个人 |
| 交易机制 | 竞价撮合 + 做市商 | 竞价撮合 |
| 特色品种 | 绿色债券、扶贫债券 | 创新创业债 |

**交易所特有交易机制**：
- **竞价撮合**（自动撮合）：类似股票市场，按价格优先、时间优先原则成交
- **做市商制度**：部分品种实行双边报价，增强流动性
- **协议大宗交易**：机构间大额交易通过系统询价完成

### 1.4 托管体系

| 托管机构 | 托管品种 | 特点 |
|---------|---------|-----|
| **中债登**（中央国债登记结算） | 国债、政金债、商业银行债、信用债、ABS | 银行间市场主体托管机构 |
| **上清所**（上海清算所） | 同业存单、短融、超短融、中票、PPN | 银行间市场清算机构 |
| **中证登**（中国证券登记结算） | 交易所债券（公司债、企业债等） | 交易所市场统一托管 |

**跨市场托管**：部分债券（如企业债）在银行间和交易所同时上市，需注意转托管操作。

## 二、经纪业务与做市业务

### 2.1 做市商制度

**定义**：经监管批准，在银行间市场持续向投资者提供双边报价（买价和卖价），并按报价成交的机构。

**做市商权利与义务**：

| 要素 | 内容 |
|-----|------|
| **权利** | 享有债券借贷便利、交易手续费优惠、参与公开市场操作 |
| **义务** | 持续报价（在显著品种上）、回复询价时间<30分钟、最小报价量要求 |
| **考核** | NAFMII对做市商进行年度评价，末位淘汰 |

**做市商双边报价示例**（以21国开15为例）：

| 方向 | 报价 | 期限 | 到期日 |
|-----|------|------|-------|
| 买价（BID） | 2.8920% | 10年 | 2031-09-22 |
| 卖价（ASK） | 2.8880% | 10年 | 2031-09-22 |

BID-ASK Spread = 2.8920% - 2.8880% = 4bp（优质利率债利差较小）

### 2.2 经纪业务

**定义**：经纪商接受客户委托，撮合交易双方（不参与自营），收取佣金的业务模式。

**经纪 vs 做市对比**：

| 对比维度 | 做市商 | 经纪商 |
|---------|-------|-------|
| 角色 | 参与交易，自担风险 | 中介，不自担风险 |
| 报价 | 有义务主动报价 | 无报价义务 |
| 利润来源 | 买卖价差 | 佣金 |
| 适用范围 | 主要在银行间市场 | 交易所+银行间 |

### 2.3 债券借贷

**定义**：债券融入方将债券出质给融出方，融入方支付利息，约定未来归还同质债券的业务。

**典型用途**：
- **做空策略**：借入债券并卖出，等价格下跌后买回还券
- **交割履约**：解决临时性债券缺口（如逼空行情下的交割需求）
- **套利交易**：配合利率互换等衍生品进行利差交易

**债券借贷要素**：

| 要素 | 说明 |
|-----|------|
| 期限 | 1天~1年（最常见1~7天） |
| 费率 | 年化利率（通常低于债券票面利率） |
| 质押率 | 折算后通常90%~95% |
| 担保品 | 现金或利率债 |

### 2.4 质押式回购 vs 买断式回购

| 对比维度 | 质押式回购 | 买断式回购 |
|---------|-----------|-----------|
| 债券所有权 | 所有权不转移，质押冻结 | 所有权转移给对方 |
| 到期还券 | 质押解除，还本付息 | 买回同质债券 |
| 信用风险 | 逆回购方承担正回购方信用风险 | 双方均承担对手方信用风险 |
| 担保品管理 | 质押债券继续产生利息收益 | 买断期间债券利息归属受让方 |
| 监管资本占用 | 较低 | 较高（国际Basel III要求） |
| 交易主体 | 机构投资者 | 机构投资者（部分产品有限制） |
| 常见期限 | 1天（隔夜）、7天（GC007）、14天、1个月 | 1天~3个月为主 |

## 三、交易定价机制

### 3.1 收益率曲线体系

**国债收益率曲线**：无风险利率基准，反映市场对未来利率走势的预期

| 曲线类型 | 编制机构 | 用途 |
|---------|---------|-----|
| 中债国债收益率曲线 | 中债估值公司 | 全市场利率基准 |
| 中债政策性金融债收益率曲线 | 中债估值公司 | 政金债定价参考 |
| 上交所国债收益率曲线 | 上交所 | 交易所国债参考 |

**信用债收益率曲线**：在国债收益率曲线基础上加信用利差

**关键期限点**：
- 1年、3年、5年、7年、10年（主力期限）
- 10年以上：20年、30年（超长期限，保险机构配置为主）

### 3.2 中债估值定价逻辑

**中债估值**（CVT Value）是银行间市场最重要的参考价格，由中债估值公司每日发布。

**估值方法**：

| 方法 | 适用范围 |
|-----|---------|
| **报价类法** | 有活跃报价的债券，取最优买价和卖价的中间价 |
| **交易类法** | 有真实成交的债券，取当日加权平均成交价 |
| **曲线推算法** | 无直接报价/成交，通过对应期限收益率曲线推算 |
| **模型定价法** | 结构化债券（含权、浮动利率）使用二叉树/蒙特卡洛模型 |

**估值价格 ≠ 成交价格**：
- 估值是参考基准，实际成交可能偏离
- 偏离过大（>3%）通常有特殊原因（流动性折价、信用变化）

### 3.3 信用利差分析

**信用利差** = 信用债收益率 - 同期限国债收益率

**利差拆解**：

```
信用利差 = 实际无风险溢价 + 流动性溢价 + 信用风险溢价
                        ↓              ↓              ↓
                  （期限+避险）    （变现难度）    （违约风险）
```

**各评级信用利差参考区间**（以5年期为例）：

| 评级 | 利差区间（bp） | 市场状态 |
|-----|-------------|---------|
| AAA | 40~80bp | 优质主体，流动性好 |
| AAA- | 60~100bp | 大型国企 |
| AA+ | 80~130bp | 中等优质城投/产业债 |
| AA | 120~180bp | 主流城投区间 |
| AA- | 160~250bp | 弱资质主体 |
| A+及以下 | >250bp | 高收益债（HY） |

**利差驱动因素**：

| 因素 | 影响方向 | 说明 |
|-----|---------|-----|
| 宏观利率下行 | 利差收窄 | 货币宽松→配置需求增加 |
| 信用风险事件 | 利差走阔 | 违约→抛售→利差扩大 |
| 供给冲击 | 利差走阔 | 发行放量→流动性紧张 |
| 监管政策 | 结构性影响 | 城投融资政策收紧→城投利差走阔 |

## 四、流动性评估指标

### 4.1 换手率

**日换手率** = 当日成交量 ÷ 债券余额 × 100%

| 换手率水平 | 流动性评价 | 价格代表性 |
|---------|---------|-----------|
| >15%/日 | 极高（主力品种） | 价格发现功能强 |
| 5%~15%/日 | 较高 | 具有参考价值 |
| 1%~5%/日 | 一般 | 价格可能有失真 |
| <1%/日 | 低流动性 | 成交不连续，价格代表性差 |
| 无成交 | 僵死状态 | 只能参考估值 |

**月换手率**：
- 活跃债券：月换手率>100%（债券到期前可换手多次）
- 不活跃债券：月换手率<20%

### 4.2 买卖价差（BID/ASK Spread）

**绝对价差** = ASK价格 - BID价格
**相对价差** =（ASK - BID）/ 中间价 × 100bp

| 债券类型 | 相对价差参考 |
|---------|------------|
| 国债（10年） | 1~3bp |
| 政金债（10年） | 2~5bp |
| AAA信用债（5年） | 3~8bp |
| AA+城投债（5年） | 5~15bp |
| AA城投债（5年） | 10~25bp |
| AA-及以下 | 25bp以上 |

### 4.3 成交量排名与市场情绪

**重点关注**：
- 单日/周度成交量排名前20的活跃券（反映机构关注度）
- 成交量突然放大的债券（可能有机构调仓或风险事件）
- 连续多日无成交的债券（流动性枯竭信号）

## 五、债券久期与凸性实战

### 5.1 修正久期

**修正久期（Modified Duration）**：衡量债券价格对收益率变化的敏感度。

**计算公式**：

```
修正久期 = 麦考利久期 ÷（1 + 到期收益率/每年付息次数）
```

**实战含义**：
- 修正久期为5年：收益率上升1%，债券价格下跌约5%
- 修正久期为3年：收益率上升1%，债券价格下跌约3%

**不同债券的久期特征**：

| 债券类型 | 典型久期 | 风险特征 |
|---------|---------|---------|
| 短期融资券（1年以内） | <1年 | 利率风险极低 |
| 中期票据（3~5年） | 2.5~4.5年 | 中等利率风险 |
| 公司债（5~7年） | 4~6年 | 较高利率风险 |
| 超长期国债（30年） | 15~20年 | 利率风险极高 |

### 5.2 凸性对冲

**凸性（Convexity）**：债券价格-收益率曲线的弯曲程度。

**为什么需要凸性对冲**：
- 久期是线性近似，实际债券价格变化是非线性的
- 收益率大幅变动时，凸性效应显著
- 凸性为正的债券，在收益率大幅下行时涨幅超过久期预测

**对冲策略**：

| 策略 | 操作 | 适用场景 |
|-----|------|---------|
| 久期免疫 | 组合久期 = 目标久期 | 锁定收益，避免利率波动 |
| 子弹型策略 | 集中在某一期限 | 预期利率不变 |
| 哑铃型策略 | 同时持有短端+长端 | 兼顾流动性和收益 |
| 阶梯型策略 | 平均分布在各期限 | 适合保守配置 |

### 5.3 利率风险免疫策略

**免疫（Immunization）**：构造债券组合，使利率变化对组合价值的影响相互抵消。

**操作步骤**：
1. 确定负债久期（需要匹配的目标）
2. 选择或构建债券组合，使组合久期 = 负债久期
3. 定期再平衡（久期会随时间变化）

## 六、信用债波段交易

### 6.1 信用事件驱动交易

**信用事件类型与市场反应**：

| 信用事件 | 典型反应 | 交易机会 |
|---------|---------|---------|
| **评级下调** | 债券收益率跳升，利差扩大50~200bp | 降级前做空，降级后超跌买入 |
| **展望负面** | 收益率温和走阔20~50bp | 短线做空或观望 |
| **列入评级观察** | 市场开始抛售 | 预警信号，观察是否真的下调 |
| **负面舆情** | 个券收益率走阔，板块联动 | 板块内选择性买入优质主体 |
| **技术性违约** | 个券暴跌，板块恐慌 | 超跌后价值凸显 |
| **正式违约** | 违约主体债券价格跌至20~30元以下 | 垃圾债投资机会（高风险） |

### 6.2 评级下调交易时间轴

**典型路径**：

| 阶段 | 时间 | 市场信号 | 交易动作 |
|-----|------|---------|---------|
| 预警期 | 违约前3~6个月 | 列入评级观察（Watch Negative） | 减持，降低仓位 |
| 确认期 | 违约前1~3个月 | 评级下调1~2小节 | 清仓或做空 |
| 冲击期 | 违约前后1个月 | 利差急速走阔，流动性枯竭 | 观望，等待市场情绪稳定 |
| 出清期 | 违约后1~3个月 | 价格企稳，违约处置进展明确 | 价值投资者分批介入 |

### 6.3 城投债波段交易策略

**城投债的特殊性**：
- 信用分析依赖政府信仰，非纯财务分析
- 区域联动效应强（一个城投出事，同区域其他平台遭抛售）
- 银行配置盘对城投债有天然偏好（稳定持有为主）

**城投波段策略**：

| 策略 | 描述 | 风险 |
|-----|------|-----|
| 区域轮动 | 超配经济强省城投，低配弱省城投 | 政策风险（城投融资政策突变） |
| 等级利差收窄 | 买入AA城投，等待利差收窄至AA+水平 | 信用风险未改善 |
| 事件驱动 | 负面事件冲击导致超跌买入 | 事件进一步恶化的风险 |
| 久期波段 | 在利率下行周期拉长久期至7年 | 利率反转风险 |

## 七、固收+产品设计

### 7.1 产品类型与收益风险特征

| 产品类型 | 股票仓位 | 预期收益（年化） | 最大回撤 | 适合投资者 |
|---------|---------|----------------|---------|---------|
| **纯债基金** | 0% | 3%~5% | <3% | 保守型、机构资金 |
| **一级债基** | 0%（不可投二级） | 4%~7% | <5% | 稳健型 |
| **二级债基** | ≤20% | 5%~10% | <10% | 平衡型 |
| **可转债基金** | 至少60%可转债 | 8%~15% | <15% | 积极型 |
| **美元债基金（QDII）** | 海外债券 | 4%~8% | <10% | 全球化配置需求 |

### 7.2 二级债基设计要点

**资产配置框架**：

| 资产类别 | 配置比例 | 策略 |
|---------|---------|-----|
| 利率债 | 20%~40% | 久期管理，获取资本利得 |
| 信用债（AAA~AA+） | 40%~60% | 票息收益为主 |
| 可转债 | 0%~20% | 弹性收益来源 |
| 股票 | 0%~20% | 增强收益 |
| 衍生品（利率互换等） | 0%~10% | 对冲利率风险 |

**风险控制指标**：
- 最大回撤目标：≤8%
- 年化波动率：≤6%
- 久期控制：≤5年（纯债部分）
- 信用集中度：单一主体不超过净资产5%

### 7.3 城投债在固收+中的配置价值

**城投债的吸引力**：
- 票息相对较高（比同评级产业债高30~50bp）
- 违约率低（公开市场违约率<0.5%）
- 与国债利差稳定（相比产业债波动小）

**配置禁区**：
- 不允许配置AA-以下债券（内部评级规定）
- 不允许配置债务率>120%的区域城投（部分机构要求更严）
- 不允许配置有负面舆情的个券

## 八、案例：某AA+城投债一级发行到二级交易全流程

### 案例主体信息

- **债券名称**：21湖州城投MTN001
- **主体评级**：AA+（联合资信）
- **发行规模**：10亿元
- **期限**：5年
- **票面利率**：4.5%（2021年6月发行）
- **募集资金用途**：偿还银行借款 + 补充流动资金

### 全流程时间轴

| 时间节点 | 阶段 | 关键事件 | 市场环境 |
|---------|------|---------|---------|
| **2021年4月** | 启动阶段 | 主承（某头部券商）进场尽调，准备注册材料 | 城投融资政策宽松，市场情绪较好 |
| **2021年5月** | 注册阶段 | NAFMII接受注册，批文有效期2年 | 利率区间：4.0%~4.8%（投标区间） |
| **2021年6月15日** | 发行日 | 簿记建档：票面利率4.5%，全场倍数2.3倍 | 同日10年国债收益率3.12% |
| **2021年6月18日** | 上市日 | 银行间市场上市，当日中债估值99.8元 | 上市首日收益率4.52%，微幅破面 |
| **2021年Q3** | 二级交易初期 | 月均成交量约5000万元，换手率5%/月 | 7月城投融资边际收紧，收益率上行至4.7% |
| **2021年Q4** | 配置需求旺盛 | 银行理财净值化转型→配置盘加大 | 收益率回落至4.4%，成交量放大 |
| **2022年3月** | 负面冲击 | 审计署披露部分城投债务问题，市场情绪谨慎 | 收益率跳升至4.9%，成交量萎缩 |
| **2022年6月** | 恢复期 | 政策明确"保障城投合理融资需求"，情绪修复 | 收益率回落至4.5%，成交量恢复 |
| **2022年12月** | 估值调整 | 中债估值调整至100.2元（净值修复） | 附近AAA~AA+利差收窄 |
| **2023年全年** | 稳健交易 | 收益率在4.3%~4.8%区间波动 | 月均成交量稳定在8000万元左右 |
| **2024年** | 到期前一年 | 投资者开始关注偿付安排 | 收益率4.2%，接近同期限国债利差 |
| **2026年6月** | 到期兑付 | 按面值偿还本金10亿元 + 最后一期利息 | 债券正常到期履约 |

### 关键交易时点分析

**发行时点选择**：
- 发行人选择在利率相对高点（4.5%）发行，反映对后市的谨慎
- 2.3倍全场倍数说明市场需求尚可，但非火爆
- AA+城投相对同期限国债利差约138bp（4.5%-3.12%），处于当时正常区间

**二级交易阶段**：
- 主要买方：城商行资管产品（配置盘）+ 基金公司专户
- 主要卖方：券商自营（波段操作）
- 单笔交易规模：1000万~5000万为主

**收益率影响因素**：
- 2022年3月收益率跳升：市场担心城投债务风险→机构减持
- 2022年6月回落：政策信号明确→配置盘重新入场
- 2024年接近到期：信用风险担忧消退→收益率向利率债收敛
