# ABS 市场全景参考

## 一、ABS 市场三大体系概述

中国资产证券化市场经过十余年发展，已形成**信贷 ABS**、**企业 ABS** 和 **资产支持票据（ABN）** 三大监管体系并存的市场格局。三大体系在监管机构、审核方式、交易场所、基础资产类型等方面存在显著差异。

### 1.1 信贷资产证券化（信贷 ABS）

| 维度 | 详细说明 |
|------|----------|
| **监管机构** | 中国人民银行、国家金融监督管理总局（原银保监会） |
| **审核方式** | 注册制（向央行注册，银保监备案） |
| **交易场所** | 银行间债券市场 |
| **登记托管** | 中央国债登记结算有限责任公司（中债登） |
| **SPV形式** | 特定目的信托 |
| **发起机构** | 银行业金融机构（商业银行、政策性银行、金融租赁公司、汽车金融公司、消费金融公司等） |
| **核心法规** | 《信贷资产证券化试点管理办法》（中国人民银行公告〔2005〕第7号）、《关于信贷资产证券化备案登记工作流程的通知》（银监办便函〔2014〕1092号） |

**主要基础资产类型：**

| 基础资产类别 | 特征说明 | 代表性项目 |
|-------------|----------|-----------|
| 个人住房抵押贷款（RMBS） | 期限长（10-30年）、分散度高、违约率低 | 建元、工元系列 |
| 个人汽车抵押贷款（Auto ABS） | 期限短（3-5年）、现金流稳定、早偿可预测 | 福元、融腾系列 |
| 个人消费贷款 | 小额分散、期限短、利率较高 | 和享、橙益系列 |
| 企业贷款（CLO） | 大额集中、影子评级重要、结构化分层 | 元信系列 |
| 不良资产（NPL ABS） | 折扣率定价、回收率建模、清收时间预估 | 建鑫、工元至诚系列 |
| 微小企业贷款 | 分散度中等、政策鼓励方向 | 兴瑞、浦鑫系列 |
| 信用卡分期应收款 | 期限极短、循环购买结构常见 | 企鑫系列 |
| 金融租赁应收款 | 租赁物担保、期限中等（3-8年） | 交融系列 |

### 1.2 企业资产证券化（企业 ABS）

| 维度 | 详细说明 |
|------|----------|
| **监管机构** | 中国证券监督管理委员会（证监会） |
| **审核方式** | 交易所挂牌条件确认（或报价系统确认）+ 基金业协会备案 |
| **交易场所** | 上海证券交易所、深圳证券交易所、机构间私募产品报价与服务系统 |
| **登记托管** | 中国证券登记结算有限责任公司（中证登） |
| **SPV形式** | 资产支持专项计划 |
| **原始权益人** | 非金融企业（包括保理公司、融资租赁公司、房地产企业、互联网平台等） |
| **核心法规** | 《证券公司及基金管理公司子公司资产证券化业务管理规定》（证监会公告〔2014〕49号） |

**主要基础资产类型：**

| 基础资产类别 | 特征说明 | 代表性类型 |
|-------------|----------|-----------|
| 融资租赁应收款 | 规模体量大、底层分散、租赁物担保 | 远东租赁系列 |
| 应收账款/保理资产 | 账期短、循环购买、核心企业信用重要 | 万科供应链、京东保理 |
| 小额贷款/消费金融 | 高度分散、大数据风控、早偿率高 | 蚂蚁花呗/借呗、京东白条 |
| 商业地产抵押贷款（CMBS） | 商业物业运营收入抵押、单笔大额 | 金融街CMBS |
| 类REITs | 权益型不动产证券化、私募属性 | 保利租赁住房REITs |
| 基础设施收费权 | 现金流稳定可预测、特许经营属性 | 高速公路收费收益权 |
| 门票/收费收益权 | 原始权益人经营风险高、信用增级需求大 | 景区门票ABS |
| 信托受益权（双SPV） | 穿透底层资产认定、两层结构设计 | 信托受益权ABS |
| PPP项目资产 | 政府付费/可行性缺口补助、政策推动 | PPP项目ABS |
| 知识产权 | 许可费收入、估值难度大 | 电影票房、专利许可费ABS |

### 1.3 资产支持票据（ABN）

| 维度 | 详细说明 |
|------|----------|
| **监管机构** | 中国银行间市场交易商协会（NAFMII） |
| **审核方式** | 注册制（向交易商协会注册） |
| **交易场所** | 银行间债券市场 |
| **登记托管** | 银行间市场清算所股份有限公司（上清所） |
| **SPV形式** | 特定目的信托（信托型ABN）/ 特定目的载体（其他SPV） |
| **发起机构/发行人** | 非金融企业 |
| **核心法规** | 《银行间债券市场非金融企业资产支持票据指引》（交易商协会公告〔2017〕27号） |

**主要基础资产类型：**

| 基础资产类别 | 特征说明 | 市场占比特征 |
|-------------|----------|-------------|
| 融资租赁应收款 | 与交易所融资租赁ABS趋同 | 占比最高 |
| 应收账款 | 供应链金融、应收工程款等 | 稳定发展 |
| 基础设施收费权 | 与交易所类ABS相似 | 政策推动 |
| 房地产类CMBN/类REITs | 商业物业抵押 | 资产筛选要求严格 |
| 保理资产 | 反向保理供应链类 | 增长显著 |
| 小微贷款 | 银行系消金公司作为发行载体 | 创新方向 |

## 二、三大市场发行规模对比（2020-2025Q1）

| 年度 | 信贷 ABS（亿元） | 企业 ABS（亿元） | ABN（亿元） | 合计（亿元） | 增速 |
|------|:------------:|:------------:|:---------:|:----------:|:----:|
| 2020 | 8,041 | 15,587 | 5,109 | 28,737 | +23.1% |
| 2021 | 8,815 | 16,173 | 6,454 | 31,442 | +9.4% |
| 2022 | 4,567 | 12,087 | 5,776 | 22,430 | -28.7% |
| 2023 | 3,485 | 10,973 | 5,218 | 19,676 | -12.3% |
| 2024 | 2,703 | 10,834 | 5,927 | 19,464 | -1.1% |
| 2025Q1 | 548 | 2,410 | 1,313 | 4,271 | -（基期影响） |

**趋势分析：**
- 信贷 ABS 规模持续收缩，主因：银行信贷资产出表需求减弱、RMBS发行意愿降低、不良资产ABS成为主要增量来源
- 企业 ABS 相对平稳，融资租赁、供应链、消费金融三大类仍为核心支柱
- ABN 逆势增长，受益于部分融资租赁类项目由交易所转向银行间、ABN产品创新（如ABCP）
- 2022-2023年整体萎缩受多重因素影响：宏观经济下行导致企业融资需求减弱、房地产行业风险传导、监管收紧部分基础资产准入

## 三、参与机构体系

### 3.1 核心参与方角色与职责

| 参与机构类型 | 角色定位 | 核心职责 | 代表性机构（示例） |
|-------------|---------|---------|-----------------|
| **原始权益人/发起机构** | 资产出让方 | 转让基础资产、提供资产数据、履行资产服务职责 | 商业银行、融资租赁公司、消费金融公司、保理公司 |
| **计划管理人/受托机构** | 核心管理人 | 设立SPV、产品设计、文件编制、申报沟通、存续管理 | 中信证券、中金公司、招商证券、华泰证券、平安信托、中信信托、华润信托 |
| **托管银行** | 资金监管方 | 托管专项计划/信托财产、资金划付、对账单提供 | 工商银行、建设银行、招商银行、兴业银行 |
| **评级机构** | 信用评判方 | 基础资产信用质量评估、分层评级、跟踪评级 | 中诚信国际、联合资信/联合评级、大公国际、新世纪评级 |
| **律师事务所** | 法律合规方 | 交易文件审查、法律意见书、尽职调查、真实出售认定 | 金杜、中伦、君合、海问、竞天公诚 |
| **会计师事务所** | 财务审计方 | 基础资产执行商定程序、现金流模型复核、税务意见 | 普华永道、德勤、安永、毕马威、立信、天健 |
| **主承销商** | 销售执行方 | 路演定价、簿记建档、投资者沟通（信贷ABS和ABN适用） | 各大商业银行及券商 |
| **资产服务机构** | 资产运营方 | 基础资产回收管理、本息催收、逾期监测、定期报告 | 多数由原始权益人兼任 |
| **登记托管机构** | 基础设施方 | 登记确权、本息兑付、持有人名册管理 | 中债登（信贷ABS）、中证登（企业ABS）、上清所（ABN） |

### 3.2 中介机构选聘原则

1. **评级机构选聘考量：**
   - 需具有银行间债券市场/交易所市场评级业务资质
   - 对于首次开展某类基础资产ABS，建议选择在该领域经验丰富的机构
   - 需评估评级分析师团队规模与项目经验匹配度
   - 双评级要求：部分产品（如信贷ABS）及特定投资者要求双评级

2. **律所选聘考量：**
   - 需具备证券法律业务资格
   - 需核查律所及签字律师是否存在监管处罚记录
   - 对复杂结构（如双SPV、跨境资产）需考量相关领域经验

3. **会计师事务所选聘考量：**
   - 需具备证券期货相关业务资格
   - 现金流模型复核需具备精算或风险管理背景团队
   - 税务意见需具备资产证券化税务处理经验

## 四、2023-2025 监管政策趋势

### 4.1 制度框架完善

| 政策/法规 | 发布时间 | 核心内容 | 影响分析 |
|----------|---------|---------|---------|
| 《关于规范信托公司信托业务分类的通知》（银保监规〔2023〕1号） | 2023.03 | 明确资产证券化服务信托作为信托业务独立分类 | 规范信托公司在ABS中的角色，推动标准化发展 |
| 沪深交易所全面注册制配套规则修订 | 2023年 | 企业ABS挂牌条件确认审核流程优化 | 审核周期缩短、信息披露要求更细 |
| 《银行间债券市场企业资产证券化业务规则》 | 2023年修订 | ABN注册发行信息披露标准化 | 统一ABN市场披露标准 |
| 金融监管总局《关于促进消费金融公司和汽车金融公司发展的通知》 | 2024年 | 鼓励消金/汽金通过ABS拓宽融资渠道 | 利好消费信贷及汽车金融类ABS发行 |
| 《关于推进基础设施领域不动产投资信托基金（REITs）常态化发行的通知》 | 2024年 | 推动公募REITs与类REITs/ABS协调衔接 | 基础设施类ABS与公募REITs形成递进通道 |
| 《关于金融支持房地产平稳健康发展的通知》及配套 | 2024年 | 支持优质房地产企业发行ABS融资 | 房地产供应链ABS恢复性增长 |
| 银行间市场交易商协会《ABCP试点扩容方案》 | 2024年 | ABCP（资产支持商业票据）试点范围扩大 | 短期滚动发行工具创新，降低企业发行成本 |
| 《非银行金融机构行政许可事项实施办法》修订 | 2025年 | 简化金融租赁公司、消费金融公司ABS发行备案流程 | 降低中小金融机构发行门槛 |

### 4.2 监管方向性趋势

1. **审慎监管理念强化：**
   - 穿透式监管：要求穿透至底层资产，核查基础资产真实性
   - 实质重于形式：防范以ABS名义进行监管套利
   - 信息披露趋严：资产池逐笔信息透明度要求提升

2. **市场分层管理：**
   - 合格投资者制度完善：ABS优先档与次级档投资者适当性管理
   - 储架发行制度优化：优质发行人获得更多便利
   - 负面清单管理：限制特定类型基础资产（如依赖单一主体信用的债权类资产）

3. **风险防控重点：**
   - 原始权益人信用风险隔离：防止资产服务机构违约对资产池的影响
   - 循环购买资产质量管控：购买标准与持续购买期监控
   - 流动性风险管理：现金储备、流动性支持安排
   - 房地产行业风险：房地产类ABS穿透核查底层资产质量

4. **创新方向引导：**
   - 绿色ABS：绿色债券标准（2023版）与ABS结合，鼓励绿色资产入池
   - 普惠金融ABS：小微贷款、涉农资产证券化政策支持
   - 知识产权ABS：制度创新助力科技型企业融资
   - 基础设施REITs与ABS协同：推动不动产证券化体系完善
   - ABCP常态化发展：短期循环型证券化工具的推广

### 4.3 投资者结构变化

| 投资者类型 | 信贷 ABS | 企业 ABS | ABN | 趋势 |
|-----------|:------:|:------:|:---:|------|
| 商业银行 | ★★★★★ | ★★★ | ★★★★ | 信贷ABS主力；对企业ABS/ABN配置增加 |
| 银行理财 | ★★ | ★★★★ | ★★★ | 资管新规后配置结构调整中 |
| 公募基金 | ★★ | ★★★ | ★★ | 对优先级配置需求稳定 |
| 券商资管 | ★★ | ★★ | ★★ | 定制化配置需求 |
| 保险机构 | ★★★ | ★★ | ★★ | 期限匹配需求驱动配置 |
| 信托公司 | ★ | ★★ | ★ | 自营投资规模有限 |
| 财务公司 | ★ | ★★ | ★ | 在特定期限段配置 |
| 境外投资者 | ★★ | ★ | ★★ | 通过CIBM/Bond Connect参与 |

## 五、三大体系核心差异对比

| 对比维度 | 信贷 ABS | 企业 ABS | ABN |
|----------|----------|----------|-----|
| **上位法** | 《信托法》+ 央行/金监局规定 | 《证券法》+ 证监会规定 | 《信托法》+ 交易商协会自律规则 |
| **SPV实体** | 特定目的信托 | 资产支持专项计划 | 特定目的信托（为主） |
| **法律主体地位** | 信托财产（无独立法人资格） | 专项计划（无独立法人资格） | 信托财产（无独立法人资格） |
| **发行方式** | 公募 | 私募（面向合格投资者） | 公募（银行间市场定向/公开均可） |
| **审核核心机制** | 注册制（央行）+ 备案制（金监局） | 交易所确认 + 基金业协会备案 | 注册制（交易商协会） |
| **审批周期参考** | 3-6个月 | 2-4个月 | 2-4个月 |
| **信息披露平台** | 中国货币网、中债信息网 | 证券交易所网站、基金业协会 | 中国货币网、交易商协会 |
| **持有人会议** | 合同约定 | 计划说明书约定 | 合同约定 |
| **风险自留要求** | ≥5%（垂直/水平/L型任选） | 无强制要求（市场化选择） | 无强制要求（市场化选择） |
| **发行载体管理机构准入** | 金融机构（信托公司） | 证券公司/基金子公司 | 信托公司等 |

## 六、ABS 市场地理分布与区域特征

| 区域 | 信贷ABS发起机构分布 | 企业ABS原始权益人分布 | 特征说明 |
|------|-------------------|---------------------|---------|
| 华东（沪苏浙） | 股份制银行、消费金融公司密集 | 融资租赁、保理、互联网平台集中 | ABS发行最活跃区域 |
| 华北（京津冀） | 国有大行总行集中 | 央企供应链、基建类ABS | 政策型/基础设施类ABS集中 |
| 华南（粤闽琼） | 区域性银行 | 房地产供应链、消费金融 | 房地产类占比较高 |
| 中西部 | 区域性银行 | 基础设施收费收益权类 | 基建类ABS特色明显 |
| 东北 | 较少 | 较少 | ABS市场参与度较低 |

## 七、2025年度 ABS 市场展望

1. **总量企稳：** 预计2025年全市场发行规模1.9-2.1万亿元，与2024年基本持平
2. **结构分化：** 信贷ABS继续小幅下行（RMBS恢复不确定性），企业ABS平稳，ABN增长趋势延续
3. **政策驱动：** 基础设施类、绿色类、普惠金融类ABS获得更多政策支持
4. **科技赋能：** 大数据+区块链技术在基础资产筛选与存续期监控中的应用深化
5. **投资者扩容：** 保险机构、银行理财的ABS配置比例有望提升
6. **国际化：** 跨境ABS产品、熊猫ABS、自贸区ABS等创新品种探索
