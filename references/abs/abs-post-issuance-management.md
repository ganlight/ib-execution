# ABS 存续期管理参考

## 一、存续期管理体系总览

ABS 存续期管理是从证券发行完成（起息日）至全部证券兑付完毕（终止日）期间的全生命周期管理活动。不同于股票和普通债券，ABS 的存续期管理横跨 **资产服务监督、信息披露、信用风险监测、触发事件管理、持有人会议、评级跟踪、服务商管理** 等七大领域，管理复杂度远超传统债券。

### 存续期管理核心原则

| 原则 | 说明 |
|------|------|
| **持续独立性** | 计划管理人/受托机构必须持续独立履职，不能在存续期沦为"橡皮图章" |
| **主动监测** | 不能"买入持有睡大觉"，必须建立主动监测和预警机制 |
| **信息披露完整性** | 披露的及时性、准确性、完整性缺一不可 |
| **持有人利益至上** | 所有存续期决策以维护持有人合法权益为首要目标 |
| **程序合规** | 重大事项严格按交易文件约定的审议和决策程序执行 |

## 二、信息披露体系

### 2.1 定期信息披露要求

#### 年度报告

| 披露内容 | 报告主体 | 时间要求 | 核心内容 |
|---------|---------|---------|---------|
| 年度资产管理报告 | 计划管理人/受托机构 | 每年 4月30日前 | SPV 运营状况、资产池表现、现金流分配、中介机构履职 |
| 年度资产服务机构报告 | 资产服务机构 | 每年 4月30日前 | 基础资产回收管理、催收行动、逾期/违约统计 |
| 年度托管报告 | 托管银行 | 每年 4月30日前 | 专项计划/信托账户资金收支、余额核对 |
| 年度审计报告 | 会计师事务所 | 每年 4月30日前 | 对管理人、服务机构的履职情况进行审计 |
| 年度跟踪评级报告 | 评级机构 | 每年 6月30日前（信贷ABS）/ 每年 4月30日前（企业ABS） | 信用等级更新、资产池信用质量变化分析 |

#### 半年度报告

| 披露内容 | 报告主体 | 时间要求 |
|---------|---------|---------|
| 半年度资产管理报告 | 计划管理人 | 上半年结束之日起2个月内 |
| 半年度托管报告 | 托管银行 | 随管理人半年度报告同步 |

#### 季度报告

| 披露内容 | 报告主体 | 时间要求 |
|---------|---------|---------|
| 季度资产管理报告 | 计划管理人 | 每季度结束后1个月内（部分产品要求） |

### 2.2 临时信息披露——重大事项公告

发生以下重大事项时，管理人须在事项发生后 **2个交易日内（企业ABS）/ 按合同约定期限内（信贷ABS/ABN）** 向持有人公告：

| 类别 | 重大事项 | 报告要求 |
|------|---------|---------|
| **参与机构变动** | 原始权益人/计划管理人/托管银行/资产服务机构/评级机构发生变更 | 即时公告 |
| **信用事件** | 基础资产发生重大违约（单一债务人占资产池总额3%以上的违约） | 即时公告 |
| | 累计违约率达到预设预警阈值 | 即时公告 + 持续跟踪 |
| | 优先级证券评级下调 | 即时公告 |
| **触发事件** | 加速清偿事件/违约事件发生 | 即时公告 |
| | 信用触发机制被激活 | 即时公告 |
| **法律/合规事件** | 专项计划/信托涉及重大诉讼、仲裁 | 即时公告 |
| | 原始权益人收到重大行政处罚 | 即时公告 |
| | 交易文件发生重大修改 | 公告修改内容 |
| **财务/经营风险** | 原始权益人丧失清偿能力、破产、解散 | 即时公告 |
| | 差额支付承诺人/担保人参级下调至约定标准以下 | 即时公告 |
| | 资产服务机构持续经营能力存在重大不确定性 | 即时公告 |
| **持有人权益** | 持有人会议召开 | 至少提前10个交易日公告 |
| | 持有人会议决议 | 决议通过后2个交易日内公告 |
| | 收益分配 | 分配前公告分配方案 |
| **其他** | 其他可能对证券本息偿付产生重大影响的事项 | 即时公告 |

## 三、资产服务机构报告审核要点

### 3.1 服务机构报告核心审核维度

| 审核维度 | 核查要点 | 警示信号 |
|---------|---------|---------|
| **回收数据真实性** | 实际回收金额与历史趋势是否吻合 | 回收率突然大幅改善或恶化，无合理商业解释 |
| **逾期统计口径** | 逾期天数计算是否准确，是否与合同约定一致 | 将逾期重新分类为"展期"或"重组"以美化逾期率 |
| **违约认定标准** | 违约认定是否严格按交易文件约定标准执行 | 无理由推迟违约认定、变相掩盖不良资产 |
| **早偿统计** | 早偿数据是否完整准确 | 早偿数据的统计偏差导致现金流预测失真 |
| **催收记录** | 催收行动是否有记录、是否及时 | 催收行动发生在逾期后超过60天才启动 |
| **服务费计提** | 服务费计提是否与合同约定费率一致 | 服务费率异常调整、附加费用无合同依据 |

### 3.2 服务机构服务质量指标（Service KPIs）

| 指标 | 计算方式 | 合格范围 | 预警范围 |
|------|---------|:------:|:------:|
| 回款及时率 | 期内按时收回金额/应收金额 | ≥ 95% | < 90% |
| 逾期催收覆盖率 | 已启动催收的逾期笔数/全部逾期笔数 | ≥ 98% | < 95% |
| 服务报告及时率 | 按时提交服务报告的期数/全部期数 | 100% | < 100% |
| 信息差错率 | 错报漏报笔数/总笔数 | < 0.5% | > 1% |
| 投诉占比 | 有理由投诉笔数/总服务笔数 | < 1% | > 3% |

### 3.3 服务机构定期现场检查清单

| 检查领域 | 具体检查事项 | 频率 |
|---------|-------------|:--:|
| IT系统 | 核心业务系统数据完整性、安全性、容灾备份 | 年度 |
| 催收流程 | 催收策略执行情况、外包催收机构管理 | 半年度 |
| 档案管理 | 基础资产合同、担保文件保管完整性 | 年度 |
| 资金管理 | 收款账户与自有账户的资金隔离 | 季度 |
| 人员配置 | 服务团队人员稳定性、专业技能 | 半年度 |
| 内控审计 | 服务机构内部审计是否覆盖ABS服务业务 | 年度 |

## 四、信用触发事件监测体系

| | 早偿率（CPR/PSA） | 月度 | 服务机构月报 |
| | 回收率（累计回收/累计违约本金） | 季度 | 服务机构季报 |

| | 差额支付承诺人/担保人评级 | 按评级机构跟踪周期 | 评级报告/公开信息 |
| | 资产服务机构评级 | 按评级机构跟踪周期 | 评级报告/公开信息 |
| | 托管银行评级 | 按评级机构跟踪周期 | 评级报告/公开信息 |

| | 服务机构违法违规记录 | 月度 | 公开信息监控 |
| | 基础资产涉诉情况 | 月度 | 债务人涉诉信息排查 |

### 4.2 触发事件的响应机制

| 触发级别 | 响应措施 | 动作时间要求 |
|:------:|---------|:----------:|
| **黄色预警** | 加强监测频率，向持有人发送预警提示函 | 知悉后5个工作日内 |
| | 约谈相关参与机构，要求其提供情况说明和整改计划 | 知悉后10个工作日内 |
| **橙色警报** | 召开持有人会议审议应对方案 | 知悉后20个工作日内 |
| | 执行交易文件约定的橙色级别应对措施（如暂停次级收益分配、启动服务商更换程序） | 按持有人会议决议期限 |
| | 聘请独立顾问（如法律顾问、专项审计） | 按持有人会议决议期限 |
| **红色警报** | 全面启动加速清偿程序 | 触发条件满足时立即启动 |
| | 对违约方提起法律诉讼/仲裁 | 按法律程序启动 |
| | 向监管机构报告 | 知悉后2个工作日内 |

## 五、持有人会议

### 5.1 持有人会议召集条件

| 序号 | 召集条件 | 发起方 |
|:--:|---------|-------|
| 1 | 拟变更计划管理人/受托机构 | 管理人/受托机构 |
| 2 | 拟更换资产服务机构 | 管理人/受托机构 |
| 3 | 拟修改交易文件中对持有人权益有重大影响的条款 | 管理人/受托机构 |
| 4 | 发生加速清偿事件 | 管理人/受托机构 |
| 5 | 拟变更信用增级方式或增信提供方 | 管理人/受托机构 |
| 6 | 拟变更优先级证券的偿付安排 | 管理人/受托机构 |
| 7 | 单独或合计持有优先级证券未偿本金余额10%以上的持有人提议 | 持有人 |
| 8 | 差额支付承诺人/担保人发生信用等级重大调整 | 管理人/受托机构 |
| 9 | 专项计划/信托提前终止 | 管理人/受托机构 |
| 10 | 交易文件约定的其他召集情形 | 按约定 |

### 5.2 持有人会议议事程序

| 程序步骤 | 时间/比例要求 | 说明 |
|---------|:-----------:|------|
| 1. 提议召集 | — | 满足召集条件的提议方书面提出 |
| 2. 发布会议通知 | 会议召开日至少提前10个交易日 | 公告会议时间、地点、议程、议案 |
| 3. 债权登记日 | 会议通知中确定 | 登记日持有证券的持有人享有表决权 |
| 4. 会议召开 | 按通知安排 | 可采取现场、通讯或线上方式 |
| 5. 表决与决议 | 须经持有有表决权证券未偿本金余额50%以上的持有人同意 | 特别事项（变更偿付安排/增信方式）可能需要更高比例 |

### 5.3 有控制权的资产支持证券持有人

在企业 ABS 中，优先级与次级持有人对某些事项具有不同的表决权重：

| 事项类型 | 控制权归属 | 表决规则 |
|---------|:--------:|---------|
| 变更管理人/托管银行 | 全体（优先级+次级） | 各档分别表决或联合表决（按约定） |
| 变更偿付安排 | 优先级持有人单独 | 仅优先级持有人表决，次级无表决权 |
| 启动加速清偿 | 优先级持有人单独 | 仅优先级持有人表决 |
| 放弃追究违约责任 | 全体 | 各档分别表决 |
| 提前终止专项计划 | 全体 | 须各档持有人一致同意（或约定高门槛） |

## 六、评级跟踪与下调应对

### 6.1 评级跟踪流程

| 阶段 | 时间 | 评级机构工作内容 | 管理人配合事项 |
|------|------|---------------|-------------|
| 定期跟踪 | 每年度1次 | 审核最新资产池数据，更新信用分析模型，出具跟踪评级报告 | 提供每期服务机构报告、资产池逐笔数据 |
| 不定期跟踪 | 重大事项发生后 | 评估事项对评级的影响，决定是否调整评级 | 2个工作日内通知评级机构 |
| 评级展望调整 | 持续监测 | 给予正面/稳定/负面展望 | 关注评级展望变化并评估影响 |

### 6.2 评级下调触发条件

| 触发条件 | 对应措施 |
|---------|---------|
| 优先级评级下调2个子级以上 | 召开持有人会议通报，评估是否需要采取保护措施 |
| 优先级评级下调至A+以下 | 启动加速清偿程序（如交易文件约定） |
| 优先级评级下调至BBB-以下 | 触发管理人或托管银行更换程序（如交易文件约定） |
| 服务商/管理机构评级下调至约定下限以下 | 在约定期限内更换 |
| 担保人/增信方评级下调 | 评级下调至低于目标评级时，须补充增信或提供替代方案 |

### 6.3 评级下调应对预案

| 应对措施 | 适用条件 | 操作流程 |
|---------|---------|---------|
| **补充信息披露** | 评级下调的触发因素可解释、可逆转 | 向持有人详细说明下调原因和影响分析 |
| **增信补偿** | 因增信提供方评级下调导致 | 要求增信方提供额外增信或引入替代增信 |
| **加速清偿** | 优先级评级跌至投资级以下 | 按交易文件约定启动加速清偿程序 |
| **合成风险转移** | 存在可行的对冲工具 | 通过信用风险缓释工具转移部分风险（中资ABS市场仍未普及） |
| **寻求债券持有人指令** | 需要持有人决策的重大不确定性 | 召开持有人会议，交由持有人决策后续行动 |

## 七、清算流程与清算报告

### 7.1 清算启动条件

| 条件类型 | 具体条件 | 发起方 |
|---------|---------|-------|
| **正常到期** | 优先级证券本金全部偿付完毕、信托/专项计划期限届满 | 管理人主动启动 |
| **提前终止** | 交易文件约定的提前终止事由发生 | 管理人或按持有人会议决议 |
| **加速清偿完毕** | 加速清偿后全部优先级本息偿付完毕 | 管理人主动启动 |
| **清仓回购** | 发起机构行使清仓回购权 | 发起机构发起 |
| **强制清算** | 法院裁定或监管要求 | 法院/监管机构 |

### 7.2 清算工作流程

| 步骤 | 工作内容 | 负责方 | 时间要求 |
|:--:|---------|-------|:------:|
| 1 | 确定清算基准日 | 管理人 | 终止条件满足后确定 |
| 2 | 组建清算组 | 管理人 + 托管银行 + 外部顾问（如需要） | 5个工作日内 |
| 3 | 资产清查与盘点 | 核实资产池剩余资产、账户余额 | 清算组 | 按资产规模复杂程度 |
| 4 | 剩余资产处置 | 出售、清收或按份额分配 | 清算组 | 按处置难度 |
| 5 | 编制清算方案 | 确定分配顺序和方案 | 清算组 | 处置完成后 |
| 6 | 持有人会议审议（如需要） | 审议清算方案 | 持有人会议 | 按会议程序 |
| 7 | 执行清算分配 | 按清算顺序向各方分配 | 清算组 | 方案通过后 |
| 8 | 出具清算报告 | 清算过程、结果、分配明细 | 清算组 | 清算分配完成后 |
| 9 | 注销登记 | 中债登/中证登注销证券登记 | 管理人 | 清算报告出具后 |
| 10 | 账户注销及归档 | 注销托管账户、所有文件归档 | 管理人 + 托管银行 | 证券注销后 |

### 7.3 清算报告必备内容

| 序号 | 必备内容 | 说明 |
|:--:|---------|------|
| 1 | 清算基准日和清算依据 | 说明启动清算的法律/合同依据 |
| 2 | 资产池清算基准日状况 | 剩余基础资产清单、价值评估 |
| 3 | 清算期间收支明细 | 清算期间的资产回收、费用支出 |
| 4 | 清算分配方案 | 按瀑布顺序的分配详情 |
| 5 | 各档证券清偿情况 | 优先级、中间级、次级的所得分配 |
| 6 | 清算结论 | 是否全额清偿、如有亏损的承担方 |
| 7 | 无未决诉讼/仲裁声明 | 确认不存在可能影响清算结果的未决争议 |

## 八、服务商更换

### 8.1 服务商更换触发条件

| 序号 | 触发条件 | 类型 |
|:--:|---------|:--:|
| 1 | 资产服务机构丧失持续经营能力（破产、解散、被接管） | 强制性 |
| 2 | 资产服务机构被主管机关吊销业务资质 | 强制性 |
| 3 | 资产服务机构严重违反服务合同（如挪用资金、篡改数据） | 强制性 |
| 4 | 资产服务机构评级下调至约定不可接受水平 | 交易文件约定 |
| 5 | 持有人会议决议更换服务商 | 持有人决议 |
| 6 | 资产服务机构连续多期违约通知（服务质量严重不达标） | 交易文件约定 |

### 8.2 服务商更换流程

| 步骤 | 工作内容 | 时间要求 |
|:--:|---------|:------:|
| 1 | 原服务商发出解任通知 | 触发条件满足后立即发出 |
| 2 | 管理人/受托机构启动备用服务商或公开选聘 | 解任通知发出后5个工作日内 |
| 3 | 与新服务商签署服务合同 | 选聘完成后签署 |
| 4 | 资产数据和档案移交 | 签署合同后30天内完成（合同可约定更短期限） |
| 5 | 收款账户变更/系统对接 | 移交同时进行 |
| 6 | 持有人公告/监管报告 | 更换完成后2个工作日内公告 |

### 8.3 备用服务机构安排

| 安排方式 | 特点 | 成本与实操性 |
|---------|------|------------|
| **事先约定备用服务商** | 发行时即签订备用服务协议，确定触发条件和方法 | 最佳方案，但增加前期安排成本 |
| **公开选聘** | 解任后通过公开方式选聘 | 耗时长、不确定性高 |
| **发起机构/母公司承接** | 由发起机构或母公司作为过渡服务商 | 过渡性安排，存在利益冲突疑虑 |
| **主承销商/管理人过渡接手** | 管理人（券商/信托）暂时接管服务职能 | 短期应急方案，非长久之计 |

## 十、存续期管理常见风险与缓释措施

| 风险类型 | 风险表现 | 缓释措施 |
|---------|---------|---------|
| **服务机构"道德风险"** | 服务机构因与原始权益人为同一主体，对资产池表现恶化的信息选择性披露 | 独立审计、多源交叉核验数据、管理人独立分析 |
| **"僵尸管理"** | 管理人缺乏主动监测意识，仅在定期报告到期前集中处理 | 建立月度监测机制、专人负责存续期管理 |
| **持有人"搭便车"** | 持有人分散、无人愿意主动承担监督成本 | 管理人作为受托人必须主动履职，不能依赖持有人监督 |
| **评级滞后** | 评级机构跟踪评级不及时，评级调整晚于市场反映 | 管理人须在评级机构之前识别出潜在降级风险 |
| **诉讼时效** | 对债务人、担保人的追索超过诉讼时效而丧失胜诉权 | 建立诉讼时效台账，提前采取时效中断措施 |
| **文档管理疏漏** | 重要交易文件、催收记录、服务机构报告缺失或保管不善 | 建立电子+纸质双备份档案管理体系 |
