# ABS 信托/SPV 结构设计参考

## 一、SPV 基本法律框架

特殊目的载体（Special Purpose Vehicle, SPV）是资产证券化的核心法律工具，其制度价值在于将基础资产与原始权益人的信用风险隔离开来，实现"资产信用"对"主体信用"的替代。中国法律体系下，SPV 的法律形态因监管体系不同而存在差异，但其核心功能一致：**持有基础资产、发行资产支持证券、隔离破产风险**。

### 1.1 三大 SPV 形态对照

| 维度 | 信贷 ABS（特定目的信托） | 企业 ABS（资产支持专项计划） | ABN（特定目的信托） |
|------|----------------------|--------------------------|-------------------|
| 法律依据 | 《信托法》第2条、第15-16条 | 证监会规定（证监会公告〔2014〕49号） | 《信托法》+ 交易商协会指引 |
| 法律定性 | 信托法律关系 | 委托代理+合同约定关系（非信托） | 信托法律关系 |
| 财产独立性 | 信托财产独立于委托人、受托人、受益人 | 计划财产独立于管理人、原始权益人、投资者 | 同信贷ABS信托 |
| 载体监管 | 受托机构（信托公司）须持牌 | 管理人（证券公司/基金子公司）须持牌 | 受托机构（信托公司）须持牌 |
| 发行载体 | 信托受益权分层结构化 | 专项计划份额 | 信托受益权分层结构化 |
| 是否具有诉讼主体资格 | 受托机构代为行使 | 管理人代为行使 | 受托机构代为行使 |

## 二、信贷 ABS 信托结构

### 2.1 委托人 → 受托人 → 受益人基本模型

```
┌──────────────┐    资产转让合同    ┌──────────────┐    发行证券    ┌──────────────┐
│  发起机构    │ ────────────────→ │  受托机构    │ ──────────→ │  投资者      │
│ （委托人）   │ ←──────────────── │ （信托公司）  │ ←────────── │ （受益人）   │
└──────┬───────┘    支付转让对价    └──────┬───────┘    认购资金    └──────────────┘
       │                                  │
       │ 兼任资产服务机构                  │
       │ (收取本息、催收、管理)             │
       ↓                                  ↓
┌──────────────┐                   ┌──────────────┐
│  借款人/     │                   │  资金保管机构 │
│  债务人      │                   │ （托管银行）  │
└──────────────┘                   └──────────────┘
```

### 2.2 关键交易文件体系

| 序号 | 文件名称 | 签署方 | 核心内容 |
|:--:|---------|-------|---------|
| 1 | **信托合同** | 发起机构与受托机构 | 信托设立、信托财产范围、信托目的、分配顺序 |
| 2 | **资产转让协议** | 发起机构与受托机构 | 资产交割方式、转让通知、回购/赎回条款 |
| 3 | **服务合同** | 受托机构与贷款服务机构 | 服务机构职责、服务报酬、服务报告 |
| 4 | **资金保管合同** | 受托机构与资金保管机构 | 账户开立、资金收付、对账 |
| 5 | **承销协议** | 发起机构（或受托机构）与主承销商 | 承销方式、承销费率、簿记建档 |
| 6 | **评级合同** | 受托机构与评级机构 | 首次评级、跟踪评级、评级费用 |
| 7 | **法律意见书** | 发行人律师出具 | 信托有效性、资产转让真实性、合规性 |
| 8 | **会计意见书** | 会计师出具 | 资产出表认定、终止确认判断 |

### 2.3 信托财产交付与风险转移时点

| 交付阶段 | 法律效果 | 操作要点 |
|---------|---------|---------|
| **信托合同签署** | 信托设立（具备法律约束力） | 合同生效日自各方签章之日 |
| **信托财产交付日（Cut-off Date/初始起算日）** | 资产池截止计算时点 | 该日之后的资产回收归属于信托财产 |
| **信托生效日（Closing Date/交割日）** | 资产权利正式转移至信托 | 取得转让对价、完成交割手续 |
| **信托财产交付公告** | 向债务人公示资产转让事实 | 可选择在全国性报纸或债务人逐一通知 |

## 三、企业 ABS 专项计划结构

### 3.1 资产支持专项计划基本结构

资产支持专项计划是证监会体系下的特有概念，本质上是一种**合同集合体**（非法律实体），通过一系列交易文件的安排实现类似信托的风险隔离效果。

```
┌──────────────┐    资产买卖协议    ┌──────────────┐    发行证券    ┌──────────────┐
│ 原始权益人   │ ────────────────→ │ 专项计划     │ ──────────→ │ 资产支持证券 │
│              │ ←──────────────── │ （管理人）   │ ←────────── │   持有人     │
└──────┬───────┘    支付转让对价    └──────┬───────┘    认购资金    └──────────────┘
       │                                  │
       │ 兼任资产服务机构                  │
       │                                  │
       ↓                                  ↓
┌──────────────┐                   ┌──────────────┐
│  债务人      │                   │  托管银行     │
└──────────────┘                   └──────────────┘
```

### 3.2 专项计划主要交易文件

| 序号 | 文件名称 | 签署方 | 核心内容 |
|:--:|---------|-------|---------|
| 1 | **计划说明书** | 管理人出具 | 产品概述、交易结构、基础资产、风险揭示 |
| 2 | **资产买卖协议** | 原始权益人与管理人 | 资产交割、陈述与保证、回购义务、瑕疵资产处理 |
| 3 | **标准条款** | 管理人与持有人 | 持有人会议、有控制权的资产支持证券持有人 |
| 4 | **托管协议** | 管理人与托管银行 | 托管账户开立与管理、资金划付指令 |
| 5 | **服务协议** | 管理人与资产服务机构 | 服务标准、服务报酬、服务报告、定期审计 |
| 6 | **监管协议** | 管理人、托管银行、资产服务机构 | 监管账户设置、资金归集路径 |
| 7 | **认购协议** | 管理人与投资者 | 认购方式、风险揭示 |
| 8 | **法律意见书** | 管理人律师出具 | 交易结构合法性、资产转让有效性 |

### 3.3 专项计划与信托结构的关键差异

| 比较维度 | 资产支持专项计划 | 特定目的信托 |
|---------|:------------:|:----------:|
| 财产独立性依据 | 合同约定 + 证监会规章（法律层级较低） | 《信托法》的法定财产独立（法律层级高） |
| 破产隔离效力 | 依赖合同安排 + 监管保护，司法认定尚未完全统一 | 法定破产隔离，经司法实践广泛确认 |
| 对管理人的依赖 | 高度依赖管理人持续履职 | 信托财产独立于受托人，更换受托人不影响信托存续 |
| 上位法稳固性 | 以部门规章为主要依据 | 以《信托法》为法律依据 |

## 四、ABN 信托结构

### 4.1 ABN 基本信托结构

ABN 的信托结构与信贷 ABS 类似，均以《信托法》为上位法基础，但发行人限定为非金融企业。

### 4.2 ABN 结构特点

| 特点 | 说明 |
|------|------|
| 发行载体 | 特定目的信托（信托公司担任受托机构） |
| 发行人 | 受托机构以信托财产为基础发行资产支持票据 |
| 原始权益人 | 非金融企业（融资租赁、保理公司、基础设施企业等） |
| 注册制 | 交易商协会注册制，非行政审批 |
| 资产出表 | 与信贷ABS一样，可实现终止确认 |

### 4.3 ABN 与信贷 ABS 信托结构差异

| 维度 | ABN 信托 | 信贷 ABS 信托 |
|------|---------|-------------|
| 委托人 | 非金融企业 | 银行业金融机构 |
| 风险自留 | 无强制要求 | 发起机构须自留 ≥ 5% |
| 主承销商角色 | 重要（负责注册文件准备和发行） | 重要 |
| 适用法律框架 | 信托法 + 交易商协会自律规则 | 信托法 + 央行规章 + 金监局规定 |

## 五、真实出售与破产隔离

### 5.1 真实出售（True Sale）的判定标准

真实出售是破产隔离的前提，只有被法律认定为"真实出售"的资产转让，才能在原始权益人破产时将基础资产排除在破产财产之外。

#### 中国法下的真实出售判定维度

| 判定维度 | 真实出售的典型特征 | 担保融资的典型特征 | 中国实践中的判定依据 |
|---------|-----------------|-----------------|-------------------|
| **追索权** | 受让人对转让人无追索权或仅有有限的合规追索 | 受让人保持对转让人的全额追索权（即转让人担保受让人收回款项） | 交易文件中是否保留了显著超出正常陈述与保证范围的追索权 |
| **赎回权** | 转让人无权利也无义务赎回转出的资产 | 转让人有义务在约定条件下回购资产 | 合同中是否含有义务性回购条款（非选择性清仓回购） |
| **对价公允性** | 转让价格基于公允市场价格协商确定 | 转让价格与资产的公允价值无直接关联 | 定价机制是否独立于发起机构融资成本 |
| **剩余收益归属** | 资产回收超过转让款的部分归属于受让人（信托） | 超过约定回收的部分归还转让人 | 超额利差和剩余收益的权利归属 |
| **会计处理** | 转让人终止确认该资产（出表） | 转让人作为担保借款处理 | 会计师基于企业会计准则第23号（金融资产转移）的判定 |
| **控制权转移** | 受让人取得对资产的完全控制权 | 转让人保留对资产的实际控制权 | 受让人能否在未得转让人同意的情况下处置资产 |
| **服务商安排** | 转让人可继续担任服务商（但仅作为代理人） | 转让人以所有者身份继续管理资产 | 服务协议中服务商的权利边界 |

### 5.2 破产隔离（Bankruptcy Remoteness）的构成要件

破产隔离的目标是：**无论原始权益人、管理人/受托人、还是资产服务机构进入破产程序，基础资产均不受影响，证券持有人的权益持续得到保障。**

#### 五项核心构成要件

| 要件 | 法律/操作安排 | 中国实践中的考量 |
|------|-------------|---------------|
| **一、真实出售** | 基础资产的转让构成法律意义上的真实出售而非担保融资 | 会计师出具终止确认意见（出表）、律师出具真实出售法律意见 |
| **二、SPV财产独立性** | SPV财产独立于各参与方的固有财产 | 信托财产独立性有《信托法》保障；专项计划财产的独立性由证监会规定保障 |
| **三、有限追索权** | 证券持有人仅在SPV财产范围内受偿，对发起机构/管理人无追索权 | 在法律文件和募集说明书中明确证券持有人的追索权范围 |
| **四、不合并实体（Non-Consolidation）** | 原始权益人的破产不会导致SPV资产与原始权益人资产合并 | 律师出具不合并法律意见（中国实践中该意见的约束力存在争议） |
| **五、SPV自身不破产** | SPV本身不得主动申请破产或被动进入破产程序 | 有限合伙SPV在公司章程中设置破产表决的特别门槛（如全体投资人一致同意） |

### 5.3 中国司法实践中破产隔离的风险点

| 风险领域 | 风险描述 | 缓释措施 |
|---------|---------|---------|
| **专项计划财产的司法先例不足** | 资产支持专项计划"破产隔离"效力尚未经最高法院司法确认 | 完善交易文件，尽可能将实质争议降至最低 |
| **资产转让通知债务人** | 《民法典》第546条规定债权转让须通知债务人，否则对债务人不发生效力 | 设计通知安排（包括但不限于权利完善通知） |
| **管理人破产风险** | 如管理人（证券公司）破产，专项计划财产是否确定排除在破产财产外 | 专项计划财产单独记账、单独托管 |
| **原始权益人"自我交易"风险** | 原始权益人同时兼任服务机构时，资金混同风险 | 定期扫款机制、监管账户、资金归集路径独立 |
| **司法冻结/扣划风险** | 基础资产因原始权益人涉诉而被司法冻结 | 资产交割后及时变更登记/通知 |

## 六、循环购买结构设计

### 6.1 循环购买的基本原理

循环购买结构（Revolving Structure）是解决短期资产与长期证券期限错配的核心工具。在循环期内，基础资产到期回收的本金不立即分配给证券持有人，而是用于持续购买新的合格基础资产，维持资产池的规模稳定。

**生命周期阶段：**

```
循环期（Revolving Period）         摊还期（Amortization Period）
│← ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ →│← ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ →│
│  持续购买新资产           │  本金逐期偿付              │
│  仅付优先级利息           │  不购买新资产              │
│  资产池规模维持           │  资产池规模递减              │
└─────────────────────────┴─────────────────────────────┘
        发行日                循环期终止日                   到期日
```

### 6.2 循环购买标准（Eligibility Criteria）

持续购买阶段拟购入资产必须同时满足以下标准：

| 序号 | 合格标准 | 量化门槛（典型参数） | 
|:--:|---------|------------------|
| 1 | 资产类型 | 与初始资产池同一类型 |
| 2 | 加权平均利率 | 不低于初始资产池 WAC - 50bp |
| 3 | 加权平均剩余期限 | 不超过初始资产池 WAM + 6个月 |
| 4 | 债务人/借款人信用 | 影子评级不低于初始池下限 |
| 5 | 行业/地区集中度 | 不超过初始池相应上限 |
| 6 | 单一债务人集中度 | 不超过初始池上限（购入后不违反集中度限制） |
| 7 | 账龄 | 不低于 3 个月（排除新发放贷款） |
| 8 | 逾期情况 | 历史无逾期超过 30 天记录 |
| 9 | 合规性 | 不涉及负面清单行业/领域 |
| 10 | 可转让性 | 不包含限制转让条款 |

**每次购买后的资产池测试：**
- 现金与待收总额足够覆盖下一期的优先级本息和费用
- 加权平均利率未低于最低要求
- 前五大债务人/行业集中度未突破上限
- 加权平均影子评级未弱于最低要求

### 6.3 循环购买触发提前摊还事件

在循环期内，一旦触发以下事件之一，循环期立即终止，转入提前摊还：

| 序号 | 触发事件 | 设定逻辑 |
|:--:|---------|---------|
| 1 | 累计违约率 > 约定阈值 | 资产池质量已明显恶化，继续购买扩大风险 |
| 2 | 合格资产供给不足连续超过 X 期 | 无法找到合格资产，资金闲置 |
| 3 | 单一债务人违约且敞口 > 资产池总额 3% | 重大信用事件 |
| 4 | 资产服务机构评级下调至不可接受水平 | 服务商风险上升 |
| 5 | 超额利差连续3期 < 0.5% | 收入端持续恶化 |
| 6 | 原始权益人/发起机构丧失清偿能力 | 机构信用风险 |

### 6.4 持续购买不合格资产的处理

当已购入的资产在后续监测中被发现不再满足合格标准时：

| 处理方式 | 适用条件 | 操作机制 |
|---------|---------|---------|
| **不合格资产回购** | 因原始权益人违反陈述与保证而导致 | 原始权益人按约定价格强制回购不合格资产 |
| **清仓回购（Clean-up Call）** | 资产池规模降至初始值 10% 以下 | 发起机构有权（非义务）回购全部剩余资产以终止信托/专项计划 |
| **自动转次级** | 资产符合特定客观不合格指标（如逾期超过约定天数） | 该资产自动调整至次级分配顺序 |

## 七、双 SPV 结构（信托受益权 ABS）

### 7.1 结构原理

双 SPV 结构是企业 ABS 中为处理某些难以直接转让的基础资产而设计的特殊交易架构。

```
原始权益人 → SPV1（单一资金信托）→ 借款人/融资人
                 ↓ 信托受益权
             SPV2（资产支持专项计划）→ 证券持有人
```

### 7.2 双 SPV 的适用场景

| 场景 | 基础资产痛点 | 双SPV解决方式 |
|------|------------|-------------|
| 委托贷款 | 委托贷款债权难以直接转让 | 先以自有资金通过信托发放委托贷款，形成信托受益权 |
| 收费收益权 | 未来现金流，缺乏确定的既有债权 | 先设立财产权信托持有收费权，形成信托受益权 |
| 物业运营收入 | 收入受经营波动影响大 | 先设立信托持有物业运营收入权 |
| 限制转让合同 | 底层合同有禁止转让条款 | 通过信托实现权利重整 |

### 7.3 双 SPV 的风险穿透要点

在双 SPV 结构中，监管审核和评级机构关注的"穿透"问题：

| 穿透维度 | 分析要点 |
|---------|---------|
| 底层信用主体 | 最终还款来源是单一主体还是分散的债务人群体 |
| 基础资产同质性 | SPV1 持有的资产是否独立于原始权益人 |
| 破产隔离 | 双重隔离是否均有效 |
| 融资人识别 | 如底层实质为单一融资人，"ABS" 被质疑为"结构融资" |

## 八、SPV 终止与清算

### 8.1 信托/专项计划终止条件

| 条件 | 触发来源 | 后果 |
|------|---------|------|
| 证券全部到期并完成兑付 | 正常到期 | 正常终止 |
| 资产池全部提前清偿 | 加速清偿 | 提前终止 |
| 发起机构行使清仓回购权 | 发起机构选择 | 提前终止 |
| 资产池规模过低无法继续有效运作 | 经济性考量 | 提前终止 |
| 信托/专项计划期限届满 | 合同约定 | 到期终止 |

### 8.2 清算财产分配顺序

| 顺序 | 项目 |
|:--:|------|
| 1 | 清算费用 |
| 2 | 应付税费 |
| 3 | 应付服务费、管理费 |
| 4 | 优先级证券未偿本息 |
| 5 | 中间级证券未偿本息 |
| 6 | 次级证券剩余收益 |
