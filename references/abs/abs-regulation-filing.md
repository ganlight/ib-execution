# ABS 监管申报体系参考

## 一、中国 ABS 三大监管体系框架

中国资产证券化实行**分业监管、分类审核**制度。信贷 ABS、企业 ABS、ABN 三大体系分别对应不同的监管机构、审核流程和申报材料要求。理解并熟练掌握各体系的申报规则，是投行 ABS 业务操作的基本功。

---

## 二、银保监体系——信贷资产证券化

### 2.1 制度演变与现行审核机制

| 时期 | 制度阶段 | 核心特点 |
|------|---------|---------|
| 2005-2008 | 试点审批制（第一阶段） | 国务院批准试点额度；央行、银监会联合审批每单项目 |
| 2012-2014 | 试点审批制（第二阶段） | 试点重启，额度管理延续 |
| 2014.11 至今 | 备案制 + 注册制 | 银监会取消信贷ABS业务资格审批，改为备案登记；央行推行注册制 |

**现行核心制度：**
- **央行注册制：** 发起机构需向央行申请注册额度，额度有效期2年，有效期内可分期发行
- **金监局（原银保监会）备案制：** 每单发行前向金监局对应监管部备案登记
- **双线并行但互相衔接**

### 2.2 央行注册制流程

```mermaid
发起机构向央行提交注册申请 → 央行受理审核（约30工作日）→ 注册通过/取得注册额度 →
在注册额度有效期内（2年）分期发行 → 每期发行前向央行进行发行前备案
```

**注册申请核心材料清单：**

| 序号 | 材料名称 | 出具方 | 关键要求 |
|:--:|---------|-------|---------|
| 1 | 注册申请书 | 发起机构 | 含注册额度、基础资产类型、发行计划 |
| 2 | 信贷资产证券化业务开展报告 | 发起机构 | 过往ABS业务开展情况 |
| 3 | 董事会决议/有权机构决议 | 发起机构 | 同意开展ABS业务 |
| 4 | 内控制度文件 | 发起机构 | 包括业务流程、风险管理制度 |
| 5 | 受托机构合作协议（如有） | 发起机构、受托机构 | 已确定受托机构的证明 |

### 2.3 金监局备案制——每单发行前备案

**备案流程：**

1. 确定受托机构和中介机构
2. 完成资产池筛选和交易结构设计
3. 完成全部交易文件定稿，取得各中介机构意见
4. 向金监局对应监管部室提交备案材料
5. 金监局完成备案登记（收到材料后约5-10个工作日）
6. 取得备案登记后启动发行

**单期备案材料清单：**

| 序号 | 材料 | 说明 |
|:--:|------|------|
| 1 | 备案登记表 | 填写产品基本信息、交易结构要素 |
| 2 | 信托合同草案 | 核心交易文件 |
| 3 | 资产转让协议草案 | 资产交割安排 |
| 4 | 服务合同草案 | 资产服务机构安排 |
| 5 | 资金保管合同草案 | 资金托管安排 |
| 6 | 主承销协议草案 | 销售安排 |
| 7 | 评级报告（草案） | 分层评级结果 |
| 8 | 法律意见书（草案） | 交易结构合法合规性 |
| 9 | 会计意见书（草案） | 资产出表判断 |
| 10 | 募集说明书草案 | 信息披露文件 |
| 11 | 受托机构出具的尽职调查报告 | 信托事务尽职调查 |
| 12 | 发起机构出具的尽职调查报告 | 基础资产尽职调查 |

### 2.4 信贷 ABS 审核关注要点

| 关注领域 | 监管关注要点 | 应对策略 |
|---------|------------|---------|
| 基础资产合规性 | 是否符合信贷政策、不涉及限制性领域 | 逐笔核查贷款用途和行业归属 |
| 风险自留 | 发起机构是否按规定自留不低于5%的风险 | 明确自留比例、方式（水平/垂直/L型）和持有期 |
| 信息披露 | 资产池逐笔信息是否充分披露 | 按央行信息披露指引逐项核查 |
| 产品定价 | 优先级、次级定价是否合理 | 提供独立估值意见 |
| 投资者适当性 | 次级档投资者是否为合格投资者 | 建立次级档投资者适当性管理机制 |
| 服务机构能力 | 贷款服务机构的管理能力和系统支持 | 提供服务机构的系统评估报告 |

## 三、证监会/交易所体系——企业资产证券化

### 3.1 挂牌条件确认流程

```
管理人向交易所提交挂牌条件确认申请 →
交易所受理（5个工作日内决定是否受理）→
首轮审核（约10-20个工作日）→
反馈意见（如有）→
管理人反馈回复 →
交易所审议（挂牌条件确认委员会）→
出具无异议函（或终止确认）
```

### 3.2 申报材料完整目录

| 序号 | 材料名称 | 出具方 | 内容说明 |
|:--:|---------|-------|---------|
| 1 | 挂牌条件确认申请书 | 管理人 | 产品基本面要素 |
| 2 | 计划说明书 | 管理人 | 产品完整信息披露（150-300页级别） |
| 3 | 主要交易合同文本 | 管理人牵头、各方签署 | 资产买卖协议、托管协议、服务协议、标准条款 |
| 4 | 法律意见书 | 律师事务所 | 交易结构合法性、资产转让有效性、专项计划设立合规性 |
| 5 | 信用评级报告 | 评级机构 | 分层评级分析、压力测试 |
| 6 | 基础资产执行商定程序报告 | 会计师事务所 | 资产池数据准确性和一致性核查 |
| 7 | 现金流预测报告/审核意见 | 会计师事务所/评估机构 | 未来现金流预测合理性 |
| 8 | 差额支付承诺函/担保合同（如适用） | 增信提供方 | 外部增信文件 |
| 9 | 原始权益人最近三年审计报告 | 会计师事务所 | 财务状况 |
| 10 | 管理人合规核查意见 | 管理人内核 | 管理人内部合规审查意见 |
| 11 | 原始权益人有权机构决议 | 原始权益人 | 董事会/股东会决议 |
| 12 | 尽职调查报告 | 管理人 | 全面尽职调查结论 |
| 13 | 特定原始权益人的持续经营能力说明 | 原始权益人 | 适用于收费收益权类ABS |
| 14 | 关于专项计划会计处理意见（如需） | 会计师事务所 | 资产出表判断 |
| 15 | 其他相关文件 | — | 按需补充 |

### 3.3 交易所反馈回复要点

| 反馈类型 | 典型问题 | 回复策略 |
|---------|---------|---------|
| **基础资产合规性** | "请说明基础资产的形成是否符合相关法律法规" | 逐笔核查形成过程、合同合法性、发票真实性 |
| **集中度风险** | "前五大债务人占比较高，请补充说明集中度风险缓释措施" | 解释行业特征、大债务人信用分析、设置集中度触发机制 |
| **现金流预测** | "现金流预测假设过于乐观，请说明合理性" | 提供历史数据佐证、增加保守假设情景、压力测试 |
| **破产隔离** | "请说明专项计划财产的独立性安排" | 完善账户设置、资金归集路径、权利完善事件 |
| **原始权益人风险** | "原始权益人经营状况恶化对专项计划的影响" | 补充原始权益人经营分析、设置资产服务机构替换机制 |
| **循环购买** | "请说明循环购买资产标准和执行机制" | 明确量化合格标准、自动化执行触发、购买前测试 |
| **信息披露** | "特定情况下专项计划可能面临的风险揭示不足" | 补充极端情景风险分析、权力完善事件披露 |

### 3.4 基金业协会备案

交易所出具无异议函后：

1. 专项计划设立（发行完成）
2. 管理人于设立日起5个工作日内向基金业协会提交备案材料
3. 基金业协会完成备案并出具《备案确认函》
4. 备案完成后正式挂牌转让

## 四、交易商协会 ABN 体系

### 4.1 注册制流程

```
发行人（受托机构）准备注册文件 →
通过主承销商向交易商协会提交注册 →
交易商协会受理并审核 →
注册会议审议（如适用）→
注册通过，取得《接受注册通知书》→
注册额度有效期2年 →
有效期内分期发行 →
每期发行前完成发行前报告
```

### 4.2 ABN 注册文件清单

| 序号 | 文件名称 | 说明 |
|:--:|---------|------|
| 1 | 注册报告 | 注册额度、发行计划 |
| 2 | 募集说明书 | 参照交易商协会ABN信息披露指引 |
| 3 | 法律意见书 | 信托有效性、合规性 |
| 4 | 评级报告 | 分层评级 |
| 5 | 信托合同（草案） | 核心交易文件 |
| 6 | 资产转让协议（草案） | 资产交割 |
| 7 | 服务合同（草案） | 服务机构安排 |
| 8 | 资金保管合同（草案） | 托管安排 |
| 9 | 承销协议（草案） | 销售安排 |
| 10 | 发行人有权机构决议 | 受托机构决议 |
| 11 | 发行人与中介机构的相关协议 | — |
| 12 | 会计师关于基础资产的专项报告 | 资产数据核查 |
| 13 | 尽职调查报告 | 主承销商/受托机构出具 |

### 4.3 发行前报告

| 报告内容 | 目的 |
|---------|------|
| 发行条款（金额、期限、利率区间） | 确认符合注册条件 |
| 簿记建档安排 | 发行方式说明 |
| 最新资产池数据 | 确认资产池未发生重大不利变化 |
| 无重大不利变化声明 | 发起机构/发行人出具 |

## 五、三大体系对比表

| 对比维度 | 信贷 ABS | 企业 ABS | ABN |
|----------|----------|----------|-----|
| **审核机构** | 央行（注册）+ 金监局（备案） | 交易所（确认）+ 基金业协会（备案） | 交易商协会（注册） |
| **审核方式** | 注册制 + 备案制 | 挂牌条件确认（类审核制） | 注册制 |
| **审核周期参考** | 3-6 个月（含两阶段） | 2-4 个月 | 2-4 个月 |
| **审核关注核心** | 基础资产质量、发起机构资质、风险自留、资产出表 | 基础资产真实性、现金流预测合理性、破产隔离 | 基础资产质量、交易结构合规性 |
| **储架发行支持** | 是（注册额度制） | 是（储架发行） | 是（注册额度制） |
| **注册/储架额度有效期** | 2 年 | 24 个月 | 2 年 |
| **申报前是否可预沟通** | 可（与央行/金监局） | 可（交易所预沟通） | 可（交易商协会预沟通） |
| **反馈轮次** | 1-2 轮 | 1-3 轮 | 1-2 轮 |
| **首次发行 vs 后续发行** | 首次注册审核更全面 | 储架下后续发行简化 | 首次注册审核更全面 |
| **发行方式** | 公募 | 私募（面向合格投资者） | 公募/定向 |
| **风险自留要求** | 强制 ≥ 5% | 无强制 | 无强制 |
| **发行载体** | 特定目的信托 | 资产支持专项计划 | 特定目的信托（为主） |
| **信息披露平台** | 中国货币网/中债信息网 | 交易所网站/基金业协会 | 中国货币网/交易商协会 |
| **产品命名规范** | ××年/第×期信贷资产支持证券 | ××资产支持专项计划 | ××年第×期资产支持票据 |

## 六、储架发行（Shelf Registration）

### 6.1 储架发行的定义与优势

储架发行是指发行人在获得一次性注册/确认后，在注册有效期内可分期、多次发行ABS的机制。

| 优势 | 说明 |
|------|------|
| 提高发行效率 | 后续批次无须重复完整审批流程 |
| 降低综合成本 | 多期共享前期尽调和审核成果 |
| 择时发行 | 灵活选择有利的发行窗口 |
| 额度管理 | 发行人获得确定的融资额度，便于资产负债管理 |

### 6.2 企业 ABS 储架发行条件

| 条件维度 | 具体要求 |
|---------|---------|
| 基础资产同质性 | 各期基础资产类型、合格标准、交易结构保持一致 |
| 原始权益人资质 | 经营状况良好、无重大违法违规、内部控制有效 |
| 主体信用 | 原始权益人主体评级通常不低于AA（非硬性要求，但影响审批效率） |
| 资产供给能力 | 能证明未来24个月内具备持续提供合格资产的能力 |
| 过往发行经验 | 如有同类ABS发行并进入正常存续期，有助于获批 |
| 信息披露完备 | 储架文件充分披露储架发行安排和各期发行的通用条款 |

### 6.3 储架下后续发行简化流程

| 简化事项 | 说明 |
|---------|------|
| 申报文件 | 仅需提交当期的计划说明书和基础资产清单 |
| 审核流程 | 交易所快速审核通道（通常1-2周出具备案确认） |
| 交易文件 | 如无重大修改，沿用储架中已确认的交易文件 |
| 中介机构意见 | 可仅就当期资产更新相关意见 |

### 6.4 储架发行中的风险控制机制

| 机制 | 运作方式 |
|------|---------|
| 累计违约率监控 | 各期设置累计违约率上限，任一期的违约率超限自动暂停后续发行 |
| 重大不利变化报告 | 原始权益人发生重大不利变化时，应暂停发行并报告交易所 |
| 储架额度上限 | 未偿本金余额合计不得超过注册总额度 |

## 七、关注要点 Top 10（申报审核阶段）

| 排名 | 关注要点 | 风险等级 | 常见问题及应对 |
|:--:|---------|:------:|-------------|
| **1** | **基础资产真实性** | ★★★★★ | 是否存在虚假资产/循环融资/构造贸易背景？→ 逐笔穿透核查合同+发票+流水三单匹配 |
| **2** | **现金流预测合理性** | ★★★★★ | 预测假设是否过于乐观？→ 提供充分的历史回测数据 + 多情景敏感性分析 |
| **3** | **破产隔离有效性** | ★★★★ | 真实出售认定 + 资金归集路径独立 + 权利完善事件 |
| **4** | **集中度风险** | ★★★★ | 单一债务人/行业/地区集中度是否超标？→ 设置集中度触发机制 |
| **5** | **增信措施充分性** | ★★★★ | CE 是否覆盖目标评级下的压力情景损失？→ 提供压力测试详情 |
| **6** | **原始权益人/资产服务机构经营稳定性** | ★★★★ | 无原始权益人=无新资产供给（对循环型）→ 评估经营可持续性 |
| **7** | **关联交易与利益冲突** | ★★★ | 原始权益人→服务机构→管理人的关联关系？→ 充分披露 + 建立防火墙 |
| **8** | **信息披露完整性** | ★★★ | 是否充分揭示各类风险？→ 按监管指引逐项核对披露清单 |
| **9** | **循环购买机制可行性** | ★★★ | 合格标准是否可实现？资产供给是否充足？→ 提供历史数据证明 |
| **10** | **税务合规性** | ★★★ | 增值税/所得税处理是否合规？→ 取得会计师税务意见 |

## 八、跨体系发行路径选择策略

### 8.1 发行路径决策矩阵

| 决策因子 | 推荐体系 | 原因 |
|---------|:------:|------|
| 发起机构为银行 | 信贷 ABS | 银行在银行间市场有天然优势 |
| 发起机构为非金融企业 | 企业 ABS 或 ABN | 非金融企业无法在信贷ABS体系发行 |
| 寻求公开募集 | 信贷 ABS 或 ABN | 企业ABS只能向合格投资者非公开发行 |
| 追求快速审核 | ABN 或企业 ABS | 注册制流程通常较快 |
| 资产出表需求强 | 信贷 ABS 或 ABN（信托SPV） | 信托结构在出表认定上更具优势 |
| 交易所挂牌流动 | 企业 ABS | 交易所挂牌提供更活跃的二级市场 |
| 双SPV结构 | 企业 ABS | 双SPV在专项计划体系下有成熟实践 |
| 储架发行 | 三个体系均支持 | 各体系均有储架通道 |
| 发起机构为融资租赁公司 | 企业 ABS 或 ABN | 两者均有大量实践案例 |

## 九、申报过程中的常见误区

| 误区 | 正确认知 |
|------|---------|
| "备案制/注册制意味着审核放松" | 实际上监管对信息披露和风险揭示的要求更高 |
| "审核人员只看材料，不关注底层实质" | 穿透式监管下，审核人员会追问底层资产的合规性和质量 |
| "历史发行经验是申报成功的充分条件" | 历史经验仅作参考，每单项目独立审核 |
| "交易结构越复杂越显得专业" | 复杂的结构反而引发更多监管追问；简洁高效的方案更受认可 |
| "法律意见书只是例行公事" | 法律意见书是审核中最重要的第三方独立意见之一 |
