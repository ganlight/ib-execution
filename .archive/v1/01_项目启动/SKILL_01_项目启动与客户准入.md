# SKILL_01：项目启动与客户准入

## 概述

本项目启动阶段涵盖从首次客户接触到正式启动尽职调查之间的全流程管理。标准投行项目的启动阶段通常为 **2-4周**，核心目标是建立清晰的合作框架、明确双方权利义务、完成初步准入审查。

---

## 一、项目启动生命周期

```
首次接触 → 保密协议签订 → 初步方案建议 → 委任书签订 → 项目启动会 → KYC/反洗钱 → 利益冲突检查 → 启动尽调
  ↓            ↓               ↓              ↓            ↓           ↓            ↓           ↓
了解客户    NDA签署          Pitch Book     Engagement    Kick-off     AML检查     COI检查    Stage 02
           (1-3天)          (1-2周)        Letter       Meeting      (并行)      (并行)      Gate
                                          (1-2周)       (1天)                   
```

## 二、委任书（Engagement Letter）流程

### 2.1 委任书核心条款

| 条款 | 要点 | 常见争议点 |
|------|------|------------|
| 服务范围 | 明确产品/市场/规模 | 是否包含超额配售/绿鞋 |
| 费用结构 | 承销费率、成功费、固定费 | 费率基准（募集总额 vs 净额） |
| 费用报销 | 差旅、律师、印刷等 | 报销上限与预审批机制 |
| 排他性 | 独家委任期限 | 排他期长短与续期条件 |
| 终止条款 | 双方终止条件与通知期限 | 尾部费用（tail fee）条款 |
| 责任限制 | 赔偿条款、责任上限 | 故意/重大过失除外 |
| 争议解决 | 仲裁/诉讼、管辖法律 | 仲裁地与语言 |

### 2.2 费用结构谈判要点

- **IPO承销费**：A股通常为募集资金总额的 2%-5%（视规模而定），港股 2.5%-3.5%，美股 6%-7%
- **财务顾问费**：M&A通常 1%-3%（Lehman公式），分签约费/交割费
- **成功费 vs 固定费**：成功费占比越高，对客户越友好，但对投行风险越大
- **分段支付**：签约费(20-30%) + 申报受理(30-40%) + 发行完成(30-50%)

## 四、初步范围界定

### 4.1 核心信息获取

| 信息类别 | 具体内容 | 来源 |
|----------|----------|------|
| 公司基本情况 | 注册资本、股权结构、主营业务 | 工商登记/管理层访谈 |
| 财务概况 | 近3年收入/利润/资产/负债 | 审计报告/管理报表 |
| 融资意向 | 目标产品/上市地/规模/时间表 | 管理层 |
| 行业定位 | 细分行业、主要竞争对手 | 管理层/行业研究 |
| 合规状况 | 重大诉讼/处罚/监管关注 | 律师/管理层访谈 |

### 4.2 监管环境初步筛查

- **A股IPO**：判断公司是否符合注册制各板块定位（科创属性/三创四新/大盘蓝筹）
- **再融资**：核查距上次融资间隔（一般需≥18个月）、募集资金使用进度
- **重大资产重组**：判断是否构成重组上市、借壳上市等监管红线
- **港股上市**：核查VIE架构合规性、外资准入限制
- **行业限制**：房地产、教育、游戏等行业的特殊监管政策

## 六、进入Stage 02的门槛条件

以下条件**全部满足**后方可进入尽职调查阶段：

| 编号 | 门槛条件 | 责任人 |
|------|----------|--------|
| G01 | 委任书已正式签署 | 项目负责人 |
| G02 | NDA已签署（如适用） | 项目负责人 |
| G03 | KYC/反洗钱审查已完成且通过 | 合规部 |
| G04 | 利益冲突检查已完成且已解除或制定信息隔离方案 | 合规部/法律部 |
| G05 | 项目组成员全部确定且已完成内部培训 | 项目负责人 |
| G06 | 首次项目启动会（Kick-off Meeting）已完成 | 项目负责人 |
| G07 | 初步时间表和工作计划已确定 | 执行负责人 |
| G08 | 客户已提供基本文件清单中≥80%的文件 | 执行负责人 |
| G09 | 监管环境筛查无重大不可逾越障碍 | 法律/合规 |
| G10 | 项目已通过公司内部立项审批 | 立项委员会 |

## 八、风险提示与常见陷阱

1. **委任书签署拖延**：建议在方案建议阶段即启动委任书谈判，避免成为瓶颈
2. **范围蔓延（Scope Creep）**：如客户要求增加服务范围（如增加上市地），需在委任书修订或签订补充协议
3. **KYC不通过**：尽调启动前务必完成KYC，避免投入资源后无法推进
4. **利益冲突未及时发现**：务必在所有项目组成员确定后强制完成COI问卷
5. **信息不对称**：首次会议获取的信息可能与实际尽调发现存在重大差异，预留调整空间
6. **中介机构协调**：律师和审计师的选定与委派应在启动阶段完成，避免后续更换
7. **监管政策变化**：关注注册制改革、行业监管等最新政策动态
