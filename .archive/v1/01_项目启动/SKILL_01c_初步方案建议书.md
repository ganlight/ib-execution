# SKILL_01c：初步方案建议书（Pitch Book）

## 概述

初步方案建议书（Pitch Book）是投行向潜在客户展示专业能力、建立信任关系的核心文件。它同时承担三大功能：**展示投行对客户及行业的理解、提出最具吸引力的交易方案、彰显投行执行能力和差异化优势**。

---

## 一、Pitch Book 标准结构

### 推荐结构（共8-10个部分）

```
┌─────────────────────────────────────────┐
│  1. 封面（Cover Page）                    │
│  2. 目录（Table of Contents）             │
│  3. 执行摘要（Executive Summary）          │
│  4. 投行介绍与团队（Firm & Team）          │
│  5. 公司与行业分析（Company & Industry）    │
│  6. 可比公司/交易分析（Comparables）        │
│  7. 建议方案与时间表（Proposal & Timeline） │
│  8. 估值分析（Valuation）                  │
│  9. 费用方案（Fee Proposal）               │
│  10. 附录（Appendix）                      │
└─────────────────────────────────────────┘
```

## 二、各部分编写指南

### 2.1 执行摘要（Executive Summary）

**篇幅**：1-2页（最多3页）

**必须包含**：
- 一句话推荐方案（如"建议在科创板上市，募集8-10亿元"）
- 推荐理由（3-5条核心论点，每条1-2句话）
- 预期时间表（关键里程碑日期）
- 投行核心优势（2-3条）
- 预期估值范围

**关键原则**：
- 重要信息前置：决策者在最初几页就要看到核心结论
- 定制化：绝不能使用通用模板语言
- 以数字说话：具体数值比形容词有力

### 2.2 投行介绍与团队

**内容要素**：

| 小节 | 内容 |
|------|------|
| 投行概览 | 行业排名、牌照、全球布局 |
| 行业经验 | 同行业/同类项目历史交易清单（务必具体） |
| 项目团队 | 核心成员照片+简历（每人3-5条亮点） |
| 项目执行能力 | 项目数量和规模数据 |

**差异化技巧**：
- 团队简历突出与目标客户最相关的经验
- 用LOGO墙/交易墓碑展示行业实力
- 列举近2-3年可比交易比长篇历史更有效

### 2.3 公司与行业分析

这是Pitch Book中**最需要定制化的部分**。

**行业分析框架**：

```
宏观环境（政策/经济/技术/社会）
    ↓
行业概况（规模/增速/阶段/集中度）
    ↓
竞争格局（主要玩家/市场份额/竞争要素）
    ↓
发展趋势（技术路径/商业模式/监管方向）
    ↓
公司定位分析与SWOT
```

**公司分析要点**：
- 展示对客户的深入了解（核心产品/技术/客户/财务）
- 精准定位公司在产业链中的位置
- 对标行业标杆，指出差距和机遇
- 避免过度吹捧，违和感是Pitch最大杀手

### 2.4 可比公司分析

| 类别 | 选取原则 | 数量 |
|------|----------|------|
| 核心可比 | 业务高度相似、同一细分行业 | 3-5家 |
| 扩展可比 | 业务有一定相似性、相邻行业 | 2-3家 |
| 交易可比 | 近期/历史可比交易 | 2-3笔 |

**展示内容**：
- 估值倍数表（PE/PB/PS/EV-EBITDA/EV-Revenue）
- 财务数据对比（收入/利润/增速/利润率）
- 二级市场表现（近1年股价走势叠加图）

### 2.5 建议方案与时间表

| 要素 | 内容要求 |
|------|----------|
| 目标市场/板块 | 明确推荐并给出理由 |
| 发行规模 | 新股比例、募集金额估算 |
| 定价方式 | 询价/定价/竞价 |
| 时间表 | 里程碑甘特图（至少精确到月） |
| 关键假设 | 信息披露时间表背后的前提 |
| 特殊安排 | 如战略配售、超额配售（绿鞋） |

### 2.6 估值分析

- DCF估值（核心假设：收入增速/利润率/WACC/终值增长率）
- 可比公司估值（倍数范围 + 对应目标估值区间）
- 可比交易估值（溢价率参考）
- 敏感性分析（至少做收入和WACC二维敏感表）
- **估值结论**：给出估值区间（如"20-25亿元"），而非单一数值

### 2.7 费用方案

| 费用组成 | 说明 |
|----------|------|
| 承销费/顾问费 | 费率+计算基础 |
| 成功费 vs 固定费 | 结构说明 |
| 费用报销 | 报销类目+上限 |
| 总费用估算 | 基于发行规模假设的计算 |

**注意事项**：
- 费率宜以区间呈现（允许谈判空间）
- 明确说明费用触发条件（如成功费仅在完成发行时支付）
- 如适用，注明"以上费率待监管审批"

## 三、Pitch Book 设计原则

### 3.1 视觉规范

1. **一致性**：全篇统一配色方案、字体、图表风格
2. **清晰度**：每页3-5个要点，关键词加粗/高亮
3. **图表化**：能用图表不用表格，能用表格不用段落文字
4. **留白**：每页至少30%留白，避免信息过载
5. **品牌**：使用投行自身品牌VI，保持专业统一

### 3.2 内容原则

- **用户思维**：把客户最关心的问题放在最前面
- **数据支撑**：每个观点有市场数据/案例/财据支撑
- **叙事逻辑**：每部分之间要有承上启下的过渡
- **对比说服**：标杆对比/方案对比比单一方案更有说服力

## 四、关键信息传达清单

Pitch Book 完成后，检查以下关键信息是否传达：

- [ ] 我们理解客户所在行业（深度>广度）
- [ ] 我们知道客户的核心竞争优势是什么
- [ ] 我们清楚客户当前面临的挑战
- [ ] 我们提出的方案是有理有据的最优解
- [ ] 我们有能力执行、有经验佐证
- [ ] 我们团队的组合是最匹配的
- [ ] 我们的估值建议是合理且可实现的
- [ ] 我们的收费是透明且有竞争力的
- [ ] 我们对关键风险有清醒的认知并经得起追问

## 五、常见Pitch错误及避坑指南

### 🔴 致命错误

| 错误 | 后果 | 修正 |
|------|------|------|
| 公司/数据张冠李戴 | 信任崩塌，无法挽回 | 三重核查制度（作者/审阅者/终审者） |
| 行业数据过时（>2年） | 不专业印象 | 确保引用的行业数据是最新的 |
| 估值过于离谱 | 客户直接忽视 | 对标行业内可比基准 |
| 使用上一客户Pitch的残留文字 | 极度不尊重 | 全文检索前任客户名称 |
| 拼写/语法错误 | 不专业 | 至少2轮校对 |

### 🟡 常见不足

| 问题 | 表现 | 改进方向 |
|------|------|----------|
| 信息过载 | 50+页但无重点 | 精简至30页核心+附录 |
| 缺乏观点 | 只有行业知识搬运 | 加入投行独立判断 |
| 方案一刀切 | 只提供一个方案 | 提供A/B方案对比 |
| 忽视竞争 | 不提竞争对手优势 | 主动对比并差异化 |
| 团队亮点不突出 | 通用简历堆砌 | 突出与本次项目最相关的2-3条 |

## 六、竞争对手差异化策略

### 6.1 差异化的五个维度

| 差异化维度 | 如何做到 |
|------------|----------|
| 行业深度 | 展示比竞品更精准的行业理解和更深的客户洞察 |
| 执行速度 | 展示比竞品更实际可行的时间表 + 历史执行速度数据 |
| 团队匹配 | 安排在该领域最具经验的MD亲自挂帅 |
| 交易创新 | 提出竞品没有想到的交易结构/融资方案 |
| 生态资源 | 展示投行整体的投资者关系/研究/分销能力资源 |

### 6.2 应对竞品常见的攻击

| 竞品攻击点 | 提前防御策略 |
|------------|-------------|
| "XX投行规模更大" | 展示行业专注度：我们虽然不是最大，但在你这个行业最专业 |
| "XX投行做过更大项目" | 强调大项目 ≠ 好服务，中等项目反而获得高层关注度更高 |
| "XX费率更低" | 价值论证：执行质量 = 更好的定价 + 更顺利的审批 |
| "XX有母公司生态" | 中立性论证：独立性反而避免利益冲突 |

## 七、Pitch Book 大纲模板

```
═══ [公司名称] [产品类型]建议方案 ═══

Part 1: 执行摘要 (1-2页)
  - 推荐方案一句话
  - 3-5条核心推荐理由
  - 估值区间
  - 项目时间表概览

Part 2: 市场与行业 (3-5页)
  - 宏观环境趋势
  - 行业规模与增速
  - 竞争格局分析
  - 行业资本化程度

Part 3: 公司分析 (3-5页)
  - 公司发展历程
  - 核心业务/产品
  - 商业模式拆解
  - 竞争优势(SWOT)
  - 财务摘要

Part 4: 估值与可比分析 (4-6页)
  - 可比公司估值表
  - DCF估值
  - 可比交易分析
  - 估值结论与敏感性

Part 5: 建议方案 (3-5页)
  - 板块/市场选择论证
  - 发行规模与结构
  - 定价方式建议
  - 募集资金用途建议
  - 详细时间表

Part 6: 投行团队与经验 (2-3页)
  - 投行行业排名/资质
  - 相关交易记录
  - 核心团队介绍

Part 7: 费用方案 (1页)
  - 费率结构
  - 费用估算

Part 8: 附录
  - 可比公司详细数据
  - 行业数据来源
  - 免责声明
```

## 八、Pitch Book 质量自检清单

- [ ] 全文搜索无上一客户名称残留
- [ ] 所有数据标注来源和日期
- [ ] 估值假设有明确的依据和可比参照
- [ ] 时间表每个里程碑有明确的完成标准
- [ ] 费用方案与最终委任书草案一致
- [ ] PDF版本少于30MB（便于邮件传输）
- [ ] 所有图表在黑白打印下仍可读
- [ ] 已做至少2轮内部审阅（同行审阅+上级审阅）
- [ ] 团队简历已获本人确认
- [ ] 中文版已完成（如需双语）
