# SKILL_06c：并购交割

## 一、概述

并购交割是重组获批后，按照交易协议完成股权/资产转移、对价支付和相关法律手续的最后环节。交割是风险最高的阶段之一——任何一件交割前提条件未满足，都可能导致交易无法完成甚至引发纠纷。本技能提供交割全程管理指南。

---

## 二、交割前置条件核查（Condition Precedent Checklist）

### 2.0 交割前置条件总览

交割前置条件（Conditions Precedent, CP）是交易协议中约定的、交割前必须满足的各项条件。CP不满足 = 交割不能启动。部分CP可在交割日当天满足（"交割日CP"），部分须提前满足（"交割前CP"）。以下按类别逐一梳理。

### 2.1 监管审批类（A类CP）

```
□ 01-证监会/交易所核准：
    本次重组的核准/注册决定已出具，且未被撤销、暂停或附加条件
    核查方法：查阅核准文件正本，确认有效期及任何限制条件

□ 02-反垄断审查：
    （若触发申报门槛）商务部/市监总局反垄断局审查已通过
    或取得"不实施进一步审查"通知书
    （若不触发）取得律师出具的"不构成经营者集中"意见

□ 03-国家安全审查：
    （若涉及敏感行业/外资收购）外商投资安全审查已通过
    核查方法：审查外资安审办公室出具的审查决定

□ 04-外资准入审查：
    （若涉及外商投资负面清单行业）商务部/地方商务部门审查已通过
    核查方法：取得外商投资企业变更备案回执或核准文件

□ 05-行业准入审批：
    （若涉及特许经营/许可证行业）相关行业主管部门同意本次交易的批复
    示例：金融行业→银保监会/证监会；电信→工信部；医疗→卫健委/药监局

□ 06-国资监管批复（若涉及国有资产）：
    国资委/财政部门对本次交易的批准文件
    核查重点：批复范围是否覆盖全部交易标的、评估备案是否完成

□ 07-国有产权交易：
    （若涉及国有产权转让）已在产权交易所挂牌并取得成交确认书
    核查方法：取得产权交易所出具的成交确认文件

□ 08-所有审批均在有效期内，未被暂停或撤销
    核查方法：逐项审批文件复查有效期，预留审批延期缓冲时间
```

### 2.2 交易双方内部审批类（B类CP）

```
□ 09-上市公司董事会审议通过：
    历次董事会决议文件（方案调整、草案、最终方案）
    核查方法：查阅董事会决议正本，确认出席人数/表决比例合规

□ 10-上市公司股东大会审议通过：
    本次重大资产重组的股东大会决议
    核查重点：①回避表决情况（关联股东是否回避）；
             ②表决比例（是否满足特别决议>2/3）；
             ③中小股东表决情况单独披露

□ 11-交易对方已履行内部决策程序：
    标的公司原股东的董事会/股东会决议
    核查方法：根据标的公司章程及注册地法律要求核查

□ 12-标的公司已履行内部决策程序：
    （若标的公司为被收购对象）标的公司董事会/股东会同意本次交易
    核查重点：是否存在少数股东异议/优先购买权行使

□ 13-债权人同意：
    （若交易触发借款合同中的控制权变更条款Control Change）
    取得所有触发条款的债权人书面同意
    核查方法：逐笔核查借款合同的控制权变更条款
```

### 2.3 第三方同意类（C类CP）

```
□ 14-重大合同的交易对方出具同意函：
    （若交易触发合同中的控制权变更条款）
    核查范围：前十大客户合同、前十大供应商合同、合资合同、合作协议
    核查方法：逐份合同排查是否含Change of Control条款

□ 15-合资合作伙伴同意：
    （若标的公司存在合资企业）合资方书面同意本次交易
    核查重点：合资方是否行使优先购买权/否决权

□ 16-银行/金融机构同意：
    贷款银行对控制权变更的书面同意
    核查方法：重点核查含"交叉违约"条款的贷款合同

□ 17-核心客户/供应商确认：
    确认不因本次重组变更合作条款
    核查重点：对于依赖少数大客户的标的公司，此项尤为关键

□ 18-知识产权许可方同意：
    （若标的公司核心技术/品牌系许可使用）许可方同意本次交易后继续许可
```

### 2.4 状态确认类（D类CP）

```
□ 19-交易双方陈述与保证（Reps & Warranties）在交割日仍真实准确：
    各方出具"陈述与保证重申函"（Bring-down Certificate）
    核查方法：对照交易协议中的R&W清单逐条确认

□ 20-未发生重大不利变化（Material Adverse Change, MAC）：
    核查期间：自估值基准日至交割日
    核查维度：财务业绩、业务运营、监管环境、核心客户、核心人员
    如发生MAC事件，评估是否触发协议中的MAC退出条款

□ 21-交易协议未终止且完全有效：
    确认交易协议未被任何一方通知终止
    确认协议中的Long Stop Date未到期

□ 22-标的公司资产/业务/人员无重大变化：
    资产无重大毁损/灭失，业务持续正常经营，核心人员未离职
    核查方法：交割日前实地走访+管理层访谈

□ 23-不存在限制或禁止交割的法院禁令或政府命令：
    核查是否存在与本次交易相关的诉讼、保全措施、强制执行
    核查方法：中国执行信息公开网+裁判文书网检索
```

### 2.5 中介意见类（E类CP）

```
□ 24-为本次交易出具的交割法律意见书：
    发行人律师出具，涵盖：
    - 本次交易的批准及授权
    - 交割条件是否已全部满足
    - 标的资产权属是否清晰、是否存在过户障碍
    - 本次交易是否符合相关法律规定

□ 25-交割审计报告：
    （若协议约定须进行交割日审计）
    审计师出具，涵盖：
    - 标的公司自基准日至交割日的财务报表
    - 过渡期损益的审计确认
    - 交割日净资产确认（与评估基准日对比）

□ 26-独立财务顾问核查意见：
    保荐人/独立财务顾问出具，涵盖：
    - 本次重组实施情况
    - 资产过户/股权过户完成情况
    - 相关信息披露是否合规
```

## 三、交割日流程

### 3.1 交割前一周准备

```
交割日前一周：
  □ 召开交割准备会，确认所有CP状态
    参会方：保荐人、律师、上市公司、交易对方
    议程：逐条确认CP完成情况 → 确定剩余CP的完成时间和责任人 → 签署交割日程安排确认函

  □ 准备交割文件包
    ├── 交割确认函（各方签字页预先签好）
    ├── 股权/资产过户材料包（工商变更全套材料）
    ├── 新增股份登记材料（如发行股份购买资产）
    ├── 对价款支付指令书（银行件）
    ├── 标的公司印章/证照/财务资料移交清单
    └── 交割公告稿（初稿，待交割完成信息填入）

  □ 资金到位确认（收购方）
    确认收购方对价支付资金已到位/银行授信已获批
    核查：银行存款证明/银行承诺函 x 预计对价总额

交割日前一日：
  □ 最后一轮CP逐条确认（全天）
  □ 签署交割确认函各方完成内部签批（预留充分时间应对签字流程）
  □ 股权过户/资产过户材料准备完毕
  □ 银行大额支付系统测试通过（单笔金额>5亿需提前报备）
```

### 3.2 交割日当日完整流程

```
交割日标准流程（通常从9:00开始，至17:00完成）：

09:00-09:30 │ Step 1：交割启动会
            │  - 各方代表到场或线上一并确认
            │  - 保荐人宣读本届CP完成情况及任何未完成CP的豁免安排
            │  - 确认交割可启动

09:30-10:00 │ Step 2：签署交割确认函
            │  - 各方授权代表签署交割确认函（确认CP全部满足或未满足CP已豁免）
            │  - 律师见证签署过程

10:00-11:30 │ Step 3：股权/资产过户启动
            │  └─ 见下方3.3节详细流程

11:30-12:00 │ Step 4：首期对价款支付
            │  - 收购方按约定向交易对方支付首期对价（通常50%-70%）
            │  - 银行系统处理时间：大额支付系统工作日9:00-17:00
            │  - 取得银行转账凭证

12:00-14:00 │ 午休 / 等待资金到账确认

14:00-15:00 │ Step 5：获取工商变更受理回执（若已递交）
            │  - 取得市场监管局出具的《受理通知书》
            │  - 确认受理日期即为股权变更的登记申请日

15:00-16:00 │ Step 6：标的公司印章、证照、财务资料移交
            │  - 按《移交清单》逐项核对
            │  - 印章类：公章、财务章、法人章、合同专用章、发票专用章
            │  - 证照类：营业执照正副本、开户许可证、资质证书
            │  - 财务类：银行U盾/密码器、税控盘、网银密码、财务报表、账册
            │  - 签署《资料交接确认书》一式三份（交接双方+见证方各执一份）

16:00-17:00 │ Step 7：新管理层入驻沟通
            │  - 召开标的公司员工大会/核心团队沟通会
            │  - 介绍新任董事长/总经理/财务负责人
            │  - 说明整合计划基本方向

17:00      │ 交割完成，各方代表签署《交割完成确认书》
```

### 3.3 股权转让登记详细流程

**有限责任公司**：工商变更登记（股东变更登记）

```
办理机构：标的公司注册地市场监督管理局

准备材料清单：
  □ 《公司登记（备案）申请书》
  □ 股权转让协议 / 增资协议（原件）
  □ 股东会决议（转让方/受让方均须出具）
  □ 章程修正案 / 新公司章程
  □ 股权转让完税证明（税务局出具的个人所得税/企业所得税完税凭证）
  □ 新股东的主体资格证明（营业执照复印件，加盖公章）
  □ 新法定代表人/董事/监事/经理的任职文件及身份证明
  □ 营业执照正副本（原件）

办理流程：
  1. 税务局办理股权转让纳税申报（转让方）
  2. 取得完税证明（通常3-5个工作日）
  3. 向市监局提交工商变更全套材料
  4. 市监局受理（3-5个工作日）
  5. 取得变更后的新《营业执照》

完成标志：取得变更后的《营业执照》（统一社会信用代码不变，但登记股东信息已更新）
```

**股份公司股份**：

```
非上市股份公司：
  - 变更股东名册（公司内部，在董事会秘书处登记）
  - 公司章程备案（如涉及章程修改）
  - 无需工商变更登记（股份公司股份转让不以工商登记为生效要件）

上市公司股份：
  - 协议转让 / 要约收购 / 集中竞价等方式交易
  - 中国证券登记结算有限责任公司办理过户登记
  - 交易所合规审查（限售期、信息披露等）
```

### 3.4 资产过户详细流程

| 资产类型 | 过户机构 | 主要材料 | 办理周期 |
|:---|:---|:---|:---:|
| **土地使用权** | 不动产登记中心 | 土地证原件、转让协议、完税证明、双方营业执照 | 15-30个工作日 |
| **房屋所有权** | 不动产登记中心 | 房产证原件、转让协议、契税完税证明、测绘报告 | 15-30个工作日 |
| **专利权** | 国家知识产权局 | 专利转让协议、著录项目变更申报书、双方主体证明 | 1-3个月 |
| **商标权** | 国家知识产权局商标局 | 商标转让协议、转让申请书、双方营业执照 | 4-8个月 |
| **软件著作权** | 中国版权保护中心 | 转让协议、申请表、身份证明 | 1-2个月 |
| **车辆** | 车管所 | 行驶证、登记证书、交易发票、双方身份证明 | 1-3个工作日 |
| **一般设备/存货** | —（双方现场交接） | 资产交接确认书（双方签字盖章） | 即时 |

**风险提示**：知识产权过户周期长，建议在交割协议中约定：
- 过户完成前，转让方不可撤销地许可受让方无偿使用；
- 以提交过户申请（取得受理回执）替代正式过户完成作为对价支付节点之一。

### 3.5 对价款支付

| 支付时间点 | 支付比例 | 前提条件（释放条件） | 支付方式 |
|:---|:---:|:---|:---|
| 交割日 | 50%-70% | CP全部满足 + 交割确认函签署 | 银行大额支付系统 |
| 工商变更受理 | 10%-20% | 取得市场监管局《受理通知书》 | 银行大额支付系统 |
| 工商变更完成 | 10%-15% | 取得新《营业执照》 | 银行转账 |
| 过渡期审计后 | 5%-10% | 过渡期审计报告出具 + 损益结算确认 | 银行转账 |
| 业绩承诺期满 | 视协议 | 业绩承诺全部完成 | 银行转账 |

> ⚠️ **最佳实践**：建议留存不低于10%的尾款至全部资产过户完成（含知识产权），以应对交割后可能出现的资产瑕疵和权属纠纷。

### 3.6 过渡期损益归属确认

```
过渡期定义：评估/审计基准日至交割日之间的期间

损益归属的常见约定模式：
  ├── 模式A：过渡期收益归收购方，亏损由转让方补足（对收购方最有利）
  ├── 模式B：过渡期收益归转让方，亏损由转让方补足（常见于国资项目）
  ├── 模式C：过渡期损益均由收购方承担（简单但风险较高）
  └── 模式D：过渡期收益归收购方但超出承诺盈利部分可返还转让方（激励型）

确认流程：
  1. 聘请审计师出具过渡期专项审计报告
  2. 确认过渡期净利润（扣非后）
  3. 与协议约定的损益归属条款对比
  4. 如转让方需补足亏损 → 计算补足金额 → 在对价尾款中抵扣
```

## 五、基准日审计与交割日审计的差异

### 5.1 两种审计的核心区别

| 维度 | 基准日审计 | 交割日审计 |
|:---|:---|:---|
| **审计基准日** | 评估基准日（如2025.12.31） | 实际交割日（如2026.10.15） |
| **目的** | 为交易定价提供财务基础 | 确认过渡期损益 + 交割日净资产 |
| **触发条件** | 交易协议签署前必须完成 | 协议约定是否需要进行 |
| **审计范围** | 标的公司最近两年一期或三年一期 | 基准日至交割日之间的期间 |
| **报告类型** | 标准审计报告（含附注） | 专项审计报告（可简化） |
| **法律效力** | 交易定价基础 | 过渡期损益结算依据 |
| **是否必须** | 是（上市公司重大资产重组强制要求） | 视协议约定（建议执行） |
| **审计师** | 申报会计师 | 可与基准日审计师相同 |

### 5.2 建议的交割审计要点

```
1. 核查过渡期内所有重大交易是否经正常商业判断
2. 确认过渡期内不存在"突击花钱"、关联方非公允交易
3. 确认标的公司完整保留了基准日的资产和业务
4. 确认过渡期内无大额资产处置/负债新增
5. 对比基准日净资产 vs 交割日净资产，计算增减变化
6. 确认影响过渡期损益的重大事项（如大额客户合同变更）
```

## 六、对赌条款触发后的交割处置

### 6.1 交割阶段的对赌相关问题

```
问题1：业绩承诺期尚未开始，但标的公司交割前已出现业绩下滑
  → 策略：在交割确认函中明确记载该事实
  → 保留未来调用MAC条款的权利
  → 在交割后减值测试中考虑该因素

问题2：交割前发现标的公司存在未披露负债
  → 策略：①触发陈述与保证不实条款
         ②要求转让方承担该负债
         ③在对价尾款中预留等额保证金
         ④签署补充协议明确追偿权

问题3：核心管理层在交割前夕离职
  → 策略：①评估是否触发MAC条款
         ②要求转让方追加补偿
         ③修改对价支付节奏（增大尾款比例）
```

### 6.2 对赌期开始前的保护措施

| 保护措施 | 说明 | 执行时机 |
|:---|:---|:---|
| 要求承诺方质押股份 | 将部分股份质押给收购方作为业绩补偿担保 | 交割日至业绩承诺期开始前 |
| 尾款留存>20% | 延长尾款支付时间，覆盖业绩承诺期第一年 | 交割对价支付时 |
| 竞业限制强化 | 确保核心管理层离职后不能从事竞争业务 | 交割前签署 |
| 过渡期行为约束 | 限制标的公司在过渡期内的大额决策（需收购方同意） | 签署交割协议时约定 |

## 七、交割风险管理

### 7.1 常见交割延迟原因

| 延迟原因 | 发生概率 | 建议 |
|---------|---------|------|
| 工商变更审核排队 | ⭐⭐⭐⭐ | 提前预约+加急通道 |
| 资金调配未及时到位 | ⭐⭐⭐ | 提前3个工作日确认资金路径 |
| 第三方同意尚未取得 | ⭐⭐⭐ | 申报阶段即启动沟通 |
| MAC事件发生 | ⭐⭐ | 密切关注行业和对手方动态 |
| CP文件签字体例争议 | ⭐⭐ | 提前草拟，减少最后时刻改字 |
| 知识产权过户耗时超预期 | ⭐⭐⭐ | 提前2-3个月启动，先取受理回执 |
| 股权转让完税证明办理延迟 | ⭐⭐⭐ | 提前沟通税务局，了解当地办理规程 |

### 7.2 交割失败预案

- 协议中应约定：如交割在约定截止日（Long Stop Date）前未完成，任一方可终止协议
- 约定违约责任（违约金/分手费）
- 设定交割延长期（Extension）条件和程序
- 分手费/反向分手费的触发条件和计算方式：
  - **正向分手费**（收购方支付）：因收购方原因导致交易失败，如收购方融资失败、收购方股东大会否决
  - **反向分手费**（转让方/标的方支付）：因标的方原因导致交易失败，如CP不满足、陈述与保证重大不实
  - 金额通常为交易对价的1%-5%

### 7.3 交割应急预案

| 情景 | 应对措施 |
|:---|:---|
| 交割日资金未到位 | ①确认延迟时间（如果是技术性问题vs实质性困难）；②如无法当日解决，签署延期交割协议；③启用分手费条款（如超过Long Stop Date） |
| 工商变更当日无法办理 | ①先签署交割确认函（法律上视为交割完成）；②次日加急办理工商变更；③在交割公告中说明"工商变更登记正在办理中" |
| 交割日发现标的公司重大不利变化 | ①暂停支付首期对价；②召开紧急联席会议；③律师评估是否触发MAC条款；④评估是否终止交易 |
| 标的公司印章/资料移交时发现缺失 | ①立即追查，确认是遗失还是被故意扣留；②如遗失→公告挂失+重新刻制；③如被扣留→法律手段追索+影响交割完成确认的判断 |
