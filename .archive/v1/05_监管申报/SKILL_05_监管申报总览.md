# SKILL_05：监管申报总览

## 一、概述

监管申报是将尽调成果和撰写文件转化为正式申报材料，并向监管机构报送的环节。不同产品的申报流程、审核机制和监管机构存在显著差异。本技能提供申报流程的总览性框架和各产品线共性方法论。

---

## 二、三大产品线申报路径全景图

### 2.1 IPO 申报路径（注册制）

```
┌──────────────┐    ┌──────────────┐    ┌──────────────┐
│ 地方证监局   │───→│   交易所     │───→│   交易所     │
│ 辅导验收     │    │   受理申报   │    │  审核问询    │
└──────────────┘    └──────────────┘    └──────┬───────┘
                                                │
              ┌──────────────────────────────────┘
              ▼
┌──────────────┐    ┌──────────────┐    ┌──────────────┐
│  上市委      │───→│   证监会     │───→│   发行上市   │
│  审议        │    │   注册       │    │              │
└──────────────┘    └──────────────┘    └──────────────┘
```

**关键节点**：
1. **辅导备案与验收**（T-6至T-3个月）：地方证监局辅导监管，出具辅导验收报告
2. **交易所受理**（T+0至T+5）：申报文件接收、形式审查、出具受理通知
3. **审核问询**（T+5至T+6个月）：多轮问询（通常1-3轮），回复+更新招股书
4. **上市委审议**（T+6至T+8个月）：上市委会议审议、投票表决
5. **证监会注册**（T+8至T+12个月）：注册审阅、注册问询（如有）、注册批文

### 2.2 债券申报路径（多头注册体系）

```
                ┌──────────────────────────────┐
                │        债券发行人             │
                └──────────────┬───────────────┘
                               │
          ┌────────────────────┼────────────────────┐
          ▼                    ▼                    ▼
┌──────────────────┐  ┌──────────────────┐  ┌──────────────────┐
│   NAFMII注册     │  │   证监会注册     │  │   发改委注册     │
│ （银行间市场）    │  │  （交易所公司债） │  │  （企业债）      │
├──────────────────┤  ├──────────────────┤  ├──────────────────┤
│ MTN 中期票据     │  │ 大公募公司债     │  │ 一般企业债       │
│ CP 短期融资券    │  │ 小公募公司债     │  │ 专项企业债       │
│ SCP 超短期融资券 │  │ 非公开发行公司债 │  │ 项目收益债       │

│ ABN/ABCP        │  │ REITs            │  │                  │

```

**各体系注册要点**：

| 注册体系 | 审核机构 | 注册框架 | 典型时限 | 核心文件 |
|---------|---------|---------|---------|---------|
| NAFMII | 交易商协会注册办公室 | DFI/TDFI/单品种 | 20工作日（反馈时限） | 募集说明书 |
| 证监会 | 交易所审核+证监会注册 | 大公募/小公募 | 2-4个月 | 募集说明书 |
| 发改委 | 中央结算公司预审+发改委终审 | 额度制 | 3-6个月 | 募集说明书+募投项目文件 |

> **详细债券注册体系**见 `references/bond/bond-registration-comparison.md`

### 2.3 并购重组申报路径

```
┌──────────────┐    ┌──────────────┐    ┌──────────────┐
│  董事会       │───→│  股东大会     │───→│   交易所     │
│  审议方案     │    │  审议方案     │    │  受理申报    │
└──────────────┘    └──────────────┘    └──────┬───────┘
                                                │
              ┌──────────────────────────────────┘
              ▼
┌──────────────┐    ┌──────────────┐    ┌──────────────┐
│  交易所      │───→│ 并购重组委   │───→│   证监会     │
│  审核问询    │    │  审议        │    │  注册/核准   │
└──────────────┘    └──────────────┘    └──────────────┘
```

**关键节点**：
1. **董事会审议**：首次董事会审议重组预案/草案，签署附生效条件的交易协议
2. **股东大会审议**：关联股东回避表决，2/3以上通过
3. **交易所受理**：提交全套申报文件，5个工作日内形式审查
4. **审核问询**：通常1-2轮，重点关注估值和业绩承诺
5. **并购重组委审议**：重组委会议审议，投票表决
6. **证监会注册/核准**：注册制（发股购买资产）或核准制（其他）

## 三、各产品线申报流程对比

### 3.1 总体对比表

| 维度 | IPO | 债券 | 并购重组 |
|-----|-----|------|---------|
| **监管机构** | 证监会 + 交易所（双重审核） | NAFMII/证监会/交易所（视品种） | 证监会并购重组委/交易所 |
| **审核制度** | 注册制（交易所审核+证监会注册） | 注册制（银行间/交易所各有规则） | 核准制（部分转为注册制） |
| **核心审核文件** | 招股说明书 | 募集说明书 | 重组报告书 |
| **审核时限** | 3-6个月（受理至注册） | 视品种而定 | 3-6个月 |
| **反馈次数** | 2-3轮（平均） | 1-2轮 | 1-2轮 |
| **上市委/重组委审议** | 有（上市委会议） | 无 | 有（重组委会议） |
| **注册有效期** | 1年（批文有效期） | 2年（批文有效期） | 1年（批文有效期） |
| **申报前辅导** | 必须（地方证监局） | 不需要 | 不需要 |
| **行业限制** | 各板块有定位要求 | 无 | 无（仅需合规） |
| **业绩要求** | 各板块有盈利/市值要求 | 无硬性盈利要求 | 标的需符合对应要求 |

### 3.2 IPO申报路径

```
改制设立 → 辅导备案 → 辅导验收 → 申报受理 → 
交易所审核（含多轮问询）→ 上市委审议 → 证监会注册 → 
发行上市
```

主板/创业板/科创版的具体差异见 **SKILL_05a** 和 **SKILL_05b**。

### 3.3 债券申报路径

> **→ 委托给 ib-bond-executor 技能处理**。

不同债券品种对应的审核路径：银行间（NAFMII注册）→ 交易所（证监会注册或交易所审核）→ 发改委（企业债审批）。具体流程见 ib-bond-executor SKILL.md。

### 3.4 并购重组申报路径

```
董事会决议 → 股东大会决议 → 申报受理 → 
交易所审核 → 重组委审议（发股购买资产）→ 证监会核准/注册 → 
实施交割
```

详细流程见 **SKILL_05d**。

```

│ 受理     │ 首轮回复  │ 二/三轮   │ 上市委     │ 注册     │
```

**周期弹性最大的节点**：
- 🔴 **注册阶段**：证监会节奏不完全可控，占25%+时间
- 🟡 **首轮回复准备**：取决于发行人和中介配合效率
- 🟡 **多轮问询**：若出现第三轮，至少要增加1-1.5个月

## 七、申报前内核机制

### 7.1 内核流程概述

内核（Internal Control Review）是投行申报前的**内部最后防线**。内核委员会独立于项目组，对申报材料的质量、完整性、合规性进行全面审查。无论IPO、债券还是重组，申报前必须通过内核。

```
项目组申报材料准备 → 内核申请提交 → 内核材料分发（T-10天）→
内核委员独立审阅 → 内核会议 → 投票表决 → 
通过 → 内核意见反馈 → 项目组逐条落实 → 内核负责人复核 → 准予申报
不通过 → 退回修改 → 30天后重新上内核
```

### 7.2 内核准备的15项必备文件清单

| 序号 | 文件名称 | 责任方 | 提交时限 | 必备/可选 |
|-----|---------|--------|---------|----------|
| 1 | **招股说明书/募集说明书/重组报告书**（内核稿V0.5） | 项目组 | 会前10个工作日 | 🔴 必备 |
| 2 | **保荐工作报告**（含尽调过程与结论） | 项目组 | 会前10个工作日 | 🔴 必备 |
| 3 | **发行保荐书**（内核稿） | 项目组 | 会前10个工作日 | 🔴 必备 |
| 4 | **内核申请报告**（项目概述+核心问题说明） | 项目组 | 会前10个工作日 | 🔴 必备 |
| 5 | **审计报告/审阅报告**（三年一期） | 会计师 | 会前10个工作日 | 🔴 必备 |
| 6 | **内部控制鉴证报告** | 会计师 | 会前10个工作日 | 🔴 必备 |
| 7 | **法律意见书**（内核稿） | 律师 | 会前10个工作日 | 🔴 必备 |
| 8 | **律师工作报告**（内核稿） | 律师 | 会前10个工作日 | 🔴 必备 |
| 9 | **尽调工作底稿索引/目录** | 项目组 | 会前10个工作日 | 🔴 必备 |
| 10 | **项目组尽调问题"负面清单"**（自查未达标事项汇总） | 项目组 | 会前10个工作日 | 🔴 必备 |
| 11 | **重大风险事项专项说明**（至少3个核心风险点深度分析） | 项目组 | 会前7个工作日 | 🟡 强烈建议 |
| 12 | **同行业可比公司审核案例汇编**（5个以上案例） | 项目组 | 会前7个工作日 | 🟡 强烈建议 |
| 13 | **反馈/问询预判问题集**（至少30个预判问题+回复思路） | 项目组 | 会前5个工作日 | 🔵 建议 |
| 14 | **内核问题回复底稿**（过往历次内核会议的回复记录） | 项目组 | 会前5个工作日 | 🔵 建议 |
| 15 | **签字页原件/复印件**（如董事签字页、测试签字页） | 项目组 | 会前3个工作日 | 🔵 建议 |

### 7.3 内核会议议程

```
┌─────────────────────────────────────────────────────────────┐
│              XX证券内核委员会第XX次会议                        │
│                                                                │
│  时间：202X年X月X日  09:30-12:00                               │
│  地点：XX会议室 / 视频会议                                     │
│  内核委员（5人）：                                             │
│    委员A（法律背景/召集人）、委员B（财务背景）                   │
│    委员C（行业背景）、委员D（合规背景）、委员E（风控背景）       │
│  列席：项目组（保荐代表人2人+项目组成员）、风控部代表            │
└─────────────────────────────────────────────────────────────┘

【09:30-09:45】议程一：内核委员独立审阅意见汇总
  - 秘书处汇总各委员书面初审意见（已提前3天提交）
  - 集中呈现共性问题和各委员特别关注问题
  - 重要性问题标红：涉及上市条件/重大风险/信息披露真实性

【09:45-10:30】议程二：项目组陈述与核心问题回应（45分钟）
  - 2.1 项目基本情况概述（5分钟）：
    · 发行人概要/行业定位/板块选择依据
    · 主要财务数据（收入/利润/ROE/资产负债率）
    · 本次发行方案（发行规模/发行股数/募投项目）
  - 2.2 尽调过程与发现（10分钟）：
    · 尽调方法体系与覆盖范围
    · 重大发现及已采取措施
    · 尽调中存在的客观限制
  - 2.3 核心风险与应对措施（15分钟）：
    · TOP 5风险因素及其缓释措施
    · "负面清单"事项逐一回应
  - 2.4 委员关切问题集中回应（15分钟）：
    · 逐条回应初审意见中的重点关注问题
    · 展示补充尽调/补充分析的结论

【10:30-11:00】议程三：内核委员质询（30分钟）
  - 委员逐一提问（不限范围）
  - 项目组现场回答（可借助资料、但须当场给出核心结论）
  - 如涉及后续补充工作：明确回复时限
  - 如存在争议性判断：可记入会议纪要，后续专项讨论

【11:00-11:15】议程四：委员闭门讨论，项目组退场（15分钟）
  - 委员独立讨论
  - 形成投票前的共识性问题清单

【11:15-11:30】议程五：投票表决与结果宣布（15分钟）
  - 投票方式：记名投票
  - 通过门槛：4/5同意（≥4票同意）
  - 表决选项：
    · "同意申报"（同意内核通过，准予申报）
    · "附条件同意"（需完成特定条件，经复核后申报）
    · "暂缓表决"（需补充尽调或论证，30天后重新内核）
    · "不同意申报"（内核否决，须退回调整方案）

【11:30-11:45】议程六：内核意见整合（15分钟）
  - 整理内核意见（可分"必须落实"和"建议关注"两档）
  - 明确落实后的复核机制（谁复核/何时复核）
  - 后续申报时间表确认

【11:45-12:00】议程七：内核会议纪要签确认
  - 召集人/秘书处/项目组三方签确认
```

### 7.4 内核常见否决/暂缓情形

| 情形 | 内核结果 | 预防措施 |
|------|---------|---------|
| 财务数据存在重大矛盾 | 🔴 否决 | 申报前完成交叉比对数据库 |
| 核心尽调程序缺失/未完成 | 🔴 否决 | 内核前逐项自查尽调清单 |
| 招股书存在重大遗漏陈述 | 🟡 暂缓 | 使用尽调问题清单逐条对照 |
| 负面清单事项未充分论证风险 | 🟡 暂缓 | 专项分析+缓释措施 |
| 中介机构意见存在显著分歧 | 🔴 否决 | 内核前协调统一口径 |
| 签章不完整/文件有格式问题 | 🟡 暂缓（快修可会前解决） | 双人复核签章页 |

- 为发行人管理预期

## 九、常见延误因素及应对

### 9.1 内部可控制因素

| 延误因素 | 风险等级 | 应对措施 |
|---------|---------|---------|
| 反馈回复准备耗时过长 | 🔴高 | 收到反馈前即预判回复方向，提前准备 |
| 补充审计/尽调耗时 | 🟡中 | 申报前充分预估，先补充后回复 |
| 中介沟通不顺畅 | 🟡中 | 建立书面确认机制，减少电话沟通 |
| 发行人配合度低 | 🔴高 | 申报前签订配合协议，明确时间表 |

### 9.2 外部不可控因素

| 延误因素 | 应对措施 |
|---------|---------|
| 监管审核负荷重（排队） | 跟踪审核员进度，适时适当催促 |
| 政策变化 | 密切关注政策动向，及时调整方案 |
| 行业舆情/媒体质疑 | 建立危机应对预案，及时回应 |

## 十一、申报前最终核查

无论哪个产品线，申报前必须完成以下共同核查：

- [ ] 所有申报文件均已完成内部审批（质控+内核）
- [ ] 签字页全部签署完毕
- [ ] 电子版和纸质版文件内容一致
- [ ] 关键数据在所有文件中保持一致
- [ ] 所有文件均在有效期内（审计报告6个月、法律意见书无固定有效期但需刷新）
- [ ] 发行人/中介机构签章完备
- [ ] 申报文件格式符合监管机构要求

### 11.1 各产品线差异化核查要点

**IPO额外核查**：
- [ ] 辅导验收报告已出具（地方证监局出具）
- [ ] 板块定位专项说明已完成（科创板/创业板）
- [ ] 保荐人内核会议纪要及表决结果完备

- [ ] 受托管理协议已签署
- [ ] 持有人会议规则已定
- [ ] 信用评级报告（如适用）

**重组额外核查**：
- [ ] 股东大会关联方回避表决记录完备
- [ ] 交易协议已签署（附生效条件）
- [ ] 独立董事意见已出具
- [ ] 独立财务顾问核查意见已出具
