# SKILL_03：方案设计总览

## 概述

方案设计阶段是在完成尽职调查后，基于公司实际情况和监管环境，为发行人量身定制最优资本市场方案的核心阶段。本阶段输出：**产品选择 + 板块选择 + 发行结构 + 估值区间 + 时间表**。

### 1.1 产品选择关键因素

| 因素 | IPO优先 | 再融资优先 | 并购优先 | 债券优先 |
|------|---------|-----------|----------|----------|
| 上市状态 | 未上市 | 已上市 | 已上市 | 不限 |
| 融资规模 | 大额（>3亿） | 中等 | 视交易规模 | 不限 |
| 盈利情况 | 较好/允许未盈利(科创板) | 较好 | 标的有协同 | 稳定现金流 |
| 股东退出意愿 | 强 | 弱/无 | 战略整合 | 无 |
| 估值水平 | 高（享受新股溢价）| 中 | 视协同 | 不关注 |
| 时间要求 | 长（6-18月） | 中（3-6月） | 长（6-12月） | 短（1-3月） |
| 品牌/治理 | 大提升 | 小提升 | — | — |

## 二、板块选择决策矩阵

### 2.1 四大板块关键差异

| 维度 | 主板 | 科创板 | 创业板 | 北交所 |
|------|------|--------|--------|--------|
| **定位** | 大盘蓝筹/行业龙头 | 硬科技/科创属性 | 三创四新/成长型创新 | 创新型中小企业 |
| **财务标准** | 3套（盈利+现金流+营收） | 5套（含未盈利标准） | 3套（含未盈利标准） | 4套（含低盈利标准） |
| **审核周期** | 6-12月 | 3-6月 | 6-12月 | 2-4月 |
| **估值水平** | 中等 | 较高 | 中高 | 较低 |
| **流动性** | 好 | 好 | 好 | 一般 |
| **投资者门槛** | 无 | 50万 | 10万 | 50万 |
| **涨跌幅限制** | ±10%(前5日无) | ±20% | ±20% | ±30% |
| **再融资便利** | 高 | 中 | 中 | 低 |
| **退市标准** | 较严 | 严 | 严 | 较严 |
| **适合企业阶段** | 成熟期 | 成长期/成熟期 | 成长期 | 早期/成长期 |

### 2.2 板块选择决策流程

```
Step 1: 行业定位检查
  ├── 是否符合科创板行业定位？（新一代信息技术/高端装备/新材料/新能源/节能环保/生物医药）
  │     └── 是 → 进一步检查科创属性指标
  ├── 是否符合创业板"三创四新"定位？
  │     └── 是 → 进一步检查创新指标
  ├── 是否为创新型中小企业且净利润<5000万？
  │     └── 是 → 北交所
  └── 大盘蓝筹/行业龙头/成熟企业？
        └── 是 → 主板

Step 2: 财务指标匹配
  逐套对照各板块财务标准 → 见 SKILL_03a

Step 3: 估值与流动性权衡
  盈利好+规模大 → 主板（稳定）
  科创属性强 → 科创板（估值溢价）
  成长性好+创新 → 创业板（兼顾）
  规模偏小+早期 → 北交所（门槛低）

Step 4: 股东偏好
  是否有退出诉求？→ 流动性优先
  是否希望保持高估值？→ 科创板/创业板
  是否有国际化需求？→ 考虑H股/红筹
```

## 五、关键的方案设计假设

在方案设计启动前，必须与客户管理层确认以下假设：

- [ ] 上市前是否需要进行重组（剥离非核心业务/清理股权）？
- [ ] 大股东/管理层是否有明确的减持/退出计划？
- [ ] 是否引入Pre-IPO战略投资者？
- [ ] 股权激励（ESOP）是否需要设立/调整？
- [ ] 募集资金的优先使用方向？
- [ ] 客户对上市时间的要求（最晚时间窗口）？
- [ ] 客户对控股权的底线（上市后最低持股比例）？
- [ ] 是否考虑"A+H"或"先H后A"等跨市场方案？

## 六、方案汇报准备清单

向客户汇报方案前，必须完成以下准备：

- [ ] 至少准备2套可行方案的对比分析
- [ ] 每套方案包含完整的：板块选择理由+发行结构+估值区间+时间表+利弊分析
- [ ] 推荐方案有明确且有力的理由
- [ ] 所有估值假设均有可比公司/可比交易作为支撑
- [ ] 涉及监管判断的事项已咨询律师或合规
- [ ] 方案PPT不超过30页核心内容
- [ ] 准备了可能被追问的关键问题及回答
- [ ] 涉及敏感的估值/定价讨论已做好预沟通
