6.5 优先认购权：在公司新增资本时，按实缴出资比例优先认缴出资（法律、行政法规及本章另有约定的除外）；

6.6 优先购买权：其他股东拟对外转让股份时，在同等条件下享有优先购买权（本协议或公司章程另有约定的除外）；

6.7 诉权：认为公司股东大会、董事会的决议违反法律、行政法规时，可以请求人民法院撤销或认定无效；

6.8 股份转让权：在符合法律、本协议和公司章程限制的前提下，转让其持有的股份。

## 第三章 股份锁定与转让限制

### 第七条 股份锁定承诺

#### 7.1 发起人股份锁定

各发起人（原始股东）承诺，其持有的股份公司股份按以下安排进行锁定：

| 股东类别 | 锁定期 | 锁定说明 | 适用法规 |
|---------|-------|---------|---------|
| 创始人/控股股股东 | 上市后36个月 | 自上市之日起36个月内不得转让或委托他人管理 | 《证券法》 |
| 其他发起人 | 上市后12个月 | 自上市之日起12个月内不得转让 | 《证券法》 |
| 董事/监事/高级管理人员 | 任职期间+离职后半年 | 任职期间每年转让不超过25%；离职后半年内不得转让 | 《公司法》 |
| 员工持股平台 | 参照创始人 | 视同实际控制人/控股股东，锁定36个月 | 交易所规定 |

#### 7.2 股份锁定例外情形

以下情形不视为违反股份锁定承诺：
- 继承、遗赠、离婚财产分割
- 向持股平台转让（触发本协议相关约定）
- 司法强制执行（应提前书面通知保荐机构）

#### 7.3 股份锁定承诺函

各发起人应在本协议签署后[5]个工作日内，就股份锁定安排向股份公司、保荐机构出具书面承诺函。

### 第八条 股份转让限制

#### 8.1 一般转让限制

**发起人之间的转让：**
发起人之间相互转让其持有的全部或部分股份，无需经其他发起人同意，但应当书面通知股份公司并办理变更登记。

**向发起人以外的第三方转让：**
发起人拟将其持有的股份全部或部分转让给发起人以外的第三方（以下简称"受让方"）时，须遵守以下程序：

1. **书面通知**：转让发起人（以下简称"转让方"）应提前[30]日以书面形式通知其他发起人，载明拟转让股份的数量、价格、付款方式、受让方基本情况等事项；

2. **其他发起人同意**：上述转让须经其他发起人[过半数/全体]书面同意方可进行；

3. **优先购买权行使**：其他发起人在收到转让通知后[20]日内，有权以同等条件优先购买全部或部分拟转让股份；逾期未书面答复或未足额行使优先购买权的，视为放弃优先购买权；

4. **共同出售权触发**：若拟转让股份导致公司控制权变更或涉及投资人股东转让持有的公司股份，应同时触发本协议第五章规定的共同出售权（Drag-Along）或随售权（Tag-Along）条款。

#### 8.2 特定情形下的转让限制

以下特定情形下的转让，须经全体发起人一致书面同意方可进行：

1. **导致控制权变更的转让**：任何转让行为导致公司实际控制人变更或新增实际控制人的；
2. **国有股权/外资股权变动**：涉及国有股权向非国有单位转让，或涉及外资准入特别管理措施的；
3. **特殊股东资格限制**：如转让涉及须经监管部门批准的股东资格（如金融机构股东资格），需取得相应批准；
4. **IPO审核期间的转让**：股份公司IPO审核期间（自申报之日起至上市之日止），除经保荐机构书面同意外，主要股东不得转让股份。

#### 8.3 董事、监事、高级管理人员股份转让特别约定

担任股份公司董事、监事、高级管理人员的发起人，在转让其持有的公司股份时，还须遵守以下特别约定：

- 任职期间每年转让的股份不得超过其所持有公司股份总数的25%；
- 离职后半年内，不得转让其所持有的公司股份；
- 在任期届满前离职的，应当在其就任时确定的任期内和任期届满后六个月内，继续遵守上述转让限制；
- 上述人员在申报期内及上市后的股份转让，应按照《公司法》、《证券法》及证券交易所的相关规定执行，并提前通知保荐机构。

## 第四章 公司治理

### 第九条 股东大会

#### 9.1 股东大会的职权

股东大会是股份公司的最高权力机构，行使下列职权：

1. 决定公司的经营方针和投资计划；
2. 选举和更换非由职工代表担任的董事、监事，决定有关董事、监事的报酬事项；
3. 审议批准董事会的报告；
4. 审议批准监事会或者监事的报告；
5. 审议批准公司的年度财务预算方案、决算方案；
6. 审议批准公司的利润分配方案和弥补亏损方案；
7. 对公司增加或者减少注册资本作出决议；
8. 对发行公司债券作出决议；
9. 对公司合并、分立、解散、清算或者变更公司形式作出决议；
10. 修改公司章程；
11. 对公司聘用、解聘会计师事务所作出决议；
12. 审议批准本章程第四十一条规定的担保事项；
13. 审议公司在一年内购买、出售重大资产超过公司最近一期经审计总资产30%的事项；
14. 审议批准变更募集资金用途事项；
15. 审议股权激励计划；
16. 法律、行政法规、部门规章或本章程规定的其他职权。

#### 9.2 股东大会的决议机制

**普通决议事项：**
下列事项经出席股东大会的股东（包括股东代理人）所持表决权的过半数通过：
- 本章程第9.1条第1项至第6项所列事项；
- 董事会和监事会的工作报告；
- 董事会拟定的利润分配方案和弥补亏损方案；
- 董事会和监事会成员的任免及其报酬和支付方法；
- 公司年度预算方案、决算方案；
- 除法律、行政法规规定或者本章程规定应当以特别决议通过以外的其他事项。

**特别决议事项：**
下列事项须经出席股东大会的股东（包括股东代理人）所持表决权的2/3以上通过：
- 本章程修改；
- 公司增加或者减少注册资本；
- 公司的合并、分立、解散、清算或者变更公司形式；
- 本章程规定的担保事项；
- 上市公司发行股票、可转换公司债券、优先股等；
- 重大资产重组；
- 法律、行政法规或本章程规定的，以及股东大会以普通决议认定会对公司产生重大影响的、需要以特别决议通过的其他事项。

   - 独立董事：[X]名（占董事会成员1/3以上）
2. **投资人董事席位**：[XX投资人]有权推荐[1]名候选人担任股份公司董事，该候选人经股东大会选举产生；该董事在董事会中应代表投资人股东的利益。

#### 10.2 董事的选举和任期

- 董事由股东大会选举或更换，任期[3]年，任期届满可连选连任；
- 董事任期届满前，股东大会不得无故解除其职务；
- 独立董事的任职资格和独立性应符合中国证监会的相关规定。

#### 10.3 董事会职权

董事会对股东大会负责，行使下列职权：
1. 召集股东大会，并向股东大会报告工作；
2. 执行股东大会的决议；
3. 决定公司的经营计划和投资方案；
4. 制订公司的年度财务预算方案、决算方案；
5. 制订公司的利润分配方案和弥补亏损方案；
6. 制订公司增加或者减少注册资本、发行债券或其他证券及上市方案；
7. 拟订公司重大收购、收购本公司股票或者合并、分立、解散及变更公司形式的方案；
8. 在股东大会授权范围内，决定公司对外投资、收购出售资产、资产抵押、对外担保事项、委托理财、关联交易等事项；
9. 决定公司内部管理机构的设置；
10. 聘任或者解聘公司经理、董事会秘书；根据经理的提名，聘任或者解聘公司副经理、财务负责人等高级管理人员，并决定其报酬事项和奖惩事项；
11. 制订公司的基本管理制度；
12. 制订本章程的修改方案；
13. 管理公司信息披露事项；
14. 向股东大会提请聘请或更换为公司审计的会计师事务所；
15. 听取公司经理的工作汇报并检查经理的工作；
16. 法律、行政法规、部门规章或本章程授予的其他职权。

#### 11.3 监事会职权

监事会行使下列职权：
1. 检查公司财务；
2. 对董事、高级管理人员执行公司职务的行为进行监督，对违反法律、行政法规、本章程或者股东大会决议的董事、高级管理人员提出罢免的建议；
3. 当董事、高级管理人员的行为损害公司的利益时，要求董事、高级管理人员予以纠正；
4. 提议召开临时股东大会，在董事会不履行《公司法》规定的召集和主持股东大会职责时召集和主持股东大会；
5. 向股东大会提出提案；
6. 依照《公司法》第一百五十一条的规定，对董事、高级管理人员提起诉讼；
7. 发现公司经营情况异常，可以进行调查；必要时，可以聘请会计师事务所、律师事务所等专业机构协助其工作。

### 第十二条 高级管理人员

#### 12.1 高级管理人员范围

股份公司的高级管理人员包括：
- 总经理（总裁）1名
- 副总经理（副总裁）[X]名
- 财务负责人（财务总监）1名
- 董事会秘书1名
- 公司章程规定的其他人员

#### 12.2 总经理的职权

总经理对董事会负责，行使下列职权：
1. 主持公司的生产经营管理工作，组织实施董事会决议；
2. 组织实施公司年度经营计划和投资方案；
3. 拟订公司内部管理机构设置方案；
4. 拟订公司的基本管理制度；
5. 制定公司的具体规章；
6. 提请聘任或者解聘公司副经理、财务负责人；
7. 决定聘任或者解聘除应由董事会决定聘任或者解聘以外的负责管理人员；
8. 董事会授予的其他职权。

#### 12.3 财务负责人

- 负责公司资金管理
- 参与公司重大经营决策
- 配合审计机构工作

## 第五章 特殊权利条款

### 第十三条 反稀释保护

#### 13.1 全棘轮条款（Full Ratchet）

若股份公司后续增资的价格低于本轮投资人股东的认购价格（以下简称"全棘轮调整"），则本轮投资人股东的持股比例应自动调整为按全棘轮调整后的价格计算所能持有的股份数量（以下简称"全棘轮调整"），差额由股份公司以无偿发行股份的方式补足。

**计算公式：**
全棘轮调整后投资人持股数 = 原持股数 ×（原认购价格 / 全棘轮调整价格）

#### 13.2 广义加权平均反稀释（Broad-Based Weighted Average）

若股份公司后续增资的价格低于本轮投资人股东的认购价格，则按以下公式调整投资人持股数量：

```
调整后持股数 = 原持股数 × [(A + B) / (A + C)]

其中：
A = 股份公司发行在外股份总数（含已行权的期权、认股权，下同）
B = 本轮投资人按原认购价格可认购的股份数（即：本次投资总额 ÷ 原认购价格）
C = 后续新增股份数（即：后续融资总额 ÷ 后续融资认购价格）
```

#### 13.3 狭义加权平均反稀释（Narrow-Based Weighted Average）

若股份公司后续增资的价格低于本轮投资人股东的认购价格，则按以下公式调整投资人持股数量：

```
调整后持股数 = 原持股数 × [(A + B) / (A + C)]

其中：
A = 股份公司发行在外股份总数（不含期权、认股权，仅计算实际发行的股份）
B = 本轮投资人按原认购价格可认购的股份数
C = 后续新增股份数
```

#### 13.4 条款适用说明

1. 本协议签署方同意采用**[第13.1款全棘轮 / 第13.2款广义加权平均 / 第13.3款狭义加权平均]**作为本次投资的反稀释保护机制；
2. 反稀释调整不适用于：员工股权激励计划项下的股份发行、公司为进行合并或收购而发行的股份、按法定程序减资等情形；
3. 反稀释调整适用于本轮投资完成后的所有后续融资。

### 第十四条 优先认购权

#### 14.1 后续融资中的优先认购权

若股份公司拟进行后续增资（发行新股或老股转让），现有投资人股东有权按其持股比例优先认购全部或部分新增股份。

#### 14.2 通知与行权

1. 股份公司拟进行后续融资时，应至少提前[20]个工作日向各投资人股东发出书面通知（以下简称"融资通知"），载明拟融资的总额、每股价格、融资期限、资金用途等；
2. 各投资人股东应在收到融资通知后[15]个工作日内以书面形式回复是否行使优先认购权及拟认购的股份数量；
3. 若投资人股东未在上述期限内回复，视为放弃本次优先认购权。

### 第十五条 优先购买权

#### 15.1 触发条件

当任一发起人（"转让方"）拟向发起人以外的第三方（"受让方"）转让其持有的全部或部分股份时，其他发起人（"优先购买权人"）享有在同等条件下的优先购买权。

#### 15.2 通知程序

1. 转让方应提前[30]日向各优先购买权人发出书面转让通知，载明拟转让股份的数量、价格、付款方式、受让方基本情况等；
2. 各优先购买权人应在收到通知后[20]日内书面回复是否行使优先购买权及拟购买的股份数量；
3. 若多个优先购买权人拟行使优先购买权，各优先购买权人按其持股比例分配拟优先购买的股份；
4. 若优先购买权人合计拟购买的股份数量低于拟转让股份数量，剩余股份可转让给受让方。

### 第十六条 共同出售权（Drag-Along）

#### 16.1 触发条件

若[创始人/核心管理团队]（以下简称"Dragging Party"）拟向第三方转让其持有的公司股份，且该转让将导致：
1. 公司控制权发生变更；或
2. 第三方将取得公司[30]%以上的股份；

则[投资人股东]（以下简称"Dragged Party"）有权（但无义务）按照与Dragging Party相同的条款和条件，按持股比例向该第三方共同出售其持有的公司股份。

#### 16.2 通知程序

1. Dragging Party在收到第三方收购意向后[5]个工作日内，应书面通知Dragged Party，载明第三方信息、拟转让的股份数量、转让价格、付款方式、交割时间等全部条款；
2. Dragged Party应在收到通知后[10]个工作日内书面回复是否行使共同出售权；
3. 若Dragged Party行使共同出售权，其应配合办理股份转让所需的一切手续。

### 第十七条 随售权（Tag-Along）

#### 17.1 触发条件

若任一[创始人/投资人]（以下简称"Tagging Party"）拟向第三方转让其持有的公司股份，且该转让触发Drag-Along条款或该转让股数超过其持股总数的[10]%，则其他股东（以下简称"Tag-Along Holders"）有权（但无义务）按同等条款和比例随该Tagging Party共同出售其持有的股份。

#### 17.2 通知与行权

1. Tagging Party在转让通知中应告知Tag-Along Holders其有权行使随售权；
2. Tag-Along Holders应在收到通知后[10]个工作日内书面回复是否行使随售权及拟随售的股份数量；
3. 若第三方不同意接受Tag-Along Holders随售，则Tagging Party不得进行该次转让。

### 第十八条 投资人特别保护条款

#### 18.1 投资人董事推荐权

[XX投资人]（持股[XX]%）有权向股份公司董事会推荐[1]名董事候选人，该候选人由股东大会选举产生。

#### 18.2 重大事项一票否决权

在股份公司IPO前，以下事项须经[XX投资人]委派董事的书面同意方可实施：

1. 公司业务范围的重大变更；
2. 公司注册资本的增加或减少；
3. 公司合并、分立、解散或重组；
4. 公司章程的修改；
5. 超过人民币[X]万元的关联交易；
6. 对外担保（不包括对控股子公司的担保）；
7. 对外借款或融资（不包括银行授信范围内的常规融资）；
8. 股权激励计划；
9. 利润分配方案；
10. 更换会计师事务所。

#### 18.3 知情权

[XX投资人]享有以下知情权：

1. 每季度结束后[30]日内，收到公司季度经营简报；
2. 每半年结束后[45]日内，收到公司未经审计的半年度财务报表；
3. 每年度结束后[90]日内，收到公司经审计的年度财务报表、审计报告及董事会年度报告；
4. [60]天前收到股东大会会议的书面通知；
5. 及时获知公司面临的重大诉讼、仲裁或行政处罚；
6. 及时获知可能导致公司财务状况发生重大不利变化的事项。

## 第六章 竞业禁止与保密

### 第十九条 竞业禁止

#### 19.1 创始人及核心人员的竞业禁止义务

[创始人A]、[创始人B]以及股份公司[高级管理人员]（以下简称"竞业禁止义务人"）承诺：

1. 在持有股份公司股份期间，未经股份公司书面同意，不直接或间接从事与股份公司业务有竞争或可能有竞争的业务（以下简称"竞争业务"）；
2. 不直接或间接投资于从事竞争业务的企业或组织（持股比例低于[5]%的上市公司股份除外）；
3. 不直接或间接为竞争业务提供技术、渠道、客户、信息等资源；
4. 不以任何形式挖角股份公司的核心员工。

#### 19.2 竞业禁止的期限

竞业禁止义务自本协议签署之日起至以下情形之一发生时终止：
1. 竞业禁止义务人不再持有股份公司股份；且
2. 竞业禁止义务人不再担任股份公司董事、监事或高级管理人员；且
3. 竞业禁止义务人离职满[24]个月。

#### 19.3 违约责任

若竞业禁止义务人违反竞业禁止约定，应当向股份公司支付违约金，金额为：
- 一次性违约金：人民币[X]万元；
- 返还违约期间所获得的全部收益（工资、奖金、分红等）；
- 赔偿股份公司因此遭受的全部损失。

### 第二十条 保密义务

#### 20.1 保密范围

各发起人及股份公司应对以下信息承担保密义务：

1. 本协议的内容、各方达成的各项安排；
2. 股份公司的商业秘密，包括但不限于技术信息、经营信息、财务信息、客户信息；
3. 各方在本协议谈判过程中相互提供的保密信息；
4. 股份公司的未公开重大事项；
5. 各方约定的其他保密信息。

#### 20.2 保密义务的例外

以下情形不视为违反保密义务：
1. 信息已公开（非因保密义务人的原因）；
2. 信息为保密义务人在收到信息前已合法拥有；
3. 信息为保密义务人从有权披露的第三方合法获取；
4. 法律或监管部门要求披露。

#### 20.3 保密义务的期限

保密义务自本协议签署之日起生效，并在本协议终止后继续有效[3]年。

本协议项下的通知，应采用书面形式，以专人送达、挂号邮寄、特快专递或电子邮件的方式发送至各发起人在本协议中载明的联系方式。自发送之日起[3]个工作日视为送达。

### 第三十条 解释规则
