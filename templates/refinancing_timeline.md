## 第二章 【阶段一：启动与准备】T-6至T-4个月

### 2.1 第1-2周（W1-W2）：项目启动

#### 2.1.1 内部决策

**主要工作：**
- 发行人管理层就再融资事项进行内部讨论，形成初步方案
- 发行人董事会就再融资事项进行初步审议（可免公告）
- 确定再融资的必要性、可行性和基本方案
- 识别需要决策的重大事项（如发行规模、定价、募投项目）

**输出文件：**
- 董事会会议纪要（内部）
- 再融资初步方案（内部）
- 发行人基本情况简介（供中介机构使用）

**责任分工：**
| 事项 | 责任方 | 协助方 |
|------|--------|--------|
| 内部决策 | 发行人董事会/管理层 | 董事会秘书 |
| 信息整理 | 发行人董事会秘书 | 财务总监 |

#### 2.1.2 中介机构选定

**主要工作：**
- 根据再融资品种和规模，选定保荐机构/主承销商
- 选定发行人律师、会计师事务所
- 如涉及募投项目，选聘可研机构、环评机构等

**选定标准：**
| 中介机构 | 选定标准 | 备注 |
|---------|---------|------|
| 保荐机构 | 具有相应再融资项目经验、行业研究能力、承销能力 | 需确认是否具有保荐资格 |
| 律师 | 具有证券法律服务经验 | 至少1名主办律师具有类似项目经验 |
| 会计师 | 具有证券从业资格 | 连续从事证券业务年限 |

**输出文件：**
- 中介机构选定名单
- 中介机构利益冲突检索报告

#### 2.1.3 保密协议与NDA签署

**主要工作：**
- 各中介机构与发行人签署保密协议
- 各中介机构之间签署保密协议（如需要）
- 确认内幕信息知情人名单

**输出文件：**
- 保密协议（多方签署）
- 内幕信息知情人名单及承诺函

#### 2.1.4 利益冲突检索

**主要工作：**
- 保荐机构进行利益冲突检索，确认与发行人之间不存在影响独立性的情形
- 律师、会计师确认利益冲突情况
- 如存在潜在冲突，评估是否影响项目承接

**输出文件：**
- 利益冲突检索报告

### 2.2 第3-4周（W3-W4）：尽职调查启动

#### 2.2.1 保荐机构进场

**主要工作：**
- 保荐机构项目组正式进场
- 召集中介机构协调会（Kick-off Meeting），明确分工和时间表
- 向发行人发送全面尽调清单
- 初步了解发行人基本情况

**中介协调会常规议题：**
1. 项目背景和目标介绍
2. 各中介机构分工确认
3. 尽调范围和重点确认
4. 时间表和关键节点确认
5. 发行人配合事项和联系人确认
6. 沟通机制和报告机制确认

**输出文件：**
- 中介协调会会议纪要
- 全面尽调清单
- 各中介机构分工表
- 项目时间表（确认版）

#### 2.2.2 再融资品种确定

**主要工作：**
根据发行人情况和资本市场环境，确定再融资品种：

**定增（定向增发）：**
- 优点：程序相对简单、发行效率高、锁定期灵活
- 缺点：稀释原股东股权、需找到合格投资者
- 适用情形：大股东参与/战略投资者引入、融资规模较大

**可转债（可转换公司债券）：**
- 优点：初始转股溢价、对原股东股权稀释较为渐进
- 缺点：发行条件较高、审核周期相对较长、转股不确定性
- 适用情形：优质上市公司、估值稳定、希望渐进融资

**配股：**
- 优点：面向原股东，股权分散时有利于股东参与
- 缺点：原股东需出资参与，否则权益被稀释
- 适用情形：希望回馈现有股东、股权结构相对稳定

**决策因素：**
| 因素 | 考虑要点 |
|------|---------|
| 发行目的 | 融资规模、募投项目类型 |
| 股东结构 | 大股东意愿、原股东参与能力 |
| 资本市场环境 | 当前估值水平、板块走势 |
| 审核难度 | 各品种审核要求、周期差异 |
| 公司财务状况 | 资产负债率、现金流情况 |

#### 2.2.3 募投项目方向初步确定

**主要工作：**
- 根据发行人战略规划，初步确定募投项目方向
- 进行募投项目初步可行性研究
- 初步测算项目投资规模和资金需求

**募投项目方向分类：**
| 类别 | 示例 | 审核关注点 |
|------|------|---------|
| 产能扩张 | 新建生产线、扩产项目 | 市场容量、产能消化能力 |
| 研发投入 | 研发中心建设、研发项目 | 技术先进性、研发成果转化 |
| 补充流动资金 | 营运资金补充 | 必要性、规模合理性 |
| 偿还银行贷款 | 有息负债偿还 | 优化财务结构 |
| 收购资产 | 产业链并购 | 协同效应、估值合理性 |

#### 2.2.4 财务基准日确定

**主要工作：**
- 确定审计基准日（财务数据的截止日期）
- 原则上选择月末或年末，便于财务核算
- 确认报告期范围（如：三年一期或一年一期，根据再融资品种确定）

**基准日选择要点：**
| 基准日类型 | 优势 | 劣势 | 建议 |
|-----------|------|------|------|
| 季度末（3/6/9/12月末） | 数据可得性强 | 可能需审慎考虑季节性 | 优先推荐 |
| 半年度末（6/12月末） | 审核机构认可度高 | 等待时间较长 | 定增常用 |
| 年度末 | 数据最完整 | 等待时间最长 | 年度再融资 |

### 2.3 第5-8周（W5-W8）：全面尽调

#### 2.3.1 法律尽职调查

**主要范围：**
| 核查领域 | 主要内容 | 底稿编号 |
|---------|---------|---------|
| 发行人主体资格 | 设立、存续、股权结构 | DD-02 |
| 规范性情况 | 公司治理、合法合规 | DD-02 |
| 主要资产 | 土地、房产、知识产权、应收账款 | DD-02 |
| 重大合同 | 采购、销售、借款、担保 | DD-02 |
| 关联交易 | 关联方、关联交易的完整性和公允性 | DD-02 |
| 诉讼仲裁 | 未决诉讼、仲裁、行政处罚 | DD-02 |
| 劳动人事 | 劳动合同、社保公积金 | DD-02 |
| 同业竞争 | 与控股股东、实际控制人的同业竞争情况 | DD-02 |

**律师核查重点：**
1. 发行人最近三年及一期是否存在重大违法违规行为
2. 发行人股权是否清晰，是否存在代持、质押、冻结等受限情形
3. 关联交易是否必要、公允，决策程序是否合规
4. 同业竞争是否已采取有效措施解决或作出承诺

#### 2.3.2 财务尽职调查

**主要范围：**
| 核查领域 | 主要内容 | 底稿编号 |
|---------|---------|---------|
| 财务报表 | 资产负债表、利润表、现金流量表分析 | DD-03 |
| 资产质量 | 应收账款、存货、固定资产 | DD-03 |
| 收入确认 | 收入确认政策、毛利率分析 | DD-03 |
| 成本费用 | 期间费用构成及变动趋势 | DD-03 |
| 关联方往来 | 关联资金往来是否规范 | DD-03 |
| 财务内控 | 内控制度是否健全有效 | DD-03 |
| 会计政策 | 会计政策是否一致，是否符合准则要求 | DD-03 |

**财务核查重点：**
1. 收入确认的真实性、完整性，是否存在跨期调节
2. 应收账款和存货的真实性，减值计提是否充分
3. 毛利率变动的合理性
4. 关联方交易的必要性和定价公允性
5. 资金管理的规范性，是否存在被关联方占用情形

#### 2.3.3 业务尽职调查

## 第三章 【阶段二：方案设计与文件起草】T-4至T-2个月

### 3.1 第9-12周（W9-W12）：预案公告与文件起草

#### 3.1.1 再融资预案公告

**主要工作：**
- 发行人召开董事会，审议通过再融资预案
- 预案内容（根据《上市公司证券发行管理办法》/《创业板上市公司证券发行注册管理办法》等规定）：

**预案主要内容：**
1. 本次发行的背景和目的
2. 本次发行证券的类型、面值、上市地点
3. 发行数量、发行价格及定价方式
4. 发行对象及其基本情况
5. 募集资金投向
6. 本次发行是否构成关联交易
7. 本次发行对公司的影响分析
8. 本次发行的相关风险
9. 公司利润分配政策
10. 其他应披露的重大事项

**输出文件：**
- 董事会决议
- 再融资预案（公告稿）
- 募集资金使用可行性分析报告（初稿）

#### 3.1.2 募投项目可行性研究

**主要工作：**
- 完成募投项目可行性研究报告
- 募投项目备案/核准（如适用）
- 募投项目环境影响评价（如需）
- 募投项目用地落实（如涉及）

**可研报告主要章节：**
| 章节 | 主要内容 |
|------|---------|
| 项目概况 | 项目背景、建设内容、投资规模 |
| 市场分析 | 市场需求、竞争格局、项目前景 |
| 技术方案 | 技术路线、设备选型、工艺流程 |
| 建设方案 | 建设地点、规模、进度安排 |
| 投资估算 | 总投资构成、铺底流动资金 |
| 经济效益 | 收入预测、成本费用、利润测算 |
| 风险分析 | 市场、技术、资金、政策风险 |
| 结论与建议 | 可行性结论、政策建议 |

**备案/核准前置事项：**
| 项目类型 | 前置审批/备案 | 负责部门 |
|---------|-------------|---------|
| 新建项目 | 发改委备案/核准 | 发改委 |
| 工业项目 | 环评批复 | 生态环境局 |
| 用地项目 | 土地证/租赁协议 | 规划和自然资源局 |
| 技改项目 | 技改备案 | 经信委/发改委 |

#### 3.1.3 保荐机构内核稿撰写

**主要工作：**
- 保荐机构各专业组根据尽调结果，撰写各专业核查意见
- 汇总形成保荐书/上市保荐书初稿
- 汇总形成证券发行保荐工作报告

**保荐书主要内容：**
1. 保荐机构及其保荐代表人声明
2. 本次证券发行基本情况
3. 保荐机构承诺事项
4. 对本次证券发行的推荐意见
5. 对发行人的持续督导安排

#### 3.2.2 内核会

**内核会流程：**
| 环节 | 内容 | 时间 |
|------|------|------|
| 项目组汇报 | 项目基本情况、核查情况、问题及解决方案 | 30分钟 |
| 内核委员提问 | 内核委员就项目情况进行提问 | 30分钟 |
| 项目组回复 | 项目组就内核委员提问进行回复 | 20分钟 |
| 内核委员评议 | 内核委员内部讨论，形成内核意见 | 15分钟 |
| 宣布内核结果 | 内核召集人宣布内核决议 | 5分钟 |

**内核决议类型：**
| 决议类型 | 说明 | 后续要求 |
|---------|------|---------|
| 通过 | 内核委员过半数同意 | 内核意见落实后可以申报 |
| 有条件通过 | 需落实特定条件 | 落实后经内核联系人确认可以申报 |
| 否决 | 内核委员过半数反对 | 修改后重新申请内核 |

#### 3.2.3 内核意见修改

**主要工作：**

### 4.2 T+1至T+3个月：审核问询

#### 4.2.1 受理与预审

**主要工作：**
- 交易所受理后5个工作日内发出受理函
- 交易所预审员对申报文件进行预审
- 预审员可能就申报材料完整性、规范性提出反馈

**审核阶段流程（注册制）：**
```
申报→受理（5WD）→预审→问询→回复→审核中心审核→上市委审议→向证监会报送审核意见→证监会注册
```

**审核周期参考：**
| 阶段 | 平均周期 | 备注 |
|------|---------|------|
| 交易所审核 | 3-6个月 | 不含问询回复时间 |
| 问询回复 | 20个工作日/轮 | 可申请延期 |
| 证监会注册 | 20个工作日 | |

#### 4.2.2 审核问询

**问询轮次：**
- 第一轮问询：通常在受理后[1-2]个月内发出
- 第二轮问询（如有）：针对回复进一步问询
- 第三轮问询（较少见）：针对重点问题深入问询

**问询内容分类：**
| 类别 | 常见问题 |
|------|---------|
| 发行条件 | 是否符合板块定位、发行条件 |
| 规范性问题 | 历史沿革、合规情况 |
| 信息披露问题 | 披露完整性、准确性 |
| 财务问题 | 业绩波动、会计处理、持续盈利能力 |
| 募投问题 | 可行性、必要性、效益预测 |

#### 4.2.3 第一轮问询回复

**回复流程：**
1. 收到问询函后[20]个工作日内提交回复
2. 如需申请延期，不得超过[20]个工作日
3. 问询函全文在交易所网站披露

**回复格式要求：**
| 项目 | 要求 |
|------|------|
| 格式 | 通过交易所审核问询系统提交 |
| 份数 | 1份（电子） |
| 签字盖章 | 律师鉴证意见、保荐机构核查意见 |
| 附件 | 相关证明材料 |

#### 4.2.4 审核中心审核

**主要工作：**
- 审核中心审核人员对问询回复及补充材料进行审核
- 审核中心形成审核初步意见
- 提交上市委审议

#### 4.2.5 上市委审议

**审议形式：**
- 会议审议：由上市委委员（通常5-7名）出席
- 视频或现场：视情况而定

**审议流程：**
1. 项目负责人汇报（15分钟）
2. 委员提问（15-20分钟）
3. 发行人代表/保荐代表人回复
4. 委员合议
5. 宣布审议结果

**审议结果：**
| 结果 | 含义 |
|------|------|
| 通过 | 可以进入注册程序 |
| 暂缓审议 | 需进一步落实特定事项 |
| 未通过 | 终止审核 |

### 4.3 T+4至T+6个月：注册/核准与批文

#### 4.3.1 提交注册

**主要工作：**
- 交易所审核通过后，向证监会提交注册申请
- 证监会收到注册申请后[20]个工作日内决定是否注册

#### 4.3.2 证监会注册

**注册审核内容：**
- 是否符合发行条件
- 是否符合信息披露要求
- 是否符合持续监管要求

**注册批文：**
- 取得注册批文后[12]个月内完成发行
- 超期需重新申请注册

## 第七章 注意事项与特别提示

### 7.1 财务数据时效要求

| 要求 | 说明 |
|------|------|
| 审计报告有效期 | 自审计报告签署日起6个月（特殊情况可申请延期） |
| 逾期处理 | 如审计报告逾期，需补充审计或出具审阅报告 |
| 季报影响 | 审核期间应关注季报披露要求 |

### 7.2 窗口期限制

**不得启动发行的窗口期：**
- 上市公司季度报告/半年报/年度报告公告前[10]个工作日内
- 重大事项决策过程中
- 股价异常波动期间

**定向增发的特别窗口期：**
- 定增董事会决议日距前次募资到位日原则上不少于18个月（再融资新规）
- 发行价格不得低于定价基准日前20日均价的80%

### 7.3 锁定期要求

| 品种 | 锁定期 | 适用对象 |
|------|-------|---------|
| 定增（一般） | 6个月 | 询价发行对象 |
| 定增（战略投资者） | 18个月 | 引入战略投资者 |
| 定增（大股东/关联方） | 18个月或36个月 | 大股东、实际控制人、控股股东 |
| 可转债 | 无锁定期 | 上市即可流通 |
| 配股 | 无锁定期 | 原股东按比例认购部分 |

### 7.4 募投项目注意事项

1. **备案/环评前置**：开工前必须取得全部前置审批
2. **土地落实**：募投用地须权属清晰，不存在法律障碍
3. **效益测算**：经济效益测算须合理，不存在夸大
4. **补流比例**：募集资金用于补充流动资金的比例须符合规定（原则上不超过30%）

### 7.5 信息披露注意事项

1. **同步披露**：申报文件内容须与发行人披露信息一致
2. **重大变化**：审核期间发生重大不利变化的，须及时报告并披露
3. **舆情管理**：关注媒体舆情，及时回应质疑
